# <a name="publishing-your-hello-world-application-with-visual-studio-2017"></a>Публикация приложения Hello World с помощью Visual Studio 2017

Изучая руководство [Создание приложения "Hello World" на C# с помощью .NET Core в Visual Studio 2017](with-visual-studio-2017.md), вы создали консольное приложение Hello World. В следующем руководстве [Отладка приложения Hello World на C# в Visual Studio 2017](debugging-with-visual-studio-2017.md) вы протестировали его с помощью отладчика Visual Studio. Теперь вы уверены, что приложение работает правильно, и его можно опубликовать для выполнения другими пользователями. Публикация создает набор файлов, необходимых для запуска приложения. Чтобы развернуть приложение, скопируйте эти файлы на целевой компьютер.

Действия для публикации и запуска приложения. 

1. Убедитесь, что в Visual Studio настроен режим создания версии выпуска. Если это не так, измените параметр конфигурации сборки на панели инструментов, указав **Выпуск** вместо **Отладка**, как показано на следующем рисунке.

   ![Изображение](media/release.jpg)

1. Откройте окно консоли. Например, в текстовом поле **Задайте любой вопрос** на панели задач Windows введите `Command Prompt`, а затем выберите приложение **командной строки**, чтобы открыть окно консоли.

1. Щелкните правой кнопкой мыши проект HelloWorld (это не то же самое, что решение HelloWorld) и выберите в меню действие **Публикация**. Также можно выбрать **Публикация HelloWorld** из раздела **Сборка** в главном меню Visual Studio.

1. Когда появится диалоговое окно **Публикация**, представленное на следующем рисунке, создайте новый профиль публикации, выбрав действие **Создать новый профиль.**

1. В диалоговом окне **Выбрать целевой объект публикации**, которое показано на следующем рисунке, нажмите кнопку **ОК**, чтобы опубликовать приложение в локальной файловой системе. Приложение будет размещено в подкаталоге bin\release\PublishOutput основного каталога проекта приложения.

1. Завершив создание профиля публикации, выберите кнопку **Публикация** в диалоговом окне **Публикация**, которое показано на следующем рисунке.

   ![Изображение](media/publish-2.jpg)

1. Как вы видите на следующем рисунке, публикация создает в качестве выходных данных три файла приложения. Чтобы развернуть это приложение, просто скопируйте эти файлы на целевую систему:

      - HelloWorld.dll
   
      - HelloWorld.deps.json

      - HelloWorld.runtimeconfig.json

   Четвертый файл (HelloWorld.pdb) содержит отладочные символы. Его не нужно распространять вместе с приложением, но желательно сохранить его на случай, если придется выполнять отладку опубликованной версии приложения.

   ![Изображение](media/pub-files.jpg)

Процесс публикации создает платформенно-зависимое развертывание. Опубликованное приложение будет выполняться на любой платформе, поддерживаемой .NET Core, при условии, что в целевой системе установлена .NET Core. Пользователи могут запускать приложение командой `dotnet.exe HelloWorld.dll` из окна консоли.

Дополнительные сведения о публикации и развертывании приложений .NET Core см. в статье [Развертывание приложений .NET Core](../../core/deploying/index.md).
