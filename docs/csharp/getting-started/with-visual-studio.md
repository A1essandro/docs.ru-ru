# <a name="building-a-c-hello-world-application-with-net-core-in-visual-studio-2017"></a>Создание приложения "Hello World" на C# с помощью .NET Core в Visual Studio 2017 #

В этой статье содержится пошаговое описание процессов построения, отладки и публикации простого консольного приложения .NET Core с помощью Visual Studio 2017. Visual Studio 2017 предоставляет полнофункциональную среду для разработки приложений .NET Core. Если само приложение не имеет зависимостей от конкретной платформы, его можно выполнять на любой официально поддерживаемой платформе .NET Core и на любой системе, где установлена .NET Core.

## <a name="prerequisites"></a>Предварительные требования ##

- [Visual Studio 2017](https://www.visualstudio.com/downloads/) с установленной рабочей нагрузкой "Кроссплатформенная разработка .NET Core". 

Дополнительные сведения см. в разделе [Visual Studio 2017](../../core/windows-prerequisites.md) в статье о необходимых компонентах Windows.

## <a name="a-simple-hello-world-application"></a>Создание простого приложения "Hello World" ##

Для начала давайте создадим простое консольное приложение "Hello World". Ниже приведены инструкции.

1. Откройте Visual Studio, а затем в меню **Файл** выберите пункты **Создать** > **Проект**. В диалоговом окне **Новый проект** слева разверните узел **Visual C#** и выберите узел **.NET Core**.

2. На панели справа выберите **Консольное приложение (.NET Core)**. Введите имя проекта (`HelloWorld`) и убедитесь, что установлен флажок **Создать каталог для решения**, как показано на следующем рисунке.

   ![Снимок экрана с диалоговым окном создания проекта, где выбран вариант консольного приложения](./media/with-visual-studio/vs_newproject.jpg)
   
3. Нажмите кнопку **ОК** . Visual Studio отображает среду разработки в окне кода, как показано на следующем рисунке. Шаблон консольного приложения C# для .NET Core автоматически определяет класс `Program` с одним методом `Main`, который принимает в качестве аргумента массив @System.String. `Main` является точкой входа в приложение. Этот метод автоматически вызывается средой выполнения при запуске приложения. Все аргументы, предоставленные в командной строке при запуске приложения, доступны через массив *args*.

   ![Visual Studio и новый проект Hello World](./media/with-visual-studio/vs_devenv.jpg)

   Шаблон создает простейший вариант приложения "Hello World", в котором вызывается метод @System.Console.WriteLine(System.String) для вывода литеральной строки "Hello World!" в окно консоли. Теперь запустите программу в режиме отладки, нажав на панели инструментов кнопку HelloWorld с зеленой стрелкой. Но при этом окно консоли отображается очень ненадолго и сразу закрывается. Это происходит потому, что метод `Main` и приложение в целом завершаются сразу, как только будет выполнена единственная инструкция в методе `Main`.

4. Давайте добавим в приложения некоторую задержку, чтобы окно консоли не закрывалось сразу. Добавьте следующий код сразу после вызова метода @System.Console.WriteLine(System.String):

   ```csharp
   Console.Write("Press any key to continue...");
   Console.ReadKey(true);
   ```
   Этот код предлагает пользователю нажать любую клавишу и приостанавливает работу программы до нажатия клавиши.

5. В строке меню последовательно выберите **Сборка**и **Собрать решение**. Это действие компилирует программу в промежуточный язык IL, который затем преобразуется в двоичный код JIT-компилятором.

6. Запустите программу, нажав на панели инструментов кнопку HelloWorld с зеленой стрелкой. Результат показан на примере ниже.

   ![Изображение](./media/with-visual-studio/simple_hello.jpg)

7. Нажмите любую клавишу, чтобы закрыть окно.

## <a name="enhancing-the-hello-world-application"></a>Расширение приложения "Hello World" ##

Теперь мы расширим это приложение, чтобы оно предлагало пользователю ввести имя, а затем отображало в окне консоли полученное имя, а также текущие дату и время. Выполните следующие действия, чтобы изменить и протестировать программу.

1. Введите код C#, представленный в окне кода ниже, между открывающей скобкой сразу за строкой `public static void Main(string[] args)` и первой закрывающей скобкой.

   [!CODE [GettingStarted#1](../../../samples/snippets/csharp/getting_started/with_visual_studio/helloworld.cs#1)]

   На рисунке ниже показано, как должно выглядеть окно кода.

   ![Выполнение модифицированной программы](./media/with-visual-studio/codewindow.jpg)

   Теперь код выдает строку "What is your name?" (Как вас зовут?) в консоль и ожидает, чтобы пользователь ввел строку текста и нажал клавишу ВВОД. Приложение сохраняет полученную строку в переменной с именем `name`. Оно также получает значение свойства @System.DateTime.Now, которое содержит текущее локальное время, и присваивает его переменной с именем `date`. Затем с помощью [строки составного формата](../../standard/base-types/composite-format.md) эти данные выводятся в консоль.

2. Скомпилируйте программу, выбрав действие **Сборка** > **Собрать решение**. Это действие компилирует программу в промежуточный язык IL, который затем преобразуется в двоичный код JIT-компилятором.

3. Запустите программу в режиме отладки, выбрав на панели инструментов кнопку с зеленой стрелкой, нажав клавишу F5 или выбрав пункт меню **Отладка** > **Начать отладку**. Когда вы введете имя в ответ на приглашение и нажмете клавишу ВВОД, окно консоли должно выглядеть примерно так:

   ![Выполнение модифицированной программы](./media/with-visual-studio/console.jpg)

4. Для закрытия консольного окна нажмите любую клавишу. Режим отладки будет завершен.

Итак, вы создали и выполнили простое приложение. Чтобы приложение достигло профессионального уровня, нужно выполнить еще несколько шагов для подготовки приложения к выпуску.

- Сведения об отладке приложения см. в статье [Debugging the Hello World Application](debugging-with-visual-studio-2017.md) (Отладка приложения Hello World)

- Сведения о разработке и публикации распространяемый версии приложения см. в статье [Publishing the Hello World application](publishing-with-visual-studio-2017.md) (Публикация приложения Hello World).
