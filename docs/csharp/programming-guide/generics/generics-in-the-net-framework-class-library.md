# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Универсальные шаблоны в библиотеке классов платформы .NET Framework (Руководство по программированию в C#)
Библиотека классов в платформе .NET Framework версии 2.0 предоставляет новое пространство имен <xref:System.Collections.Generic>, которое содержит несколько готовых к использованию универсальных классов коллекций и связанных интерфейсов. Другие пространства имен, такие как <xref:System>, также предоставляют новые универсальные интерфейсы, например <xref:System.IComparable%601>. Эти классы и интерфейсы более эффективны и строго типизированы, чем неуниверсальные классы коллекций в более ранних выпусках платформы .NET Framework. Перед разработкой и реализацией собственных пользовательских классов коллекций нужно решить, будет ли класс использоваться либо наследоваться из одного из классов библиотеки классов .NET Framework.
