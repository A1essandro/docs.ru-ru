<span data-ttu-id="a7f8b-104">Начиная с версии 7, язык C# поддерживает значения, возвращаемые по ссылке (возвращаемые ссылочные значения).</span><span class="sxs-lookup"><span data-stu-id="a7f8b-104">Starting with C# 7, C# supports reference return values (ref returns).</span></span> <span data-ttu-id="a7f8b-105">Возвращаемое ссылочное значение позволяет методу вернуть вызывающей стороне ссылку на переменную, а не фиксированное значение.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-105">A reference return value allows a method to return a reference to a variable, rather than a value, back to a caller.</span></span> <span data-ttu-id="a7f8b-106">После этого вызывающий может самостоятельно решить, как обрабатывать полученную переменную: по значению или по ссылке.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-106">The caller can then choose to treat the returned variable as if it were returned by value or by reference.</span></span> <span data-ttu-id="a7f8b-107">Вызывающий может создать новую переменную и присвоить ей ссылку на полученное значение (локальная ссылочная переменная).</span><span class="sxs-lookup"><span data-stu-id="a7f8b-107">The caller can create a new variable that is itself a reference to the returned value, called a ref local.</span></span>

## <a name="what-is-a-reference-return-value"></a><span data-ttu-id="a7f8b-108">Что представляет собой возвращаемое ссылочное значение?</span><span class="sxs-lookup"><span data-stu-id="a7f8b-108">What is a reference return value?</span></span>

<span data-ttu-id="a7f8b-109">Большинство разработчиков прекрасно знакомы с передачей аргумента в вызываемый метод *по ссылке*.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-109">Most developers are familiar with passing an argument to a called method *by reference*.</span></span> <span data-ttu-id="a7f8b-110">Список аргументов вызываемого метода включает переменную, переданную по ссылке, и вызывающий может отследить любые изменения этого значения в вызываемом методе.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-110">A called method's argument list includes a variable passed by reference, and any changes made to its value by the called method are observed by the caller.</span></span> <span data-ttu-id="a7f8b-111">*Возвращаемое ссылочное значение* означает, что метод возвращает *ссылку* (или псевдоним) на некоторую переменную, в область действия которой входит этот метод и время существования которой должно продолжаться после того, как метод возвращает это значение.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-111">A *reference return value* means that a method returns a *reference* (or an alias) to some variable whose scope includes the method and whose lifetime must extend beyond the return of the method.</span></span> <span data-ttu-id="a7f8b-112">Все изменения, которые вызывающий производит с возвращаемым значением метода, применяются к возвращенной переменной.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-112">Modifications to the method's return value by the caller are made to the variable that is returned by the method.</span></span>

<span data-ttu-id="a7f8b-113">Если для метода объявлено *возвращаемое ссылочное значение*, значит он возвращает псевдоним переменной.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-113">Declaring that a method returns a *reference return value* indicates that the method returns an alias to a variable.</span></span> <span data-ttu-id="a7f8b-114">Чаще всего это нужно для того, чтобы вызывающий код получил псевдоним для доступа к этой переменной, в том числе для ее изменения.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-114">The design intent is often that the calling code should have access to that variable through the alias, including to modify it.</span></span> <span data-ttu-id="a7f8b-115">Следовательно, методы с возвращаемыми ссылочными значениями не могут иметь тип возвращаемого значения `void`.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-115">It follows that methods returning by reference cannot have the return type `void`.</span></span>

<span data-ttu-id="a7f8b-116">Есть несколько ограничений для выражений, которые можно использовать в качестве возвращаемых ссылочных значений метода.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-116">There are some restrictions on the expression that a method can return as a reference return value.</span></span> <span data-ttu-id="a7f8b-117">Сюда входит следующее.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-117">These include:</span></span>

- <span data-ttu-id="a7f8b-122">Возвращаемое значение не может быть литералом `null`.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-122">The return value cannot be the literal `null`.</span></span> <span data-ttu-id="a7f8b-123">При попытке возвратить `null` возникает ошибка компилятора CS8156 "Выражение невозможно использовать в данном контексте, так как его невозможно вернуть по ссылке".</span><span class="sxs-lookup"><span data-stu-id="a7f8b-123">Attempting to return `null` generates compiler error CS8156, "An expression cannot be used in this context because it may not be returned by reference."</span></span>

   <span data-ttu-id="a7f8b-124">Метод с возвращаемым ссылочным значением может возвращать псевдоним для переменной с текущим значением null (которой не присвоены экземпляры) или [тип, допускающий значение null](../nullable-types/index.md).</span><span class="sxs-lookup"><span data-stu-id="a7f8b-124">A method with a ref return can return an alias to a variable whose value is currently the null (uninstantiated) value or a [nullable type](../nullable-types/index.md) for a value type.</span></span>
 
- <span data-ttu-id="a7f8b-125">Возвращаемое значение не может быть константой, элементом перечисления, полученным по значению, возвращаемым значением свойства, а также методом `class` или `struct`.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-125">The return value cannot be a constant, an enumeration member, the by-value return value from a property, or a method of a `class` or `struct`.</span></span> <span data-ttu-id="a7f8b-126">При попытке возвратить такие значения возникает ошибка компилятора CS8156 "Выражение невозможно использовать в данном контексте, так как его невозможно вернуть по ссылке".</span><span class="sxs-lookup"><span data-stu-id="a7f8b-126">Attempting to return these generates compiler error CS8156, "An expression cannot be used in this context because it may not be returned by reference."</span></span>

## <a name="defining-a-ref-return-value"></a><span data-ttu-id="a7f8b-128">Определение возвращаемого ссылочного значения</span><span class="sxs-lookup"><span data-stu-id="a7f8b-128">Defining a ref return value</span></span>

<span data-ttu-id="a7f8b-129">Чтобы определить возвращаемое ссылочное значение, необходимо добавить ключевое слово [ref](../../language-reference/keywords/ref.md) к типу возвращаемого значения в сигнатуре метода.</span><span class="sxs-lookup"><span data-stu-id="a7f8b-129">You define a ref return value by adding the [ref](../../language-reference/keywords/ref.md) keyword to the return type of the method signature.</span></span> <span data-ttu-id="a7f8b-130">Например, следующая сигнатура указывает, что свойство `GetContactInformation` возвращает вызывающему объекту ссылку на объект `Person`:</span><span class="sxs-lookup"><span data-stu-id="a7f8b-130">For example, the following signature indicates that the `GetContactInformation` property returns a reference to a `Person` object to the caller:</span></span>
