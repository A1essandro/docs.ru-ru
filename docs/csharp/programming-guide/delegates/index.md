## <a name="delegates-overview"></a><span data-ttu-id="d39c8-119">Общие сведения о делегатах</span><span class="sxs-lookup"><span data-stu-id="d39c8-119">Delegates Overview</span></span>  
 <span data-ttu-id="d39c8-120">Делегаты имеют следующие свойства.</span><span class="sxs-lookup"><span data-stu-id="d39c8-120">Delegates have the following properties:</span></span>
