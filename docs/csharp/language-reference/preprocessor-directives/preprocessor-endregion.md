# #endregion (справочник по C#)
`#endregion` Отмечает конец блока [\#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md).  Примеры.  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
