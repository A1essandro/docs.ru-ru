# <a name="new-c-reference"></a>new (справочник по C#)
В C# `new` ключевое слово можно использовать в качестве оператора, модификатора или ограничения.  
  
 [Оператор new](../../../csharp/language-reference/keywords/new-operator.md)  
 Применяется для создания объектов и вызова конструкторов.  
  
 [Модификатор new](../../../csharp/language-reference/keywords/new-modifier.md)  
 Используется, чтобы скрыть наследуемый член от члена базового класса.  
  
 [Ограничение new](../../../csharp/language-reference/keywords/new-constraint.md)  
 Применяется для ограничения типов, которые могут использоваться в качестве аргументов для параметра типа в универсальном объявлении.
