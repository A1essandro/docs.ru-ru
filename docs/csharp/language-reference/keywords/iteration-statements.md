# <a name="iteration-statements-c-reference"></a>Операторы перебора (Справочник по C#)

При помощи операторов итерации можно создавать циклы. Операторы итерации могут инициировать многократное выполнение встроенных операторов на основании условия прекращения цикла. Эти операторы выполняются по порядку, если только не встречается [оператор jump](jump-statements.md).

В операторах итерации могут использоваться следующие ключевые слова:

- [do](do.md)

- [for](for.md)

- [foreach, in](foreach-in.md)

- [while](while.md)
