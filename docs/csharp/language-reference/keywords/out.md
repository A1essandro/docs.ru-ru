# <a name="out-c-reference"></a>out (Справочник по C#)

Ключевое слово `out` можно использовать в двух контекстах:

- В качестве [модификатора параметра](out-parameter-modifier.md), который позволяет передавать аргумент в метод по ссылке, а не по значению.

- В [объявлениях параметров универсального типа](out-generic-modifier.md) для интерфейсов и делегатов, что указывает, что параметр типа является ковариантным.
