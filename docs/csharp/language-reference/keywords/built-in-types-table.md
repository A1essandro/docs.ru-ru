## <a name="remarks"></a><span data-ttu-id="f0867-122">Примечания</span><span class="sxs-lookup"><span data-stu-id="f0867-122">Remarks</span></span>

<span data-ttu-id="f0867-123">Все типы в таблице, за исключением `object` и `string`, считаются простыми.</span><span class="sxs-lookup"><span data-stu-id="f0867-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>  
  
<span data-ttu-id="f0867-124">Ключевые слова типов C# и их псевдонимы являются взаимозаменяемыми.</span><span class="sxs-lookup"><span data-stu-id="f0867-124">The C# type keywords and their aliases are interchangeable.</span></span> <span data-ttu-id="f0867-125">Например, можно объявить целочисленную переменную с помощью любого из следующих объявлений:</span><span class="sxs-lookup"><span data-stu-id="f0867-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>  

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="f0867-126">Используйте оператор [typeof](typeof.md), чтобы получить экземпляр <xref:System.Type?displayProperty=nameWithType>, представляющий указанный тип:</span><span class="sxs-lookup"><span data-stu-id="f0867-126">Use the [typeof](typeof.md) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
