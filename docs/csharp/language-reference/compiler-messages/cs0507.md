# <a name="compiler-error-cs0507"></a><span data-ttu-id="ec8de-102">Ошибка компилятора CS0507</span><span class="sxs-lookup"><span data-stu-id="ec8de-102">Compiler Error CS0507</span></span>
<span data-ttu-id="ec8de-103">"функция1": невозможно изменить модификаторы доступа при переопределении "уровень доступа", унаследованном из "функция2"</span><span class="sxs-lookup"><span data-stu-id="ec8de-103">'function1' : cannot change access modifiers when overriding 'access' inherited member 'function2'</span></span>

## <a name="example"></a><span data-ttu-id="ec8de-105">Пример</span><span class="sxs-lookup"><span data-stu-id="ec8de-105">Example</span></span>  
 <span data-ttu-id="ec8de-106">В следующем примере возникает ошибка CS0507.</span><span class="sxs-lookup"><span data-stu-id="ec8de-106">The following sample generates CS0507.</span></span>  
  
```csharp  
// CS0507.cs  
abstract public class clx  
{  
   virtual protected void f() {}  
}  
  
public class cly : clx  
{  
   public override void f() {}   // CS0507  
   public static void Main() {}  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ec8de-107">Пример</span><span class="sxs-lookup"><span data-stu-id="ec8de-107">Example</span></span>  
 <span data-ttu-id="ec8de-108">Кроме того, ошибка CS0507 может возникать в том случае, если в классе предпринимается попытка переопределить метод, помеченный как `protected internal` в ссылочных метаданных.</span><span class="sxs-lookup"><span data-stu-id="ec8de-108">CS0507 can also occur if a class attempts to override a method marked as `protected internal` defined in referenced metadata.</span></span> <span data-ttu-id="ec8de-109">В этом случае переопределяющий метод следует пометить как `protected`.</span><span class="sxs-lookup"><span data-stu-id="ec8de-109">In this situation, the overriding method should be marked as `protected`.</span></span>  
  
```csharp  
// CS0507_b.cs  
// compile with: /target:library  
abstract public class clx  
{  
   virtual protected internal void f() {}  
}  
```
