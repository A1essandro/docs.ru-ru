# <a name="compiler-error-cs0266"></a><span data-ttu-id="940e0-102">Ошибка компилятора CS0266</span><span class="sxs-lookup"><span data-stu-id="940e0-102">Compiler Error CS0266</span></span>
<span data-ttu-id="940e0-103">Невозможно неявным образом преобразовать тип "type1" в "type2".</span><span class="sxs-lookup"><span data-stu-id="940e0-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="940e0-104">Существует явное преобразование (отсутствует приведение?)</span><span class="sxs-lookup"><span data-stu-id="940e0-104">An explicit conversion exists (are you missing a cast?)</span></span>  
  
 <span data-ttu-id="940e0-105">Эта ошибка возникает, когда код пытается выполнить преобразование между двумя типами, которые не могут быть преобразованы неявно, но доступно явное преобразование.</span><span class="sxs-lookup"><span data-stu-id="940e0-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="940e0-106">Дополнительные сведения см. в статье [Casting and Type Conversions](../../../csharp/programming-guide/types/casting-and-type-conversions.md) (Приведение и преобразование типов).</span><span class="sxs-lookup"><span data-stu-id="940e0-106">For more information, see [Casting and Type Conversions](../../../csharp/programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="940e0-107">Следующий код — примеры ситуаций, приводящих к ошибке CS0266.</span><span class="sxs-lookup"><span data-stu-id="940e0-107">The following code shows examples that generate CS0266.</span></span>  
  
```csharp  
// CS0266.cs  
class MyClass  
{  
    public static void Main()  
    {  
        // You cannot implicitly convert a double to an integer.  
        double d = 3.2;  
  
        // The following line causes compiler error CS0266.  
        int i1 = d;  
  
        // However, you can resolve the error by using an explicit conversion.  
        int i2 = (int)d;  
  
        // You cannot implicitly convert an object to a class type.  
        object obj = "MyString";  
  
        // The following assignment statement causes error CS0266.  
        MyClass myClass = obj;    
  
        // You can resolve the error by using an explicit conversion.  
        MyClass c = ( MyClass )obj;  
  
        // You cannot implicitly convert a base class object to a derived class type.  
        MyClass mc = new MyClass();  
        DerivedClass dc = new DerivedClass();  
  
        // The following line causes compiler error CS0266.  
        dc = mc;  
  
        // You can resolve the error by using an explicit conversion.  
        dc = (DerivedClass)mc;  
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
