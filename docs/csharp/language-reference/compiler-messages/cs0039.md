<span data-ttu-id="59614-103">Не удается преобразовать тип "тип1" в "тип2" с помощью преобразования ссылок, упаковки-преобразования, распаковки-преобразования, преобразования в оболочку или преобразования типа Null</span><span class="sxs-lookup"><span data-stu-id="59614-103">Cannot convert type 'type1' to 'type2' via a reference conversion, boxing conversion, unboxing conversion, wrapping conversion, or null type conversion</span></span>  
  
 <span data-ttu-id="59614-104">Преобразование с использованием оператора [as](../../../csharp/language-reference/keywords/as.md) допускается посредством наследования, преобразования ссылок и упаковки-преобразования.</span><span class="sxs-lookup"><span data-stu-id="59614-104">A conversion with the [as](../../../csharp/language-reference/keywords/as.md) operator is allowed by inheritance, reference conversions, and boxing conversions.</span></span> <span data-ttu-id="59614-105">Дополнительные сведения см. в разделе [Операторы преобразования](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="59614-105">For more information, see [Conversion Operators](../../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="59614-106">Пример</span><span class="sxs-lookup"><span data-stu-id="59614-106">Example</span></span>  
 <span data-ttu-id="59614-107">Следующий пример приводит к возникновению ошибки CS0039.</span><span class="sxs-lookup"><span data-stu-id="59614-107">The following example generates CS0039.</span></span>  
  
```csharp  
// CS0039.cs  
using System;  
class A  
{  
}  
class B: A  
{  
}  
class C: A  
{  
}  
class M  
{  
    static void Main()  
    {  
        A a = new C();  
        B b = new B();  
        C c;  
  
        // This is valid; there is a built-in reference  
        // conversion from A to C.  
        c = a as C;    
  
        //The following generates CS0039; there is no  
        // built-in reference conversion from B to C.  
        c = b as C;  // CS0039  
    }  
}  
```
