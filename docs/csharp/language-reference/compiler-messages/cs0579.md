# Compiler Error CS0579
Повторяющийся атрибут 'attribute'  
  
 Один атрибут нельзя использовать несколько раз, если для этого атрибута не установлен параметр **AllowMultiple\=true** в его атрибуте [AttributeUsage](../Topic/AttributeUsage%20\(C%23%20and%20Visual%20Basic\).md).  
  
## Пример  
 Следующий пример проводит к генерированию ошибки CS0579.  
  
```c#  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
