# <a name="compiler-error-cs0188"></a><span data-ttu-id="ab2b7-102">Ошибка компилятора CS0188</span><span class="sxs-lookup"><span data-stu-id="ab2b7-102">Compiler Error CS0188</span></span>
<span data-ttu-id="ab2b7-103">Невозможно использовать объект "this", пока не будут назначены все его поля</span><span class="sxs-lookup"><span data-stu-id="ab2b7-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="ab2b7-104">Все поля в `struct` должны быть присвоены конструктором до того, как конструктор сможет вызвать метод в `struct`.</span><span class="sxs-lookup"><span data-stu-id="ab2b7-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="ab2b7-105">Если эта ошибка появляется при попытке инициализировать свойство в конструкторе структуры, измените параметр структуры, указав резервное поле вместо самого свойства.</span><span class="sxs-lookup"><span data-stu-id="ab2b7-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="ab2b7-106">В структурах не рекомендуется использовать автоматически реализуемые свойства, поскольку они не содержат резервное поле и поэтому не могут инициализироваться из конструктора.</span><span class="sxs-lookup"><span data-stu-id="ab2b7-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="ab2b7-107">Дополнительные сведения см. в разделе [Использование структур](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="ab2b7-107">For more information, see [Using Structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="ab2b7-108">Пример</span><span class="sxs-lookup"><span data-stu-id="ab2b7-108">Example</span></span>  
 <span data-ttu-id="ab2b7-109">В следующем примере возникает ошибка CS0188:</span><span class="sxs-lookup"><span data-stu-id="ab2b7-109">The following sample generates CS0188:</span></span>  
  
```csharp  
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
