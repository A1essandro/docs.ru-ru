# <a name="compiler-error-cs1946"></a><span data-ttu-id="c0c31-102">Ошибка компилятора CS1946</span><span class="sxs-lookup"><span data-stu-id="c0c31-102">Compiler Error CS1946</span></span>
<span data-ttu-id="c0c31-103">Выражение анонимного метода не может быть преобразовано в дерево выражения.</span><span class="sxs-lookup"><span data-stu-id="c0c31-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
 <span data-ttu-id="c0c31-104">Анонимный метод представляет собой набор инструкций, однако дерево выражения не должно содержать инструкции.</span><span class="sxs-lookup"><span data-stu-id="c0c31-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="c0c31-105">Таким образом, анонимный метод не может быть представлен в виде дерева выражения.</span><span class="sxs-lookup"><span data-stu-id="c0c31-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>

1.  <span data-ttu-id="c0c31-107">Замените анонимный метод лямбда-выражением.</span><span class="sxs-lookup"><span data-stu-id="c0c31-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c0c31-108">Пример</span><span class="sxs-lookup"><span data-stu-id="c0c31-108">Example</span></span>  
 <span data-ttu-id="c0c31-109">В следующем примере возникает ошибка CS1946:</span><span class="sxs-lookup"><span data-stu-id="c0c31-109">The following example generates CS1946:</span></span>  
  
```csharp  
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
