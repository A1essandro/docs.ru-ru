# <a name="compiler-error-cs0051"></a>Ошибка компилятора CS0051
Несогласованность по доступности: доступность типа параметра "тип" ниже доступности метода "метод"  
  
 Тип возврата и все типы, указанные в списке формальных параметров метода, должны быть менее доступны, чем сам метод. Убедитесь, что используемые в сигнатурах методов типы не получили случайно уровень доступности private из-за пропуска модификатора `public`. Дополнительные сведения см. в статье [Модификаторы доступа](../../../csharp/programming-guide/classes-and-structs/access-modifiers.md).  
  
## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки CS0051:  
  
```csharp  
// CS0051.cs  
public class A  
{  
    // Try making B public since F is public  
    // B is implicitly private here  
    class B  
    {  
    }  
  
    public static void F(B b)  // CS0051  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
