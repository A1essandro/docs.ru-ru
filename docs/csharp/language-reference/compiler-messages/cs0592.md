# <a name="compiler-error-cs0592"></a><span data-ttu-id="4c8f1-102">Ошибка компилятора CS0592</span><span class="sxs-lookup"><span data-stu-id="4c8f1-102">Compiler Error CS0592</span></span>
<span data-ttu-id="4c8f1-103">Атрибут "атрибут" не допускается для этого типа объявления.</span><span class="sxs-lookup"><span data-stu-id="4c8f1-103">Attribute 'attribute' is not valid on this declaration type.</span></span> <span data-ttu-id="4c8f1-104">Он допустим только для объявлений "тип".</span><span class="sxs-lookup"><span data-stu-id="4c8f1-104">It is valid on 'type' declarations only.</span></span>  
  
 <span data-ttu-id="4c8f1-105">При определении атрибута с помощью значения `AttributeTargets` определяются конструкции, к которым он может быть применен.</span><span class="sxs-lookup"><span data-stu-id="4c8f1-105">When you define an attribute, you define what constructs it can be applied to by specifying an `AttributeTargets` value.</span></span> <span data-ttu-id="4c8f1-106">В следующем примере атрибут `MyAttribute` может применяться только к интерфейсам, поскольку атрибут `AttributeUsage` имеет значение `AttributeTargets.Iterface`.</span><span class="sxs-lookup"><span data-stu-id="4c8f1-106">In the following example, the `MyAttribute` attribute can be applied to interfaces only, because the `AttributeUsage` attribute specifies `AttributeTargets.Iterface`.</span></span> <span data-ttu-id="4c8f1-107">Ошибка возникает из-за того, что этот атрибут применяется к классу (класс `A`).</span><span class="sxs-lookup"><span data-stu-id="4c8f1-107">The error is generated because the attribute is applied to a class (class `A`).</span></span>  
  
## <a name="example"></a><span data-ttu-id="4c8f1-108">Пример</span><span class="sxs-lookup"><span data-stu-id="4c8f1-108">Example</span></span>  
 <span data-ttu-id="4c8f1-109">В следующем примере возникает ошибка CS0592:</span><span class="sxs-lookup"><span data-stu-id="4c8f1-109">The following sample generates CS0592:</span></span>  
  
```csharp  
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute   
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.   
public class A    
{  
    public static void Main()  
    {  
    }  
}  
```
