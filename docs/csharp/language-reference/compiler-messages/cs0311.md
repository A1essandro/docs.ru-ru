# <a name="compiler-error-cs0311"></a><span data-ttu-id="e2d57-102">Ошибка компилятора CS0311</span><span class="sxs-lookup"><span data-stu-id="e2d57-102">Compiler Error CS0311</span></span>
<span data-ttu-id="e2d57-103">Тип type1 нельзя использовать в качестве параметра типа T в универсальном типе или методе \<name>.</span><span class="sxs-lookup"><span data-stu-id="e2d57-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="e2d57-104">Отсутствует неявное преобразование ссылок из типа type1 в тип type2.</span><span class="sxs-lookup"><span data-stu-id="e2d57-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="e2d57-105">Когда к параметру универсального типа применяется ограничение, должно существовать неявное преобразование идентификатора или ссылки из определенного аргумента в тип, установленный для этого ограничения.</span><span class="sxs-lookup"><span data-stu-id="e2d57-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e2d57-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e2d57-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="e2d57-107">Измените аргумент, который используете для создания класса.</span><span class="sxs-lookup"><span data-stu-id="e2d57-107">Change the argument you are using to create the class.</span></span>  
  
2.  <span data-ttu-id="e2d57-108">Если вы являетесь владельцем класса, вы можете удалить ограничение или внести некоторые изменения, чтобы добавить неявное преобразование ссылки или идентификатора.</span><span class="sxs-lookup"><span data-stu-id="e2d57-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="e2d57-109">Например, второй тип можно унаследовать от первого.</span><span class="sxs-lookup"><span data-stu-id="e2d57-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e2d57-110">Пример</span><span class="sxs-lookup"><span data-stu-id="e2d57-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B {}  
class C {}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="e2d57-111">Если эта ошибка возникает при попытке использования аргумента типа значения, следует помнить, что неявное числовое преобразование (например, из `short` в `int`) не соответствует параметру универсального типа.</span><span class="sxs-lookup"><span data-stu-id="e2d57-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
