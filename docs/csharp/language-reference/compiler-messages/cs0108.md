# <a name="compiler-warning-level-2-cs0108"></a><span data-ttu-id="1d3b5-102">Предупреждение компилятора (уровень 2) CS0108</span><span class="sxs-lookup"><span data-stu-id="1d3b5-102">Compiler Warning (level 2) CS0108</span></span>

<span data-ttu-id="1d3b5-103">"член1" скрывает унаследованный член "член2"</span><span class="sxs-lookup"><span data-stu-id="1d3b5-103">'member1' hides inherited member 'member2'.</span></span> <span data-ttu-id="1d3b5-104">Если скрытие было выполнено намеренно, используйте ключевое слово new.</span><span class="sxs-lookup"><span data-stu-id="1d3b5-104">Use the new keyword if hiding was intended.</span></span>  
  
 <span data-ttu-id="1d3b5-105">Была объявлена переменная с тем же именем, что и переменная базового класса.</span><span class="sxs-lookup"><span data-stu-id="1d3b5-105">A variable was declared with the same name as a variable in a base class.</span></span> <span data-ttu-id="1d3b5-106">Тем не менее ключевое слово [new](../../../csharp/language-reference/keywords/new.md) не использовалось.</span><span class="sxs-lookup"><span data-stu-id="1d3b5-106">However, the [new](../../../csharp/language-reference/keywords/new.md) keyword was not used.</span></span> <span data-ttu-id="1d3b5-107">Это предупреждение свидетельствует о необходимости использовать ключевое слово **new**. Переменная объявлена так, как это делается при использовании ключевого слова **new** в объявлении.</span><span class="sxs-lookup"><span data-stu-id="1d3b5-107">This warning informs you that you should use **new**; the variable is declared as if **new** had been used in the declaration.</span></span>  
  
 <span data-ttu-id="1d3b5-108">В следующем примере возникает ошибка CS0108:</span><span class="sxs-lookup"><span data-stu-id="1d3b5-108">The following sample generates CS0108:</span></span>  

```csharp
// CS0108.cs  
// compile with: /W:2  
using System;  
  
namespace x  
{  
   public class clx  
   {  
      public int i = 1;  
   }  
  
   public class cly : clx  
   {  
      public static int i = 2;   // CS0108, use the new keyword  
      // Use the following line instead:  
      // public static new int i = 2;  
  
      public static void Main()  
      {  
         Console.WriteLine(i);  
      }  
   }  
}  
```
