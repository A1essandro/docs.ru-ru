# <a name="compiler-error-cs1921"></a><span data-ttu-id="f1cde-102">Ошибка компилятора CS1921</span><span class="sxs-lookup"><span data-stu-id="f1cde-102">Compiler Error CS1921</span></span>
<span data-ttu-id="f1cde-103">Наиболее подходящий перегруженный метод для "метод" имеет неправильную сигнатуру элемента инициализатора.</span><span class="sxs-lookup"><span data-stu-id="f1cde-103">The best overloaded method match for 'method' has wrong signature for the initializer element.</span></span> <span data-ttu-id="f1cde-104">Инициализируемый метод Add должен быть доступным методом экземпляра.</span><span class="sxs-lookup"><span data-stu-id="f1cde-104">The initializable Add must be an accessible instance method.</span></span>  
  
 <span data-ttu-id="f1cde-105">Эта ошибка возникает при попытке использовать инициализатор коллекции с классом, не имеющим открытого нестатического метода `Add`.</span><span class="sxs-lookup"><span data-stu-id="f1cde-105">This error is generated when you try to use a collection initializer with a class that has no public non-static `Add` method.</span></span> <span data-ttu-id="f1cde-106">Если метод `Add` недоступен из-за установленного уровня защиты (`private`, `protected`, `internal`), возникнет ошибка CS0122, которая, как правило, означает, что метод определен как `static`.</span><span class="sxs-lookup"><span data-stu-id="f1cde-106">If the `Add` method is not accessible because of its protection level (`private`, `protected`, `internal`) then you will get CS0122, so that this error probably means that the method is defined as `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f1cde-107">Пример</span><span class="sxs-lookup"><span data-stu-id="f1cde-107">Example</span></span>  
 <span data-ttu-id="f1cde-108">В следующем примере возникает ошибка CS1921:</span><span class="sxs-lookup"><span data-stu-id="f1cde-108">The following example generates CS1921:</span></span>  
  
```csharp  
// cs1921.cs  
using System.Collections;  
public class C : CollectionBase  
{  
    public static void Add(int i)  
    {  
    }  
}

    public static void Main()  
    {  
        var collection = new C { 1, 2, 3 }; // CS1921  
    }  
}  
```
