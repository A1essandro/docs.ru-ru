# <a name="compiler-error-cs7003"></a>CS7003 ошибки компилятора
Неожиданное использование несвязанного универсального имени.

Эта ошибка возникает при использовании универсального типа, которым требуется один параметр типа без передачи любое имя универсального параметра типа в угловых скобках. Такое использование может быть объявление переменной или создания экземпляра объекта.

## <a name="to-correct-this-error"></a>Исправление ошибки  
  
Укажите одно имя параметра типа в угловых скобках, при использовании универсального типа.  

 ## <a name="example"></a>Пример  
 Следующий пример приводит к возникновению ошибки CS7003:  
  
```csharp  
// CS7003.cs  
    class Program
    {
        static void Main(string[] args)
        {
            var myVar1 = new MyGenericClass<>();  //CS7003

            MyGenericClass<> var2;                //CS7003
        }
    }

    public class MyGenericClass<T> { }
```
