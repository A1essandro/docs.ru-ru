# <a name="compiler-warning-level-1-cs1060"></a><span data-ttu-id="74d0d-102">Предупреждение компилятора (уровень 1) CS1060</span><span class="sxs-lookup"><span data-stu-id="74d0d-102">Compiler Warning (level 1) CS1060</span></span>
<span data-ttu-id="74d0d-103">Использование поля "имя", которому, возможно, не присвоено значение.</span><span class="sxs-lookup"><span data-stu-id="74d0d-103">Use of possibly unassigned field 'name'.</span></span> <span data-ttu-id="74d0d-104">Переменным экземпляра структуры, которой не присвоено значение, значения изначально не присваиваются.</span><span class="sxs-lookup"><span data-stu-id="74d0d-104">Struct instance variables are initially unassigned if struct is unassigned.</span></span>  
  
 <span data-ttu-id="74d0d-105">Если члены структуры не инициализируются явно, при инициализации им присваивается значение по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="74d0d-105">Struct members are initialized to their default value if you do not explicitly initialize them.</span></span> <span data-ttu-id="74d0d-106">Для типов классов и других ссылочных типов значением по умолчанию является NULL.</span><span class="sxs-lookup"><span data-stu-id="74d0d-106">The default value for class types (and other reference types) is null.</span></span> <span data-ttu-id="74d0d-107">Если класс не был инициализирован до попытки получить доступ к нему, во время выполнения возникнет исключение `NullReferenceException`.</span><span class="sxs-lookup"><span data-stu-id="74d0d-107">If the class is not initialized before any attempt to access it, a `NullReferenceException` will be thrown at runtime.</span></span> <span data-ttu-id="74d0d-108">Компилятору не удается однозначно определить, будет ли член класса инициализирован или нет, в связи с чем CS1060 — это предупреждение, а не ошибка.</span><span class="sxs-lookup"><span data-stu-id="74d0d-108">The compiler cannot determine definitively whether the class member will be initialized or not, and so CS1060 is a warning and not an error.</span></span>

## <a name="example"></a><span data-ttu-id="74d0d-111">Пример</span><span class="sxs-lookup"><span data-stu-id="74d0d-111">Example</span></span>  
 <span data-ttu-id="74d0d-112">В следующем коде возникает предупреждение CS1060, поскольку тип класса `U` является членом `struct S`, но не был инициализирован.</span><span class="sxs-lookup"><span data-stu-id="74d0d-112">The following code generates CS1060 because the class type `U` is a member of the `struct S` but is never initialized.</span></span>  
  
```csharp  
// cs1060.cs  
namespace CS1060  
{      
    public class U  
    {  
        public int i;  
    }  
  
    public struct S  
    {  
        public U u;  
        // Add constructor to correct the error.  
        //public S(int val)  
        //{  
        //    u = new U() { i = val };  
        //}  
    }  
    public class Test  
    {  
        static void Main()  
        {  
            S s;  
            s.u.i = 5;  // CS1060  
  
            //Try these lines instead, and uncomment the constructor in S  
            // S s = new S(0);  
            // s.u.i = 5;  
  
        }  
    }    
}  
```
