 <span data-ttu-id="ad220-107">В следующем примере возникает ошибка CS0050. {  
```csharp  
}

{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
