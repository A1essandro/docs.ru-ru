# <a name="compiler-error-cs0116"></a><span data-ttu-id="6e6a9-102">Ошибка компилятора CS0116</span><span class="sxs-lookup"><span data-stu-id="6e6a9-102">Compiler Error CS0116</span></span>
<span data-ttu-id="6e6a9-103">Пространство имен напрямую не включает в себя такие члены, как поля или методы</span><span class="sxs-lookup"><span data-stu-id="6e6a9-103">A namespace does not directly contain members such as fields or methods</span></span>  
  
 <span data-ttu-id="6e6a9-104">В [пространстве имен](../../../csharp/language-reference/keywords/namespace.md) компилятор принимает только классы, структуры, объединения, перечисления, интерфейсы и делегаты.</span><span class="sxs-lookup"><span data-stu-id="6e6a9-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="6e6a9-105">С этой ошибкой часто сталкиваются разработчики, имеющие опыт программирования на C/C++, поскольку в C# методы и переменные должны объявляться и определяться внутри структуры или класса.</span><span class="sxs-lookup"><span data-stu-id="6e6a9-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="6e6a9-106">Дополнительные сведения см. в разделе [Общая структура программы C#](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="6e6a9-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="6e6a9-107">Пример</span><span class="sxs-lookup"><span data-stu-id="6e6a9-107">Example</span></span>  
 <span data-ttu-id="6e6a9-108">В следующем примере возникает ошибка CS0116:</span><span class="sxs-lookup"><span data-stu-id="6e6a9-108">The following sample generates CS0116:</span></span>  
  
```csharp  
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
