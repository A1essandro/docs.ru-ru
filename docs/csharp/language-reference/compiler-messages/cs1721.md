# <a name="compiler-error-cs1721"></a><span data-ttu-id="70ab4-102">Ошибка компилятора CS1721</span><span class="sxs-lookup"><span data-stu-id="70ab4-102">Compiler Error CS1721</span></span>
<span data-ttu-id="70ab4-103">Класс "класс" не может иметь несколько базовых классов: "класс1" и "класс2"</span><span class="sxs-lookup"><span data-stu-id="70ab4-103">Class 'class' cannot have multiple base classes: 'class_1' and 'class_2'</span></span>  
  
 <span data-ttu-id="70ab4-104">Чаще всего это сообщение об ошибке связано с попыткой использовать множественное наследование.</span><span class="sxs-lookup"><span data-stu-id="70ab4-104">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="70ab4-105">Класс в C# может напрямую наследоваться только от одного класса.</span><span class="sxs-lookup"><span data-stu-id="70ab4-105">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="70ab4-106">Тем не менее класс может реализовывать любое число интерфейсов.</span><span class="sxs-lookup"><span data-stu-id="70ab4-106">However, a class can implement any number of interfaces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="70ab4-107">Пример</span><span class="sxs-lookup"><span data-stu-id="70ab4-107">Example</span></span>  
 <span data-ttu-id="70ab4-108">В примере показан один из способов того, в котором CS1721 создается, а затем показано два возможных способа устранения ошибки.</span><span class="sxs-lookup"><span data-stu-id="70ab4-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```
