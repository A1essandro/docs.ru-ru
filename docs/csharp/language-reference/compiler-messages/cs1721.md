# <a name="compiler-error-cs1721"></a><span data-ttu-id="0f54d-102">Ошибка компилятора CS1721</span><span class="sxs-lookup"><span data-stu-id="0f54d-102">Compiler Error CS1721</span></span>
<span data-ttu-id="0f54d-103">Класс "класс" не может иметь несколько базовых классов: "класс1" и "класс2"</span><span class="sxs-lookup"><span data-stu-id="0f54d-103">Class 'class' cannot have multiple base classes: 'class_1' and 'class_2'</span></span>  
  
 <span data-ttu-id="0f54d-104">Чаще всего это сообщение об ошибке связано с попыткой использовать множественное наследование.</span><span class="sxs-lookup"><span data-stu-id="0f54d-104">The most common cause of this error message is attempting to use multiple inheritance.</span></span> <span data-ttu-id="0f54d-105">Класс в C# может напрямую наследоваться только от одного класса.</span><span class="sxs-lookup"><span data-stu-id="0f54d-105">A class in C# may only inherit directly from one class.</span></span> <span data-ttu-id="0f54d-106">Тем не менее класс может реализовывать любое число интерфейсов.</span><span class="sxs-lookup"><span data-stu-id="0f54d-106">However, a class can implement any number of interfaces.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0f54d-107">Пример</span><span class="sxs-lookup"><span data-stu-id="0f54d-107">Example</span></span>  
 <span data-ttu-id="0f54d-108">В следующем примере показан один из сценариев, в которых возникает ошибка CS1721, и приводятся два возможных способа ее устранения.</span><span class="sxs-lookup"><span data-stu-id="0f54d-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```
