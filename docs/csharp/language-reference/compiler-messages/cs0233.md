# <a name="compiler-error-cs0233"></a><span data-ttu-id="94249-102">Ошибка компилятора CS0233</span><span class="sxs-lookup"><span data-stu-id="94249-102">Compiler Error CS0233</span></span>
<span data-ttu-id="94249-103">"идентификатор" не имеет предопределенного размера, поэтому оператор sizeof может использоваться только в небезопасном (unsafe) контексте (рекомендуется использование System.Runtime.InteropServices.Marshal.SizeOf)</span><span class="sxs-lookup"><span data-stu-id="94249-103">'identifier' does not have a predefined size, therefore sizeof can only be used in an unsafe context (consider using System.Runtime.InteropServices.Marshal.SizeOf)</span></span>  
  
 <span data-ttu-id="94249-104">Оператор [sizeof](../../../csharp/language-reference/keywords/sizeof.md) можно использовать только для типов, которые представляют собой константы времени компиляции.</span><span class="sxs-lookup"><span data-stu-id="94249-104">The [sizeof](../../../csharp/language-reference/keywords/sizeof.md) operator can only be used for types that are compile-time constants.</span></span> <span data-ttu-id="94249-105">При получении этой ошибки убедитесь, что размер идентификатора можно определить во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="94249-105">If you are getting this error, make sure that the size of the identifier can be determined at compile time.</span></span> <span data-ttu-id="94249-106">Если это невозможно, используйте <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> вместо `sizeof`.</span><span class="sxs-lookup"><span data-stu-id="94249-106">If it cannot, then use <xref:System.Runtime.InteropServices.Marshal.SizeOf%2A> instead of `sizeof`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="94249-107">Пример</span><span class="sxs-lookup"><span data-stu-id="94249-107">Example</span></span>  
 <span data-ttu-id="94249-108">В следующем примере возникает ошибка CS0233:</span><span class="sxs-lookup"><span data-stu-id="94249-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
