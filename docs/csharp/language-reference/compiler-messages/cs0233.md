# <a name="compiler-error-cs0233"></a><span data-ttu-id="3011a-102">Ошибка компилятора CS0233</span><span class="sxs-lookup"><span data-stu-id="3011a-102">Compiler Error CS0233</span></span>

## <a name="example"></a><span data-ttu-id="3011a-107">Пример</span><span class="sxs-lookup"><span data-stu-id="3011a-107">Example</span></span>  
 <span data-ttu-id="3011a-108">В следующем примере возникает ошибка CS0233:</span><span class="sxs-lookup"><span data-stu-id="3011a-108">The following example generates CS0233:</span></span>  
  
```csharp  
// CS0233.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Sequential)]  
public struct S  
{  
    public int a;  
}  
  
public class MyClass  
{  
    public static void Main()  
    {  
        S myS = new S();  
        Console.WriteLine(sizeof(S));   // CS0233  
        // Try the following line instead:  
        // Console.WriteLine(Marshal.SizeOf(myS));  
   }  
}  
```
