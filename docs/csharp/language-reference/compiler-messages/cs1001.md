# <a name="compiler-error-cs1001"></a><span data-ttu-id="96bdd-102">Ошибка компилятора CS1001</span><span class="sxs-lookup"><span data-stu-id="96bdd-102">Compiler Error CS1001</span></span>
<span data-ttu-id="96bdd-103">Требуется идентификатор</span><span class="sxs-lookup"><span data-stu-id="96bdd-103">Identifier expected</span></span>  
  
 <span data-ttu-id="96bdd-104">Не указан идентификатор.</span><span class="sxs-lookup"><span data-stu-id="96bdd-104">You did not supply an identifier.</span></span> <span data-ttu-id="96bdd-105">Идентификатор представляет собой указываемое вами имя класса, структуры, пространства имен, метода, переменной и т. д.</span><span class="sxs-lookup"><span data-stu-id="96bdd-105">An identifier is the name of a class, struct, namespace, method, variable and so on that you provide.</span></span>  
  
 <span data-ttu-id="96bdd-106">В следующем примере объявляется простой класс без имени:</span><span class="sxs-lookup"><span data-stu-id="96bdd-106">The following example declares a simple class but does not give the class a name:</span></span>  
  
```csharp  
//cs1001.cs  
public class              //CS1001  
    {  
        public int Num {get; set;}  
        void MethodA(){}  
    }  
```  
  
 <span data-ttu-id="96bdd-107">В следующем примере возникает ошибка CS1001, поскольку при объявлении перечисления необходимо указать его члены:</span><span class="sxs-lookup"><span data-stu-id="96bdd-107">The following sample generates CS1001 because, when declaring an enum, you must specify members:</span></span>  
  
```csharp  
// CS1001.cs  
public class clx  
{  
   enum Colors : int  
   {  
      'a', 'b' // CS1001, 'a' is not a valid int identifier  
       // The following line shows examples of valid identifiers:  
       // Blue, Red, Orange  
   };  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="96bdd-108">Имена параметров являются обязательными даже в том случае, если компилятор не использует их, например в определении интерфейса.</span><span class="sxs-lookup"><span data-stu-id="96bdd-108">Parameter names are required even if the compiler doesn't use them, for example, in an interface definition.</span></span> <span data-ttu-id="96bdd-109">Они необходимы, чтобы другие программисты, использующие интерфейс, могли понять значение параметров.</span><span class="sxs-lookup"><span data-stu-id="96bdd-109">These parameters are required so that programmers who are consuming the interface know something about the what the parameters mean.</span></span>  
  
```csharp  
// CS1001-2.cs  
// compile with: /target:library  
interface IMyTest  
{  
   void TestFunc1(int, int);  // CS1001  
   // Use the following line instead:  
   // void TestFunc1(int a, int b);  
}  
  
class CMyTest : IMyTest  
{  
   void IMyTest.TestFunc1(int a, int b)  
   {  
   }  
}  
```
