## <a name="example"></a><span data-ttu-id="c55d1-107">Пример</span><span class="sxs-lookup"><span data-stu-id="c55d1-107">Example</span></span>

```csharp

interface ITest  
{  
    event MyEvent Clicked;  
}

{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead

    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
