## <a name="example"></a><span data-ttu-id="05862-107">Пример</span><span class="sxs-lookup"><span data-stu-id="05862-107">Example</span></span>  

 <span data-ttu-id="05862-108">Следующий пример приводит к возникновению ошибки CS0417:</span><span class="sxs-lookup"><span data-stu-id="05862-108">The following example generates CS0417:</span></span>  

```csharp

    // The following line causes CS0417.

}  
```
