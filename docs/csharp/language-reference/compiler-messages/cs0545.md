# <a name="compiler-error-cs0545"></a><span data-ttu-id="c7d58-102">Ошибка компилятора CS0545</span><span class="sxs-lookup"><span data-stu-id="c7d58-102">Compiler Error CS0545</span></span>
<span data-ttu-id="c7d58-103">"функция": переопределение невозможно, так как "свойство" не имеет функции доступа get, доступной для переопределения</span><span class="sxs-lookup"><span data-stu-id="c7d58-103">'function' : cannot override because 'property' does not have an overridable get accessor</span></span>  
  
 <span data-ttu-id="c7d58-104">Была предпринята попытка указать переопределение для метода доступа свойства, в то время как базовый класс не имеет такого указания переопределения.</span><span class="sxs-lookup"><span data-stu-id="c7d58-104">A try was made to define an override for a property accessor when the base class has no such definition to override.</span></span> <span data-ttu-id="c7d58-105">Эту ошибку можно устранить следующими способами:</span><span class="sxs-lookup"><span data-stu-id="c7d58-105">You can resolve this error by:</span></span>  
  
-   <span data-ttu-id="c7d58-106">Добавление метода доступа `set` в базовый класс.</span><span class="sxs-lookup"><span data-stu-id="c7d58-106">Adding a `set` accessor in the base class.</span></span>

-   <span data-ttu-id="c7d58-108">Скрытие свойства базового класса путем добавления ключевого слова [new](../../../csharp/language-reference/keywords/new.md) к свойству производного класса.</span><span class="sxs-lookup"><span data-stu-id="c7d58-108">Hiding the base class property by adding the [new](../../../csharp/language-reference/keywords/new.md) keyword to a property in a derived class.</span></span>  
  
-   <span data-ttu-id="c7d58-109">Присвоение свойству базового класса модификатора [virtual](../../../csharp/language-reference/keywords/virtual.md).</span><span class="sxs-lookup"><span data-stu-id="c7d58-109">Making the base class property [virtual](../../../csharp/language-reference/keywords/virtual.md).</span></span>  
  
 <span data-ttu-id="c7d58-110">Дополнительные сведения см. в разделе [Использование свойств](../../../csharp/programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="c7d58-110">For more information, see [Using Properties](../../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="c7d58-111">Пример</span><span class="sxs-lookup"><span data-stu-id="c7d58-111">Example</span></span>  
 <span data-ttu-id="c7d58-112">В следующем примере возникает ошибка CS0545.</span><span class="sxs-lookup"><span data-stu-id="c7d58-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
