# <a name="compiler-error-cs1936"></a><span data-ttu-id="8a309-102">Ошибка компилятора CS1936</span><span class="sxs-lookup"><span data-stu-id="8a309-102">Compiler Error CS1936</span></span>

<span data-ttu-id="8a309-103">Не удалось найти реализацию шаблона запроса для исходного типа "тип".</span><span class="sxs-lookup"><span data-stu-id="8a309-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="8a309-104">Метод "метод" не найден.</span><span class="sxs-lookup"><span data-stu-id="8a309-104">'method' not found.</span></span>  
  
<span data-ttu-id="8a309-105">Для отправки запроса к исходному типу этот тип должен реализовывать стандартные методы операторов запросов, которые вызываются в запросе.</span><span class="sxs-lookup"><span data-stu-id="8a309-105">In order to query a source type, that type must implement the standard query operator methods that you are invoking in the query.</span></span> <span data-ttu-id="8a309-106">Реализация может иметь вид членов класса или методов расширения, которые вводятся в область действия с помощью соответствующей директивы `using`.</span><span class="sxs-lookup"><span data-stu-id="8a309-106">The implementation can be either in the form of class members or extension methods that are brought into scope with the appropriate `using` directive.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="8a309-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="8a309-107">To correct this error</span></span>

- <span data-ttu-id="8a309-108">Убедитесь, что запрашивается коллекция объектов, а не отдельный объект.</span><span class="sxs-lookup"><span data-stu-id="8a309-108">Make sure that you are querying a collection of objects, not an individual object.</span></span>  
  
- <span data-ttu-id="8a309-109">Убедитесь, что указаны необходимые директивы `using`.</span><span class="sxs-lookup"><span data-stu-id="8a309-109">Make sure that you have specified the necessary `using` directives.</span></span>  

## <a name="example"></a><span data-ttu-id="8a309-110">Пример</span><span class="sxs-lookup"><span data-stu-id="8a309-110">Example</span></span>

<span data-ttu-id="8a309-111">Следующий пример приводит к возникновению ошибки CS1936.</span><span class="sxs-lookup"><span data-stu-id="8a309-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="8a309-112">Эта ошибка обычно возникает, если случайно запрашивается объект какого-то типа вместо коллекции таких объектов.</span><span class="sxs-lookup"><span data-stu-id="8a309-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
