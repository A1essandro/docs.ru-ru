# <a name="compiler-error-cs0016"></a>Ошибка компилятора CS0016
Не удалось произвести запись в выходной файл "файл" — "причина"  
  
 Компилятору не удалось выполнить запись в выходной файл. Проверьте путь к файлу и убедитесь, что он существует. Если в этом месте уже существует ранее построенный файл, убедитесь, что он не защищен от записи и не заблокирован какими-либо процессами. Например, перед попыткой выполнить построение убедитесь, что исполняемый файл не выполняется.
