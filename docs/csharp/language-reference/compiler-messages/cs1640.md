# <a name="compiler-error-cs1640"></a><span data-ttu-id="12782-102">Ошибка компилятора CS1640</span><span class="sxs-lookup"><span data-stu-id="12782-102">Compiler Error CS1640</span></span>
<span data-ttu-id="12782-103">Оператор foreach не может использоваться с переменными типа "тип", так как он реализует несколько экземпляров "интерфейс"; рекомендуется предварительное приведение к определенной реализации интерфейса</span><span class="sxs-lookup"><span data-stu-id="12782-103">foreach statement cannot operate on variables of type 'type' because it implements multiple instantiations of 'interface', try casting to a specific interface instantiation</span></span>  
  
 <span data-ttu-id="12782-104">Тип наследуется от двух или более экземпляров IEnumerator\<T>, то есть уникального перечисления этого типа, которое может использоваться инструкцией `foreach`, не существует.</span><span class="sxs-lookup"><span data-stu-id="12782-104">The type inherits from two or more instances of IEnumerator\<T>, which means there is not a unique enumeration of the type that `foreach` could use.</span></span> <span data-ttu-id="12782-105">Укажите тип IEnumerator\<T> или используйте другую конструкцию цикла.</span><span class="sxs-lookup"><span data-stu-id="12782-105">Specify the type of IEnumerator\<T> or use another looping construct.</span></span>  
  
## <a name="example"></a><span data-ttu-id="12782-106">Пример</span><span class="sxs-lookup"><span data-stu-id="12782-106">Example</span></span>  
 <span data-ttu-id="12782-107">В следующем примере возникает ошибка CS1640:</span><span class="sxs-lookup"><span data-stu-id="12782-107">The following sample generates CS1640:</span></span>  
  
```csharp  
// CS1640.cs  
  
using System;  
using System.Collections;  
using System.Collections.Generic;  
  
public class C : IEnumerable, IEnumerable<int>, IEnumerable<string>  
{  
    IEnumerator<int> IEnumerable<int>.GetEnumerator()  
    {  
        yield break;  
    }  
  
    IEnumerator<string> IEnumerable<string>.GetEnumerator()  
    {  
        yield break;  
    }  
  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        return (IEnumerator)((IEnumerable<string>)this).GetEnumerator();  
    }  
}  
  
public class Test  
{  
    public static int Main()  
    {  
        foreach (int i in new C()){}    // CS1640  
  
        // Try specifing the type of IEnumerable<T>  
        // foreach (int i in (IEnumerable<int>)new C()){}  
        return 1;  
    }  
}  
```
