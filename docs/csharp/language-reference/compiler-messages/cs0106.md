## <a name="example"></a><span data-ttu-id="b0528-115">Пример</span><span class="sxs-lookup"><span data-stu-id="b0528-115">Example</span></span>

```csharp

{  
   interface I  
   {  
      void m();  
      static public void f();   // CS0106

   {  
      public void I.m() {}   // CS0106  
      public static void Main() {}  
   }  
}
```
