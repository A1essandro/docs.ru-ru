 <span data-ttu-id="2829c-105">Как правило, эта ошибка возникает в тех случаях, когда вместо выражения во встроенное выражение на странице ASP.NET помещается инструкция.</span><span class="sxs-lookup"><span data-stu-id="2829c-105">A common cause of this error is placing a statement, rather than an expression, within an inline expression in an ASP.NET page.</span></span> <span data-ttu-id="2829c-106">Например, следующий код содержит ошибку:</span><span class="sxs-lookup"><span data-stu-id="2829c-106">For example, the following is incorrect:</span></span>
