# <a name="compiler-error-cs1708"></a><span data-ttu-id="866a0-102">Ошибка компилятора CS1708</span><span class="sxs-lookup"><span data-stu-id="866a0-102">Compiler Error CS1708</span></span>
<span data-ttu-id="866a0-103">Доступ к буферам фиксированного размера разрешен только через локальные переменные или поля</span><span class="sxs-lookup"><span data-stu-id="866a0-103">Fixed size buffers can only be accessed through locals or fields</span></span>  
  
 <span data-ttu-id="866a0-104">Новая функция в C# 2.0 — способность определять встроенный массив внутри `struct`.</span><span class="sxs-lookup"><span data-stu-id="866a0-104">A new feature in C# 2.0 is the ability to define an in-line array inside of a `struct`.</span></span> <span data-ttu-id="866a0-105">Доступ к таким массивам может осуществляться только через локальные переменные или поля, и ссылаться на них как на промежуточные значения слева от выражения нельзя.</span><span class="sxs-lookup"><span data-stu-id="866a0-105">Such arrays can only be accessed via local variables or fields, and may not be referenced as intermediate values on the left-hand side of an expression.</span></span> <span data-ttu-id="866a0-106">Кроме того, доступ к массивам нельзя получить через поля, которые являются `static` или `readonly`.</span><span class="sxs-lookup"><span data-stu-id="866a0-106">Also, the arrays cannot be accessed by fields that are `static` or `readonly`.</span></span>  
  
 <span data-ttu-id="866a0-107">Чтобы устранить эту ошибку, определите переменную массива и присвойте встроенный массив переменной.</span><span class="sxs-lookup"><span data-stu-id="866a0-107">To resolve this error, define an array variable, and assign the in-line array to the variable.</span></span> <span data-ttu-id="866a0-108">Кроме того, можно удалить модификатор `static` или `readonly` из поля, представляющего встроенный массив.</span><span class="sxs-lookup"><span data-stu-id="866a0-108">Or, remove the `static` or `readonly` modifier from the field representing the in-line array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="866a0-109">Пример</span><span class="sxs-lookup"><span data-stu-id="866a0-109">Example</span></span>  
 <span data-ttu-id="866a0-110">В следующем примере возникает ошибка CS1708.</span><span class="sxs-lookup"><span data-stu-id="866a0-110">The following sample generates CS1708.</span></span>  
  
```csharp  
// CS1708.cs  
// compile with: /unsafe  
using System;  
  
unsafe public struct S  
{  
    public fixed char name[10];  
}  
  
public unsafe class C  
{  
    public S UnsafeMethod()  
    {  
        S myS = new S();  
        return myS;  
    }  
  
    static void Main()  
    {  
        C myC = new C();  
        myC.UnsafeMethod().name[3] = 'a';  // CS1708  
        // Uncomment the following 2 lines to resolve:  
        // S myS = myC.UnsafeMethod();  
        // myS.name[3] = 'a';  
  
        // The field cannot be static.  
        C._s1.name[3] = 'a';  // CS1708  
  
        // The field cannot be readonly.  
        myC._s2.name[3] = 'a';  // CS1708  
    }  
  
    static readonly S _s1;  
    public readonly S _s2;  
}  
```
