# <a name="compiler-warning-level-4-cs0429"></a><span data-ttu-id="e3ff7-102">Предупреждение компилятора (уровень 4) CS0429</span><span class="sxs-lookup"><span data-stu-id="e3ff7-102">Compiler Warning (level 4) CS0429</span></span>
<span data-ttu-id="e3ff7-103">Обнаружен недостижимый код в выражении</span><span class="sxs-lookup"><span data-stu-id="e3ff7-103">Unreachable expression code detected</span></span>  
  
 <span data-ttu-id="e3ff7-104">Эта ошибка возникает в тех случаях, когда невозможно достижение какой-либо части выражения в коде.</span><span class="sxs-lookup"><span data-stu-id="e3ff7-104">This error occurs whenever part of an expression in your code is unreachable.</span></span> <span data-ttu-id="e3ff7-105">В следующем примере условие `false && myTest()` соответствует этому критерию, поскольку метод `myTest()` никогда не будет выполнен из-за того, что левая часть операции `&&` всегда будет возвращать значение false.</span><span class="sxs-lookup"><span data-stu-id="e3ff7-105">In the following example, the condition `false && myTest()` meets this criteria because the `myTest()` method will never get evaluated due to the fact that the left side of the `&&` operation is always false.</span></span> <span data-ttu-id="e3ff7-106">Как только оператор `&&` вычисляет инструкцию `false` как значение false, обработка прекращается, и вычисление правой части выражения ни при каких условиях не производится.</span><span class="sxs-lookup"><span data-stu-id="e3ff7-106">As soon as the `&&` operator evaluates the `false` statement as false, it stops the evaluation, and will never evaluate the right side.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e3ff7-107">Пример</span><span class="sxs-lookup"><span data-stu-id="e3ff7-107">Example</span></span>

```csharp  
// CS0429.cs  
public class cs0429   
{  
    public static void Main()   
    {  
        if (false && myTest())  // CS0429  
        // Try the following line instead:  
        // if (true && myTest())  
        {  
        }  
        else  
        {  
            int i = 0;  
            i++;  
        }  
    }  
  
    static bool myTest() { return true; }  
}  
```
