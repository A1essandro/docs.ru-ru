## <a name="example"></a><span data-ttu-id="de0c5-108">Пример</span><span class="sxs-lookup"><span data-stu-id="de0c5-108">Example</span></span>  
 <span data-ttu-id="de0c5-109">В следующем примере возникает ошибка CS0122:</span><span class="sxs-lookup"><span data-stu-id="de0c5-109">The following sample generates CS0122:</span></span>  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
