 <span data-ttu-id="bf5bb-248">Арифметические операторы ([+](../../../csharp/language-reference/operators/addition-operator.md), [-](../../../csharp/language-reference/operators/subtraction-operator.md), [*](../../../csharp/language-reference/operators/multiplication-operator.md), [/](../../../csharp/language-reference/operators/division-operator.md)) могут возвращать результаты, выходящие за пределы диапазона возможных значений для используемого числового типа.</span><span class="sxs-lookup"><span data-stu-id="bf5bb-248">The arithmetic operators ([+](../../../csharp/language-reference/operators/addition-operator.md), [-](../../../csharp/language-reference/operators/subtraction-operator.md), [*](../../../csharp/language-reference/operators/multiplication-operator.md), [/](../../../csharp/language-reference/operators/division-operator.md)) can produce results that are outside the range of possible values for the numeric type involved.</span></span> <span data-ttu-id="bf5bb-249">Подробнее см. в разделе, посвященном конкретному оператору, однако в общем случае применяются следующие правила.</span><span class="sxs-lookup"><span data-stu-id="bf5bb-249">You should refer to the section on a particular operator for details, but in general:</span></span>
