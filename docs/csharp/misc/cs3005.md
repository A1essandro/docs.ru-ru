## <a name="example"></a><span data-ttu-id="44ae3-106">Пример</span><span class="sxs-lookup"><span data-stu-id="44ae3-106">Example</span></span>

```csharp  
public class a  
{  
    public static int a1 = 0;  
    public static int A1 = 1;   // CS3005  
  
    public static void Main()  
    {  
        Console.WriteLine(a1);  
        Console.WriteLine(A1);  
    }  
}  
```
