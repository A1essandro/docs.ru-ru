# <a name="compiler-warning-level-1-cs3004"></a>Предупреждение компилятора (уровень 1) CS3004
Смешанные и разложенные знаки Юникода не соответствуют CLS  
  
 В соответствии со спецификацией CLS в идентификаторах [public](../../csharp/language-reference/keywords/public.md), [protected](../../csharp/language-reference/keywords/protected.md)или `protected internal` допускаются только составные символы ЮНИКОДА. Дополнительные сведения о соответствии спецификации CLS см. в разделах [Создание кода, совместимого с CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) и [Независимость от языка и независимые от языка компоненты](../../../docs/standard/language-independence-and-language-independent-components.md).
