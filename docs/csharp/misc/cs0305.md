## <a name="example"></a><span data-ttu-id="1111a-106">Пример</span><span class="sxs-lookup"><span data-stu-id="1111a-106">Example</span></span>  
 <span data-ttu-id="1111a-107">Следующий пример приводит к возникновению ошибки CS0305:</span><span class="sxs-lookup"><span data-stu-id="1111a-107">The following sample generates CS0305.</span></span>  
  
```  
// CS0305.cs  
public class MyList<T> {}  
public class MyClass<T> {}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
