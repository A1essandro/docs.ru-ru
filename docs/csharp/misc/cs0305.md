## <a name="example"></a><span data-ttu-id="428f7-106">Пример</span><span class="sxs-lookup"><span data-stu-id="428f7-106">Example</span></span>

```csharp

{  
   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
