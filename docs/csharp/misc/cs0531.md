 <span data-ttu-id="3d533-104">Методы, объявленные в [интерфейсе](../../csharp/language-reference/keywords/interface.md) , должны быть реализованы в классе, который наследует от него, а не в самом интерфейсе.</span><span class="sxs-lookup"><span data-stu-id="3d533-104">Methods that are declared in an [interface](../../csharp/language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="3d533-105">В следующем примере возникает ошибка CS0531:</span><span class="sxs-lookup"><span data-stu-id="3d533-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
