# <a name="compiler-error-cs1025"></a><span data-ttu-id="08b98-102">Ошибка компилятора CS1025</span><span class="sxs-lookup"><span data-stu-id="08b98-102">Compiler Error CS1025</span></span>
<span data-ttu-id="08b98-103">Требуется однострочный комментарий или признак конца строки.</span><span class="sxs-lookup"><span data-stu-id="08b98-103">Single-line comment or end-of-line expected</span></span>  
  
 <span data-ttu-id="08b98-104">Строка с [директивой препроцессора](../../csharp/language-reference/preprocessor-directives/index.md) не может содержать многострочный комментарий.</span><span class="sxs-lookup"><span data-stu-id="08b98-104">A line with a [preprocessor directive](../../csharp/language-reference/preprocessor-directives/index.md) cannot have a multiline comment.</span></span>  
  
 <span data-ttu-id="08b98-105">Следующий пример приводит к возникновению ошибки CS1025:</span><span class="sxs-lookup"><span data-stu-id="08b98-105">The following sample generates CS1025:</span></span>  
  
```csharp

*/   // CS1025  
#endif   // this is a good comment  
```  
  
 <span data-ttu-id="08b98-106">Ошибка CS1025 также может возникать при попытке использования недопустимой директивы препроцессора, например:</span><span class="sxs-lookup"><span data-stu-id="08b98-106">CS1025 could also occur if you attempt some invalid preprocessor directive, as follows:</span></span>  
  
```csharp

class Sample  
{  
   static void Main()  
   {  
      #if a 1   // CS1025, invalid syntax  
         System.Console.WriteLine("Hello, World!");  
      #endif  
   }  
}  
```
