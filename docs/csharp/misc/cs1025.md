 <span data-ttu-id="01b46-104">Строка с [директивой препроцессора](../../csharp/language-reference/preprocessor-directives/index.md) не может содержать многострочный комментарий.</span><span class="sxs-lookup"><span data-stu-id="01b46-104">A line with a [preprocessor directive](../../csharp/language-reference/preprocessor-directives/index.md) cannot have a multiline comment.</span></span>

 <span data-ttu-id="01b46-106">Ошибка CS1025 также может возникать при попытке использования недопустимой директивы препроцессора, например:</span><span class="sxs-lookup"><span data-stu-id="01b46-106">CS1025 could also occur if you attempt some invalid preprocessor directive, as follows:</span></span>

```  
{  
   static void Main()  
   {

      #endif  
   }  
}  
```
