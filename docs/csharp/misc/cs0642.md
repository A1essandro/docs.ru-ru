# <a name="compiler-warning-level-3-cs0642"></a><span data-ttu-id="75634-102">Предупреждение компилятора (уровень 3) CS0642</span><span class="sxs-lookup"><span data-stu-id="75634-102">Compiler Warning (level 3) CS0642</span></span>
<span data-ttu-id="75634-103">Возможно, ошибочный пустой оператор</span><span class="sxs-lookup"><span data-stu-id="75634-103">Possible mistaken empty statement</span></span>  
  
 <span data-ttu-id="75634-104">Точка с запятой, стоящая после условного оператора, может быть причиной изменения предполагаемого способа выполнения кода.</span><span class="sxs-lookup"><span data-stu-id="75634-104">A semicolon after a conditional statement may cause your code to execute differently than intended.</span></span>  
  
 <span data-ttu-id="75634-105">Можно использовать **/nowarn** параметр компилятора или `#pragmas warning` для отключения этого предупреждения; в разделе [/nowarn (параметры компилятора C#)](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) или [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) для получения дополнительной информации.</span><span class="sxs-lookup"><span data-stu-id="75634-105">You can use **/nowarn** compiler option or `#pragmas warning` to disable this warning; see [/nowarn (C# Compiler Options)](../../csharp/language-reference/compiler-options/nowarn-compiler-option.md) or [#pragma warning](../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) for more information.</span></span>  
  
 <span data-ttu-id="75634-106">При компиляции следующего примера будет выдано предупреждение CS0642:</span><span class="sxs-lookup"><span data-stu-id="75634-106">The following sample generates CS0642:</span></span>  
  
```csharp  
// CS0642.cs  
// compile with: /W:3  
class MyClass  
{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?  
      {  
         System.Console.WriteLine (i);  
      }  
   }  
}  
```
