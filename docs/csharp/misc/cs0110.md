# <a name="compiler-error-cs0110"></a><span data-ttu-id="09f05-102">Ошибка компилятора CS0110</span><span class="sxs-lookup"><span data-stu-id="09f05-102">Compiler Error CS0110</span></span>
<span data-ttu-id="09f05-103">При оценке постоянного значения для "объявление константы" используется циклическое определение</span><span class="sxs-lookup"><span data-stu-id="09f05-103">The evaluation of the constant value for 'const declaration' involves a circular definition</span></span>  
  
 <span data-ttu-id="09f05-104">Объявление переменной [const](../../csharp/language-reference/keywords/const.md) (`a`) не может ссылаться на другую переменную const (`b`), которая также ссылается (`a`).</span><span class="sxs-lookup"><span data-stu-id="09f05-104">The declaration of a [const](../../csharp/language-reference/keywords/const.md) variable (`a`) cannot reference another const variable (`b`) that also references (`a`).</span></span>  
  
 <span data-ttu-id="09f05-105">Следующий пример приводит к возникновению ошибки CS0110:</span><span class="sxs-lookup"><span data-stu-id="09f05-105">The following sample generates CS0110:</span></span>  
  
```csharp  
// CS0110.cs  
namespace MyNamespace  
{  
   public class A  
   {  
      public static void Main()  
      {  
      }  
   }  
  
   public class B : A  
   {  
      public const int i = c + 1;   // CS0110, c already references i  
      public const int c = i + 1;  
      // the following line would be OK  
      // public const int c = 10;  
   }  
}  
```
