# <a name="compiler-error-cs0452"></a><span data-ttu-id="7ab6d-102">Ошибка компилятора CS0452</span><span class="sxs-lookup"><span data-stu-id="7ab6d-102">Compiler Error CS0452</span></span>
<span data-ttu-id="7ab6d-103">Тип "имя_типа" должен быть ссылочным типом для его использования в качестве параметра "имя_параметра" в универсальном типе или методе "идентификатор_универсального_объекта".</span><span class="sxs-lookup"><span data-stu-id="7ab6d-103">The type 'type name' must be a reference type in order to use it as parameter 'parameter name' in the generic type or method 'identifier of generic'</span></span>  
  
 <span data-ttu-id="7ab6d-104">Эта ошибка возникает при передаче типа значения, такого как `struct` или `int` , в качестве параметра в универсальный тип или метод, имеющий ограничение ссылочного типа.</span><span class="sxs-lookup"><span data-stu-id="7ab6d-104">This error occurs when you pass a value type such as a `struct` or `int` as a parameter to a generic type or method that has a reference type constraint.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7ab6d-105">Пример</span><span class="sxs-lookup"><span data-stu-id="7ab6d-105">Example</span></span>  
 <span data-ttu-id="7ab6d-106">Следующий код приводит к ошибке CS0452.</span><span class="sxs-lookup"><span data-stu-id="7ab6d-106">The following code generates error CS0452.</span></span>  
  
```csharp  
// CS0452.cs

public class Derived1 : BaseClass<int> { } // CS0452  
public class Derived2<S> : BaseClass<S> where S : struct { } // CS0452  
```
