# <a name="compiler-warning-level-1-cs0612"></a><span data-ttu-id="de35c-102">Предупреждение компилятора (уровень 1) CS0612</span><span class="sxs-lookup"><span data-stu-id="de35c-102">Compiler Warning (level 1) CS0612</span></span>
<span data-ttu-id="de35c-103">"член" устарел</span><span class="sxs-lookup"><span data-stu-id="de35c-103">'member' is obsolete</span></span>  
  
 <span data-ttu-id="de35c-104">Конструктор классов пометил член атрибутом [Obsolete](http://msdn.microsoft.com/en-us/05e99cd0-bda6-4f79-a890-1ca093b4b488) .</span><span class="sxs-lookup"><span data-stu-id="de35c-104">The class designer marked a member with the [Obsolete](http://msdn.microsoft.com/en-us/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute.</span></span> <span data-ttu-id="de35c-105">Это означает, что этот член может не поддерживаться в будущих версиях данного класса.</span><span class="sxs-lookup"><span data-stu-id="de35c-105">This means that the member might not be supported in a future version of the class.</span></span>  
  
 <span data-ttu-id="de35c-106">В приведенном ниже примере показано, как при доступе к нерекомендуемому члену создается предупреждение CS0612.</span><span class="sxs-lookup"><span data-stu-id="de35c-106">The following sample shows how accessing an obsolete member generates CS0612:</span></span>  
  
```  
// CS0612.cs  
// compile with: /W:1  
using System;  
  
class MyClass  
{  
   [Obsolete]  
   public static void ObsoleteMethod()  
   {  
   }  
  
   [Obsolete]  
   public static int ObsoleteField;  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
      MyClass.ObsoleteMethod();    // CS0612 here: method is deprecated  
      MyClass.ObsoleteField = 0;   // CS0612 here: field is deprecated  
   }  
}  
```
