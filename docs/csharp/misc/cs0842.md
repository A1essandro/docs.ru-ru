## <a name="example"></a><span data-ttu-id="853b3-108">Пример</span><span class="sxs-lookup"><span data-stu-id="853b3-108">Example</span></span>

```  
{  
    [StructLayout(LayoutKind.Explicit)]

        {  
            return 1;  
        }  
    }  
}  
```
