## <a name="example"></a><span data-ttu-id="fb757-106">Пример</span><span class="sxs-lookup"><span data-stu-id="fb757-106">Example</span></span>  
 <span data-ttu-id="fb757-107">В следующем примере возникает ошибка CS3006.</span><span class="sxs-lookup"><span data-stu-id="fb757-107">The following example generates CS3006.</span></span> <span data-ttu-id="fb757-108">Чтобы устранить это предупреждение, закомментируйте атрибут уровня сборки или удалите одно из определений метода.</span><span class="sxs-lookup"><span data-stu-id="fb757-108">To resolve this warning, comment out the assembly-level attribute or remove one of the method definitions.</span></span>  
  
```csharp

using System;  
  
[assembly: CLSCompliant(true)]  
public class MyClass  
{  
    public void f(int i)  
    {  
    }  
  
    public void f(ref int i)   // CS3006  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
