# <a name="compiler-error-cs0432"></a><span data-ttu-id="05ab0-102">Ошибка компилятора CS0432</span><span class="sxs-lookup"><span data-stu-id="05ab0-102">Compiler Error CS0432</span></span>
<span data-ttu-id="05ab0-103">Псевдоним "идентификатор" не найден</span><span class="sxs-lookup"><span data-stu-id="05ab0-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="05ab0-104">Эта ошибка возникает при использовании "::" справа от идентификатора, который не является псевдонимом.</span><span class="sxs-lookup"><span data-stu-id="05ab0-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="05ab0-105">Для устранения этой ошибки используйте оператор ".".</span><span class="sxs-lookup"><span data-stu-id="05ab0-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="05ab0-106">Следующий пример приводит к возникновению ошибки CS0432:</span><span class="sxs-lookup"><span data-stu-id="05ab0-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
