# <a name="compiler-error-cs0243"></a><span data-ttu-id="a18e7-102">Ошибка компилятора CS0243</span><span class="sxs-lookup"><span data-stu-id="a18e7-102">Compiler Error CS0243</span></span>
<span data-ttu-id="a18e7-103">Атрибут Conditional в методе "метод" недопустим, так как он является методом переопределения</span><span class="sxs-lookup"><span data-stu-id="a18e7-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="a18e7-104">[Условного](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) атрибут не поддерживается в методе, который помечен [переопределить](../../csharp/language-reference/keywords/override.md) ключевое слово.</span><span class="sxs-lookup"><span data-stu-id="a18e7-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="a18e7-105">Дополнительные сведения см. в разделе [знать, когда с помощью переопределений и новые ключевые слова](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="a18e7-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="a18e7-106">Компилятор никогда не привязывается к методам переопределения; он привязывается только к базовому методу, и среда CLR вызывает переопределение соответствующим образом.</span><span class="sxs-lookup"><span data-stu-id="a18e7-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="a18e7-107">В следующем примере возникает ошибка CS0243:</span><span class="sxs-lookup"><span data-stu-id="a18e7-107">The following sample generates CS0243:</span></span>  
  
```  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
