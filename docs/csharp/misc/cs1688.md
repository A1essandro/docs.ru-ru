# <a name="compiler-error-cs1688"></a><span data-ttu-id="801d2-102">Ошибка компилятора CS1688</span><span class="sxs-lookup"><span data-stu-id="801d2-102">Compiler Error CS1688</span></span>
<span data-ttu-id="801d2-103">Невозможно преобразовать блок анонимного метода без списка параметров в делегат "делегат", поскольку он имеет один или несколько выходных параметров</span><span class="sxs-lookup"><span data-stu-id="801d2-103">Cannot convert anonymous method block without a parameter list to delegate type 'delegate' because it has one or more out parameters</span></span>  
  
 <span data-ttu-id="801d2-104">В большинстве случаев компилятор позволяет опускать параметры в блоке анонимного метода.</span><span class="sxs-lookup"><span data-stu-id="801d2-104">The compiler allows parameters to be omitted from an anonymous method block in most cases.</span></span> <span data-ttu-id="801d2-105">Эта ошибка возникает при отсутствии списка параметров в блоке анонимного метода и при наличии параметра `out` у делегата.</span><span class="sxs-lookup"><span data-stu-id="801d2-105">This error arises when the anonymous method block does not have a parameter list, but the delegate has an `out` parameter.</span></span> <span data-ttu-id="801d2-106">Компилятор не допускает возникновение такой ситуации, так как игнорирование имеющегося параметра `out` почти наверняка вызовет появление ошибок.</span><span class="sxs-lookup"><span data-stu-id="801d2-106">The compiler does not allow this situation because it would need to ignore the presence of the `out` parameter, which is unlikely to be the correct behavior.</span></span>
