# <a name="compiler-error-cs1642"></a><span data-ttu-id="02773-102">Ошибка компилятора CS1642</span><span class="sxs-lookup"><span data-stu-id="02773-102">Compiler Error CS1642</span></span>
<span data-ttu-id="02773-103">Поля буферов фиксированного размера могут быть только членами структур.</span><span class="sxs-lookup"><span data-stu-id="02773-103">Fixed size buffer fields may only be members of structs.</span></span>  
  
 <span data-ttu-id="02773-104">Эта ошибка возникает, если поле буфера фиксированного размера используется в `class`, а не в `struct`.</span><span class="sxs-lookup"><span data-stu-id="02773-104">This error occurs if you use a fixed size buffer field in a `class`, instead of a `struct`.</span></span> <span data-ttu-id="02773-105">Чтобы устранить эту ошибку, измените `class` на `struct` или объявите это поле как обычный массив.</span><span class="sxs-lookup"><span data-stu-id="02773-105">To resolve this error, change the `class` to a `struct` or declare the field as an ordinary array.</span></span>  
  
## <a name="example"></a><span data-ttu-id="02773-106">Пример</span><span class="sxs-lookup"><span data-stu-id="02773-106">Example</span></span>  
 <span data-ttu-id="02773-107">Следующий пример приводит к возникновению ошибки CS1642.</span><span class="sxs-lookup"><span data-stu-id="02773-107">The following sample generates CS1642.</span></span>  
  
```csharp  
// CS1642.cs  
// compile with: /unsafe /target:library  
unsafe class C  
{  
   fixed int a[10];   // CS1642  
}  
  
unsafe struct D  
{  
    fixed int a[10];  
}  
  
unsafe class E  
{  
   public int[] a = null;  
}  
```
