# <a name="compiler-error-cs0314"></a><span data-ttu-id="0bc5b-102">Ошибка компилятора CS0314</span><span class="sxs-lookup"><span data-stu-id="0bc5b-102">Compiler Error CS0314</span></span>
<span data-ttu-id="0bc5b-103">Тип "тип1" не может быть использован как параметр-тип "имя" в универсальном типе или методе "имя".</span><span class="sxs-lookup"><span data-stu-id="0bc5b-103">The type 'type1' cannot be used as type parameter 'name' in the generic type or method 'name'.</span></span> <span data-ttu-id="0bc5b-104">Нет преобразования-упаковки или преобразования параметра-типа из "тип1" в "тип2".</span><span class="sxs-lookup"><span data-stu-id="0bc5b-104">There is no boxing conversion or type parameter conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="0bc5b-105">Если универсальным типом используется ограниченный параметр типа, то новый класс также должен удовлетворять тем же ограничениям.</span><span class="sxs-lookup"><span data-stu-id="0bc5b-105">When a generic type uses a type parameter that is constrained, the new class must also satisfy those same constraints.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0bc5b-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="0bc5b-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="0bc5b-107">В приведенном ниже примере добавьте `where T : ClassConstraint` в класс `B`.</span><span class="sxs-lookup"><span data-stu-id="0bc5b-107">In the example that follows, add `where T : ClassConstraint` to class `B`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0bc5b-108">Пример</span><span class="sxs-lookup"><span data-stu-id="0bc5b-108">Example</span></span>  
 <span data-ttu-id="0bc5b-109">Следующий код приводит к возникновению ошибки CS0314:</span><span class="sxs-lookup"><span data-stu-id="0bc5b-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
