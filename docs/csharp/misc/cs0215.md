# <a name="compiler-error-cs0215"></a><span data-ttu-id="eb326-102">Ошибка компилятора CS0215</span><span class="sxs-lookup"><span data-stu-id="eb326-102">Compiler Error CS0215</span></span>
<span data-ttu-id="eb326-103">Тип возвращаемого значения операторов Истина и Ложь должен быть логическим.</span><span class="sxs-lookup"><span data-stu-id="eb326-103">The return type of operator True or False must be bool</span></span>  
  
 <span data-ttu-id="eb326-104">Пользовательские операторы [true](../../csharp/language-reference/keywords/true.md) и [false](../../csharp/language-reference/keywords/false.md) должны иметь тип возврата [bool](../../csharp/language-reference/keywords/bool.md).</span><span class="sxs-lookup"><span data-stu-id="eb326-104">User-defined [true](../../csharp/language-reference/keywords/true.md) and [false](../../csharp/language-reference/keywords/false.md) operators must have a return type of [bool](../../csharp/language-reference/keywords/bool.md).</span></span> <span data-ttu-id="eb326-105">Дополнительные сведения см. в разделе [Перегружаемые операторы](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span><span class="sxs-lookup"><span data-stu-id="eb326-105">For more information, see [Overloadable Operators](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span></span>  
  
 <span data-ttu-id="eb326-106">В следующем примере возникает ошибка CS0215:</span><span class="sxs-lookup"><span data-stu-id="eb326-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
