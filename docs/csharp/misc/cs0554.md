 <span data-ttu-id="9c14f-104">Пользовательские преобразования в значения производного класса не допускаются; в операторе подобного типа нет необходимости.</span><span class="sxs-lookup"><span data-stu-id="9c14f-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>

 <span data-ttu-id="9c14f-106">В следующем примере возникает ошибка CS0554:</span><span class="sxs-lookup"><span data-stu-id="9c14f-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
