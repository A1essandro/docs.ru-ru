 <span data-ttu-id="2dea8-104">Разделяемый метод должен иметь определяющее объявление, которое определяет сигнатуру метода (имя, тип возвращаемого значения и параметры).</span><span class="sxs-lookup"><span data-stu-id="2dea8-104">A partial method must have a defining declaration that defines the signature (name, return type and parameters) of the method.</span></span> <span data-ttu-id="2dea8-105">Реализация тела метода необязательна.</span><span class="sxs-lookup"><span data-stu-id="2dea8-105">The implementation or method body is optional.</span></span>
