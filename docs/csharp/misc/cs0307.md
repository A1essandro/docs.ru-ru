 <span data-ttu-id="41d2b-105">Названная конструкция не является типом или методом, единственными конструкциями, которые могут принимать универсальные аргументы.</span><span class="sxs-lookup"><span data-stu-id="41d2b-105">The construct named was not a type or a method, the only constructs that can take generic arguments.</span></span> <span data-ttu-id="41d2b-106">Удалите аргументы типа в угловых скобках.</span><span class="sxs-lookup"><span data-stu-id="41d2b-106">Remove the type arguments in angle brackets.</span></span> <span data-ttu-id="41d2b-107">Если необходим универсальный объект, объявите свою универсальную конструкцию как универсальный тип или метод.</span><span class="sxs-lookup"><span data-stu-id="41d2b-107">If a generic is needed, declare your generic construct as a generic type or method.</span></span>  
  
 <span data-ttu-id="41d2b-108">В следующем примере возникает ошибка CS0307:</span><span class="sxs-lookup"><span data-stu-id="41d2b-108">The following sample generates CS0307:</span></span>  
  
```csharp  
// CS0307.cs  
class C  
{  
   public int P { get { return 1; } }  
   public static void Main()  
   {  
      C c = new C();  
      int p = c.P<int>();  // CS0307 – C.P is a property  
      // Try this instead  
      // int p = c.P;  
   }  
}  
```
