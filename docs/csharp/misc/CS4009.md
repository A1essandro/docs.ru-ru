# <a name="compiler-error-cs4009"></a><span data-ttu-id="c6684-102">CS4009 ошибки компилятора</span><span class="sxs-lookup"><span data-stu-id="c6684-102">Compiler Error CS4009</span></span>

<span data-ttu-id="c6684-103">«Type.Method»: точка входа не может быть помечен `async` модификатор.</span><span class="sxs-lookup"><span data-stu-id="c6684-103">'Type.Method': an entry point cannot be marked with the `async` modifier.</span></span>

<span data-ttu-id="c6684-104">Нельзя использовать `async` ключевое слово в точке входа приложения (обычно `Main` метода).</span><span class="sxs-lookup"><span data-stu-id="c6684-104">You cannot use the `async` keyword in the application entry point (typically the `Main` method).</span></span>  

<span data-ttu-id="c6684-105">В следующем примере создается CS4009:</span><span class="sxs-lookup"><span data-stu-id="c6684-105">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```
