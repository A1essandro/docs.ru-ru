<span data-ttu-id="57642-103">"конструктор": статический конструктор не должен иметь параметры</span><span class="sxs-lookup"><span data-stu-id="57642-103">'constructor' : a static constructor must be parameterless</span></span>  
  
 <span data-ttu-id="57642-104">[Статический](../../csharp/language-reference/keywords/static.md) конструктор не может объявляться с одним или несколькими параметрами.</span><span class="sxs-lookup"><span data-stu-id="57642-104">A [static](../../csharp/language-reference/keywords/static.md) constructor cannot be declared with one or more parameters.</span></span> <span data-ttu-id="57642-105">Дополнительные сведения см. в разделе [Конструкторы](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="57642-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="57642-106">Следующий пример приводит к возникновению ошибки CS0132:</span><span class="sxs-lookup"><span data-stu-id="57642-106">The following sample generates CS0132:</span></span>  
  
```  
// CS0132.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public MyClass(int i)  
      {  
      }  
   }  
  
   public class MyClass2 : MyClass  
   {  
      static MyClass2(int i)   // CS0132  
      {  
      }  
   }  
}  
```
