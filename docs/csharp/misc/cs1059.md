## <a name="example"></a><span data-ttu-id="36d52-110">Пример</span><span class="sxs-lookup"><span data-stu-id="36d52-110">Example</span></span>  
 <span data-ttu-id="36d52-111">Приведенный ниже пример приводит к возникновению ошибки CS1059, так как `i` является не переменной, а константой, а `E` имеет тип `Enum` , элементы которого также представляют собой постоянные значения.</span><span class="sxs-lookup"><span data-stu-id="36d52-111">The following example generates CS1059 because `i` is a constant, not a variable, and `E` is an `Enum` type, whose elements are also constant values.</span></span>  
  
```  
// CS1059.cs  
    class Program  
    {  
        public enum E : sbyte  
        {  
            a = 1,  
            b = 2  
        }  
  
        static void Main(string[] args)  
        {  
            const int i = 0;  
            i++;            // CS1059  
            E test = E.a++; // CS1059  
        }  
    }  
```
