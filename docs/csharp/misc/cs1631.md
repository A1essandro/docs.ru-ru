# <a name="compiler-error-cs1631"></a><span data-ttu-id="92f80-102">Ошибка компилятора CS1631</span><span class="sxs-lookup"><span data-stu-id="92f80-102">Compiler Error CS1631</span></span>
<span data-ttu-id="92f80-103">Нельзя использовать оператор yield в теле предложения catch.</span><span class="sxs-lookup"><span data-stu-id="92f80-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="92f80-104">Использование оператора yield в теле предложения catch недопустимо.</span><span class="sxs-lookup"><span data-stu-id="92f80-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="92f80-105">Чтобы избежать этой ошибки, переместите оператор yield за пределы предложения catch.</span><span class="sxs-lookup"><span data-stu-id="92f80-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="92f80-106">Следующий пример приводит к возникновению ошибки CS1631:</span><span class="sxs-lookup"><span data-stu-id="92f80-106">The following sample generates CS1631:</span></span>  
  
```  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
