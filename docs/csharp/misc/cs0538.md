# <a name="compiler-error-cs0538"></a><span data-ttu-id="3e0e1-102">Ошибка компилятора CS0538</span><span class="sxs-lookup"><span data-stu-id="3e0e1-102">Compiler Error CS0538</span></span>
<span data-ttu-id="3e0e1-103">"имя" в явном объявлении интерфейса не является интерфейсом</span><span class="sxs-lookup"><span data-stu-id="3e0e1-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="3e0e1-104">Предпринята попытка явного объявления [интерфейса](../../csharp/language-reference/keywords/interface.md), но интерфейс не существует.</span><span class="sxs-lookup"><span data-stu-id="3e0e1-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="3e0e1-105">Следующий пример приводит к возникновению ошибки CS0538:</span><span class="sxs-lookup"><span data-stu-id="3e0e1-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
