 <span data-ttu-id="77247-104">Параметр типа использовался как аргумент атрибута, что запрещено.</span><span class="sxs-lookup"><span data-stu-id="77247-104">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="77247-105">Используйте тип, не являющийся универсальным.</span><span class="sxs-lookup"><span data-stu-id="77247-105">Use a non-generic type.</span></span>

```csharp  
// CS0416.cs  
public class MyAttribute : System.Attribute  
{  
   public MyAttribute(System.Type t)  
   {  
   }  
}

{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
