 <span data-ttu-id="5f95c-104">Переменная, объявленная в [операторе fixed](../../csharp/language-reference/keywords/fixed-statement.md) , должна быть указателем.</span><span class="sxs-lookup"><span data-stu-id="5f95c-104">The variable that you declare in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md) must be a pointer.</span></span> <span data-ttu-id="5f95c-105">Дополнительные сведения см. в разделе [Небезопасный код и указатели](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="5f95c-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="5f95c-106">Следующий пример приводит к возникновению ошибки CS0209:</span><span class="sxs-lookup"><span data-stu-id="5f95c-106">The following sample generates CS0209:</span></span>  
  
```  
// CS0209.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0209  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```
