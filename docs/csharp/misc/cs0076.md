# Ошибка компилятора CS0076
Имя перечислителя "value\_\_" зарезервировано и не может использоваться  
  
 Элемент в [перечислении](../../csharp/language-reference/keywords/enum.md) не может иметь идентификатор с именем **value\_\_**.
