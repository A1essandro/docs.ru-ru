# <a name="compiler-error-cs0035"></a><span data-ttu-id="5df91-102">Ошибка компилятора CS0035</span><span class="sxs-lookup"><span data-stu-id="5df91-102">Compiler Error CS0035</span></span>
<span data-ttu-id="5df91-103">Оператор "оператор" для операнда типа "тип" является неоднозначным</span><span class="sxs-lookup"><span data-stu-id="5df91-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="5df91-104">Компилятору доступно несколько преобразований, и ему не удается определить, какое преобразование использовать перед применением оператора.</span><span class="sxs-lookup"><span data-stu-id="5df91-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="5df91-105">Дополнительные сведения см. в разделе [шаблонного определенные пользователем преобразования](https://docs.microsoft.com/cpp/misc/templated-user-defined-conversions) и [операторы преобразования](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="5df91-105">For more information, see [Templated User Defined Conversions](https://docs.microsoft.com/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="5df91-106">Следующий пример приводит к возникновению ошибки CS0035:</span><span class="sxs-lookup"><span data-stu-id="5df91-106">The following sample generates CS0035:</span></span>  
  
```  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
