# <a name="compiler-error-cs1947"></a><span data-ttu-id="4dc47-102">Ошибка компилятора CS1947</span><span class="sxs-lookup"><span data-stu-id="4dc47-102">Compiler Error CS1947</span></span>

## <a name="example"></a><span data-ttu-id="4dc47-109">Пример</span><span class="sxs-lookup"><span data-stu-id="4dc47-109">Example</span></span>  
 <span data-ttu-id="4dc47-110">Следующий код приводит к возникновению ошибки CS1947:</span><span class="sxs-lookup"><span data-stu-id="4dc47-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
