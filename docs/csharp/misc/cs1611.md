# <a name="compiler-error-cs1611"></a><span data-ttu-id="ae108-102">Ошибка компилятора CS1611</span><span class="sxs-lookup"><span data-stu-id="ae108-102">Compiler Error CS1611</span></span>
<span data-ttu-id="ae108-103">Параметр params не может объявляться как ref или out</span><span class="sxs-lookup"><span data-stu-id="ae108-103">The params parameter cannot be declared as in ref or out</span></span>  
  
 <span data-ttu-id="ae108-104">Ключевые слова [в](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) или [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) не может использоваться с [params](../../csharp/language-reference/keywords/params.md) ключевое слово.</span><span class="sxs-lookup"><span data-stu-id="ae108-104">The keywords [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) cannot be used with the [params](../../csharp/language-reference/keywords/params.md) keyword.</span></span>  
  
 <span data-ttu-id="ae108-105">В следующем примере возникает ошибка CS1611:</span><span class="sxs-lookup"><span data-stu-id="ae108-105">The following sample generates CS1611:</span></span>  
  
```csharp  
// CS1611.cs  
public class MyClass  
{  
   public static void Test(params ref int[] a)   // CS1611, remove ref  
   {  
   }  
  
   public static void Main()  
   {  
      Test(1);  
   }  
}  
```
