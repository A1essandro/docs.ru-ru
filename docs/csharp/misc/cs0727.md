# Ошибка компилятора CS0727
Недопустимый спецификатор формата  
  
 Эта ошибка возникает в отладчике. При вводе имени переменной в окне отладчика можно через запятую указать спецификатор формата. Примеры: myInt, h или myString, nq. Данная ошибка возникает, когда компилятор не может проанализировать введенные вами данные. Чтобы устранить эту ошибку, повторно введите имя переменной, при желании указав через запятую [допустимый спецификатор формата](/visual-studio/debugger/format-specifiers-in-csharp).
