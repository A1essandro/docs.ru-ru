# <a name="compiler-error-cs0727"></a>Ошибка компилятора CS0727
Недопустимый спецификатор формата  
  
 Эта ошибка возникает в отладчике. При вводе имени переменной в окне отладчика можно через запятую указать спецификатор формата. Примеры: myInt, h или myString, nq. Данная ошибка возникает, когда компилятор не может проанализировать введенные вами данные. Чтобы устранить эту ошибку, повторно введите имя переменной, при необходимости используя запятую и [допустимым спецификатором формата](https://docs.microsoft.com/visualstudio/debugger/format-specifiers-in-csharp).
