# <a name="compiler-error-cs1583"></a>Ошибка компилятора CS1583
"файл" не является допустимым файлом ресурсов Win32  
  
 Эта ошибка возникает при указании файла с [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) параметра компилятора, который не является допустимым или правильного формата файла ресурсов Win32. В Visual Studio имя файла задается в области приложения в конструкторе проектов.
