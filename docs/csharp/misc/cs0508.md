## <a name="example"></a><span data-ttu-id="f4017-106">Пример</span><span class="sxs-lookup"><span data-stu-id="f4017-106">Example</span></span>  
 <span data-ttu-id="f4017-107">Следующий пример приводит к возникновению ошибки CS0508.</span><span class="sxs-lookup"><span data-stu-id="f4017-107">The following sample generates CS0508.</span></span>  
  
```  
// CS0508.cs  
// compile with: /target:library  
abstract public class Clx  
{  
   public int i = 0;  
   // Return type is int.  
   abstract public int F();  
}  
  
public class Cly : Clx  
{  
   public override double F()  
   {  
      return 0.0;   // CS0508  
   }  
}  
```
