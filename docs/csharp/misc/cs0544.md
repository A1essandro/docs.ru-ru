# <a name="compiler-error-cs0544"></a><span data-ttu-id="28e03-102">Ошибка компилятора CS0544</span><span class="sxs-lookup"><span data-stu-id="28e03-102">Compiler Error CS0544</span></span>
<span data-ttu-id="28e03-103">"переопределение_свойства": невозможно переопределить, поскольку "не_свойство" не является свойством</span><span class="sxs-lookup"><span data-stu-id="28e03-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="28e03-104">Предпринята попытка переопределения типа данных, не являющегося свойством, как [свойства](../../csharp/programming-guide/classes-and-structs/properties.md), что недопустимо.</span><span class="sxs-lookup"><span data-stu-id="28e03-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="28e03-105">В следующем примере возникает ошибка CS0544:</span><span class="sxs-lookup"><span data-stu-id="28e03-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
