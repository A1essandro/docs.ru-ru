# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="e1d57-102">Предупреждение компилятора (уровень 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="e1d57-102">Compiler Warning (level 2) CS3021</span></span>
<span data-ttu-id="e1d57-103">Для "тип" не требуется атрибут CLSCompliant, так как данная сборка не имеет атрибута CLSCompliant</span><span class="sxs-lookup"><span data-stu-id="e1d57-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="e1d57-104">Это предупреждение возникает, если `[CLSCompliant(false)]` отображается на класс в сборке, которая не поддерживает набор атрибутов уровня сборки CLSCompliant значение true (то есть строка `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="e1d57-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="e1d57-105">Поскольку сборка не объявляет себя CLS-совместимой, не требуется, чтобы какой-либо объект в сборке сам себя объявлял несовместимым, поскольку он считается несовместимым.</span><span class="sxs-lookup"><span data-stu-id="e1d57-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="e1d57-106">Дополнительные сведения о CLS-совместимости см. в разделе [Написание кода, совместимого с CLS](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span><span class="sxs-lookup"><span data-stu-id="e1d57-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/en-us/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>  
  
 <span data-ttu-id="e1d57-107">Чтобы избавиться от этого предупреждения, удалите атрибут или добавьте атрибут уровня сборки.</span><span class="sxs-lookup"><span data-stu-id="e1d57-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e1d57-108">Пример</span><span class="sxs-lookup"><span data-stu-id="e1d57-108">Example</span></span>  
 <span data-ttu-id="e1d57-109">Следующий пример приводит к возникновению предупреждения CS3021:</span><span class="sxs-lookup"><span data-stu-id="e1d57-109">The following example generates CS3021:</span></span>  
  
```  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
