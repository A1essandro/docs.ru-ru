# <a name="compiler-error-cs1661"></a><span data-ttu-id="0d511-102">Ошибка компилятора CS1661</span><span class="sxs-lookup"><span data-stu-id="0d511-102">Compiler Error CS1661</span></span>
<span data-ttu-id="0d511-103">Невозможно преобразовать блок анонимного метода в тип делегата "тип_делегата", так как типы параметров указанного блока не соответствуют типам параметров делегата</span><span class="sxs-lookup"><span data-stu-id="0d511-103">Cannot convert anonymous method block to delegate type 'delegate type' because the specified block's parameter types do not match the delegate parameter types</span></span>  
  
 <span data-ttu-id="0d511-104">Эта ошибка возникает, если в определении анонимного метода типы параметров анонимного метода не соответствуют типам параметров делегата.</span><span class="sxs-lookup"><span data-stu-id="0d511-104">This error occurs if, in an anonymous method definition, the parameter types of the anonymous method do not match the delegate parameter types.</span></span> <span data-ttu-id="0d511-105">Проверьте число параметров, типы параметров и все параметры ref и out, а также точное соответствие.</span><span class="sxs-lookup"><span data-stu-id="0d511-105">Check the number of parameters, the parameter types, and any ref or out parameters and verify an exact match.</span></span>

    public static void Main()

        MyDelegate d = delegate(string s) { };  // CS1661
