# <a name="compiler-error-cs0553"></a><span data-ttu-id="19029-102">Ошибка компилятора CS0553</span><span class="sxs-lookup"><span data-stu-id="19029-102">Compiler Error CS0553</span></span>
<span data-ttu-id="19029-103">"подпрограмма преобразования": определенное пользователем преобразование в базовый класс/из базового класса</span><span class="sxs-lookup"><span data-stu-id="19029-103">'conversion routine' : user defined conversion to/from base class</span></span>  
  
 <span data-ttu-id="19029-104">Пользовательские преобразования в значения базового класса не допускаются; в операторе подобного типа нет необходимости.</span><span class="sxs-lookup"><span data-stu-id="19029-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="19029-105">Следующий пример приводит к возникновению ошибки CS0553:</span><span class="sxs-lookup"><span data-stu-id="19029-105">The following sample generates CS0553:</span></span>  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
