# <a name="compiler-error-cs0411"></a><span data-ttu-id="dcb82-102">Ошибка компилятора CS0411</span><span class="sxs-lookup"><span data-stu-id="dcb82-102">Compiler Error CS0411</span></span>
<span data-ttu-id="dcb82-103">Аргументы типа для метода "метод" не могут выводиться из использования.</span><span class="sxs-lookup"><span data-stu-id="dcb82-103">The type arguments for method 'method' cannot be inferred from the usage.</span></span> <span data-ttu-id="dcb82-104">Попробуйте указать аргументы типа явным образом.</span><span class="sxs-lookup"><span data-stu-id="dcb82-104">Try specifying the type arguments explicitly.</span></span>  
  
 <span data-ttu-id="dcb82-105">Эта ошибка возникает, если универсальный метод вызывается без явного предоставления аргументов типа, и компилятор не может определить, какие аргументы типа имеются в виду.</span><span class="sxs-lookup"><span data-stu-id="dcb82-105">This error occurs if you call a generic method without explicitly providing the type arguments and the compiler cannot infer which type arguments are intended.</span></span> <span data-ttu-id="dcb82-106">Чтобы избежать этой ошибки, добавьте предполагаемые аргументы типа в угловых скобках.</span><span class="sxs-lookup"><span data-stu-id="dcb82-106">To avoid this error, add the intended type arguments in angle brackets.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dcb82-107">Пример</span><span class="sxs-lookup"><span data-stu-id="dcb82-107">Example</span></span>  
 <span data-ttu-id="dcb82-108">Следующий пример приводит к возникновению ошибки CS0411:</span><span class="sxs-lookup"><span data-stu-id="dcb82-108">The following sample generates CS0411:</span></span>  
  
```  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
