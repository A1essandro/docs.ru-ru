 <span data-ttu-id="66c32-104">При объявлении [свойства](../../csharp/programming-guide/classes-and-structs/using-properties.md)также необходимо объявить его методы доступа.</span><span class="sxs-lookup"><span data-stu-id="66c32-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="66c32-105">Однако свойство не может иметь несколько методов доступа `get` или несколько методов доступа `set` .</span><span class="sxs-lookup"><span data-stu-id="66c32-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="66c32-106">Пример</span><span class="sxs-lookup"><span data-stu-id="66c32-106">Example</span></span>  
 <span data-ttu-id="66c32-107">В следующем примере возникает ошибка CS1007:</span><span class="sxs-lookup"><span data-stu-id="66c32-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
