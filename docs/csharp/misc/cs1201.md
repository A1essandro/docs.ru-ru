# Предупреждение компилятора (уровень 1) CS1201
Функция "недопустимая функция" устарела. Вместо нее используйте функцию "допустимая функция".  
  
 Вы пытаетесь использовать устаревшую функцию. Обновите код, чтобы использовать вместо нее допустимую функцию.
