# <a name="compiler-warning-level-1-cs3008"></a>Предупреждение компилятора (уровень 1) CS3008
Идентификатор "идентификатор", отличающийся только регистром, несовместим с CLS  
  
 Объект [открытый](../../csharp/language-reference/keywords/public.md), [защищенных](../../csharp/language-reference/keywords/protected.md), или `protected internal` идентификатор нарушает совместимость с общих спецификацией (CLS), если он начинается с символа подчеркивания (\_). Дополнительные сведения о соответствии спецификации CLS см. в разделах [Создание кода, совместимого с CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) и [Независимость от языка и независимые от языка компоненты](../../../docs/standard/language-independence-and-language-independent-components.md).  
  
## <a name="example"></a>Пример  
 В следующем примере возникает ошибка CS3008:  
  
```csharp  
// CS3008.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
public class a  
{  
    public static int _a = 0;  // CS3008  
    // OK, private  
    // private static int _a1 = 0;  
  
    public static void Main()  
    {  
    }  
}  
```
