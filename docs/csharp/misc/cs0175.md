# <a name="compiler-error-cs0175"></a><span data-ttu-id="7f71b-102">Ошибка компилятора CS0175</span><span class="sxs-lookup"><span data-stu-id="7f71b-102">Compiler Error CS0175</span></span>
<span data-ttu-id="7f71b-103">Использование ключевого слова "base" в этом контексте не допускается.</span><span class="sxs-lookup"><span data-stu-id="7f71b-103">Use of keyword 'base' is not valid in this context</span></span>  
  
 <span data-ttu-id="7f71b-104">Для указания определенного члена базового класса необходимо использовать ключевое слово [base](../../csharp/language-reference/keywords/base.md) .</span><span class="sxs-lookup"><span data-stu-id="7f71b-104">The [base](../../csharp/language-reference/keywords/base.md) keyword must be used to specify a particular member of the base class.</span></span> <span data-ttu-id="7f71b-105">Дополнительные сведения см. в разделе [Конструкторы](../../csharp/programming-guide/classes-and-structs/constructors.md).</span><span class="sxs-lookup"><span data-stu-id="7f71b-105">For more information, see [Constructors](../../csharp/programming-guide/classes-and-structs/constructors.md).</span></span>  
  
 <span data-ttu-id="7f71b-106">Следующий пример приводит к возникновению ошибки CS0175:</span><span class="sxs-lookup"><span data-stu-id="7f71b-106">The following sample generates CS0175:</span></span>  
  
```  
// CS0175.cs  
using System;  
class BaseClass  
{  
    public int TestInt = 0;  
}  
  
class MyClass : BaseClass  
{  
    public static void Main()  
    {  
        MyClass aClass = new MyClass();  
        aClass.BaseTest();  
    }  
  
    public void BaseTest()  
    {  
        Console.WriteLine(base); // CS0175  
        // Try the following line instead:  
        // Console.WriteLine(base.TestInt);  
        base = 9;   // CS0175  
  
        // Try the following line instead:  
        // base.TestInt = 9;  
    }  
}  
```
