# <a name="compiler-error-cs0179"></a><span data-ttu-id="2081e-102">Ошибка компилятора CS0179</span><span class="sxs-lookup"><span data-stu-id="2081e-102">Compiler Error CS0179</span></span>
<span data-ttu-id="2081e-103">Член "член" не может иметь модификатор extern и объявлять тело.</span><span class="sxs-lookup"><span data-stu-id="2081e-103">'member' cannot be extern and declare a body</span></span>  
  
 <span data-ttu-id="2081e-104">Если член класса помечен как [extern](../../csharp/language-reference/keywords/extern.md), это означает, что определение этого члена находится в другом файле.</span><span class="sxs-lookup"><span data-stu-id="2081e-104">When a class member is marked [extern](../../csharp/language-reference/keywords/extern.md), it means that the member's definition is located in another file.</span></span> <span data-ttu-id="2081e-105">Таким образом член класса, помеченный как **extern** , не может быть определен в классе.</span><span class="sxs-lookup"><span data-stu-id="2081e-105">Therefore, a class member marked as **extern** cannot be defined in the class.</span></span> <span data-ttu-id="2081e-106">Удалите ключевое слово `extern` или удалите определение.</span><span class="sxs-lookup"><span data-stu-id="2081e-106">Either remove the `extern` keyword or delete the definition.</span></span> <span data-ttu-id="2081e-107">Дополнительные сведения см. в статье [Методы](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="2081e-107">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="2081e-108">Следующий пример приводит к возникновению ошибки CS0179:</span><span class="sxs-lookup"><span data-stu-id="2081e-108">The following sample generates CS0179:</span></span>  
  
```csharp  
// CS0179.cs  
public class MyClass  
{  
   public extern int ExternMethod(int aa)   // CS0179  
   {  
      return 0;  
   }  
   // try the following line instead  
   // public extern int ExternMethod(int aa);  
  
   public static void Main()  
   {  
   }  
}  
```
