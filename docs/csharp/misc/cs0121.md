# <a name="compiler-error-cs0121"></a><span data-ttu-id="c035e-102">Ошибка компилятора CS0121</span><span class="sxs-lookup"><span data-stu-id="c035e-102">Compiler Error CS0121</span></span>
<span data-ttu-id="c035e-103">Вызов неоднозначно для следующих методов или свойств: «метод1» и «метод2»</span><span class="sxs-lookup"><span data-stu-id="c035e-103">The call is ambiguous between the following methods or properties: 'method1' and 'method2'</span></span>  
  
 <span data-ttu-id="c035e-104">Из-за неявного преобразования компилятору не удалось вызвать одну форму перегруженного метода.</span><span class="sxs-lookup"><span data-stu-id="c035e-104">Due to implicit conversion, the compiler was not able to call one form of an overloaded method.</span></span> <span data-ttu-id="c035e-105">Эту ошибку можно устранить одним из следующих способов:</span><span class="sxs-lookup"><span data-stu-id="c035e-105">You can resolve this error in the following ways:</span></span>  
  
-   <span data-ttu-id="c035e-106">Укажите параметры метода таким образом, что неявное преобразование не выполняется.</span><span class="sxs-lookup"><span data-stu-id="c035e-106">Specify the method parameters in such a way that implicit conversion does not take place.</span></span>  
  
-   <span data-ttu-id="c035e-107">Удалите все перегрузки для метода.</span><span class="sxs-lookup"><span data-stu-id="c035e-107">Remove all overloads for the method.</span></span>  
  
-   <span data-ttu-id="c035e-108">Приведение к нужному типу перед вызовом метода.</span><span class="sxs-lookup"><span data-stu-id="c035e-108">Cast to proper type before calling method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c035e-109">Пример</span><span class="sxs-lookup"><span data-stu-id="c035e-109">Example</span></span>  
 <span data-ttu-id="c035e-110">Следующий пример приводит к возникновению ошибки CS0121:</span><span class="sxs-lookup"><span data-stu-id="c035e-110">The following sample generates CS0121:</span></span>  
  
```csharp  
// CS0121.cs  
public class C  
{  
   void f(int i, double d)   
   {  
   }  
  
   void f(double d, int i)  
   {  
   }  
  
   public static void Main()  
   {  
      C c = new C();  
  
      c.f(1, 1);   // CS0121  
      // try the following line instead  
      // c.f(1, 1.0);  
      // or  
      // c.f(1.0, 1);  
      // or  
      // c.f(1, (double)1);   // cast and specify which method to call  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="c035e-111">Пример</span><span class="sxs-lookup"><span data-stu-id="c035e-111">Example</span></span>  
 <span data-ttu-id="c035e-112">В следующем примере создается CS0121 в текущей версии Visual Studio, но не в Visual Studio 2005:</span><span class="sxs-lookup"><span data-stu-id="c035e-112">The following example produces CS0121 in the current version of Visual Studio, but not in Visual Studio 2005:</span></span>  
  
```csharp  
//CS0121_2.cs  
class Program2  
{  
  
    static int ol_invoked = 0;  
  
    delegate int D1(int x);  
    delegate T D1<T>(T x);  
    delegate T D1<T, U>(U u);  
  
    static void F(D1 d1) { ol_invoked = 1; }  
    static void F<T>(D1<T> d1t) { ol_invoked = 2; }  
    static void F<T, U>(D1<T, U> d1t) { ol_invoked = 3; }  
  
    static int Test001()  
    {  
        F(delegate(int x) { return 1; }); //CS0121  
        if (ol_invoked == 1)  
            return 0;  
        else  
            return 1;  
    }  
  
    static int Main()  
    {  
        return Test001();  
    }  
}  
```
