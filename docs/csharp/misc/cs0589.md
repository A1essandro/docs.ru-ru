# <a name="compiler-error-cs0589"></a>Ошибка компилятора CS0589
Внутренняя ошибка компилятора: этап "PARSE"  
  
 Возможно, ошибка возникает из-за сбоя компилятора при попытке проанализировать неправильный синтаксис. Если это не так, попробуйте [обращайтесь к нам](/visualstudio/ide/talk-to-us).
