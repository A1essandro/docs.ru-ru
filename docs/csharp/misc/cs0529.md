# <a name="compiler-error-cs0529"></a><span data-ttu-id="27fb2-102">Ошибка компилятора CS0529</span><span class="sxs-lookup"><span data-stu-id="27fb2-102">Compiler Error CS0529</span></span>
<span data-ttu-id="27fb2-103">Наследуемый интерфейс "интерфейс1" образует циклическую ссылку в иерархии интерфейсов "интерфейс2".</span><span class="sxs-lookup"><span data-stu-id="27fb2-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="27fb2-104">Список наследования для [интерфейса](../../csharp/language-reference/keywords/interface.md) включает прямую или косвенную ссылку на самого себя.</span><span class="sxs-lookup"><span data-stu-id="27fb2-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="27fb2-105">Интерфейс не может наследовать от самого себя.</span><span class="sxs-lookup"><span data-stu-id="27fb2-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="27fb2-106">В следующем примере возникает ошибка CS0529:</span><span class="sxs-lookup"><span data-stu-id="27fb2-106">The following sample generates CS0529:</span></span>  
  
```  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
