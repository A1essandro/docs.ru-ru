# <a name="compiler-error-cs0513"></a><span data-ttu-id="e4196-102">Ошибка компилятора CS0513</span><span class="sxs-lookup"><span data-stu-id="e4196-102">Compiler Error CS0513</span></span>
<span data-ttu-id="e4196-103">"функция" является абстрактным, но содержится в классе "класс", который не является абстрактным</span><span class="sxs-lookup"><span data-stu-id="e4196-103">'function' is abstract but it is contained in nonabstract class 'class'</span></span>

```csharp

{  
   public class clx  
   {  
      abstract public void f();   // CS0513, abstract member of nonabstract class  
   }  
}  
```
