# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="d678a-102">Предупреждение компилятора (уровень 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="d678a-102">Compiler Warning (level 2) CS0435</span></span>
<span data-ttu-id="d678a-103">Пространство имен "пространство_имен" в сборке "сборка" конфликтует с импортированным типом "тип" в сборке "сборка".</span><span class="sxs-lookup"><span data-stu-id="d678a-103">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="d678a-104">Используется пространство имен, определенное в "сборка".</span><span class="sxs-lookup"><span data-stu-id="d678a-104">Using the namespace defined in 'assembly'..</span></span>  
  
 <span data-ttu-id="d678a-105">Это предупреждение выдается, когда пространство имен в исходном файле (file_2) конфликтует с импортированным пространством имен в file_1.</span><span class="sxs-lookup"><span data-stu-id="d678a-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="d678a-106">Компилятор использует пространство имен из исходного файла.</span><span class="sxs-lookup"><span data-stu-id="d678a-106">The compiler uses the one in the source file.</span></span>

```csharp

// compile with: /t:library

{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="d678a-109">Затем скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="d678a-109">Then, compile this file:</span></span>

```csharp  
{  
   public class A { }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
