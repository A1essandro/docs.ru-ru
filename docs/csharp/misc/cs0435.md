# <a name="compiler-warning-level-2-cs0435"></a><span data-ttu-id="ef0c2-102">Предупреждение компилятора (уровень 2) CS0435</span><span class="sxs-lookup"><span data-stu-id="ef0c2-102">Compiler Warning (level 2) CS0435</span></span>
<span data-ttu-id="ef0c2-103">Пространство имен "пространство_имен" в сборке "сборка" конфликтует с импортированным типом "тип" в сборке "сборка".</span><span class="sxs-lookup"><span data-stu-id="ef0c2-103">The namespace 'namespace' in 'assembly' conflicts with the imported type 'type' in 'assembly'.</span></span> <span data-ttu-id="ef0c2-104">Используется пространство имен, определенное в "сборка".</span><span class="sxs-lookup"><span data-stu-id="ef0c2-104">Using the namespace defined in 'assembly'..</span></span>  
  
 <span data-ttu-id="ef0c2-105">Это предупреждение выдается, когда пространство имен в исходном файле (file_2) конфликтует с импортированным пространством имен в file_1.</span><span class="sxs-lookup"><span data-stu-id="ef0c2-105">This warning is issued when a namespace in a source file (file_2) conflicts with an imported type in file_1.</span></span> <span data-ttu-id="ef0c2-106">Компилятор использует пространство имен из исходного файла.</span><span class="sxs-lookup"><span data-stu-id="ef0c2-106">The compiler uses the one in the source file.</span></span>  
  
 <span data-ttu-id="ef0c2-107">Следующий пример приводит к возникновению ошибки CS0435:</span><span class="sxs-lookup"><span data-stu-id="ef0c2-107">The following example generates CS0435:</span></span>  
  
 <span data-ttu-id="ef0c2-108">Сначала скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="ef0c2-108">Compile this file first:</span></span>  
  
```csharp

// compile with: /t:library  
public class Util   
{  
   public class A { }  
}  
```  
  
 <span data-ttu-id="ef0c2-109">Затем скомпилируйте этот файл:</span><span class="sxs-lookup"><span data-stu-id="ef0c2-109">Then, compile this file:</span></span>  
  
```csharp  
// CS0435_2.cs  
// compile with: /r:CS0435_1.dll  
  
using System;  
  
namespace Util   
{  
   public class A { }  
}  
  
public class Test   
{  
   public static void Main()   
   {  
      Console.WriteLine(typeof(Util.A)); // CS0435  
   }  
}  
```
