# <a name="compiler-error-cs1625"></a><span data-ttu-id="a6bde-102">Ошибка компилятора CS1625</span><span class="sxs-lookup"><span data-stu-id="a6bde-102">Compiler Error CS1625</span></span>
<span data-ttu-id="a6bde-103">Нельзя использовать оператор yield в теле предложения finally.</span><span class="sxs-lookup"><span data-stu-id="a6bde-103">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="a6bde-104">Использование оператора yield в основной части предложения finally недопустимо.</span><span class="sxs-lookup"><span data-stu-id="a6bde-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="a6bde-105">Чтобы избежать этой ошибки, выведите оператор yield за пределы предложения finally.</span><span class="sxs-lookup"><span data-stu-id="a6bde-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="a6bde-106">Следующий пример приводит к возникновению ошибки CS1625:</span><span class="sxs-lookup"><span data-stu-id="a6bde-106">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
