# <a name="compiler-error-cs0455"></a><span data-ttu-id="0fbc7-102">Ошибка компилятора CS0455</span><span class="sxs-lookup"><span data-stu-id="0fbc7-102">Compiler Error CS0455</span></span>
<span data-ttu-id="0fbc7-103">Параметр типа "Имя параметра типа" наследует конфликтующие ограничения "Имя ограничения 1" и "Имя ограничения 2"</span><span class="sxs-lookup"><span data-stu-id="0fbc7-103">Type parameter 'Type Parameter Name' inherits conflicting constraints 'Constraint Name 1' and 'Constraint Name 2'</span></span>
