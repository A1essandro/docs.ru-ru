# <a name="compiler-warning-level-3-cs0660"></a><span data-ttu-id="b0682-102">Предупреждение компилятора (уровень 3) CS0660</span><span class="sxs-lookup"><span data-stu-id="b0682-102">Compiler Warning (level 3) CS0660</span></span>

<span data-ttu-id="b0682-103">Класс "класс" определяет оператор == или оператор !=, но не переопределяет Object.Equals(object o)</span><span class="sxs-lookup"><span data-stu-id="b0682-103">'class' defines operator == or operator != but does not override Object.Equals(object o)</span></span>  
  
<span data-ttu-id="b0682-104">Компилятор обнаружил определяемые пользователем оператор равенства или неравенства, но переопределение <xref:System.Object.Equals%2A?displayProperty=nameWithType> метод.</span><span class="sxs-lookup"><span data-stu-id="b0682-104">The compiler detected the user-defined equality or inequality operator, but no override for the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="b0682-105">Определяемый пользователем оператор равенства или неравенства подразумевает, что нужно также переопределить <xref:System.Object.Equals%2A> метод.</span><span class="sxs-lookup"><span data-stu-id="b0682-105">A user-defined equality or inequality operator implies that you also want to override the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="b0682-106">Дополнительные сведения см. в разделе [Как определению равенства значений для типа](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span><span class="sxs-lookup"><span data-stu-id="b0682-106">For more information, see [How to: Define Value Equality for a Type](../programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>
  
<span data-ttu-id="b0682-107">В следующем примере возникает ошибка CS0660:</span><span class="sxs-lookup"><span data-stu-id="b0682-107">The following sample generates CS0660:</span></span>

```csharp
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
