# <a name="compiler-error-cs0751"></a><span data-ttu-id="0e3cb-102">Ошибка компилятора CS0751</span><span class="sxs-lookup"><span data-stu-id="0e3cb-102">Compiler Error CS0751</span></span>
<span data-ttu-id="0e3cb-103">Разделяемый метод должен быть объявлен в разделяемом классе или разделяемой структуре</span><span class="sxs-lookup"><span data-stu-id="0e3cb-103">A partial method must be declared in a partial class or partial struct</span></span>  
  
 <span data-ttu-id="0e3cb-104">Невозможно объявить [разделяемый](../../csharp/language-reference/keywords/partial-method.md) метод, если он не инкапсулирован в разделяемом классе или разделяемой структуре.</span><span class="sxs-lookup"><span data-stu-id="0e3cb-104">It is not possible to declare a [partial](../../csharp/language-reference/keywords/partial-method.md) method unless it is encapsulated inside a partial class or partial struct.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0e3cb-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="0e3cb-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="0e3cb-106">Удалите модификатор `partial` из метода и предоставьте реализацию или добавьте модификатор `partial` во включающий класс или структуру.</span><span class="sxs-lookup"><span data-stu-id="0e3cb-106">Either remove the `partial` modifier from the method and provide an implementation, or else add the `partial` modifier to the enclosing class or struct.</span></span>
