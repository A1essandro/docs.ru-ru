# <a name="compiler-error-cs0205"></a><span data-ttu-id="a0024-102">Ошибка компилятора CS0205</span><span class="sxs-lookup"><span data-stu-id="a0024-102">Compiler Error CS0205</span></span>
<span data-ttu-id="a0024-103">Не удается вызвать абстрактный член базового класса: "метод"</span><span class="sxs-lookup"><span data-stu-id="a0024-103">Cannot call an abstract base member: 'method'</span></span>  
  
 <span data-ttu-id="a0024-104">Вы не можете вызвать [абстрактный](../../csharp/language-reference/keywords/abstract.md) метод, так как он не имеет тела метода.</span><span class="sxs-lookup"><span data-stu-id="a0024-104">You cannot call an [abstract](../../csharp/language-reference/keywords/abstract.md) method because it does not have a method body.</span></span> <span data-ttu-id="a0024-105">Дополнительные сведения см. в статье [Абстрактные и запечатанные классы и члены классов](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span><span class="sxs-lookup"><span data-stu-id="a0024-105">For more information, see [Abstract and Sealed Classes and Class Members](../../csharp/programming-guide/classes-and-structs/abstract-and-sealed-classes-and-class-members.md).</span></span>  
  
 <span data-ttu-id="a0024-106">В следующем примере возникает ошибка CS0205.</span><span class="sxs-lookup"><span data-stu-id="a0024-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs  
abstract public class MyClass  
{  
   abstract public void M();  
}  
  
public class MyClass2 : MyClass  
{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
