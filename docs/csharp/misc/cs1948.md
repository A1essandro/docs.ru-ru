## <a name="example"></a><span data-ttu-id="bb646-107">Пример</span><span class="sxs-lookup"><span data-stu-id="bb646-107">Example</span></span>  
 <span data-ttu-id="bb646-108">Приведенный ниже пример приводит к возникновению ошибки CS1948, так как идентификатор `T` используется для переменной диапазона и для параметра типа в методе `TestMethod`.</span><span class="sxs-lookup"><span data-stu-id="bb646-108">The following example generates CS1948 because the identifier `T` is used for the range variable and for the type parameter on method `TestMethod`:</span></span>

```csharp  
class Test  
{  
    public void TestMethod<T>(T t)  
    {  
        var x = from T in Enumerable.Range(1, 100) // CS1948  
                select T;  
    }  
}  
```
