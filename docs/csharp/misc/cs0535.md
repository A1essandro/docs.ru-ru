 <span data-ttu-id="e9995-104">[Класс](../../csharp/language-reference/keywords/class.md) является производным от [интерфейса](../../csharp/language-reference/keywords/interface.md), но не реализует один или несколько членов интерфейса.</span><span class="sxs-lookup"><span data-stu-id="e9995-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="e9995-105">Класс должен реализовывать все члены интерфейсов, от которых он является производным, или должен объявляться как `abstract`.</span><span class="sxs-lookup"><span data-stu-id="e9995-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e9995-106">Пример</span><span class="sxs-lookup"><span data-stu-id="e9995-106">Example</span></span>  
 <span data-ttu-id="e9995-107">Следующий пример приводит к возникновению ошибки CS0535.</span><span class="sxs-lookup"><span data-stu-id="e9995-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
