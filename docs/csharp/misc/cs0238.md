# <a name="compiler-error-cs0238"></a><span data-ttu-id="e387c-102">Ошибка компилятора CS0238</span><span class="sxs-lookup"><span data-stu-id="e387c-102">Compiler Error CS0238</span></span>
<span data-ttu-id="e387c-103">"элемент" не может быть запечатанным, так как не является результатом переопределения</span><span class="sxs-lookup"><span data-stu-id="e387c-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="e387c-104">Значение[sealed](../../csharp/language-reference/keywords/sealed.md) использовалось для "элемент", который не был помечен атрибутом [override](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="e387c-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="e387c-105">Дополнительные сведения см. в разделе [Наследование](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="e387c-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="e387c-106">При компиляции следующего примера возникнет ошибка CS0238:</span><span class="sxs-lookup"><span data-stu-id="e387c-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
