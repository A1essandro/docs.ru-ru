<span data-ttu-id="a008c-103">Несогласованное использование лямбда-параметра; типы параметров должны быть либо все явными, либо все неявными.</span><span class="sxs-lookup"><span data-stu-id="a008c-103">Inconsistent lambda parameter usage; all parameter types must either be explicit or implicit.</span></span>  
  
 <span data-ttu-id="a008c-104">Если лямбда-выражение имеет несколько входных параметров, не разрешается, чтобы одни параметры использовали явную типизацию, а другие неявную.</span><span class="sxs-lookup"><span data-stu-id="a008c-104">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>

 <span data-ttu-id="a008c-108">В следующем коде возникает ошибка CS0748, поскольку в лямбда-выражении только параметру `alpha` присваивается явный тип:</span><span class="sxs-lookup"><span data-stu-id="a008c-108">The following code generates CS0748 because, in the lambda expression, only `alpha` is given an explicit type:</span></span>
