# <a name="compiler-error-cs0748"></a><span data-ttu-id="521ac-102">Ошибка компилятора CS0748</span><span class="sxs-lookup"><span data-stu-id="521ac-102">Compiler Error CS0748</span></span>
<span data-ttu-id="521ac-103">Использование параметра несогласованное лямбда-выражения; типы параметров должны быть все явными, либо все неявными.</span><span class="sxs-lookup"><span data-stu-id="521ac-103">Inconsistent lambda parameter usage; parameter types must be all explicit or all implicit.</span></span>
  
 <span data-ttu-id="521ac-104">Если лямбда-выражение имеет несколько входных параметров, не разрешается, чтобы одни параметры использовали явную типизацию, а другие неявную.</span><span class="sxs-lookup"><span data-stu-id="521ac-104">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="521ac-105">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="521ac-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="521ac-106">Предоставьте всем входным параметрам неявные типы или всем им предоставьте явные типы.</span><span class="sxs-lookup"><span data-stu-id="521ac-106">Give all the input parameters implicit types, or give them all explicit types.</span></span>  
  
## <a name="example"></a><span data-ttu-id="521ac-107">Пример</span><span class="sxs-lookup"><span data-stu-id="521ac-107">Example</span></span>  
 <span data-ttu-id="521ac-108">В следующем коде возникает ошибка CS0748, поскольку в лямбда-выражении только параметру `alpha` присваивается явный тип:</span><span class="sxs-lookup"><span data-stu-id="521ac-108">The following code generates CS0748 because, in the lambda expression, only `alpha` is given an explicit type:</span></span>

    delegate double D(int x, int y);
