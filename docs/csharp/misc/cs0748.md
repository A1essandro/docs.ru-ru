<span data-ttu-id="0605e-103">Использование параметра несогласованное лямбда-выражения; типы параметров должны быть все явными, либо все неявными.</span><span class="sxs-lookup"><span data-stu-id="0605e-103">Inconsistent lambda parameter usage; parameter types must be all explicit or all implicit.</span></span>
  
<span data-ttu-id="0605e-104">Если лямбда-выражение имеет несколько входных параметров, не разрешается, чтобы одни параметры использовали явную типизацию, а другие неявную.</span><span class="sxs-lookup"><span data-stu-id="0605e-104">If a lambda expression has multiple input parameters, some parameters cannot use implicit typing while others use explicit typing.</span></span>

<span data-ttu-id="0605e-105">Чтобы исправить эту ошибку, опустите все объявления типа параметра или явно указать тип для всех параметров.</span><span class="sxs-lookup"><span data-stu-id="0605e-105">To correct this error, either omit all parameter type declarations or explicitly specify the type of all parameters.</span></span>
