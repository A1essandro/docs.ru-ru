# <a name="compiler-error-cs0558"></a><span data-ttu-id="dd548-102">Ошибка компилятора CS0558</span><span class="sxs-lookup"><span data-stu-id="dd548-102">Compiler Error CS0558</span></span>
<span data-ttu-id="dd548-103">Определенный пользователем оператор "оператор" должен быть объявлен как статический и общий</span><span class="sxs-lookup"><span data-stu-id="dd548-103">User-defined operator 'operator' must be declared static and public</span></span>  
  
 <span data-ttu-id="dd548-104">В пользовательских операторах должны быть указаны оба **модификатора** доступа, **static** и [public](../../csharp/language-reference/keywords/modifiers.md) .</span><span class="sxs-lookup"><span data-stu-id="dd548-104">Both the **static** and **public** access [modifiers](../../csharp/language-reference/keywords/modifiers.md) must be specified on user-defined operators.</span></span>  
  
 <span data-ttu-id="dd548-105">Следующий пример приводит к возникновению ошибки CS0558:</span><span class="sxs-lookup"><span data-stu-id="dd548-105">The following sample generates CS0558:</span></span>  
  
```csharp  
// CS0558.cs  
namespace x  
{  
   public class ii  
   {  
      public class iii  
      {  
         static implicit operator int(iii aa)   // CS0558, add public  
         {  
            return 0;  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
