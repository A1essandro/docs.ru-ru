# <a name="compiler-error-cs0112"></a><span data-ttu-id="4a69d-102">Ошибка компилятора CS0112</span><span class="sxs-lookup"><span data-stu-id="4a69d-102">Compiler Error CS0112</span></span>
<span data-ttu-id="4a69d-103">Статический член "функция" не может быть помечен как override, virtual или abstract</span><span class="sxs-lookup"><span data-stu-id="4a69d-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="4a69d-104">Любое объявление метода, которое использует ключевое слово `override`, **virtual**или **abstract** , не может использовать ключевое слово **static** .</span><span class="sxs-lookup"><span data-stu-id="4a69d-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="4a69d-105">Дополнительные сведения см. в статье [Методы](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="4a69d-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="4a69d-106">При компиляции следующего примера возникнет ошибка CS0112:</span><span class="sxs-lookup"><span data-stu-id="4a69d-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
