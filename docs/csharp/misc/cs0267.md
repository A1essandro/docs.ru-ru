 <span data-ttu-id="280c0-107">Следующий пример приводит к возникновению ошибки CS0267:</span><span class="sxs-lookup"><span data-stu-id="280c0-107">The following sample generates CS0267:</span></span>  
  
```csharp

{  
   public MyClass()  
   {  
   }  
}

// public partial class MyClass

   public void Foo()  
   {  
   }

   {  
   }  
}  
```
