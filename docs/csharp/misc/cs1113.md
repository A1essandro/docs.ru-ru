## <a name="example"></a><span data-ttu-id="aed01-109">Пример</span><span class="sxs-lookup"><span data-stu-id="aed01-109">Example</span></span>  
 <span data-ttu-id="aed01-110">В следующем примере возникает ошибка CS1113:</span><span class="sxs-lookup"><span data-stu-id="aed01-110">The following example generates CS1113:</span></span>  
  
```  
// cs1113.cs  
using System;  
public static class Extensions  
{  
    public static S ExtMethod(this S s)  
    {  
        return s;  
    }  
}  
  
public struct S  
{  
}  
  
public class Test  
{  
    static int Main()  
    {  
        Func<S> f = new S().ExtMethod; // CS1113  
        return 1;  
    }  
}  
```
