## <a name="example"></a><span data-ttu-id="86727-108">Пример</span><span class="sxs-lookup"><span data-stu-id="86727-108">Example</span></span>  
 <span data-ttu-id="86727-109">В следующем примере возникает ошибка CS3017:</span><span class="sxs-lookup"><span data-stu-id="86727-109">The following example generates CS3017:</span></span>
