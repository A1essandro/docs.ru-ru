## <a name="example"></a><span data-ttu-id="74c01-108">Пример</span><span class="sxs-lookup"><span data-stu-id="74c01-108">Example</span></span>  
 <span data-ttu-id="74c01-109">В следующем примере возникает ошибка CS3017:</span><span class="sxs-lookup"><span data-stu-id="74c01-109">The following example generates CS3017:</span></span>  
  
```csharp  
// CS3017.cs  
// compile with: /target:module

[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]

{  
    static void Main() {}  
}  
```
