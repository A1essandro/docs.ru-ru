## <a name="example"></a><span data-ttu-id="ada15-108">Пример</span><span class="sxs-lookup"><span data-stu-id="ada15-108">Example</span></span>  
 <span data-ttu-id="ada15-109">В следующем примере возникает ошибка CS3017:</span><span class="sxs-lookup"><span data-stu-id="ada15-109">The following example generates CS3017:</span></span>  
  
```  
// CS3017.cs  
// compile with: /target:module  
  
using System;  
  
[module: CLSCompliant(true)]  
[assembly: CLSCompliant(false)]  // CS3017  
// Try this line instead:  
// [assembly: CLSCompliant(true)]  
class C  
{  
    static void Main() {}  
}  
```
