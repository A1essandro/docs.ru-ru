# <a name="compiler-error-cs0764"></a><span data-ttu-id="e8185-102">Ошибка компилятора CS0764</span><span class="sxs-lookup"><span data-stu-id="e8185-102">Compiler Error CS0764</span></span>
<span data-ttu-id="e8185-103">Либо оба объявления разделяемого метода должны иметь модификаторы unsafe, либо ни одно из объявлений не должно иметь модификатора unsafe.</span><span class="sxs-lookup"><span data-stu-id="e8185-103">Both partial method declarations must be unsafe or neither may be unsafe</span></span>  
  
 <span data-ttu-id="e8185-104">Разделяемый метод состоит из определяющего объявления (сигнатуры) и необязательного реализующего объявления (тела).</span><span class="sxs-lookup"><span data-stu-id="e8185-104">A partial method consists of a defining declaration (signature) and an optional implementing declaration (body).</span></span> <span data-ttu-id="e8185-105">Если определяющее объявление содержит модификатор `unsafe` , этот модификатор должен содержаться и в реализующем объявлении.</span><span class="sxs-lookup"><span data-stu-id="e8185-105">If the defining declaration has the `unsafe` modifier, the implementing declaration must also have it.</span></span> <span data-ttu-id="e8185-106">И наоборот, если реализующее объявление содержит модификатор `unsafe` , этот модификатор должен содержаться и в определяющем объявлении.</span><span class="sxs-lookup"><span data-stu-id="e8185-106">Conversely, if the implementing declaration has the `unsafe` modifier, the defining declaration must also.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e8185-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e8185-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="e8185-108">Если определяющее объявление не содержит ошибок, добавьте или удалите модификатор `unsafe` в реализующем объявлении для соответствия с определяющим объявлением.</span><span class="sxs-lookup"><span data-stu-id="e8185-108">Assuming that the defining declaration is correct, add or remove the `unsafe` modifier from the implementing declaration to match the defining declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e8185-109">Пример</span><span class="sxs-lookup"><span data-stu-id="e8185-109">Example</span></span>  
  
```csharp  
// cs0764.cs  
//  Compile with: /target:library /unsafe  
public partial class C  
{  
    partial void Part();  
    unsafe partial void Part() //CS0764

    }

    {  
        return 1;  
    }  
}  
```
