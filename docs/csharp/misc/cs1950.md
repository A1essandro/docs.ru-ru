# <a name="compiler-error-cs1950"></a><span data-ttu-id="601eb-102">Ошибка компилятора CS1950</span><span class="sxs-lookup"><span data-stu-id="601eb-102">Compiler Error CS1950</span></span>
<span data-ttu-id="601eb-103">Наиболее подходящий перегруженный метод Add "имя" для инициализатора коллекции содержит недопустимые аргументы.</span><span class="sxs-lookup"><span data-stu-id="601eb-103">The best overloaded Add method 'name' for the collection initializer has some invalid arguments.</span></span>  
  
 <span data-ttu-id="601eb-104">Для поддержки инициализаторов коллекций класс должен реализовывать IEnumerable и иметь открытый метод `Add` .</span><span class="sxs-lookup"><span data-stu-id="601eb-104">To support collection initializers, a class must implement IEnumerable and have a public `Add` method.</span></span> <span data-ttu-id="601eb-105">Для инициализации типа с помощью инициализатора коллекции входной параметр метода `Add` должен быть совместим с типом объекта, который вы пытаетесь добавить.</span><span class="sxs-lookup"><span data-stu-id="601eb-105">To initialize the type by using a collection initializer, the input parameter of the `Add` method must be compatible with the type of the object you are trying to add.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="601eb-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="601eb-106">To correct this error</span></span>  
  
-   <span data-ttu-id="601eb-107">Используйте совместимый тип в инициализаторе коллекции.</span><span class="sxs-lookup"><span data-stu-id="601eb-107">Use a compatible type in the collection initializer.</span></span>  
  
-   <span data-ttu-id="601eb-108">Измените входной параметр и/или доступность метода `Add` в типе коллекции.</span><span class="sxs-lookup"><span data-stu-id="601eb-108">Modify the input parameter and/or accessibility of the `Add` method in the collection type.</span></span>  
  
-   <span data-ttu-id="601eb-109">Добавьте новый метод `Add` с входным параметром, соответствующим тому, что вы передаете.</span><span class="sxs-lookup"><span data-stu-id="601eb-109">Add a new `Add` method with an input parameter that matches what you are passing in.</span></span>  
  
-   <span data-ttu-id="601eb-110">Сделайте ваш класс коллекции универсальным, чтобы он мог иметь метод `Add` , который принимает любой переданный в него тип.</span><span class="sxs-lookup"><span data-stu-id="601eb-110">Make your collection class generic so that it can have an `Add` method that accepts any type you pass in.</span></span>
