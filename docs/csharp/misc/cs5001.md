# <a name="compiler-error-cs5001"></a><span data-ttu-id="b080c-102">Ошибка компилятора CS5001</span><span class="sxs-lookup"><span data-stu-id="b080c-102">Compiler Error CS5001</span></span>
<span data-ttu-id="b080c-103">Программа "программа" не содержит статического метода Main, подходящего для точки входа</span><span class="sxs-lookup"><span data-stu-id="b080c-103">Program 'program' does not contain a static 'Main' method suitable for an entry point</span></span>  
  
 <span data-ttu-id="b080c-104">Эта ошибка возникает, когда не статический [Main](../../csharp/programming-guide/main-and-command-args/index.md) метод с правильной сигнатуры находится в коде, который создает исполняемый файл.</span><span class="sxs-lookup"><span data-stu-id="b080c-104">This error occurs when no static [Main](../../csharp/programming-guide/main-and-command-args/index.md) method with a correct signature is found in the code that produces an executable file.</span></span> <span data-ttu-id="b080c-105">Кроме того, эта ошибка происходит, если функция точки входа ( `Main`) определена с использованием неправильного регистра, например со строчной буквы: `main`.</span><span class="sxs-lookup"><span data-stu-id="b080c-105">This error also occurs if the entry point function, `Main`, is defined with the wrong case, such as lower-case `main`.</span></span>  
  
-   <span data-ttu-id="b080c-106">`Main`должен быть объявлен как статический и он должен вернуть [void](../../csharp/language-reference/keywords/void.md) или [int](../../csharp/language-reference/keywords/int.md), и он должен иметь либо без параметров, либо иметь один параметр типа `string[]`.</span><span class="sxs-lookup"><span data-stu-id="b080c-106">`Main` must be declared as static and it must return [void](../../csharp/language-reference/keywords/void.md) or [int](../../csharp/language-reference/keywords/int.md), and it must have either no parameters or else one parameter of type `string[]`.</span></span>
