# <a name="compiler-error-cs0745"></a><span data-ttu-id="bf1c0-102">Ошибка компилятора CS0745</span><span class="sxs-lookup"><span data-stu-id="bf1c0-102">Compiler Error CS0745</span></span>
<span data-ttu-id="bf1c0-103">Требуется контекстное ключевое слово "by".</span><span class="sxs-lookup"><span data-stu-id="bf1c0-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="bf1c0-104">Шаблоном для предложения `group` является `group...by` , за которым следует необязательное ключевое слово `into`, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="bf1c0-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

## <a name="example"></a><span data-ttu-id="bf1c0-108">Пример</span><span class="sxs-lookup"><span data-stu-id="bf1c0-108">Example</span></span>  
 <span data-ttu-id="bf1c0-109">При компиляции следующего кода возникнет ошибка CS0745:</span><span class="sxs-lookup"><span data-stu-id="bf1c0-109">The following code generates CS0745:</span></span>  
  
```  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
