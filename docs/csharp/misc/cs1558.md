# <a name="compiler-error-cs1558"></a><span data-ttu-id="512e8-102">Ошибка компилятора CS1558</span><span class="sxs-lookup"><span data-stu-id="512e8-102">Compiler Error CS1558</span></span>
<span data-ttu-id="512e8-103">Класс "класс" не имеет подходящего статического метода Main</span><span class="sxs-lookup"><span data-stu-id="512e8-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="512e8-104">[/Main](../../csharp/language-reference/compiler-options/main-compiler-option.md) класс, в котором следует искать параметре компилятора **Main** метод.</span><span class="sxs-lookup"><span data-stu-id="512e8-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="512e8-105">Тем не менее [Main](../../csharp/programming-guide/main-and-command-args/index.md) не определен метод.</span><span class="sxs-lookup"><span data-stu-id="512e8-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="512e8-106">В следующем примере возникает ошибка CS1558 из-за недопустимого типа возврата.</span><span class="sxs-lookup"><span data-stu-id="512e8-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
