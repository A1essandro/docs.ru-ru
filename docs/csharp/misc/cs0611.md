<span data-ttu-id="36539-103">Элементы массива не могут иметь тип "тип"</span><span class="sxs-lookup"><span data-stu-id="36539-103">Array elements cannot be of type 'type'</span></span>  
  
 <span data-ttu-id="36539-104">Имеется несколько типов, которые нельзя использовать в качестве типа массива.</span><span class="sxs-lookup"><span data-stu-id="36539-104">There are some types that cannot be used as the type of an array.</span></span> <span data-ttu-id="36539-105">В число этих типов входит **System.ArgIterator** и **System.TypedReference**.</span><span class="sxs-lookup"><span data-stu-id="36539-105">These types include **System.TypedReference** and **System.ArgIterator**.</span></span>  
  
 <span data-ttu-id="36539-106">В следующем примере ошибка CS0611 возникает в результате использования типа **System.TypedReference** как элемента массива.</span><span class="sxs-lookup"><span data-stu-id="36539-106">The following sample generates CS0611 as a result of using **System.TypedReference** as an array element:</span></span>  
  
```  
// CS0611.cs  
public class a  
{  
   public static void Main()  
   {  
      System.TypedReference[] ao = new System.TypedReference [10];   // CS0611  
      // try the following line instead  
      // int[] ao = new int[10];  
   }  
}  
```
