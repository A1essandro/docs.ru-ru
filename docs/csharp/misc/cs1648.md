# <a name="compiler-error-cs1648"></a><span data-ttu-id="fe138-102">Ошибка компилятора CS1648</span><span class="sxs-lookup"><span data-stu-id="fe138-102">Compiler Error CS1648</span></span>
<span data-ttu-id="fe138-103">Модификация членов доступного только для чтения поля "идентификатор" возможна только в конструкторе или в инициализаторе переменной</span><span class="sxs-lookup"><span data-stu-id="fe138-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="fe138-104">Эта ошибка возникает при попытке изменить член поля, доступного только для чтения, где нельзя вносить изменения.</span><span class="sxs-lookup"><span data-stu-id="fe138-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="fe138-105">Чтобы устранить эту ошибку, при назначении полям, доступным только для чтения, ограничьтесь конструктором или инициализатором переменных или удалите ключевое слово readonly из объявления поля.</span><span class="sxs-lookup"><span data-stu-id="fe138-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="fe138-106">Следующий пример приводит к возникновению ошибки CS1648:</span><span class="sxs-lookup"><span data-stu-id="fe138-106">The following sample generates CS1648:</span></span>  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
