## <a name="example"></a><span data-ttu-id="609a0-105">Пример</span><span class="sxs-lookup"><span data-stu-id="609a0-105">Example</span></span>  
 <span data-ttu-id="609a0-106">В следующем примере возникает ошибка CS1107:</span><span class="sxs-lookup"><span data-stu-id="609a0-106">The following example generates CS1107:</span></span>  
  
```  
// cs1107.cs  
public static class Test  
{  
    // Extension methods.  
        public static void TestMethod(this this t) {} // CS1107  
  
    // Regular Instance Method  
        public void TestMethod(ref ref int i) {} // CS1107  
  
}  
```
