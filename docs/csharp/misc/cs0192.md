# <a name="compiler-error-cs0192"></a><span data-ttu-id="ceafb-102">Ошибка компилятора CS0192</span><span class="sxs-lookup"><span data-stu-id="ceafb-102">Compiler Error CS0192</span></span>
<span data-ttu-id="ceafb-103">Поля доступного только для чтения статического поля "имя" могут передаваться как параметры с ключевым словом ref или out только в статическом конструкторе</span><span class="sxs-lookup"><span data-stu-id="ceafb-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="ceafb-104">Поле (переменная), отмеченные [readonly](../../csharp/language-reference/keywords/readonly.md) ключевое слово не может быть передано [ref](../../csharp/language-reference/keywords/ref.md) или [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) параметры, за исключением внутри конструктора.</span><span class="sxs-lookup"><span data-stu-id="ceafb-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="ceafb-105">Дополнительные сведения см. в разделе [Поля](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="ceafb-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="ceafb-106">Ошибка CS0192 также возникает, если поле `readonly` является [static](../../csharp/language-reference/keywords/static.md) (статическим), а конструктор не помечен как `static`.</span><span class="sxs-lookup"><span data-stu-id="ceafb-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ceafb-107">Пример</span><span class="sxs-lookup"><span data-stu-id="ceafb-107">Example</span></span>  
 <span data-ttu-id="ceafb-108">Следующий пример приводит к возникновению ошибки CS0192:</span><span class="sxs-lookup"><span data-stu-id="ceafb-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
