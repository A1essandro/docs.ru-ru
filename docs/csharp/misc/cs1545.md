# <a name="compiler-error-cs1545"></a><span data-ttu-id="eb392-102">Ошибка компилятора CS1545</span><span class="sxs-lookup"><span data-stu-id="eb392-102">Compiler Error CS1545</span></span>
<span data-ttu-id="eb392-103">Свойство, индексатор или событие "свойство" не поддерживается в данном языке; попробуйте напрямую вызвать метод доступа set или get.</span><span class="sxs-lookup"><span data-stu-id="eb392-103">Property, indexer, or event 'property' is not supported by the language; try directly calling accessor methods 'set accessor' or 'get accessor'</span></span>  
  
 <span data-ttu-id="eb392-104">Код использует объект, имеющий [индексатор](../../csharp/programming-guide/indexers/index.md) , не являющийся индексатором по умолчанию, и пытается использовать индексированный синтаксис.</span><span class="sxs-lookup"><span data-stu-id="eb392-104">The code is consuming an object that has a non-default [indexer](../../csharp/programming-guide/indexers/index.md) and tried to use the indexed syntax.</span></span> <span data-ttu-id="eb392-105">Чтобы устранить эту ошибку, вызовите метод доступа `get` или `set` свойства.</span><span class="sxs-lookup"><span data-stu-id="eb392-105">To resolve this error, call the property's `get` or `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="eb392-106">Пример</span><span class="sxs-lookup"><span data-stu-id="eb392-106">Example</span></span>  
  
```cpp  
// CPP1545.cpp  
// compile with: /clr /LD  
// a Visual C++ program  
using namespace System;  
public ref struct Employee {  
   Employee( String^ s, int d ) {}  
  
   property String^ name {  
      String^ get() {  
         return nullptr;  
      }  
   }  
};  
  
public ref struct Manager {  
   property Employee^ Report [String^] {  
      Employee^ get(String^ s) {  
         return nullptr;  
      }  
  
      void set(String^ s, Employee^ e) {}  
   }  
};  
```  
  
## <a name="example"></a><span data-ttu-id="eb392-107">Пример</span><span class="sxs-lookup"><span data-stu-id="eb392-107">Example</span></span>  
 <span data-ttu-id="eb392-108">Следующий пример приводит к возникновению ошибки CS1545.</span><span class="sxs-lookup"><span data-stu-id="eb392-108">The following sample generates CS1545.</span></span>  
  
```csharp  
// CS1545.cs

class x {  
   public static void Main() {  
      Manager Ed = new Manager();  
      Employee Bob = new Employee("Bob Smith", 12);  
      Ed.Report[ Bob.name ] = Bob;   // CS1545  
      Ed.set_Report( Bob.name, Bob);   // OK  
   }  
}  
```
