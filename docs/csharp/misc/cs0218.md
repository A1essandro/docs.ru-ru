# <a name="compiler-error-cs0218"></a><span data-ttu-id="bdfe5-102">Ошибка компилятора CS0218</span><span class="sxs-lookup"><span data-stu-id="bdfe5-102">Compiler Error CS0218</span></span>
<span data-ttu-id="bdfe5-103">Тип ("тип") должен содержать объявления оператора true и оператора false</span><span class="sxs-lookup"><span data-stu-id="bdfe5-103">The type ('type') must contain declarations of operator true and operator false</span></span>  
  
 <span data-ttu-id="bdfe5-104">Если вы задаете оператор для определяемого пользователем типа, а затем пытаетесь использовать его как оператор краткой записи, определяемый пользователем оператор должен иметь заданный оператор true и оператор false.</span><span class="sxs-lookup"><span data-stu-id="bdfe5-104">If you define an operator for a user-defined type, and then try to use the operator as a short-circuit operator, the user-defined operator must have operator true and operator false defined.</span></span> <span data-ttu-id="bdfe5-105">Дополнительные сведения об операторах краткой записи см. в разделе [& & оператор](../../csharp/language-reference/operators/conditional-and-operator.md) и [ &#124; &#124; оператор](../../csharp/language-reference/operators/conditional-or-operator.md).</span><span class="sxs-lookup"><span data-stu-id="bdfe5-105">For more information on short-circuit operators, see [&& Operator](../../csharp/language-reference/operators/conditional-and-operator.md) and [&#124;&#124; Operator](../../csharp/language-reference/operators/conditional-or-operator.md).</span></span>  
  
 <span data-ttu-id="bdfe5-106">В следующем примере возникает ошибка CS0218:</span><span class="sxs-lookup"><span data-stu-id="bdfe5-106">The following sample generates CS0218:</span></span>  
  
```csharp  
// CS0218.cs  
using System;  
public class MyClass  
{  
   // uncomment these operator declarations to resolve this CS0218  
   /*  
   public static bool operator true (MyClass f)  
   {  
      return false;  
   }  
  
   public static bool operator false (MyClass f)  
   {  
      return false;  
   }  
   */  
  
   public static implicit operator int(MyClass x)  
   {  
      return 0;  
   }  
  
   public static MyClass operator & (MyClass f1, MyClass f2)  
   {  
      return new MyClass();  
   }  
  
   public static void Main()  
   {  
      MyClass f = new MyClass();  
      int i = f && f;   // CS0218, requires operators true and false  
   }  
}  
```
