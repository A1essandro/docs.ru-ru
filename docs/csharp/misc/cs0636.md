# <a name="compiler-error-cs0636"></a><span data-ttu-id="c8d33-102">Ошибка компилятора CS0636</span><span class="sxs-lookup"><span data-stu-id="c8d33-102">Compiler Error CS0636</span></span>
<span data-ttu-id="c8d33-103">Атрибут FieldOffset может назначаться только членам типов, для которых используется StructLayout(LayoutKind.Explicit).</span><span class="sxs-lookup"><span data-stu-id="c8d33-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="c8d33-104">Вы должны использовать атрибут **StructLayout(LayoutKind.Explicit)** в объявлении структуры, если она содержит элементы, помеченные атрибутом **FieldOffset** .</span><span class="sxs-lookup"><span data-stu-id="c8d33-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="c8d33-105">Дополнительные сведения см. в разделе [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="c8d33-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="c8d33-106">В следующем примере возникает ошибка CS0636:</span><span class="sxs-lookup"><span data-stu-id="c8d33-106">The following sample generates CS0636:</span></span>  
  
```  
// CS0636.cs  
using System;  
using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]  
struct Worksheet  
{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
