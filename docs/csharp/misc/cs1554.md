 <span data-ttu-id="82325-104">Тип возвращаемых данных для пользовательского [оператора](../../csharp/language-reference/keywords/operator.md) должен находиться перед ключевым словом operator.</span><span class="sxs-lookup"><span data-stu-id="82325-104">The return type for a user-defined [operator](../../csharp/language-reference/keywords/operator.md) must appear before the keyword operator.</span></span>

```csharp

{  
   public static operator ++ MyClass (MyClass f)    // CS1554  
   // try the following line instead  
   // public static MyClass operator ++ (MyClass f)

      return new MyClass ();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
