# <a name="compiler-error-cs0017"></a><span data-ttu-id="92bc9-102">Ошибка компилятора CS0017</span><span class="sxs-lookup"><span data-stu-id="92bc9-102">Compiler Error CS0017</span></span>
<span data-ttu-id="92bc9-103">Программа "имя выходного файла" имеет несколько заданных точек входа.</span><span class="sxs-lookup"><span data-stu-id="92bc9-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="92bc9-104">Выполните компиляцию с параметром /main, чтобы указать тип, который содержит точку входа.</span><span class="sxs-lookup"><span data-stu-id="92bc9-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="92bc9-105">Программа может иметь только один метод [Main](../../csharp/programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="92bc9-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="92bc9-106">Чтобы устранить эту ошибку, можно либо удалить в коде все методы Main, кроме одного, или с помощью параметра компилятора [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) указать, какой метод Main вы хотите использовать.</span><span class="sxs-lookup"><span data-stu-id="92bc9-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="92bc9-107">Следующий пример приводит к возникновению ошибки CS0017:</span><span class="sxs-lookup"><span data-stu-id="92bc9-107">The following sample generates CS0017:</span></span>  
  
```  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
