# <a name="compiler-error-cs0017"></a><span data-ttu-id="dfe0a-102">Ошибка компилятора CS0017</span><span class="sxs-lookup"><span data-stu-id="dfe0a-102">Compiler Error CS0017</span></span>
<span data-ttu-id="dfe0a-103">Программа "имя выходного файла" имеет несколько заданных точек входа.</span><span class="sxs-lookup"><span data-stu-id="dfe0a-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="dfe0a-104">Выполните компиляцию с параметром /main, чтобы указать тип, который содержит точку входа.</span><span class="sxs-lookup"><span data-stu-id="dfe0a-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="dfe0a-105">Программа может иметь только один [Main](../../csharp/programming-guide/main-and-command-args/index.md) метод.</span><span class="sxs-lookup"><span data-stu-id="dfe0a-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="dfe0a-106">Чтобы устранить эту ошибку, можно либо удалить все методы Main в коде, кроме одного, или можно использовать [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) параметр компилятора, чтобы указать, какой метод Main, который вы хотите использовать.</span><span class="sxs-lookup"><span data-stu-id="dfe0a-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="dfe0a-107">Следующий пример приводит к возникновению ошибки CS0017:</span><span class="sxs-lookup"><span data-stu-id="dfe0a-107">The following sample generates CS0017:</span></span>  
  
```  
// CS0017.cs  
// compile with: /target:exe  
public class clx  
{  
   static public void Main()  
   {  
   }  
}  
  
public class cly  
{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
