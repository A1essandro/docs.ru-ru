# <a name="compiler-error-cs0214"></a><span data-ttu-id="18497-102">Ошибка компилятора CS0214</span><span class="sxs-lookup"><span data-stu-id="18497-102">Compiler Error CS0214</span></span>
<span data-ttu-id="18497-103">Указатели и буферы фиксированного размера можно использовать только в небезопасном контексте.</span><span class="sxs-lookup"><span data-stu-id="18497-103">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="18497-104">Указатели могут использоваться только с ключевым словом [unsafe](../../csharp/language-reference/keywords/unsafe.md) .</span><span class="sxs-lookup"><span data-stu-id="18497-104">Pointers can only be used with the [unsafe](../../csharp/language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="18497-105">Дополнительные сведения см. в разделе [Небезопасный код и указатели](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="18497-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="18497-106">В следующем примере возникает ошибка CS0214:</span><span class="sxs-lookup"><span data-stu-id="18497-106">The following sample generates CS0214:</span></span>  
  
```  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
