# <a name="compiler-error-cs0620"></a><span data-ttu-id="c8a06-102">Ошибка компилятора CS0620</span><span class="sxs-lookup"><span data-stu-id="c8a06-102">Compiler Error CS0620</span></span>
<span data-ttu-id="c8a06-103">Индексатор не может иметь тип void.</span><span class="sxs-lookup"><span data-stu-id="c8a06-103">Indexers cannot have void type</span></span>  
  
 <span data-ttu-id="c8a06-104">Тип возвращаемого значения [индексатора](../../csharp/programming-guide/indexers/index.md) не может быть `void`.</span><span class="sxs-lookup"><span data-stu-id="c8a06-104">The return type of an [indexer](../../csharp/programming-guide/indexers/index.md) cannot be `void`.</span></span> <span data-ttu-id="c8a06-105">Индексатор должен возвращать значение.</span><span class="sxs-lookup"><span data-stu-id="c8a06-105">An indexer must return a value.</span></span>  
  
 <span data-ttu-id="c8a06-106">При компиляции следующего примера возникнет ошибка CS0620:</span><span class="sxs-lookup"><span data-stu-id="c8a06-106">The following sample generates CS0620:</span></span>  
  
```csharp  
// CS0620.cs  
class MyClass  
{  
   public static void Main()  
   {  
      MyClass test = new MyClass();  
      System.Console.WriteLine(test[2]);  
   }  
  
   void this [int intI]   // CS0620, return type cannot be void  
   {  
      get  
      {  
         // will need to return some value  
      }  
   }  
}  
```
