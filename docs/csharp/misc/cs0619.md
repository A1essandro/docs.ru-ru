# <a name="compiler-error-cs0619"></a><span data-ttu-id="89489-102">Ошибка компилятора CS0619</span><span class="sxs-lookup"><span data-stu-id="89489-102">Compiler Error CS0619</span></span>
<span data-ttu-id="89489-103">"член" устарел: "текст"</span><span class="sxs-lookup"><span data-stu-id="89489-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="89489-104">Член класса был помечен атрибутом [устарело] ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) атрибута, таким образом, будет выведено сообщение об ошибке, при ссылке на член класса.</span><span class="sxs-lookup"><span data-stu-id="89489-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="89489-105">Следующий пример приводит к возникновению ошибки CS0619:</span><span class="sxs-lookup"><span data-stu-id="89489-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
