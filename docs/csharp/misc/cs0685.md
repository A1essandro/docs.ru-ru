# <a name="compiler-error-cs0685"></a><span data-ttu-id="b4e0d-102">Ошибка компилятора CS0685</span><span class="sxs-lookup"><span data-stu-id="b4e0d-102">Compiler Error CS0685</span></span>
<span data-ttu-id="b4e0d-103">Член с атрибутом Conditional "член" не может иметь параметр out.</span><span class="sxs-lookup"><span data-stu-id="b4e0d-103">Conditional member 'member' cannot have an out parameter</span></span>  
  
 <span data-ttu-id="b4e0d-104">При использовании в методе атрибута <xref:System.Diagnostics.ConditionalAttribute> этот метод не может иметь параметр out.</span><span class="sxs-lookup"><span data-stu-id="b4e0d-104">When using the <xref:System.Diagnostics.ConditionalAttribute> attribute on a method, that method may not have an out parameter.</span></span> <span data-ttu-id="b4e0d-105">Это обусловлено тем, что значение переменной, используемой для параметра out, не может быть задано в случае, когда вызов метода компилируется в ничего.</span><span class="sxs-lookup"><span data-stu-id="b4e0d-105">This is because the value of the variable used for the out parameter would not be defined in the case that the method call is compiled to nothing.</span></span> <span data-ttu-id="b4e0d-106">Чтобы избежать этой ошибки, удалите параметр out из объявления условного метода или не используйте атрибут Conditional.</span><span class="sxs-lookup"><span data-stu-id="b4e0d-106">To avoid this error, remove the out parameter from the conditional method declaration, or don't use the Conditional Attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b4e0d-107">Пример</span><span class="sxs-lookup"><span data-stu-id="b4e0d-107">Example</span></span>  
 <span data-ttu-id="b4e0d-108">Следующий пример приводит к возникновению ошибки CS0685.</span><span class="sxs-lookup"><span data-stu-id="b4e0d-108">The following sample generates CS0685:</span></span>  
  
```csharp  
// CS0685.cs  
using System.Diagnostics;  
  
class C  
{  
    [Conditional("DEBUG")]  
    void trace(out int i)  // CS0685  
    {  
        i = 1;  
    }  
}  
```
