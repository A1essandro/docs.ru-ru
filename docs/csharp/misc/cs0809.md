<span data-ttu-id="84852-103">Устаревший член "членA" переопределяет неустаревший член "членB".</span><span class="sxs-lookup"><span data-stu-id="84852-103">Obsolete member 'memberA' overrides non-obsolete member 'memberB'.</span></span>  
  
 <span data-ttu-id="84852-104">Обычно член, помеченный как устаревший, не должен переопределять член, который не помечен как устаревший.</span><span class="sxs-lookup"><span data-stu-id="84852-104">Typically, a member that is marked as obsolete should not override a member that is not marked as obsolete.</span></span> <span data-ttu-id="84852-105">Это предупреждение создается в [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] но не в [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)].</span><span class="sxs-lookup"><span data-stu-id="84852-105">This warning is generated in [!INCLUDE[vs_orcas_long](~/includes/vs-orcas-long-md.md)] but not in [!INCLUDE[vsprvslong](~/includes/vsprvslong-md.md)].</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="84852-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="84852-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="84852-108">Пример</span><span class="sxs-lookup"><span data-stu-id="84852-108">Example</span></span>  
  
```csharp  
// CS0809.cs  
public class Base  
{  
    public virtual void Test1()  
    {  
    }  
}  
public class C : Base  
{  
    [System.Obsolete()]  
    public override void Test1() // CS0809  
    {  
    }  
}  
```
