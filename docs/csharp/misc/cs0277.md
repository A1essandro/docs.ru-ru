 <span data-ttu-id="737c1-105">Эта ошибка возникает при попытке реализовать свойство интерфейса, но реализация метода доступа свойства в классе не является открытой.</span><span class="sxs-lookup"><span data-stu-id="737c1-105">This error occurs when you try to implement a property of an interface, but the implementation of the property accessor in the class is not public.</span></span> <span data-ttu-id="737c1-106">Методы, которые реализуют члены интерфейса, должны иметь доступность уровня public.</span><span class="sxs-lookup"><span data-stu-id="737c1-106">Methods that implement interface members need to have public accessibility.</span></span> <span data-ttu-id="737c1-107">Для устранения ошибки удалите модификатор доступа в методе доступа свойства.</span><span class="sxs-lookup"><span data-stu-id="737c1-107">To resolve, remove the access modifier on the property accessor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="737c1-108">Пример</span><span class="sxs-lookup"><span data-stu-id="737c1-108">Example</span></span>  
 <span data-ttu-id="737c1-109">В следующем примере возникает ошибка CS0277.</span><span class="sxs-lookup"><span data-stu-id="737c1-109">The following example generates CS0277:</span></span>  
  
```csharp  
// CS0277.cs  
public interface MyInterface  
{  
    int Property  
    {  
        get;  
        set;  
    }  
}  
  
public class MyClass : MyInterface   // CS0277  
{  
    public int Property  
    {  
        get { return 0; }  
        // Try this instead:  
        //set { }  
        protected set { }  
    }  
}  
```
