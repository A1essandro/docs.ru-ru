 <span data-ttu-id="118ea-109">В приведенном ниже примере кода возникает ошибка CS1955, так как происходит попытка вызвать поле и свойство с помощью оператора вызова метода [()](../../csharp/language-reference/operators/invocation-operator.md).</span><span class="sxs-lookup"><span data-stu-id="118ea-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the method call operator [()](../../csharp/language-reference/operators/invocation-operator.md).</span></span> <span data-ttu-id="118ea-110">Поле или свойство вызвать невозможно, но можно обратиться к хранящемуся в нем значению с помощью оператора доступа к члену ( [.](../../csharp/language-reference/operators/member-access-operator.md)</span><span class="sxs-lookup"><span data-stu-id="118ea-110">You cannot call a field or property, but you can access the value it stores by using the member access operator ( [.](../../csharp/language-reference/operators/member-access-operator.md)</span></span> <span data-ttu-id="118ea-111">).</span><span class="sxs-lookup"><span data-stu-id="118ea-111">).</span></span>  
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
