 <span data-ttu-id="fd03a-104">Указание параметра [void](../../csharp/language-reference/keywords/void.md) , кроме указателя void, является необязательным и недопустимым.</span><span class="sxs-lookup"><span data-stu-id="fd03a-104">It is not necessary or valid to specify a [void](../../csharp/language-reference/keywords/void.md) parameter other than a void pointer.</span></span>

   {  
   }
