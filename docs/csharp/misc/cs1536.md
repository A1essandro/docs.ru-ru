# <a name="compiler-error-cs1536"></a><span data-ttu-id="eb58f-102">Ошибка компилятора CS1536</span><span class="sxs-lookup"><span data-stu-id="eb58f-102">Compiler Error CS1536</span></span>

 <span data-ttu-id="eb58f-104">Указание параметра [void](../../csharp/language-reference/keywords/void.md) , кроме указателя void, является необязательным и недопустимым.</span><span class="sxs-lookup"><span data-stu-id="eb58f-104">It is not necessary or valid to specify a [void](../../csharp/language-reference/keywords/void.md) parameter other than a void pointer.</span></span>  
  
 <span data-ttu-id="eb58f-105">При компиляции следующего примера возникнет ошибка CS1536:</span><span class="sxs-lookup"><span data-stu-id="eb58f-105">The following sample generates CS1536:</span></span>  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
