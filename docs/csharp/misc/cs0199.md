# <a name="compiler-error-cs0199"></a><span data-ttu-id="c97df-102">Ошибка компилятора CS0199</span><span class="sxs-lookup"><span data-stu-id="c97df-102">Compiler Error CS0199</span></span>
<span data-ttu-id="c97df-103">Поля доступного только для чтения статического поля "имя" могут передаваться как параметры с ключевым словом ref или out только в статическом конструкторе</span><span class="sxs-lookup"><span data-stu-id="c97df-103">Fields of static readonly field 'name' cannot be passed ref or out (except in a static constructor)</span></span>  
  
 <span data-ttu-id="c97df-104">Объект [readonly](../../csharp/language-reference/keywords/readonly.md) переменной должны иметь одинаковый [статических](../../csharp/language-reference/keywords/static.md) использование в качестве конструктора, в котором требуется передать в качестве [ref](../../csharp/language-reference/keywords/ref.md) или [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) параметра.</span><span class="sxs-lookup"><span data-stu-id="c97df-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to pass it as a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="c97df-105">Дополнительные сведения см. в разделе [Передача параметров](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="c97df-105">For more information, see [Passing Parameters](../../csharp/programming-guide/classes-and-structs/passing-parameters.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="c97df-106">Пример</span><span class="sxs-lookup"><span data-stu-id="c97df-106">Example</span></span>  
 <span data-ttu-id="c97df-107">Следующий пример приводит к возникновению ошибки CS0199:</span><span class="sxs-lookup"><span data-stu-id="c97df-107">The following sample generates CS0199:</span></span>  
  
```csharp  
// CS0199.cs  
class MyClass  
{  
    public static readonly int TestInt = 6;  
  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // CS0199, TestInt is static  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
