# <a name="compiler-error-cs0228"></a>Ошибка компилятора CS0228
"тип" не содержит определение члена "член" или недоступен  
  
 Вы могли заменить версию системы **System.Hashtable**, **System.String**или **System.Array**, и замена не содержит `member`.  
  
 В противном случае может потребоваться восстановление или переустановка Visual Studio.
