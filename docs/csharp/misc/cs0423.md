# <a name="compiler-error-cs0423"></a><span data-ttu-id="91a41-102">Ошибка компилятора CS0423</span><span class="sxs-lookup"><span data-stu-id="91a41-102">Compiler Error CS0423</span></span>
<span data-ttu-id="91a41-103">Поскольку "класс" имеет атрибут ComImport, "метод" должен быть внешним или абстрактным</span><span class="sxs-lookup"><span data-stu-id="91a41-103">Since 'class' has the ComImport attribute, 'method' must be extern or abstract</span></span>  
  
 <span data-ttu-id="91a41-104">Указание атрибута ComImport подразумевает, что реализация класса должна импортироваться из COM-модуля.</span><span class="sxs-lookup"><span data-stu-id="91a41-104">Specifying the ComImport attribute implies that the implementation for the class is to be imported from a COM module.</span></span> <span data-ttu-id="91a41-105">Дополнительные методы не могут быть определены.</span><span class="sxs-lookup"><span data-stu-id="91a41-105">Additional methods may not be defined.</span></span>  
  
 <span data-ttu-id="91a41-106">В следующем примере возникает ошибка CS0423:</span><span class="sxs-lookup"><span data-stu-id="91a41-106">The following sample generates CS0423:</span></span>  
  
```csharp  
// CS0423.cs  
  
using System.Runtime.InteropServices;  
  
[  
  ComImport,  
  Guid("7ab770c7-0e23-4d7a-8aa2-19bfad479829")  
]  
class ImageProperties  
{  
  public static void Main()  // CS0423  
  {  
    ImageProperties i = new ImageProperties();  
  }  
}  
```
