 <span data-ttu-id="2ac1f-104">**System.Void** не может использоваться в C#.</span><span class="sxs-lookup"><span data-stu-id="2ac1f-104">**System.Void** cannot be used in C#.</span></span>

```csharp  
   public static void Main()  
   {  
      System.Type t = typeof(System.Void);   // CS0673

   }  
}  
```
