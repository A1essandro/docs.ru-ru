# <a name="compiler-error-cs0308"></a><span data-ttu-id="f274f-102">Ошибка компилятора CS0308</span><span class="sxs-lookup"><span data-stu-id="f274f-102">Compiler Error CS0308</span></span>
<span data-ttu-id="f274f-103">Идентификатор "идентификатор" неуниверсального типа или метода не может использоваться с аргументами типа.</span><span class="sxs-lookup"><span data-stu-id="f274f-103">The non-generic type-or-method 'identifier' cannot be used with type arguments.</span></span>  
  
 <span data-ttu-id="f274f-104">Метод или тип не является универсальным, но он использовался с аргументами типа.</span><span class="sxs-lookup"><span data-stu-id="f274f-104">The method or type is not generic, but it was used with type arguments.</span></span> <span data-ttu-id="f274f-105">Чтобы избежать этой ошибки, удалите угловые скобки и аргументы типа или заново объявите этот метод или тип как универсальный.</span><span class="sxs-lookup"><span data-stu-id="f274f-105">To avoid this error, remove the angled brackets and type arguments, or redeclare the method or type as a generic method or type.</span></span>  
  
 <span data-ttu-id="f274f-106">В следующем примере возникает ошибка CS0308:</span><span class="sxs-lookup"><span data-stu-id="f274f-106">The following example generates CS0308:</span></span>  
  
```  
// CS0308a.cs  
class MyClass  
{  
   public void F() {}  
   public static void Main()  
   {  
      F<int>();  // CS0308 – F is not generic.  
      // Try this instead:  
      // F();  
   }  
}  
```  
  
 <span data-ttu-id="f274f-107">В следующем примере также возникает ошибка CS0308.</span><span class="sxs-lookup"><span data-stu-id="f274f-107">The following example also generates CS0308.</span></span> <span data-ttu-id="f274f-108">Чтобы устранить эту ошибку, используйте директиву using System.Collections.Generic.</span><span class="sxs-lookup"><span data-stu-id="f274f-108">To resolve the error, use the directive "using System.Collections.Generic."</span></span>  
  
```  
// CS0308b.cs  
// compile with: /t:library  
using System.Collections;  
// To resolve, uncomment the following line:  
// using System.Collections.Generic;  
public class MyStack<T>  
{  
    // Store the elements of the stack:  
    private T[] items = new T[100];  
    private int stack_counter = 0;  
  
    // Define the iterator block:  
    public IEnumerator<T> GetEnumerator()   // CS0308  
    {  
        for (int i = stack_counter - 1 ; i >= 0; i--)  
        yield return items[i];  
    }  
}  
```
