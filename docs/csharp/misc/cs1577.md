# <a name="compiler-error-cs1577"></a>Ошибка компилятора CS1577
Сбой при создании сборки — причина  
  
 Этап компиляции, на котором происходит создание сборки, завершился ошибкой после того, как компилятор C# успешно скомпилировал исходный код. См. в документации ошибок для служебной программы ALink ([Al.exe ошибки и предупреждения средства](http://msdn.microsoft.com/library/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)) для получения дополнительной информации.
