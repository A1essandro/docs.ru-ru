# <a name="compiler-error-cs1577"></a>Ошибка компилятора CS1577
Сбой при создании сборки — причина  
  
 Этап компиляции, на котором происходит создание сборки, завершился ошибкой после того, как компилятор C# успешно скомпилировал исходный код. Для получения дополнительных сведений обратитесь к документации по ошибкам служебной программы ALink ([Ошибки и предупреждения средства Al.exe](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b)).
