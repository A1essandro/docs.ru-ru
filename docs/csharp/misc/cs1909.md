# <a name="compiler-error-cs1909"></a><span data-ttu-id="8b69b-102">Ошибка компилятора CS1909</span><span class="sxs-lookup"><span data-stu-id="8b69b-102">Compiler Error CS1909</span></span>

 <span data-ttu-id="8b69b-104">Ошибка CS1909 возникает при использовании атрибута `DefaultValue` , который не применим к типу параметра.</span><span class="sxs-lookup"><span data-stu-id="8b69b-104">CS1909 is generated when you use a `DefaultValue` attribute that is not applicable on the parameter type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8b69b-105">Пример</span><span class="sxs-lookup"><span data-stu-id="8b69b-105">Example</span></span>  
 <span data-ttu-id="8b69b-106">Следующий пример приводит к возникновению ошибки CS1909:</span><span class="sxs-lookup"><span data-stu-id="8b69b-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs  
// compile with: /target:library  
using System.Runtime.InteropServices;  
  
public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
