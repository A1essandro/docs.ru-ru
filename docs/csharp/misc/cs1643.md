 <span data-ttu-id="d6d99-104">Эта ошибка возникает, если в теле делегата отсутствует оператор return или имеется оператор return, который компилятор не может проверить.</span><span class="sxs-lookup"><span data-stu-id="d6d99-104">This error occurs if a delegate body does not have a return statement, or has a return statement that the compiler is unable to verify will be reached.</span></span> <span data-ttu-id="d6d99-105">В следующем примере компилятор не пытается предсказать результат условия ветвления для проверки того, что блок анонимного метода всегда возвращает значение.</span><span class="sxs-lookup"><span data-stu-id="d6d99-105">In the example below, the compiler does not attempt to predict the result of the branching condition in order to verify that the anonymous method block always returns a value.</span></span>

```  
{  
    static void Main()  
    {  
        MyDelegate d = delegate

        };  
    }

```
