# <a name="compiler-error-cs0283"></a><span data-ttu-id="f223b-102">Ошибка компилятора CS0283</span><span class="sxs-lookup"><span data-stu-id="f223b-102">Compiler Error CS0283</span></span>
<span data-ttu-id="f223b-103">Тип "тип" не может быть объявлен как константа.</span><span class="sxs-lookup"><span data-stu-id="f223b-103">The type 'type' cannot be declared const</span></span>  
  
<span data-ttu-id="f223b-104">Тип, указанный в [константы](../language-reference/keywords/const.md) объявление должно быть `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, [перечисления](../language-reference/keywords/enum.md) тип или ссылочный тип, который назначается значение `null`.</span><span class="sxs-lookup"><span data-stu-id="f223b-104">The type specified in a [constant](../language-reference/keywords/const.md) declaration must be `byte`, `sbyte`, `ushort`, `short`, `uint`, `int`, `ulong`, `long`, `char`, `float`, `double`, `decimal`, `bool`, `string`, an [enum](../language-reference/keywords/enum.md) type, or a reference type that is assigned a value of `null`.</span></span> <span data-ttu-id="f223b-105">Каждое константное выражение должно выдавать значение целевого типа или типа, который может быть неявно преобразован в тип целевого объекта.</span><span class="sxs-lookup"><span data-stu-id="f223b-105">Each constant expression must yield a value of the target type or of a type that is implicitly convertible to the target type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f223b-106">Пример</span><span class="sxs-lookup"><span data-stu-id="f223b-106">Example</span></span>  
 <span data-ttu-id="f223b-107">Следующий пример приводит к возникновению ошибки CS0283.</span><span class="sxs-lookup"><span data-stu-id="f223b-107">The following example generates CS0283.</span></span>  
  
```csharp  
// CS0283.cs  
struct MyTest  
{  
}  
class MyClass   
{  
    // To resolve the error but retain the "const-ness",  
    // change const to readonly.  
    const MyTest test = new MyTest();   // CS0283  
  
    public static int Main() {  
        return 1;  
    }  
}  
```
