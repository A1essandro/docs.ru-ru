## <a name="example"></a><span data-ttu-id="d194e-109">Пример</span><span class="sxs-lookup"><span data-stu-id="d194e-109">Example</span></span>  
 <span data-ttu-id="d194e-110">В следующем примере возникает ошибка CS0279:</span><span class="sxs-lookup"><span data-stu-id="d194e-110">The following example generates CS0279:</span></span>

```csharp  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }

    }

        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
