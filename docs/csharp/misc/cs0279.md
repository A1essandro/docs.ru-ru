## <a name="example"></a><span data-ttu-id="a0562-109">Пример</span><span class="sxs-lookup"><span data-stu-id="a0562-109">Example</span></span>  
 <span data-ttu-id="a0562-110">В следующем примере возникает ошибка CS0279:</span><span class="sxs-lookup"><span data-stu-id="a0562-110">The following example generates CS0279:</span></span>  
  
```csharp  
// CS0279.cs  
  
using System;  
using System.Collections;  
  
public class myTest : IEnumerable  
{  
    IEnumerator IEnumerable.GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    internal IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    public static void Main()  
    {  
        foreach (int i in new myTest()) {}  // CS0279  
    }  
}  
```
