# <a name="compiler-error-cs1673"></a><span data-ttu-id="6d955-102">Ошибка компилятора CS1673</span><span class="sxs-lookup"><span data-stu-id="6d955-102">Compiler Error CS1673</span></span>
<span data-ttu-id="6d955-103">Анонимные методы, лямбда-выражения и выражения запроса внутри структуры не могут обращаться к членам экземпляров this.</span><span class="sxs-lookup"><span data-stu-id="6d955-103">Anonymous methods, lambda expressions, and query expressions inside structs cannot access instance members of 'this'.</span></span> <span data-ttu-id="6d955-104">Рекомендуется скопировать this в локальную переменную за пределами анонимного метода, лямбда-выражения или выражения запроса и использовать эту локальную переменную.</span><span class="sxs-lookup"><span data-stu-id="6d955-104">Consider copying 'this' to a local variable outside the anonymous method, lambda expression or query expression and using the local instead.</span></span>  
  
 <span data-ttu-id="6d955-105">Следующий пример приводит к возникновению ошибки CS1673:</span><span class="sxs-lookup"><span data-stu-id="6d955-105">The following sample generates CS1673:</span></span>  
  
```csharp  
// CS1673.cs  
delegate int MyDelegate();  
  
public struct S  
{  
   int member;  
  
   public int F(int i)  
   {  
       member = i;  
       // Try assigning to a local variable  
       // S s = this;  
       MyDelegate d = delegate()  
       {  
          i = this.member;  // CS1673  
          // And use the local variable instead of "this"  
          // i =  s.member;  
          return i;  
  
       };  
       return d();  
   }  
}  
  
class CMain  
{  
   public static void Main()  
   {  
   }  
}  
```
