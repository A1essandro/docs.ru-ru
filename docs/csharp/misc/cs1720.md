# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="cb49e-102">Предупреждение компилятора (уровень 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="cb49e-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="cb49e-103">Выражение всегда будет вызывать System.NullReferenceException, поскольку значение "универсальный_тип" по умолчанию равно null</span><span class="sxs-lookup"><span data-stu-id="cb49e-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="cb49e-104">Если вы записали выражение с переменной универсального типа, имеющей значение по умолчанию, которая является ссылочным типом (например, класс), возникнет эта ошибка.</span><span class="sxs-lookup"><span data-stu-id="cb49e-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="cb49e-105">Рассмотрим следующее выражение:</span><span class="sxs-lookup"><span data-stu-id="cb49e-105">Consider the following expression:</span></span>  
  
```  
default(T).ToString()  
```  
  
 <span data-ttu-id="cb49e-106">Так как `T` является ссылочным типом, его значение по умолчанию равно null, и поэтому попытка применить к нему метод <xref:System.Object.ToString%2A> вызовет <xref:System.NullReferenceException>.</span><span class="sxs-lookup"><span data-stu-id="cb49e-106">Since `T` is a reference type, its default value is null, and so attempting to apply the <xref:System.Object.ToString%2A> method to it will throw a <xref:System.NullReferenceException>.</span></span>  
  
## <a name="example"></a><span data-ttu-id="cb49e-107">Пример</span><span class="sxs-lookup"><span data-stu-id="cb49e-107">Example</span></span>  
 <span data-ttu-id="cb49e-108">Ограничение ссылки на класс для типа `T` гарантирует, что `T` является ссылочным типом.</span><span class="sxs-lookup"><span data-stu-id="cb49e-108">The class reference constraint on type `T` ensures that `T` is a reference type.</span></span>  
  
 <span data-ttu-id="cb49e-109">При компиляции следующего примера будет выдано предупреждение CS1720.</span><span class="sxs-lookup"><span data-stu-id="cb49e-109">The following sample generates CS1720.</span></span>  
  
```  
// CS1720.cs  
using System;  
public class Tester   
{  
    public static void GenericClass<T>(T t1) where T : class   
    {  
        Console.WriteLine(default(T).ToString());  // CS1720  
    }  
    public static void Main() {}  
}  
```
