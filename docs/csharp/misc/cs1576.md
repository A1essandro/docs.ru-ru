 <span data-ttu-id="a41e6-104">Компилятор обнаружил ошибку в значении, переданном в директиву [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) .</span><span class="sxs-lookup"><span data-stu-id="a41e6-104">The compiler detected an error with the value passed to the [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) directive.</span></span>  
  
 <span data-ttu-id="a41e6-105">Следующий пример приводит к возникновению ошибки CS1576:</span><span class="sxs-lookup"><span data-stu-id="a41e6-105">The following sample generates CS1576:</span></span>  
  
```  
// CS1576.cs  
public class MyClass  
{  
   static void Main()  
   {  
      #line "abc.sc"         // CS1576  
      // try the following line instead  
      //#line  101 "abc.sc"  
      intt i;  // error will be reported on line 101  
   }  
}  
```
