# <a name="compiler-error-cs0752"></a><span data-ttu-id="78828-102">Ошибка компилятора CS0752</span><span class="sxs-lookup"><span data-stu-id="78828-102">Compiler Error CS0752</span></span>
<span data-ttu-id="78828-103">Разделяемый метод не может иметь параметры out.</span><span class="sxs-lookup"><span data-stu-id="78828-103">A partial method cannot have out parameters</span></span>  
  
 <span data-ttu-id="78828-104">Разделяемый метод не может иметь [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) параметра.</span><span class="sxs-lookup"><span data-stu-id="78828-104">A partial method cannot have an [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="78828-105">Параметры out запрещены, поскольку при удалении компилятором разделяемого метода нет никакой гарантии, что параметр out когда-либо будет назначен.</span><span class="sxs-lookup"><span data-stu-id="78828-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="78828-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="78828-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="78828-107">Удалите модификатор out из параметра и используйте вместо него возвращаемое значение метода либо удалите модификатор partial из объявления метода.</span><span class="sxs-lookup"><span data-stu-id="78828-107">Remove the out modifier from the parameter and use the return value of the method instead, or else remove the partial modifier from the method declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="78828-108">Пример</span><span class="sxs-lookup"><span data-stu-id="78828-108">Example</span></span>  
 <span data-ttu-id="78828-109">В следующем коде возникает ошибка CS0752:</span><span class="sxs-lookup"><span data-stu-id="78828-109">The following code generates CS0752:</span></span>

```csharp  
// cs0752.cs  
public partial class C  
{  
    partial void Part(out int num); // CS0752  
    // try the following line instead  
    // partial void Part(int num);

    {  
        return 1;  
    }  
}  
```
