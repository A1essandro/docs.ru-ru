# <a name="compiler-error-cs0551"></a><span data-ttu-id="71f9a-102">Ошибка компилятора CS0551</span><span class="sxs-lookup"><span data-stu-id="71f9a-102">Compiler Error CS0551</span></span>
<span data-ttu-id="71f9a-103">В явной реализации интерфейса "реализация" отсутствует метод доступа "метод_доступа"</span><span class="sxs-lookup"><span data-stu-id="71f9a-103">Explicit interface implementation 'implementation' is missing accessor 'accessor'</span></span>  
  
 <span data-ttu-id="71f9a-104">Класс, который явно реализует свойство интерфейса, должен реализовывать все методы доступа, которые определяет этот интерфейс.</span><span class="sxs-lookup"><span data-stu-id="71f9a-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>  
  
 <span data-ttu-id="71f9a-105">Дополнительные сведения см. в разделе [Использование свойств](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span><span class="sxs-lookup"><span data-stu-id="71f9a-105">For more information, see [Using Properties](../../csharp/programming-guide/classes-and-structs/using-properties.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="71f9a-106">Пример</span><span class="sxs-lookup"><span data-stu-id="71f9a-106">Example</span></span>  
 <span data-ttu-id="71f9a-107">В следующем примере возникает ошибка CS0551:</span><span class="sxs-lookup"><span data-stu-id="71f9a-107">The following sample generates CS0551.</span></span>  
  
```  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i      
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
