# <a name="compiler-error-cs0711"></a><span data-ttu-id="0446a-102">Ошибка компилятора CS0711</span><span class="sxs-lookup"><span data-stu-id="0446a-102">Compiler Error CS0711</span></span>
<span data-ttu-id="0446a-103">Статические классы не могут содержать деструкторы.</span><span class="sxs-lookup"><span data-stu-id="0446a-103">Static classes cannot contain destructors</span></span>  
  
 <span data-ttu-id="0446a-104">Для статических классов нельзя создавать экземпляры, поэтому для них не нужны конструкторы и деструкторы.</span><span class="sxs-lookup"><span data-stu-id="0446a-104">A static class cannot be instantiated, hence it has no need for constructors or destructors.</span></span> <span data-ttu-id="0446a-105">Чтобы избежать возникновения этой ошибки, удалите деструкторы из статических классов или, если все же необходимо создавать и уничтожать экземпляры, сделайте соответствующие классы нестатическими.</span><span class="sxs-lookup"><span data-stu-id="0446a-105">To avoid this error, remove any destructors from static classes, or, if you really want to construct and destroy instances, make the class non-static.</span></span>  
  
 <span data-ttu-id="0446a-106">Следующий пример приводит к возникновению ошибки CS0711:</span><span class="sxs-lookup"><span data-stu-id="0446a-106">The following sample generates CS0711:</span></span>  
  
```  
// CS0711.cs  
public static class C  
{  
   ~C()  // CS0711  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
