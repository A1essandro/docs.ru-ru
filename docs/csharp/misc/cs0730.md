## <a name="example"></a><span data-ttu-id="ee4e0-105">Пример</span><span class="sxs-lookup"><span data-stu-id="ee4e0-105">Example</span></span>  
 <span data-ttu-id="ee4e0-106">Следующий пример приводит к возникновению ошибки CS0730.</span><span class="sxs-lookup"><span data-stu-id="ee4e0-106">The following sample generates CS0730.</span></span> <span data-ttu-id="ee4e0-107">Он состоит из двух файлов исходного кода.</span><span class="sxs-lookup"><span data-stu-id="ee4e0-107">It consists of two source files.</span></span> <span data-ttu-id="ee4e0-108">Сначала скомпилируйте файл библиотеки `CS0730a.cs`, а затем скомпилируйте файл `CS0730.cs` , ссылающийся на файл библиотеки.</span><span class="sxs-lookup"><span data-stu-id="ee4e0-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
