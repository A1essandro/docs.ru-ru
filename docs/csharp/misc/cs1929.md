# <a name="compiler-error-cs1929"></a><span data-ttu-id="d3bb2-102">Ошибка компилятора CS1929</span><span class="sxs-lookup"><span data-stu-id="d3bb2-102">Compiler Error CS1929</span></span>
<span data-ttu-id="d3bb2-103">Аргумент экземпляра: невозможно преобразовать из "типа A" в "тип B".</span><span class="sxs-lookup"><span data-stu-id="d3bb2-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="d3bb2-104">Эта ошибка возникает при попытке вызвать метод расширения из класса, который он не расширяет.</span><span class="sxs-lookup"><span data-stu-id="d3bb2-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="d3bb2-105">В приведенном здесь примере метод расширения определен для производного класса `A`, но не для базового класса `B`.</span><span class="sxs-lookup"><span data-stu-id="d3bb2-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d3bb2-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="d3bb2-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="d3bb2-107">Создайте новый метод расширения для типа, где необходимо вызвать его, или переместите вызов в объект типа, который расширяет существующий метод.</span><span class="sxs-lookup"><span data-stu-id="d3bb2-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d3bb2-108">Пример</span><span class="sxs-lookup"><span data-stu-id="d3bb2-108">Example</span></span>  
 <span data-ttu-id="d3bb2-109">Следующий код приводит к возникновению ошибок CS1928 и CS1929:</span><span class="sxs-lookup"><span data-stu-id="d3bb2-109">The following code generates CS1928 and CS1929:</span></span>  
  
```  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
