# <a name="compiler-error-cs0456"></a><span data-ttu-id="dd448-102">Ошибка компилятора CS0456</span><span class="sxs-lookup"><span data-stu-id="dd448-102">Compiler Error CS0456</span></span>
<span data-ttu-id="dd448-103">Параметр типа "Имя параметра типа 1" имеет ограничение struct, следовательно, "Имя параметра типа 1" не может использоваться как ограничение для "Имя параметра типа 2"</span><span class="sxs-lookup"><span data-stu-id="dd448-103">Type parameter 'Type Parameter Name 1' has the 'struct' constraint so 'Type Parameter Name 1' cannot be used as a constraint for 'Type Parameter Name 2'</span></span>  
  
 <span data-ttu-id="dd448-104">Ограничения типа значения являются неявно запечатанными, поэтому их нельзя использовать в качестве ограничений для второго параметра типа.</span><span class="sxs-lookup"><span data-stu-id="dd448-104">Value type constraints are implicitly sealed so those constraints cannot be used as constraints on a second type parameter.</span></span> <span data-ttu-id="dd448-105">Это обусловлено тем, что типы значений нельзя переопределять.</span><span class="sxs-lookup"><span data-stu-id="dd448-105">This is because value types cannot be overridden.</span></span> <span data-ttu-id="dd448-106">Для устранения этой ошибки поместите ограничение типа значения непосредственно во второй параметр типа, вместо того чтобы делать это косвенно посредством первого параметра типа.</span><span class="sxs-lookup"><span data-stu-id="dd448-106">To resolve this error, put a value type constraint directly on the second type parameter, instead of doing so indirectly by means of the first type parameter.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dd448-107">Пример</span><span class="sxs-lookup"><span data-stu-id="dd448-107">Example</span></span>  
 <span data-ttu-id="dd448-108">Следующий пример приводит к возникновению ошибки CS0456:</span><span class="sxs-lookup"><span data-stu-id="dd448-108">The following sample generates CS0456.</span></span>  
  
```csharp  
// CS0456.cs  
// compile with: /target:library  
public class GenericsErrors  
{  
   public class G5<T> where T : struct  
   {  
      public class N<U> where U : T {}   // CS0456  
      public class N2<U> where U : struct {}   // OK  
   }  
}  
```
