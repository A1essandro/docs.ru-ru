# <a name="compiler-warning-level-1-cs0672"></a><span data-ttu-id="b257b-102">Предупреждение компилятора (уровень 1) CS0672</span><span class="sxs-lookup"><span data-stu-id="b257b-102">Compiler Warning (level 1) CS0672</span></span>
<span data-ttu-id="b257b-103">Член "член1" переопределяет устаревший член "член2".</span><span class="sxs-lookup"><span data-stu-id="b257b-103">Member 'member1' overrides obsolete member 'member2.</span></span> <span data-ttu-id="b257b-104">Добавьте к "член1" атрибут Obsolete.</span><span class="sxs-lookup"><span data-stu-id="b257b-104">Add the Obsolete attribute to 'member1'</span></span>  
  
 <span data-ttu-id="b257b-105">Компилятор обнаружил ключевое слово `override` , примененное к методу, помеченному атрибутом `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="b257b-105">The compiler found an `override` to a method marked as `obsolete`.</span></span> <span data-ttu-id="b257b-106">Однако переопределяющий метод не помечен как устаревший.</span><span class="sxs-lookup"><span data-stu-id="b257b-106">However, the overriding method was not itself marked as obsolete.</span></span> <span data-ttu-id="b257b-107">В случае вызова переопределяющий метод будет продолжать вызывать ошибку [CS0612](../../csharp/misc/cs0612.md).</span><span class="sxs-lookup"><span data-stu-id="b257b-107">The overriding method will still generate [CS0612](../../csharp/misc/cs0612.md), if called.</span></span>  
  
 <span data-ttu-id="b257b-108">Проверьте объявления методов и явным образом укажите, должен ли метод (и все его переопределения) помечаться как `obsolete`.</span><span class="sxs-lookup"><span data-stu-id="b257b-108">Review your method declarations and explicitly indicate whether a method (and all of its overrides) should be marked `obsolete`.</span></span>  
  
 <span data-ttu-id="b257b-109">Следующий пример приводит к возникновению предупреждения CS0672:</span><span class="sxs-lookup"><span data-stu-id="b257b-109">The following sample generates CS0672:</span></span>  
  
```  
// CS0672.cs  
// compile with: /W:1  
class MyClass  
{  
   [System.Obsolete]  
   public virtual void ObsoleteMethod()  
   {  
   }  
}  
  
class MyClass2 : MyClass  
{  
   public override void ObsoleteMethod()   // CS0672  
   {  
   }  
}  
  
class MainClass  
{  
   static public void Main()  
   {  
   }  
}  
```
