# <a name="compiler-error-cs1621"></a><span data-ttu-id="ece59-102">Ошибка компилятора CS1621</span><span class="sxs-lookup"><span data-stu-id="ece59-102">Compiler Error CS1621</span></span>
<span data-ttu-id="ece59-103">Внутри анонимного метода или лямбда-выражения нельзя использовать оператор yield.</span><span class="sxs-lookup"><span data-stu-id="ece59-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="ece59-104">Оператор yield не может находиться в блоке анонимного метода в итераторе.</span><span class="sxs-lookup"><span data-stu-id="ece59-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ece59-105">Пример</span><span class="sxs-lookup"><span data-stu-id="ece59-105">Example</span></span>  
 <span data-ttu-id="ece59-106">Следующий пример приводит к возникновению ошибки CS1621:</span><span class="sxs-lookup"><span data-stu-id="ece59-106">The following sample generates CS1621:</span></span>  
  
```  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
