# <a name="compiler-options"></a>Параметры компилятора

В этом разделе описаны параметры командной строки компилятора компилятора F #, fsc.exe. В среде компиляции можно также управлять с помощью свойств проекта.

|**--nowarn:&lt;список типа int&gt;**|Отключает определенные предупреждения, перечисленные по номеру. Каждый номер предупреждения следует разделяйте запятыми.
