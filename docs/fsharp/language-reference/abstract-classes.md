# <a name="abstract-classes"></a><span data-ttu-id="cb367-103">Абстрактные классы</span><span class="sxs-lookup"><span data-stu-id="cb367-103">Abstract Classes</span></span>

<span data-ttu-id="cb367-104">*Абстрактные классы* — это классы, которые оставляют некоторые или все члены не требует дополнительных инструментов, которых могут быть предоставлены реализации производных классов.</span><span class="sxs-lookup"><span data-stu-id="cb367-104">*Abstract classes* are classes that leave some or all members unimplemented, so that implementations can be provided by derived classes.</span></span>

## <a name="syntax"></a><span data-ttu-id="cb367-105">Синтаксис</span><span class="sxs-lookup"><span data-stu-id="cb367-105">Syntax</span></span>

```fsharp
// Abstract class syntax.
[<AbstractClass>]
type [ accessibility-modifier ] abstract-class-name =
[ inherit base-class-or-interface-name ]
[ abstract-member-declarations-and-member-definitions ]

// Abstract member syntax.
abstract member member-name : type-signature
```
