# <a name="f-guide"></a>Руководство по языку F#

F # является функциональным языком программирования, которая выполняется на платформе .NET.  Помимо поддержки конструкций функционального программирования он также имеет возможности программирования объектов.  Делает этот гибридом функционального программирования с объектно ориентированные возможности F # прагматичное использование языка для выполнения любой задачи.

## <a name="if-youre-new-to-f"></a>Если вы не знакомы с F # #

Если вы не знакомы с F #, начинаются с [обзор языка F #](tour.md) получить общие сведения о языке и некоторые из его принципы программирования.  Если вы используете Visual Studio, шаблон учебного проекта содержит то же содержимое.

## <a name="if-youre-experienced-with-f"></a>Если у вас есть опыт работы с F # #

Если вы знаете, как F # или хотите узнать больше о конкретных языковой конструкции, см. раздел [Справочник по языку](language-reference/index.md).  Это подробное руководство по всех возможностей языка F #.

Кроме того [Справочник по основной библиотеке F #](https://msdn.microsoft.com/visualfsharpdocs/conceptual/fsharp-core-library-reference) — это отличный ресурс для изучения FSharp.Core основной библиотеки, являющийся частью F #.

## <a name="the-f-software-foundation"></a>F# Software Foundation

Несмотря на то, что корпорация Майкрософт является основным разработчиком языка F # и его инструментарий, F # также поддерживаемый независимых foundation, F # программного обеспечения Foundation (FSSF).

В задачи F# Software Foundation входит продвижение, защита и совершенствование языка программирования F#, а также поддержка и расширение международного сообщества программистов F#.

Узнать дополнительные сведения и принять участие в этой работе можно на сайте [fsharp.org](http://fsharp.org).
