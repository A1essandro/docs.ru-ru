# <a name="getting-started-with-f"></a>Приступая к работе с F# #

К работе с F# можно приступить тремя разными способами.

Если вы используете Windows, см. статью [Начало работы с F# в Visual Studio](getting-started-visual-studio.md).

Если вас интересует написание кода в упрощенной интегрированной среде разработки на базе Windows, Linux или macOS, см. статью [Начало работы с F# в Visual Studio Code с помощью Ionide](getting-started-vscode.md).

Если вы предпочитаете использовать командную строку в любой ОС, см. статью [Начало работы с F# с помощью программ командной строки](getting-started-command-line.md).
