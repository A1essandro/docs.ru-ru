# <a name="type-providers"></a><span data-ttu-id="3039b-103">Поставщики типов</span><span class="sxs-lookup"><span data-stu-id="3039b-103">Type Providers</span></span>

<span data-ttu-id="3039b-104">Поставщик типов F# — это компонент, предоставляющий типы, свойства и методы для использования в программе.</span><span class="sxs-lookup"><span data-stu-id="3039b-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="3039b-105">Поставщики типов формирования, то, что известно как **типы, предоставляемые**, которой создаются компилятором F # и основаны на внешнем источнике данных.</span><span class="sxs-lookup"><span data-stu-id="3039b-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="3039b-106">Например поставщик типов F # для SQL можно создавать типы, представляющие таблицы и столбцы в реляционной базе данных.</span><span class="sxs-lookup"><span data-stu-id="3039b-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="3039b-107">На самом деле, именно то, что [SQLProvider](https://fsprojects.github.io/SQLProvider/) и поставщик типа.</span><span class="sxs-lookup"><span data-stu-id="3039b-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="3039b-108">При условии, что типы зависят от входных параметров для поставщика типов.</span><span class="sxs-lookup"><span data-stu-id="3039b-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="3039b-109">Такие входные данные могут быть образец источника данных (например, схема файл JSON), URL-адрес, указывающий непосредственно внешней службы или строку подключения к источнику данных.</span><span class="sxs-lookup"><span data-stu-id="3039b-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="3039b-110">Поставщик типов может также гарантировать развертывание групп типов являются только по запросу. то есть они разворачиваются при программе фактически ссылок на типы.</span><span class="sxs-lookup"><span data-stu-id="3039b-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="3039b-111">Это позволяет применять прямую интеграцию по запросу широкомасштабных информационных пространств (например, рынков оперативных данных) в строго типизированном виде.</span><span class="sxs-lookup"><span data-stu-id="3039b-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="3039b-112">Поставщики типов generative и Стертой</span><span class="sxs-lookup"><span data-stu-id="3039b-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="3039b-113">Поставщики типов могут быть двух видов: Generative и Стертой.</span><span class="sxs-lookup"><span data-stu-id="3039b-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="3039b-114">Generative поставщиков типов создавать типы, которые могут быть написаны как типы .NET в сборку, в которой они создаются.</span><span class="sxs-lookup"><span data-stu-id="3039b-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="3039b-115">Это позволяет им для использования из кода в других сборках.</span><span class="sxs-lookup"><span data-stu-id="3039b-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="3039b-116">Это означает, что типизированным представлением источника данных, обычно необходимо, позволяет представлять с типами .NET.</span><span class="sxs-lookup"><span data-stu-id="3039b-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="3039b-117">Стирания поставщиков типов создавать типы, которые могут быть использованы только в сборке, или они создаются из проекта.</span><span class="sxs-lookup"><span data-stu-id="3039b-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="3039b-118">Типы, эфемерным; то есть они не записываются в сборку и не может использоваться кодом в других сборках.</span><span class="sxs-lookup"><span data-stu-id="3039b-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="3039b-119">Они могут содержать *отложенной* члены, что позволяет использовать типы из пространства потенциально бесконечный сведения.</span><span class="sxs-lookup"><span data-stu-id="3039b-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="3039b-120">Они полезны для использования небольшое подмножество источника большой и взаимосвязанных данных.</span><span class="sxs-lookup"><span data-stu-id="3039b-120">They are useful for using a small subset of a large and interconnected data source.</span></span>

- <span data-ttu-id="3039b-127">[FSharp.Data.GraphQL](https://fsprojects.github.io/FSharp.Data.GraphQL/index.html) содержит **GraphQLProvider**, который предоставляет типы, основанные на сервере GraphQL по указанному URL-адрес.</span><span class="sxs-lookup"><span data-stu-id="3039b-127">[FSharp.Data.GraphQL](https://fsprojects.github.io/FSharp.Data.GraphQL/index.html) contains the **GraphQLProvider**, which provides types based on a GraphQL server specified by URL.</span></span>
