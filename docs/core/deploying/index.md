## <a name="framework-dependent-deployments-fdd"></a><span data-ttu-id="47969-116">Развертывания, зависящие от платформы</span><span class="sxs-lookup"><span data-stu-id="47969-116">Framework-dependent deployments (FDD)</span></span>

## <a name="self-contained-deployments-scd"></a><span data-ttu-id="47969-132">Автономные развертывания</span><span class="sxs-lookup"><span data-stu-id="47969-132">Self-contained deployments (SCD)</span></span>
