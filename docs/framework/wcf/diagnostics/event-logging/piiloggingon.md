# <a name="piiloggingon"></a>PiiLoggingOn
Идентификатор: 107  
  
 Важность: ошибка  
  
 Категория: Tracing

 Данное событие указывает на включение ведения журнала PII. Будут регистрироваться конфиденциальные сведения. В событии указаны имя и ИД процесса.
