# WmiExecMethodFailed
Идентификатор: 125  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## Description  
 Это событие показывает, что запрос ExecMethod инструментария WMI не был обработан.В событии указаны исключение, имя процесса и ИД процесса.
