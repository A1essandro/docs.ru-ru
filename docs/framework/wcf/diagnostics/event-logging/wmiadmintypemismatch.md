# WmiAdminTypeMismatch
Идентификатор: 128  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## Описание  
 Это событие указывает, что имело место несоответствие типов.  Заданное свойство класса CIM не может принимать значение этого типа.  В событии указаны имя и ИД процесса.
