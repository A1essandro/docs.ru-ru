# WasDisconnected
Идентификатор: 162.  
  
 Важность: ошибка  
  
 Категория: SMSvcHost  
  
## Описание  
 Это событие указывает об отсоединении служб активации Windows \(WAS\).В событии указывается HRESULT, имя процесса и идентификатор процесса.
