# <a name="invariantassertionfailed"></a>InvariantAssertionFailed
Идентификатор: 106  
  
 Важность: ошибка  
  
 Категория: Tracing  
  
## <a name="description"></a>Описание  
 Данное событие показывает, что инвариантное утверждение ложно. В событии указаны сообщение, имя и ИД процесса.
