# WmiRegistrationFailed
Идентификатор: 126  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## Описание  
 Это событие показывает, что регистрация поставщика WMI не выполнена.В событии указаны объект WMI, ошибка, имя процесса и идентификатор процесса.
