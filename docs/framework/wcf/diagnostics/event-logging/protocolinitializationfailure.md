# <a name="protocolinitializationfailure"></a>ProtocolInitializationFailure
Идентификатор: 142  
  
 Важность: ошибка  
  
 Категория: TransactionBridge  
  
## <a name="description"></a>Описание:  
 Это событие показывает, что не удалось инициализировать службу протокола WS-Atomic Transaction. В результате функции протокола WS-Atomic Transaction отключены. В событии указаны идентификатор и имя протокола, исключение, имя и идентификатор процесса. .
