# ProtocolInitializationFailure
Идентификатор: 142  
  
 Важность: ошибка  
  
 Категория: TransactionBridge  
  
## Описание  
 Это событие показывает, что не удалось инициализировать службу протокола WS\-Atomic Transaction.В результате функции протокола WS\-Atomic Transaction отключены.В событии указаны ИД и имя протокола, исключение, имя и ИД процесса.
