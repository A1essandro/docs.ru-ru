# WmiGetObjectFailed
Идентификатор: 120  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## Описание  
 Это событие показывает, что запрос GetObject инструментария WMI не был обработан.В событии указаны объект WMI, исключение, имя процесса и идентификатор процесса.
