# ProtocolRecoveryCompleteFailure
Идентификатор: 145  
  
 Важность: ошибка  
  
 Категория: TransactionBridge  
  
## Описание  
 Это событие указывает на то, что службе протокола WS\-Atomic Transaction не удалось завершить запуск и восстановление.В результате функции протокола WS\-Atomic Transaction отключены.В событии указаны идентификатор протокола, имя протокола, исключение, имя и идентификатор процесса.
