# <a name="protocolrecoverycompletefailure"></a>ProtocolRecoveryCompleteFailure
Идентификатор: 145  
  
 Важность: ошибка  
  
 Категория: TransactionBridge  
  
## <a name="description"></a>Описание  
 Это событие указывает на то, что службе протокола WS-Atomic Transaction не удалось завершить запуск и восстановление. В результате функции протокола WS-Atomic Transaction отключены. В событии указаны идентификатор и имя протокола, исключение, имя и идентификатор процесса.
