# TraceCodeRemovedBadFilter
Идентификатор: 114  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## Описание  
 Это событие показывает, что фильтр регистрации неправильных сообщений, который обращался к тексту сообщения, был удален.В событии указаны XPath, имя процесса и идентификатор процесса.
