# <a name="wmicreateinstancefailed"></a>WmiCreateInstanceFailed
Идентификатор: 123  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## <a name="description"></a>Описание  
 Данное событие указывает, что запрос CreateInstance инструментария WMI не был обработан. В событии указаны имя класса, исключение, имя процесса и ИД процесса.
