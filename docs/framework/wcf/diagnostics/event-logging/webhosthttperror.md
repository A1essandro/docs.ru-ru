# WebHostHttpError
Идентификатор: 110  
  
 Важность: ошибка  
  
 Категория: ServiceModel  
  
## Описание  
 Данное событие указывает, что произошло исключение WebHost HTTP.  В событии указаны информация об отправителе, значение HtmlErrorMessage, исключение, имя и идентификатор процесса.
