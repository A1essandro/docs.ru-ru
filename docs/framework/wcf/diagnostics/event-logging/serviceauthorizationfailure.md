# <a name="serviceauthorizationfailure"></a>ServiceAuthorizationFailure
Идентификатор: 168  
  
 Важность: ошибка  
  
 Категория: SecurityAudit  
  
## <a name="description"></a>Описание  
 Это событие указывает на сбой авторизации службы. В событии указаны служба, действие, идентификатор клиента, AuthorizationContext, идентификатор действия и ServiceAuthorizationManager.
