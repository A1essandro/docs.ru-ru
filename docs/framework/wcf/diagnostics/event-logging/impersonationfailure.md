# ImpersonationFailure
Идентификатор: 176  
  
 Важность: ошибка  
  
 Категория: SecurityAudit  
  
## Описание  
 Это событие показывает, что олицетворение выполнить не удалось, и содержит имя метода, учетную запись клиента и идентификатор действия.
