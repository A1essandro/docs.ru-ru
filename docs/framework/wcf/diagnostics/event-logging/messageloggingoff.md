# <a name="messageloggingoff"></a>MessageLoggingOff
Идентификатор: 117  
  
 Важность: ошибка  
  
 Категория: ServiceModel

 Данное событие указывает на отключение ведения журнала сообщений. В событии указаны имя и ИД процесса.
