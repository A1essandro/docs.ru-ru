# ProtocolRecoveryBeginningFailure
Идентификатор: 144  
  
 Важность: ошибка  
  
 Категория: TransactionBridge  
  
## Описание  
 Это событие означает, что в начале восстановления произошел сбой протокола MSDTC WS\-Atomic Transaction.В результате функции протокола WS\-Atomic Transaction отключены.В событии указаны идентификатор протокола, имя протокола, исключение, имя и идентификатор процесса.
