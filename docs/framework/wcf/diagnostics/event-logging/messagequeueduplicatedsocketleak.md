# <a name="messagequeueduplicatedsocketleak"></a>MessageQueueDuplicatedSocketLeak
ИД: 165  
  
 Уровень серьезности: Error  
  
 Категория: SMSvcHost  
  
## <a name="description"></a>Описание  
 Это событие показывает, что произошла ошибка при диспетчеризации дублированного сокета. В данном процессе происходит утечка этого дескриптора. В событии указаны источник, исключение, имя процесса и ИД процесса.
