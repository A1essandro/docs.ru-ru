# StartErrorPublish
Идентификатор: 158  
  
 Важность: ошибка  
  
 Категория: SMSvcHost  
  
## Описание  
 Это событие показывает, что при запуске службы NT произошел сбой.  Не удалось опубликовать конечную точку, ожидающую передачи данных.  В событии указаны имя и ИД процесса.
