# TransportAuthenticationFailure
Идентификатор: 174  
  
 Важность: ошибка  
  
 Категория: SecurityAudit  
  
## Описание  
 Данное событие показывает, что не удалось выполнить проверку подлинности транспорта, и содержит удостоверение клиента и идентификатор действия.
