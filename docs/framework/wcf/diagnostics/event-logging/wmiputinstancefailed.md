# <a name="wmiputinstancefailed"></a>WmiPutInstanceFailed
ИД: 121  
  
 Уровень серьезности: Error  
  
 Категория: ServiceModel  
  
## <a name="description"></a>Описание:  
 Это событие показывает, что запрос PutInstance WMI не был обработан. В событии указаны объект WMI, исключение, имя процесса и идентификатор процесса.
