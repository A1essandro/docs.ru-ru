# Основное взаимодействие: структура каналов
В этом разделе перечислены все исключения, вызываемые структурой каналов [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
## Список исключений  
  
|Код ресурса|Строка ресурса|  
|-----------------|--------------------|  
|ReceiveRequestTimedOut|Время ожидания полученного запроса определенного локального адреса истекло по прошествии заданного времени.Возможно, выделенное для этой операции время было частью более длительного периода ожидания.|  
|ReceiveTimedOutNoLocalAddress|Срок ожидания операции получения истек по прошествии заданного времени.Возможно, выделенное для этой операции время было частью более длительного периода ожидания.|  
|UnableToDemuxChannel|Нет доступного канала для получения сообщения с заданным действием.|
