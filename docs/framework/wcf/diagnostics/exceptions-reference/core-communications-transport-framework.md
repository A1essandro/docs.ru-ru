# Основное взаимодействие: структура транспорта
В этом разделе перечислены все исключения, вызываемые транспортом Framework.  
  
## Список исключений  
  
|Код ресурса|Строка ресурса|  
|-----------------|--------------------|  
|Hosting\_TransportBindingNotFound|Не существует привязки протокола, соответствующей указанному адресу.Привязки протокола настраиваются на уровне сайта в службах IIS или конфигурации служб активации процесса Windows.|
