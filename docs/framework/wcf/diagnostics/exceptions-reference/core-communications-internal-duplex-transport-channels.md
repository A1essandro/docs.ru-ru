# <a name="core-communications-internal-duplex-transport-channels"></a>Основное взаимодействие: внутренние дуплексные транспортные каналы
В этом разделе перечислены все исключения, вызываемые Windows Communication Foundation (WCF) внутренними дуплексными транспортными каналами.  
  
## <a name="exception-list"></a>Список исключений  
  
|Код ресурса|Строка ресурса|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|Для отправки сообщений через составные дуплексные серверные каналы сообщение должно иметь свойство Via или заголовок To.|
