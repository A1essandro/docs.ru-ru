# Подозрительные сообщения в очереди
Имя счетчика: Queued Poison Messages.  
  
## Описание  
 Количество сообщений в этой службе, отмеченных использующим очереди транспортом как подозрительные.  
  
 Дополнительные сведения о том, в каких случаях сообщения отмечаются как подозрительные, см. в разделе [Обработка подозрительных сообщений](http://go.microsoft.com/fwlink/?LinkID=96233).
