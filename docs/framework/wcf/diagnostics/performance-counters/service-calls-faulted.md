# <a name="service-calls-faulted"></a>Служба: сбои вызовов
Имя счетчика: Calls Faulted  
  
## <a name="description"></a>Описание  
 Количество вызовов данной службы, которые возвратили сообщение о сбое. В приложениях Windows Communication Foundation (WCF), методы службы передают сведения об ошибке обработки с помощью сообщений об ошибках SOAP. Сообщения об ошибках SOAP - это типы сообщений, которые включаются в метаданные, связанные с операцией службы, и таким образом создают контракт ошибок, который клиенты могут использовать для повышения надежности и интерактивности своей работы. Поскольку сообщения об ошибках SOAP передаются клиентам в формате XML, они поддерживают возможность взаимодействия.
