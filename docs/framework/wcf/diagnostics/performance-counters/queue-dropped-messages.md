# Удаленные из очереди сообщения
Имя счетчика: Queued Messages Dropped  
  
## Описание  
 Количество сообщений, отброшенных использующим очереди транспортом в этой службе.  
  
 Дополнительные сведения о том, в каких случаях сообщения отбрасываются, см. в разделе [Обработка подозрительных сообщений](http://go.microsoft.com/fwlink/?LinkId=96233).
