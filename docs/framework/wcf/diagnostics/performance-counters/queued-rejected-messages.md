# Количество отклоненных сообщений из очереди
Имя счетчика: Queued Messages Rejected.  
  
## Описание  
 Количество сообщений, отклоненных использующим очереди транспортом в этой службе.  
  
 Дополнительные сведения о том, в каких случаях сообщения отклоняются, см. в разделе [Обработка подозрительных сообщений](http://go.microsoft.com/fwlink/?LinkID=96233).
