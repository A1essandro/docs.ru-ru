# System.ServiceModel.PerformanceCountersFailedDuringUpdate
System.ServiceModel.PerformanceCountersFailedDuringUpdate  
  
## Описание  
 Произошла ошибка при обновлении запроса указанного счетчика производительности.Данный счетчик производительности отключен.
