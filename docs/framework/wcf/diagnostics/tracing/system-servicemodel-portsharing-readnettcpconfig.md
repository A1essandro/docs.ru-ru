# System.ServiceModel.PortSharing.ReadNetTcpConfig
System.ServiceModel.PortSharing.ReadNetTcpConfig  
  
## Описание  
 Данные конфигурации для NetTcpSection были прочитаны с указанными данными: listenBacklog, maxConnectionsPendingDispatch, maxPendingAccepts, receiveTimeout и teredoEnabled.
