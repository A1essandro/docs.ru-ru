# System.ServiceModel.OverridingDuplicateConfigurationKey
System.ServiceModel.OverridingDuplicateConfigurationKey  
  
## Описание  
 Система конфигурации определила повторяющийся ключ в другой области конфигурации и переопределяет на более новое значение.
