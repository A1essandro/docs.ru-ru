# Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
Имя отправителя входящего сообщения не совпадает с именем, полученным из ссылки на конечную точку перечисления.  
  
## Описание  
 Трассируется в случае сбоя проверки безопасности сообщения.  
  
## Устранение неполадок  
 Убедитесь в правильности настройки безопасности транзакции на всех компьютерах, участвующих в транзакции.
