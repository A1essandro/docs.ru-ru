# System.ServiceModel.Channels.PeerNeighborNotFound
Не удалось выполнить операцию, связанную с соседним узлом, поскольку этот соседний узел не обнаружился в списке соседних узлов.  
  
## Описание  
 Эта ошибка происходит при подключении к соседнему узлу или закрытии соседнего узла.
