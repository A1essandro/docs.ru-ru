# System.ServiceModel.Activation.WebHostServiceActivated
System.ServiceModel.Activation.WebHostServiceActivated  
  
## Описание  
 Появляется, когда служба, размещенная в службах IIS или службе активации Windows \(WAS\), полностью активирована и начинает прослушивание сообщений.
