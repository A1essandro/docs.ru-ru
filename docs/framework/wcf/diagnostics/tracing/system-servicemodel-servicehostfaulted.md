# System.ServiceModel.ServiceHostFaulted
System.ServiceModel.ServiceHostFaulted  
  
## Описание  
 Сбой ServiceHost.  
  
 Данная трассировка показывает, что произошел сбой в одном из прослушивателей.Эту трассировку удобно использовать для отладки.
