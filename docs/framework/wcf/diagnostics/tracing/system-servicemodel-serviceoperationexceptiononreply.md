# System.ServiceModel.ServiceOperationExceptionOnReply
System.ServiceModel.ServiceOperationExceptionOnReply  
  
## Описание  
 При ответе на операцию типа "запрос\-ответ" создается исключение.  Эту трассировку удобно использовать для отладки во время разработки.
