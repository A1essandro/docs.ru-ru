# System.Runtime.Serialization.ObjectWithLargeDepth
System.Runtime.Serialization.ObjectWithLargeDepth  
  
## Описание  
 На запись объекта с рекурсивной структурой существуют ограничения при большой глубине вложения.Следует уменьшить глубину вложения объекта.
