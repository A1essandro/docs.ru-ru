# <a name="systemservicemodelchannelsconnectionpoolcloseexception"></a>System.ServiceModel.Channels.ConnectionPoolCloseException
При закрытии подключений в этом пуле подключений возникло исключение.  
  
## <a name="description"></a>Описание  
 Эта трассировка уровня ошибки указывает, что произошла ошибка при закрытии пулов соединений, используемых возможностью пулов подключений Windows Communication Foundation (WCF). Эта ошибка может быть вызвана тем, что не удалось закрыть подключение (или набор подключений) из пула в течение времени ожидания (CloseTimeout). При выдаче этого исключения все оставшиеся незакрытые подключения из этого пула прерываются. Незакрытые подключения из других пулов оставляются.
