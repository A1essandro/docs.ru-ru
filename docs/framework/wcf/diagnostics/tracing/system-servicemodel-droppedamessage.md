# System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## Описание  
 Ответное сообщение было получено, но нет необработанных запросов, ожидающих это сообщение.Сообщение удаляется средой выполнения [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
 Эта трассировка может использоваться для отладки проблемы потери сообщений.
