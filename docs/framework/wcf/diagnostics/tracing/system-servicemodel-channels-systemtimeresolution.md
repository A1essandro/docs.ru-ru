# System.ServiceModel.Channels.SystemTimeResolution
System.ServiceModel.Channels.SystemTimeResolution  
  
## Описание  
 Разрешение таймера операционной системы было определено как *n* тиков, что примерно равно *m* миллисекундам.  Здесь *n* и *m* — числовые значения.
