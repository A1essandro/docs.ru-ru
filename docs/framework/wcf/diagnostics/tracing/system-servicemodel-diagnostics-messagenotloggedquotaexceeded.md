# System.ServiceModel.Diagnostics.MessageNotLoggedQuotaExceeded
System.ServiceModel.Diagnostics.MessageNotLoggedQuotaExceeded  
  
## Описание  
 Сообщение не внесено в журнал, поскольку его размер превышает заданную квоту.
