# System.ServiceModel.Channels.PeerNeighborManagerOnline
Узел PeerNode находится в сети, что указывает на наличие одного или нескольких установленных подключений к соседним узлам.  
  
## Описание  
 Эта трассировка возникает в конце успешной попытки подключения к соседнему узлу.
