## <a name="description"></a>Описание  
 Это событие создается, если входящий запрос соответствует службе WCF, заданной в маршруте Asp.Net, с адресом %1.  
  
## <a name="message"></a>Сообщение  
 Входящий запрос соответствует службе WCF, заданной в маршруте Asp.Net, с адресом %1.
