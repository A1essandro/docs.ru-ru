# 3826 - RoutingServiceUsingExistingTransaction
## Свойства  
  
|||  
|-|-|  
|Идентификатор|3826|  
|Ключевые слова|Службы маршрутизации|  
|Уровень|Verbose|  
|Канал|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Описание  
 Это событие создается, когда служба маршрутизации будет отправлять сообщения с использованием существующей транзакции.  
  
## Сообщение  
 Служба маршрутизации будет отправлять сообщения с помощью существующей транзакции «%1».
