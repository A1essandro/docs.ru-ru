# 4804 - DiscoveryMessageReceivedAfterOperationCompleted
## Свойства  
  
|||  
|-|-|  
|Идентификатор|4804|  
|Ключевые слова|Обнаружение|  
|Уровень|Сведения|  
|Канал|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Описание  
 Это событие отправляется при получении сообщения обнаружения после завершения операции.  
  
## Сообщение  
 Сообщение %1 с messageId\=«%2» удалено клиентом DiscoveryClient, поскольку выполнена соответствующая операция %3.
