# 3415 - WebSocketConnectionRequestSendStart
## Свойства  
  
|||

|Ключевые слова|HTTP|  
|Уровень|Verbose|

## Описание  
 Это событие создается, когда запущен запрос на отправку соединения WebSocket.  
  
## Сообщение  
 Запрос соединения WebSocket к «%1» отправляет сигнал к началу.
