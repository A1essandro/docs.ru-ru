# 3824 - RoutingServiceCompletingReceiveContext
## Свойства  
  
|||  
|-|-|  
|Идентификатор|3824|  
|Ключевые слова|Службы маршрутизации|  
|Уровень|Verbose|  
|Канал|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Описание  
 Это событие создается, когда служба маршрутизации вызывает ReceiveContext.Complete для сообщения с указанным идентификатором.  
  
## Сообщение  
 Служба маршрутизации вызывает ReceiveContext.Complete для сообщения с идентификатором: «%1».
