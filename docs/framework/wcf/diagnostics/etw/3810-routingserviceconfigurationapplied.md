|-|-|  
|Идентификатор|3810|  
|Ключевые слова|Службы маршрутизации|  
|Уровень|Сведения|

## <a name="description"></a>Описание  
 Это событие создается при применении конфигурации маршрутизации к службе маршрутизации.
