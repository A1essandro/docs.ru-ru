# 3430 - WebSocketUseVersionFromClientWebSocketFactory
## Свойства  
  
|||  
|-|-|  
|Идентификатор|3430|  
|Ключевые слова|HTTP|  
|Уровень|Verbose|  
|Канал|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Описание  
 Это событие создается при использовании WebSocketVersion из фабрики клиентского соединения WebSocket.  
  
## Сообщение  
 Используется версия WebSocketVersion из фабрики клиентского соединения WebSocket типа «%1».
