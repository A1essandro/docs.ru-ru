# 4808 - DiscoveryMessageWithNoContent

## Описание  
 Это событие формируется при удалении сообщения обнаружения из\-за неверного адреса ReplyTo.  
  
## Сообщение  
 Сообщение %1 удалено, поскольку отсутствует его содержимое.
