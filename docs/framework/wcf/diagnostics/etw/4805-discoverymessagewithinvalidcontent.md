# 4805 - DiscoveryMessageWithInvalidContent
## Свойства  
  
|||  
|-|-|  
|Идентификатор|4805|  
|Ключевые слова|Обнаружение|  
|Уровень|Предупреждение|  
|Канал|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Описание  
 Это событие создается, когда сообщение обнаружения содержит недопустимое содержимое.  
  
## Сообщение  
 Сообщение %1 с messageId\="%2" удалено из\-за недопустимого содержимого.
