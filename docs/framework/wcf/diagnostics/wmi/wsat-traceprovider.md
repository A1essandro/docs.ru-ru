# WSAT_TraceProvider
WSAT\_TraceProvider  
  
## Синтаксис  
  
```  
class WSAT_TraceProvider : EventTrace  
{  
};  
```  
  
## Методы  
 Класс WSAT\_TraceProvider наследуется от класса EventTrace, но не определяет дополнительных методов и свойств.  
  
## Требования  
  
|MOF|Объявлено в файле Servicemodel.mof.|  
|---------|-----------------------------------------|  
|Пространство имен|Определено в root\\ServiceModel.|
