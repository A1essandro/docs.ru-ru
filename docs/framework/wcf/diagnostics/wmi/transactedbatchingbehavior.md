# TransactedBatchingBehavior
TransactedBatchingBehavior  
  
## Синтаксис  
  
```  
class TransactedBatchingBehavior : Behavior  
{  
  sint32 MaxBatchSize;  
};  
```  
  
## Методы  
 Класс TransactedBatchingBehavior не определяет никаких методов.  
  
## Свойства  
 Класс TransactedBatchingBehavior имеет следующее свойство.  
  
### MaxBatchSize  
 Тип данных: sint32  
  
 Тип доступа: только для чтения  
  
 Максимальный размер пакета для пакетирования с поддержкой транзакций.
