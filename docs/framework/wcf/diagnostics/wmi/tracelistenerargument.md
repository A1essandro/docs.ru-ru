# TraceListenerArgument
TraceListenerArgument  
  
## Синтаксис  
  
```  
class TraceListenerArgument  
{  
  string Name;  
  string Value;  
};  
```  
  
## Методы  
 Класс TraceListenerArgument не определяет никаких методов.  
  
## Свойства  
 Класс TraceListenerArgument имеет следующие свойства.  
  
### Имя  
 Тип данных: string  
  
 Тип доступа: только для чтения  
  
 Имя аргумента.  
  
### Значение  
 Тип данных: string  
  
 Тип доступа: только для чтения  
  
 Значение аргумента.
