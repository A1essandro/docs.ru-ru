# Класс Behavior
## Синтаксис  
  
```  
class Behavior  
{  
  string Type;  
};  
```  
  
## Методы  
 Класс Behavior не определяет никаких методов.  
  
## Свойства  
 Класс Behavior имеет следующее свойство.  
  
### Type  
 Тип данных: string  
  
 Тип доступа: только для чтения  
  
 Тип поведения.  
  
## Требования  
  
|MOF|Объявлено в файле Servicemodel.mof.|  
|---------|-----------------------------------------|  
|Пространство имен|Определено в root\\ServiceModel.|
