# WSAT_TraceEvent
WSAT\_TraceEvent  
  
## Синтаксис  
  
```  
class WSAT_TraceEvent : WSAT_TraceProvider  
{  
};  
```  
  
## Методы  
 Класс WSAT\_TraceEvent наследуется от класса WSAT\_TraceProvider, но не определяет дополнительных методов и свойств.  
  
## Требования  
  
|MOF|Объявлено в файле Servicemodel.mof.|  
|---------|-----------------------------------------|  
|Пространство имен|Определено в root\\ServiceModel.|
