# PeerResolverBindingElement
PeerResolverBindingElement  
  
## Синтаксис  
  
```  
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## Методы  
 Класс PeerResolverBindingElement не определяет никакие методы.  
  
## Свойства  
 Класс PeerResolverBindingElement имеет следующее свойство:  
  
### ReferralPolicy  
 Тип данных: string  
  
 Тип доступа: только для чтения  
  
 Определяет, каким образом отсылки распределяются между одноранговыми узлами.

|MOF|Объявлено в файле Servicemodel.mof.|  
|---------|-----------------------------------------|  
|Пространство имен|Определено в root\\ServiceModel.|
