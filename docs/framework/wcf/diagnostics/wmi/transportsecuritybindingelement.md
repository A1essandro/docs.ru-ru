# TransportSecurityBindingElement
TransportSecurityBindingElement  
  
## Синтаксис  
  
```  
class TransportSecurityBindingElement : SecurityBindingElement  
{  
};  
```  
  
## Методы  
 Класс TransportSecurityBindingElement наследуется от класса SecurityBindingElement, но не определяет дополнительные методы и свойства.
