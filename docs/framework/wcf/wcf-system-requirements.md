# Требования WCF к системе
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)] является инфраструктурой связи, используемой для создания распределенных приложений.  Ниже перечисляются требования, необходимые для работы [!INCLUDE[indigo2](../../../includes/indigo2-md.md)].  
  
## Требования к системе  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] по умолчанию устанавливается в составе [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] также может устанавливаться в системы [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)] или [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Примечание** Функциональность очереди сообщений \(MSMQ\) [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] поддерживается только в [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] и [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
