# Отзывы и предложения и сообщество
Мы будем признательны за комментарии и вопросы относительно документации по [!INCLUDE[indigo1](../../../includes/indigo1-md.md)] или по [!INCLUDE[infocard](../../../includes/infocard-md.md)].  В нижней части каждой страницы в пакет SDK имеется ссылка «Отправить комментарии к этому разделу в корпорацию Майкрософт». Щелкните ссылку «комментарии», чтобы отправить свои комментарии и предложения по определенному разделу.  
  
 Узнать дополнительные сведения и поделиться собственными методами с другими пользователями [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] можно на [форуме Windows Communication Foundation MSDN](http://go.microsoft.com/fwlink/?LinkId=186463).
