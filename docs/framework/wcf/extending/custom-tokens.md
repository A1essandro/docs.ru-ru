# Пользовательские маркеры
Хотя в [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] предусмотрена собственная поддержка сертификатов X.509, маркеров контекста безопасности, билетов Kerberos и маркеров имени пользователя в качестве учетных данных, также имеется возможность использовать пользовательские учетные данные.  
  
## В этом подразделе  
 [Как создавать пользовательский маркер](../../../../docs/framework/wcf/extending/how-to-create-a-custom-token.md)  
 Создание пользовательского маркера безопасности с помощью класса <xref:System.IdentityModel.Tokens.SecurityToken> и его интеграция с поставщиком пользовательских маркеров безопасности и структурой проверки подлинности.
