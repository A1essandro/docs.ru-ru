# <a name="custom-tokens"></a>Пользовательские маркеры
Несмотря на то, что Windows Communication Foundation (WCF) поддерживает сертификаты X.509, маркеры контекста безопасности, билетов Kerberos и маркеров имени пользователя в качестве учетных данных, является достаточно гибким, чтобы можно было использовать пользовательские учетные данные.

 Создание пользовательского маркера безопасности с помощью класса <xref:System.IdentityModel.Tokens.SecurityToken> и его интеграция с поставщиком пользовательских маркеров безопасности и структурой проверки подлинности.
