# <a name="wcf-and-net-framework-client-profile"></a>WCF и профиль клиента .NET Framework
[!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] - это облегченная версия полной версии платформы .Net Framework, созданная для клиентов, которым не требуются все возможности платформы. Платформа клиента поддерживает не все функции Windows Communication Foundation.  
  
## <a name="wcf-features-supported-by-the-net-framework-client-profile"></a>Функции WCF, поддерживаемые .Net Framework Client Profile  
 [!INCLUDE[arrowhead](../../../includes/arrowhead-md.md)] поддерживает следующие функции Windows Communication Foundation.  
  
-   Поддерживаются все функции WCF, кроме Cardspace и размещения на веб-сервере.  
  
-   Поддерживаются каналы TCP/IP удаленного взаимодействия.  
  
-   Не поддерживаются Asmx (веб-службы).
