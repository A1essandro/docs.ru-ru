# Настройка WorkflowServiceHost
В данном разделе содержатся краткие инструкции по настройке различных функций класса <xref:System.ServiceModel.Activities.WorkflowServiceHost> в конфигурации.  Здесь описывается настройка, характерная для службы рабочего процесса. Все остальные настройки выполняются аналогично настройке обычной службы [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].  
  
## В этом подразделе

 Описывает настройку отслеживания рабочего процесса для службы рабочего процесса при помощи <xref:System.ServiceModel.Activities.WorkflowServiceHost>.  
  
 [Как настроить неактивное поведение с помощью WorkflowServiceHost](../../../../docs/framework/wcf/feature-details/how-to-configure-idle-behavior-with-workflowservicehost.md)
