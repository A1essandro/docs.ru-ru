# <a name="configuring-workflowservicehost"></a>Настройка WorkflowServiceHost
В этом разделе описывается ряд коротких «как» разделы, в которых кратко описывается настройка различных характеристик <xref:System.ServiceModel.Activities.WorkflowServiceHost> в конфигурации. Здесь описывается настройка, характерная для службы рабочего процесса. Все остальные настройки выполняются аналогично настройке обычной службы [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].
