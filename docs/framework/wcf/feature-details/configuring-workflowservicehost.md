# <a name="configuring-workflowservicehost"></a>Настройка WorkflowServiceHost

Здесь описывается настройка, характерная для службы рабочего процесса. Все остальные настройки выполняются аналогично настройке обычной службы [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].
