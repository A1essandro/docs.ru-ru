 <span data-ttu-id="05129-109">Данная функция доступна только в службах [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)], размещенных в службах IIS.</span><span class="sxs-lookup"><span data-stu-id="05129-109">This feature is only available to [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] services that are hosted under IIS.</span></span> <span data-ttu-id="05129-110">Эта функция отключена по умолчанию.</span><span class="sxs-lookup"><span data-stu-id="05129-110">This feature is not enabled by default.</span></span> <span data-ttu-id="05129-111">Чтобы включить, то необходимо добавить `multipleSiteBindingsEnabled` атрибут <`serviceHostingEnvironment`> элемент в файл Web.config файл и присвойте ему значение `true`, как показано в следующем примере.</span><span class="sxs-lookup"><span data-stu-id="05129-111">To enable it you must add the `multipleSiteBindingsEnabled` attribute to the <`serviceHostingEnvironment`> element in your Web.config file and set it to `true`, as shown in the following example.</span></span>
