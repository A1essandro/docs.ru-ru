-   <span data-ttu-id="10018-117">Операции при отсутствии подключения к сети.</span><span class="sxs-lookup"><span data-stu-id="10018-117">Disconnected operations.</span></span> <span data-ttu-id="10018-118">Операции отправки, получения и обработки могут прерваться во время передачи данных по сетям с высокой задержкой или сетям с ограниченным доступом, например в случае с мобильными устройствами.</span><span class="sxs-lookup"><span data-stu-id="10018-118">Sending, receiving, and processing operations can become disconnected when communicating over high-latency networks or limited-availability networks, such as in the case of mobile devices.</span></span> <span data-ttu-id="10018-119">В очереди выполнение этих операций продолжится даже при отключении конечных точек.</span><span class="sxs-lookup"><span data-stu-id="10018-119">Queues allow these operations to continue, even when the endpoints are disconnected.</span></span> <span data-ttu-id="10018-120">При повторном подключении сообщения из очереди пересылаются в принимающее приложение.</span><span class="sxs-lookup"><span data-stu-id="10018-120">When the connection is reestablished, the queue forwards messages to the receiving application.</span></span>  
  
 <span data-ttu-id="10018-121">Чтобы использовать функцию очередей в приложении [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)], следует применить одну из стандартных привязок или создать пользовательскую привязку, если ни одна из стандартных привязок не удовлетворяет необходимым требованиям.</span><span class="sxs-lookup"><span data-stu-id="10018-121">To use the queues feature in a [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] application, you can use one of the standard bindings, or you can create a custom binding if one of the standard bindings does not satisfy your requirements.</span></span> <span data-ttu-id="10018-122">Дополнительные сведения о соответствующих стандартных привязок и выберите один см [как: обмена сообщениями с конечными точками WCF и приложениями очереди сообщений](../../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).</span><span class="sxs-lookup"><span data-stu-id="10018-122">For more information about relevant standard bindings and how to choose one, see [How to: Exchange Messages with WCF Endpoints and Message Queuing Applications](../../../../docs/framework/wcf/feature-details/how-to-exchange-messages-with-wcf-endpoints-and-message-queuing-applications.md).</span></span> <span data-ttu-id="10018-123">Дополнительные сведения о создании настраиваемых привязок см. в разделе [пользовательские привязки](../../../../docs/framework/wcf/extending/custom-bindings.md).</span><span class="sxs-lookup"><span data-stu-id="10018-123">For more information about creating custom bindings, see [Custom Bindings](../../../../docs/framework/wcf/extending/custom-bindings.md).</span></span>
