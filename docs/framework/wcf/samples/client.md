# <a name="client"></a>"Клиент";
Этот раздел содержит образцы, демонстрирующие клиентских приложений Windows Communication Foundation (WCF).

 [Взаимодействие с клиентами](../../../../docs/framework/wcf/samples/client-interoperability.md)  
 Показывает взаимодействие между [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] и другими технологиями.  
  
 [Заголовки адресов](../../../../docs/framework/wcf/samples/address-headers.md)  
 Демонстрируется передача клиентами ссылочных параметров службе с помощью [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)].

 Показывает, как клиентское приложение может создать канал с помощью <xref:System.ServiceModel.ChannelFactory> вместо созданного клиента.  
  
 [Ожидаемые исключения](../../../../docs/framework/wcf/samples/expected-exceptions.md)  
 Демонстрируется перехват ожидаемых исключений при использовании типизированного клиента.

 Демонстрируется реализация клиента, динамически извлекающего метаданные из службы для выбора конечной точки для взаимодействия.  
  
 [Предотвращение проблем при использовании операторов](../../../../docs/framework/wcf/samples/avoiding-problems-with-the-using-statement.md)  
 Показывает, как не следует использовать оператор «using» в C# для автоматической очистки ресурсов, если используется типизированный клиент.  
  
 [Типизированный клиент](../../../../docs/framework/wcf/samples/typed-client.md)  
 Показано, как для получения информации от типизированного клиента, созданные [ServiceModel Metadata Utility Tool (Svcutil.exe)](../../../../docs/framework/wcf/servicemodel-metadata-utility-tool-svcutil-exe.md).
