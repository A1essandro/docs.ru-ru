# <a name="system-provided-bindings"></a><span data-ttu-id="ea6f1-103">Привязки, предоставляемые системой</span><span class="sxs-lookup"><span data-stu-id="ea6f1-103">System-provided bindings</span></span>

<span data-ttu-id="ea6f1-104">Привязки указывают, какой механизм связи использовать при взаимодействии с конечной точкой, а также способ подключения к конечной точке.</span><span class="sxs-lookup"><span data-stu-id="ea6f1-104">Bindings specify the communication mechanism to use when talking to an endpoint and indicate how to connect to an endpoint.</span></span> <span data-ttu-id="ea6f1-105">Привязка содержит следующие элементы.</span><span class="sxs-lookup"><span data-stu-id="ea6f1-105">A binding contains the following elements:</span></span>

- <span data-ttu-id="ea6f1-106">Элемент стека протоколов определяет параметры безопасности, надежности и перемещения контекста, используемые для отправляемых конечной точке сообщений.</span><span class="sxs-lookup"><span data-stu-id="ea6f1-106">The protocol stack determines the security, reliability, and context flow settings to use for messages that are sent to the endpoint.</span></span>

- <span data-ttu-id="ea6f1-107">Элемент транспорта определяет основной транспортный протокол, используемый при отправке сообщений конечной точке, например TCP или HTTP.</span><span class="sxs-lookup"><span data-stu-id="ea6f1-107">The transport determines the underlying transport protocol to use when sending messages to the endpoint, for example, TCP or HTTP.</span></span>

- <span data-ttu-id="ea6f1-108">Способ кодирования определяет кодировку, используемую для сообщений, отправляемых в конечную точку.</span><span class="sxs-lookup"><span data-stu-id="ea6f1-108">The encoding determines the wire encoding to use for messages that are sent to the endpoint.</span></span> <span data-ttu-id="ea6f1-109">Это может быть текст, двоичная кодировка, механизм оптимизации передачи сообщений (MTOM).</span><span class="sxs-lookup"><span data-stu-id="ea6f1-109">For example, text/XML, binary, or Message Transmission Optimization Mechanism (MTOM).</span></span>
