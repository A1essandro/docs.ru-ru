## <a name="what-is-the-net-framework"></a><span data-ttu-id="1f027-105">Что такое платформа .NET Framework</span><span class="sxs-lookup"><span data-stu-id="1f027-105">What is the .NET Framework?</span></span>

<span data-ttu-id="1f027-106">Платформа .NET Framework — это управляемая среда выполнения, предоставляющая разнообразные службы запускаемым в ней приложениям.</span><span class="sxs-lookup"><span data-stu-id="1f027-106">The .NET Framework is a managed execution environment that provides a variety of services to its running apps.</span></span> <span data-ttu-id="1f027-107">Она состоит из двух основных компонентов: среды CLR — механизма, управляющего выполняющимися приложениями, и библиотеки классов .NET Framework — библиотеки проверенного кода, предназначенного для повторного использования, который разработчики могут вызывать из своих приложений.</span><span class="sxs-lookup"><span data-stu-id="1f027-107">It consists of two major components: the common language runtime (CLR), which is the execution engine that handles running apps, and the .NET Framework Class Library, which provides a library of tested, reusable code that developers can call from their own apps.</span></span> <span data-ttu-id="1f027-108">Вот какие службы предоставляет .NET Framework выполняемым в ней приложениям:</span><span class="sxs-lookup"><span data-stu-id="1f027-108">The services that the .NET Framework provides to running apps include the following:</span></span>

- <span data-ttu-id="1f027-112">Общая система типов.</span><span class="sxs-lookup"><span data-stu-id="1f027-112">A common type system.</span></span> <span data-ttu-id="1f027-113">В традиционных языках программирования базовые типы определяются компилятором, что осложняет взаимодействие между языками.</span><span class="sxs-lookup"><span data-stu-id="1f027-113">In traditional programming languages, basic types are defined by the compiler, which complicates cross-language interoperability.</span></span> <span data-ttu-id="1f027-114">В .NET Framework базовые типы определяются системой типов .NET Framework, при этом для всех языков .NET Framework используются одни и те же базовые типы.</span><span class="sxs-lookup"><span data-stu-id="1f027-114">In the .NET Framework, basic types are defined by the .NET Framework type system and are common to all languages that target the .NET Framework.</span></span>

- <span data-ttu-id="1f027-119">Взаимодействие языков.</span><span class="sxs-lookup"><span data-stu-id="1f027-119">Language interoperability.</span></span> <span data-ttu-id="1f027-120">Языковые компиляторы, ориентированные на .NET Framework, выдают промежуточный код, называемый языком CIL (Common Intermediate Language), который в свою очередь компилируется во время выполнения средой CLR.</span><span class="sxs-lookup"><span data-stu-id="1f027-120">Language compilers that target the .NET Framework emit an intermediate code named Common Intermediate Language (CIL), which, in turn, is compiled at runtime by the common language runtime.</span></span> <span data-ttu-id="1f027-121">С помощью этой функции подпрограммы, написанные на одном языке, доступны в других языках, поэтому разработчики могут создавать приложения на предпочитаемых языках.</span><span class="sxs-lookup"><span data-stu-id="1f027-121">With this feature, routines written in one language are accessible to other languages, and programmers focus on creating apps in their preferred languages.</span></span>

- <span data-ttu-id="1f027-128">Настройка для различных версий.</span><span class="sxs-lookup"><span data-stu-id="1f027-128">Multitargeting.</span></span> <span data-ttu-id="1f027-129">Используя [.NET Standard](~/docs/standard/net-standard.md), разработчики создают сборки, которые работают на нескольких платформах .NET Framework, например Windows 7, Windows 8, Windows 8.1, Windows 10, Windows Phone и Xbox 360.</span><span class="sxs-lookup"><span data-stu-id="1f027-129">By targeting [.NET Standard](~/docs/standard/net-standard.md), developers create assemblies that work on multiple .NET Framework platforms, such as Windows 7, Windows 8, Windows 8.1, Windows 10, Windows Phone, and Xbox 360.</span></span>
