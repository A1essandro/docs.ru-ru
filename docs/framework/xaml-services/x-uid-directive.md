 <span data-ttu-id="ec3be-111">`x:Uid` является дискретным из `x:Name` оба из-за указанного сценария локализации XAML, а идентификаторы, используемые для локализации не имеют зависимости от особенностей модели программирования `x:Name`.</span><span class="sxs-lookup"><span data-stu-id="ec3be-111">`x:Uid` is discrete from `x:Name` both because of the stated XAML localization scenario and so that identifiers that are used for localization have no dependencies on the programming model implications of `x:Name`.</span></span> <span data-ttu-id="ec3be-112">Кроме того `x:Name` регулируется область имен XAML;
