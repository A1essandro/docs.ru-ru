 <span data-ttu-id="5dfa1-193">В следующем примере кода задается сертификат X.509 и пользовательский тип проверки в элементе `<authentication>`.</span><span class="sxs-lookup"><span data-stu-id="5dfa1-193">The following code specifies an X.509 certificate and a custom validation type in the `<authentication>` element.</span></span>
