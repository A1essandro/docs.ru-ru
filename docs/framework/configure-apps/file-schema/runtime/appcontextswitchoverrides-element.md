## <a name="example"></a><span data-ttu-id="8495f-243">Пример</span><span class="sxs-lookup"><span data-stu-id="8495f-243">Example</span></span>  
 <span data-ttu-id="8495f-244">В следующем примере используется `AppContextSwitchOverrides` для определения переключателя совместимости одного приложения, `Switch.System.Globalization.NoAsyncCurrentCulture`, который препятствует проходящих через потоков в асинхронных вызовах методов языка и региональных параметров.</span><span class="sxs-lookup"><span data-stu-id="8495f-244">The following example uses the `AppContextSwitchOverrides` element to define a single application  compatibility switch, `Switch.System.Globalization.NoAsyncCurrentCulture`, that prevents culture from flowing across threads in asynchronous method calls.</span></span>

   </runtime>
