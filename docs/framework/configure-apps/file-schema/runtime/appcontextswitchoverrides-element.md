 <span data-ttu-id="45156-263">В следующем примере используется `AppContextSwitchOverrides` для определения переключателя совместимости одного приложения, `Switch.System.Globalization.NoAsyncCurrentCulture`, который препятствует проходящих через потоков в асинхронных вызовах методов языка и региональных параметров.</span><span class="sxs-lookup"><span data-stu-id="45156-263">The following example uses the `AppContextSwitchOverrides` element to define a single application  compatibility switch, `Switch.System.Globalization.NoAsyncCurrentCulture`, that prevents culture from flowing across threads in asynchronous method calls.</span></span>
