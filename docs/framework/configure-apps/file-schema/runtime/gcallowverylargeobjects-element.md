# Элемент &lt;gcAllowVeryLargeObjects&gt;
На 64 разрядных платформах, позволяет использовать массивы, размер которых превышает 2 гигабайта \(GB\).  
  
## Синтаксис  
  
```  
<gcAllowVeryLargeObjects    
   enabled="true|false" />  
```  
  
## Атрибуты и элементы  
 В следующих разделах описаны атрибуты, дочерние и родительские элементы.  
  
### Атрибуты  
  
|Атрибут|Описание|  
|-------------|--------------|  
|`enabled`|Обязательный атрибут.<br /><br /> Указывает, включены ли массивы, размер которых превышает 2 ГБ, в 64 разрядных платформах.|  
  
## Атрибут enabled  
  
|Значение|Описание|  
|--------------|--------------|  
|`false`|Массивы, размер которых превышает 2 ГБ, отключены.  Это значение по умолчанию.|  
|`true`|Массивы, размер которых превышает 2 ГБ, включены на 64 разрядных платформах.|  
  
### Дочерние элементы  
 Нет.  
  
### Родительские элементы  
  
|Элемент|Описание|  
|-------------|--------------|  
|`configuration`|Корневой элемент в любом файле конфигурации, используемом средой CLR и приложениями платформы .NET Framework.|  
|`runtime`|Содержит сведения о параметрах инициализации среды выполнения.|  
  
## Заметки  
 Использование этого элемента в файле конфигурации приложения включает массивы, размер которых превышает 2 ГБ в размере, но не изменяет другие ограничения на размер объекта или размера массива:  
  
-   Максимальное количество элементов в массиве \- <xref:System.UInt32.MaxValue?displayProperty=fullName>.  
  
-   Максимальный индекс в любом измерении \- 2,147,483,591 \(0x7FFFFFC7\) байтовых массивов и массивов однобайтовых структур и 2,146,435,071 \(0X7FEFFFFF\) для других типов.  
  
-   Максимальный размер для строк и других объектов, не являющихся массивами, не изменяется.  
  
> [!CAUTION]
>  Прежде чем включить эту функцию, убедитесь, что приложение не включает небезопасный код, предполагающий, что все массивы меньше 2 ГБ.  Например, небезопасный код, который использует массивы в качестве буфера, может быть восприимчивым к переполнениям буфера, если записывается на допущении, что массивы не превысят 2 ГБ.  
  
## Пример  
 Следующий пример показывает, как активировать эту особенность предыдущей версии для приложения.  
  
```  
<configuration>  
  <runtime>  
    <gcAllowVeryLargeObjects enabled="true" />  
  </runtime>  
</configuration>  
```
