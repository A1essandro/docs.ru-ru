# Keyboard Input in a Windows Forms Application
Windows Forms содержит стандартные события клавиатуры, которые позволяют реагировать на нажатия определенных клавиш, а также предоставляет способы для перехвата, изменения и получения нажатий клавиш на уровне приложения, формы, и элемента управления.

 Описывается, как сообщения клавиатуры обрабатываются и преобразуются в события клавиатуры.  
  
 [Using Keyboard Events](../../../docs/framework/winforms/using-keyboard-events.md)  
 Предоставляется информация о типах событий клавиатуры и сведениях, получаемых обработчиками событий клавиатуры.

 Предоставляется пример кода, в котором показано, как изменять значения клавиш, прежде чем они достигнут элемента управления.  
  
 [How to: Determine Which Modifier Key Was Pressed](../../../docs/framework/winforms/how-to-determine-which-modifier-key-was-pressed.md)  
 Показано, как узнать, была ли нажата клавиша SHIFT, ALT или CTRL в сочетании с другой клавишей.  
  
 [How to: Handle Keyboard Input at the Form Level](../../../docs/framework/winforms/how-to-handle-keyboard-input-at-the-form-level.md)  
 Предоставляется пример кода, который показывает, как перехватить клавиши, прежде чем они достигнут элемента управления.
