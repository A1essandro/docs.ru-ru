# Практическое руководство. Создание элемента управления с простой привязкой в форме Windows Forms
Используя *простую привязку*, можно отображать в элементе управления один элемент данных \(например, значение столбца из таблицы набора данных\).  Можно выполнить простую привязку любого свойства элемента управления к значению данных.  
  
> [!NOTE]
>  Отображаемые диалоговые окна и команды меню могут отличаться от описанных в справке в зависимости от текущих настроек или выпуска.  Чтобы изменить параметры, в меню **Сервис** выберите команду **Импорт и экспорт параметров**.  Дополнительные сведения см. в разделе [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/ru-ru/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### Чтобы выполнить простую привязку элемента управления  
  
1.  Подключение к источнику данных.  Дополнительные сведения см. в разделе [Подключение к источнику данных](../../../docs/framework/data/adonet/connecting-to-a-data-source.md).  
  
2.  В форме выделите элемент управления и отобразите окно **Свойства**.  
  
3.  Разверните свойство **\(DataBindings\)**.  
  
     Свойства, привязка к которым выполняется чаще всего, отображаются под свойством **\(DataBindings\)**.  Например, для большинства элементов управления чаще всего выполняется привязка к свойству **Text**.  
  
4.  Если свойство, к которому необходимо выполнить привязку, не является одним из часто используемых свойств, нажмите кнопку **с многоточием** \(![Снимок экрана VisualStudioEllipsesButton](../../../docs/framework/winforms/media/vbellipsesbutton.png "vbEllipsesButton")\) в поле **\(Дополнительно\)** для отображения диалогового окна **Форматирование и сложная привязка**, содержащего полный список свойств этого элемента управления.  
  
5.  Выберите свойство, привязку к которому следует выполнить, и щелкните по стрелке раскрывающегося списка **Привязка**.  
  
     Отображается список доступных источников данных.  
  
6.  Разворачивайте источник данных, к которому следует выполнить привязку, до нахождения требуемого единого элемента данных.  Например, если необходимо выполнить привязку к значению столбца в таблице набора данных, разверните имя набора данных, а затем разверните имя таблицы для отображения имен столбцов.  
  
7.  Чтобы выполнить привязку к элементу, щелкните его имя.  
  
8.  После работы в диалоговом окне **Форматирование и сложная привязка** щелкните **ОК**, чтобы вернуться в окно **Свойства**.  
  
9. Чтобы выполнить привязку дополнительных свойств элемента управления, повторите шаги 3 – 7.  
  
    > [!NOTE]
    >  Поскольку элементы управления с простой привязкой отображают только один элемент данных, в формы Windows Form с элементами управления с простой привязкой часто добавляют логику переходов.
