# Сортировка данных в элементе управления DataGridView в Windows Forms
По умолчанию пользователи могут сортировать данные в элементе управления `DataGridView`, щелкнув заголовок столбца текстового поля.  Свойство `SortMode` для определенных столбцов можно изменить так, чтобы пользователи могли сортировать по другим типам столбцов, если это имеет смысл.  Данные можно сортировать программным образом по любому столбцу или нескольким столбцам.
