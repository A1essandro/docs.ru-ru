# Общие сведения об элементе управления ToolStripStatusLabel
<xref:System.Windows.Forms.ToolStripStatusLabel> является меткой для <xref:System.Windows.Forms.StatusStrip>.  <xref:System.Windows.Forms.ToolStripStatusLabel> предоставляет область отображения для текста и изображений, не содержащую активных точек, аналогично <xref:System.Windows.Forms.Label> или <xref:System.Windows.Forms.ToolStripLabel>.  <xref:System.Windows.Forms.ToolStripStatusLabel> размещается в <xref:System.Windows.Forms.StatusStrip>.  
  
### Важные члены ToolStripStatusLabel  
  
|Имя|Описание|  
|---------|--------------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|Получает или задает значение, указывающее, заполняет ли <xref:System.Windows.Forms.ToolStripStatusLabel> автоматически доступное пространство в <xref:System.Windows.Forms.StatusStrip> при изменении размера формы.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|Получает или задает значение, указывающее, с каких сторон <xref:System.Windows.Forms.ToolStripStatusLabel> отображаются границы.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|Получает или задает стиль объекта <xref:System.Windows.Forms.ToolStripStatusLabel>.|
