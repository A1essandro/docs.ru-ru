# Класс FileDialog
Класс Windows Forms <xref:System.Windows.Forms.FileDialog> является общим базовым классом для компонентов <xref:System.Windows.Forms.OpenFileDialog> и <xref:System.Windows.Forms.SaveFileDialog>.  В класс <xref:System.Windows.Forms.FileDialog> можно внести изменения, влияющие на внешний вид и поведение этих диалоговых окон в зависимости от используемой версии ОС Windows.
