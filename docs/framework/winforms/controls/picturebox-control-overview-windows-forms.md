# <a name="picturebox-control-overview-windows-forms"></a>Общие сведения об элементе управления PictureBox (Windows Forms)
Windows Forms <xref:System.Windows.Forms.PictureBox> элемент управления используется для отображения графических объектов в формат растрового изображения, GIF, JPEG, метафайла или значок.  
  
## <a name="key-properties-and-methods"></a>Ключевые свойства и методы  
 Рисунок, отображаемый определяется <xref:System.Windows.Forms.PictureBox.Image%2A> свойство, которое можно задать во время выполнения или во время разработки. Можно также указать изображение, задав <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> свойство и затем загрузить изображение синхронно с помощью <xref:System.Windows.Forms.PictureBox.Load%2A> метода или асинхронном режиме с помощью <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> метод. <xref:System.Windows.Forms.PictureBox.SizeMode%2A> Свойство управляет как изображения и управления друг с другом. Дополнительные сведения см. в разделе [как: изменение размера или размещения изображения во время выполнения](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md).
