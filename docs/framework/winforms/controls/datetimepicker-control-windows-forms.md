# <a name="datetimepicker-control-windows-forms"></a>Элемент управления DateTimePicker (Windows Forms)
Windows Forms `DateTimePicker` управления позволяет пользователю выбрать один элемент из списка дат или времени. Используемый для представления даты, он отображается в виде двух частей: раскрывающегося списка с даты, представленной в тексте и сетки, которая появляется, если щелкнуть стрелку вниз рядом со списком.
