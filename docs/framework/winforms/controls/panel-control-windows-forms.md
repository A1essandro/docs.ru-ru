# Элемент управления Panel (Windows Forms)
Элементы управления Windows Forms `Panel` предназначены для объединения в идентифицируемые группы других элементов управления.  Обычно панели используются для разделения формы по функциям.  Элемент управления `Panel` имеет некоторое сходство с элементом управления <xref:System.Windows.Forms.GroupBox>, однако только у элемента управления `Panel` могут быть полосы прокрутки, и только элемент управления <xref:System.Windows.Forms.GroupBox> отображает заголовок.
