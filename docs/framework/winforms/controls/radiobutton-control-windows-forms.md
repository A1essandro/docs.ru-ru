# Элемент управления RadioButton (Windows Forms)
Элементы управления Windows Forms `RadioButton` \(переключатели\) обеспечивают выбор из двух или более взаимоисключающих вариантов.  Функции переключателей и флажков могут показаться схожими, но между ними есть важное отличие: в случае переключателя пользователь может выбрать лишь один вариант.
