# Практическое руководство. Копирование элементов управления между формами Windows Forms
Элемент управления можно скопировать в ту же самую или в другую форму в проекте, или же в буфер обмена для использования в других решениях.  
  
### Чтобы скопировать элемент управления, выполните следующие действия.  
  
1.  Выберите элемент управления и выберите команду **Копировать** в меню **Правка**.  
  
 Теперь этот элемент управления можно вставить в любую форму, принимающую этот тип элементов управления.  Кроме того, происходит добавление этого элемента управления в буфер обмена.
