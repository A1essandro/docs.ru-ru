# Элементы управления Windows Forms
В ходе разработки и изменения пользовательского интерфейса приложений Windows Forms требуется добавлять, выравнивать и размещать элементы управления.  Элементы управления — это объекты, которые находятся внутри объектов формы.  Каждый тип элемента управления имеет собственный набор свойств, методов и событий, соответствующих определенному назначению.  С элементами управления можно работать в конструкторе или добавлять их динамически во время выполнения с помощью кода.
