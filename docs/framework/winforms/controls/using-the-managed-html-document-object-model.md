# Использование управляемой объектной модели HTML-документов
Управляемая объектная модель HTML\-документов \(DOM\) обеспечивает оболочку, основанную на [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)], для классов HTML, предоставленных Internet Explorer.  Эти классы можно использовать для управления HTML\-страницами в элементе управления <xref:System.Windows.Forms.WebBrowser> или для создания новых страниц с нуля.
