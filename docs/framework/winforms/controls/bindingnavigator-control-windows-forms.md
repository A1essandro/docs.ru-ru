# Элемент управления BindingNavigator (Windows Forms)
The `BindingNavigator` control is the navigation and manipulation user interface \(UI\) for controls that are bound to data.  The `BindingNavigator` control enables users to navigate through and manipulate data on a Windows Form.  
  
 В этом разделе содержится обзор элемента управления `BindingNavigator` и проводятся пошаговые инструкции по его использованию для перехода по данным и перемещения в <xref:System.Data.DataSet>.
