# <a name="how-to-resize-controls-on-windows-forms"></a>Практическое руководство. Изменение размера элементов управления в формах Windows Forms
Вы можете изменить размер отдельных элементов управления, и вы можете изменить размер набора элементов управления одного или разных типов, таких как <xref:System.Windows.Forms.Button> и <xref:System.Windows.Forms.GroupBox> элементов управления.  
  
### <a name="to-resize-a-control"></a>Размер элемента управления  
  
1.  Щелкните элемент управления, чтобы изменить размер и перетащите один из восьми маркеров.  
  
    > [!NOTE]
    >  Выберите элемент управления и нажмите клавиши со стрелками, удерживая нажатой клавишу SHIFT, чтобы изменить размер элемента управления на один пиксель за раз. Нажмите клавиши со стрелками вниз и вправо, удерживая нажатыми клавиши SHIFT и CTRL, чтобы изменить размер элемента управления в больших.  
  
### <a name="to-resize-multiple-controls-on-a-form"></a>Чтобы изменить размер нескольких элементов управления в форме  
  
1.  Удерживая нажатой клавишу CTRL или SHIFT и выберите элементы управления, которые требуется изменить. Размер первый выбранный элемент управления используется для других элементов управления.  
  
2.  На **формат** меню, выберите **установить тот же размер**, а затем выберите один из четырех вариантов. Первые три команды изменения размеров элементов управления, которые соответствуют первого выбранного элемента управления.
