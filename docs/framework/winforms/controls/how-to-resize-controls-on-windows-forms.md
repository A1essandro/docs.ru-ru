### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="54f01-108">Чтобы изменить размер нескольких элементов управления в форме</span><span class="sxs-lookup"><span data-stu-id="54f01-108">To resize multiple controls on a form</span></span>  
  
1.  <span data-ttu-id="54f01-109">Удерживая нажатой клавишу CTRL или SHIFT и выберите элементы управления, которые требуется изменить.</span><span class="sxs-lookup"><span data-stu-id="54f01-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="54f01-110">Размер первый выбранный элемент управления используется для других элементов управления.</span><span class="sxs-lookup"><span data-stu-id="54f01-110">The size of the first control you select is used for the other controls.</span></span>
