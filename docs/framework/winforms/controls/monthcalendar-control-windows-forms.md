# Элемент управления MonthCalendar (Windows Forms)
Элемент управления Windows Forms `MonthCalendar` предоставляет пользователям понятный графический интерфейс для просмотра и задания сведений о дате.  Этот элемент управления выводит сетку, содержащую пронумерованные дни месяца, разбитые на столбцы по дням недели.  Перейти к другому месяцу можно, нажав кнопку со стрелкой справа или слева от заголовка месяца.  В отличие от аналогичного элемента управления <xref:System.Windows.Forms.DateTimePicker>, данный элемент управления позволяет выделить диапазон дат; однако элемент управления <xref:System.Windows.Forms.DateTimePicker> может использоваться для задания не только даты, но и времени.
