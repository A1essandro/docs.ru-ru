# Общие сведения об элементе управления ContextMenuStrip
> [!NOTE]
>  Элемент управления <xref:System.Windows.Forms.ContextMenuStrip> заменяет элемент управления <xref:System.Windows.Forms.ContextMenu> и расширяет его функциональные возможности; однако при необходимости элемент управления <xref:System.Windows.Forms.ContextMenu> можно сохранить для обратной совместимости и использования в будущем.  
  
 Контекстные меню открываются в положении указателя по щелчку правой кнопкой мыши.  Контекстные *меню* обеспечивают клиентскую область или элемент управления в положении указателя мыши.  
  
 Элемент управления <xref:System.Windows.Forms.ContextMenuStrip> предназначен для тесной работы с новыми связанными элементами управления <xref:System.Windows.Forms.ToolStrip>, но <xref:System.Windows.Forms.ContextMenuStrip> можно также легко связать с другими элементами управления.  
  
 В следующей таблице приведены основные сопутствующие классы элемента управления <xref:System.Windows.Forms.ContextMenuStrip>.  
  
|Класс|Описание|  
|-----------|--------------|  
|<xref:System.Windows.Forms.ToolStripMenuItem>|Представляет доступный для выбора параметр, отображаемый в объекте <xref:System.Windows.Forms.MenuStrip> или <xref:System.Windows.Forms.ContextMenuStrip>.|  
|<xref:System.Windows.Forms.ToolStripDropDown>|Представляет элемент управления, позволяющий пользователю выбрать один элемент из списка, отображаемого при щелчке мышью по кнопке <xref:System.Windows.Forms.ToolStripDropDownButton> или пункту меню более высокого уровня.|  
|<xref:System.Windows.Forms.ToolStripDropDownItem>|Обеспечивает базовую функциональность для элементов управления, производных от <xref:System.Windows.Forms.ToolStripItem>, служащих для отображения раскрывающихся списков при щелчке мышью.|
