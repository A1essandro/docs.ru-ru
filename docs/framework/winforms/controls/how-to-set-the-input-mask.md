# Практическое руководство. Определение маски ввода
Элемент управления текстового поля с маской является улучшенным элементом управления текстового поля, поддерживающим декларативный синтаксис для принятия или отклонения введенных пользователем данных.  Настроив свойство маски, можно указать, какие данные допустимы для ввода пользователем, не создавая в приложении настраиваемую логику проверки.  Дополнительные сведения содержатся в разделе "Примечания" для класса <xref:System.Windows.Forms.MaskedTextBox>.  
  
## Установка свойства маски вручную  
 Свойство маски можно ввести вручную при наличии опыта работы с символами, которые им поддерживаются.  Сводка символов, поддерживаемых свойством маски, приведена в разделе "Примечания" для свойства <xref:System.Windows.Forms.MaskedTextBox.Mask%2A>.  
  
#### Чтобы задать свойство маски вручную  
  
1.  В представлении **Конструктор** выберите <xref:System.Windows.Forms.MaskedTextBox>.  
  
2.  В окне **Свойства** найдите свойство <xref:System.Windows.Forms.MaskedTextBox.Mask%2A>.  
  
3.  Введите необходимую маску.  Например, введите `###`.  
  
## Использование диалогового окна "Маска ввода"  
 Диалоговое окно "Маска ввода" предоставляет несколько предопределенных масок ввода.  Можно также изменить предопределенные маски или ввести собственную маску вручную.  
  
#### Чтобы открыть диалоговое окно "Маска ввода"  
  
1.  В представлении **Конструктор** выберите <xref:System.Windows.Forms.MaskedTextBox>.  
  
    1.  Щелкните смарт\-тег, чтобы открыть панель **Задачи MaskedTextBox**.  
  
    2.  Нажмите **Задать маску**.  
  
     \- или \-  
  
    1.  В окне **Свойства** выберите свойство <xref:System.Windows.Forms.MaskedTextBox.Mask%2A>.  
  
    2.  Нажмите кнопку с многоточием в столбце значения свойства.  
  
     Появится диалоговое окно **Маска ввода**.  
  
#### Чтобы использовать диалоговое окно "Маска ввода"  
  
1.  \(Необязательно\) Выберите одну из предопределенных масок в списке.  
  
2.  \(Необязательно\) Измените предопределенную маску в поле **Маска**.  
  
3.  \(Необязательно\) Введите новую маску в поле **Маска**.  Таким образом, использование предопределенной маски необязательно.  
  
    > [!NOTE]
    >  В поле "Предварительный просмотр" отображаются символы, которые пользователь увидит в <xref:System.Windows.Forms.MaskedTextBox>.  Эти символы позволят пользователю правильно ввести данные.  
  
4.  Установите или снимите флажок **Использовать ValidatingType**.  Флажок **Использовать ValidatingType** определяет, используется ли тип данных для проверки данных, введенных пользователем.  Дополнительные сведения содержатся в свойстве <xref:System.Windows.Forms.MaskedTextBox.ValidatingType%2A>.  
  
5.  Нажмите кнопку **ОК**.  
  
     Для ввода маски используется свойство **Маска** в окне **Свойства**.
