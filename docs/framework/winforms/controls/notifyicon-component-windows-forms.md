# Компонент NotifyIcon (Windows Forms)
Компонент форм Windows Forms `NotifyIcon` выводит значки в области уведомления о состоянии в панели задач для процессов, выполняемых в фоновом режиме, являющиеся их пользовательским интерфейсом.  Примером такого процесса является антивирусная программа, доступ к которой можно получить, щелкнув значок в области уведомления о состоянии в панели задач.
