# Элемент управления LinkLabel (Windows Forms)
Элемент управления форм `LinkLabel` позволяет добавлять ссылки в стиле веб в приложения форм Windows.  Элемент управления <xref:System.Windows.Forms.Label> может использоваться для тех же целей, что и элемент управления `LinkLabel`; кроме того, имеется возможность задать часть текста в качестве ссылки на объект или веб\-страницу.
