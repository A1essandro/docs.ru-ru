# Класс SoundPlayer
Класс `SoundPlayer` позволяет облегчить добавление звуков в приложениях.  
  
 Можно также использовать класс <xref:System.Media.SystemSounds> для воспроизведения системных звуков, в том числе звукового сигнала.
