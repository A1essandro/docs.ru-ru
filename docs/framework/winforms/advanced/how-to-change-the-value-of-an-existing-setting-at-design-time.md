# How To: Change the Value of an Existing Setting at Design Time
Visual Studio можно использовать для изменения существующих в проекте значений параметров.  
  
### Чтобы изменить значение существующего параметра во время разработки в C\#, выполните следующие действия.  
  
1.  В **обозревателе решений** разверните узел **Свойства** проекта.  
  
2.  Дважды щелкните SETTINGS\-файл, в который требуется добавить новый параметр.  Имя этого файла по умолчанию — Settings.settings.  
  
3.  В конструкторе параметров найдите параметр, для которого требуется изменить значение, и введите новое значение в столбце значения.  
  
### Чтобы изменить значение существующего параметра во время конструирования в Visual Basic, выполните следующие действия.  
  
1.  В **обозревателе решений** щелкните правой кнопкой мыши узел проекта и выберите пункт **Свойства**.  
  
2.  На странице **Свойства** перейдите на вкладку **Параметры**.  
  
3.  В конструкторе параметров найдите параметр, для которого требуется изменить значение, и введите новое значение в столбце значения.
