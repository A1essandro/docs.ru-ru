# <a name="recoloring-images"></a>Перекрашивание изображений
Перекрашивание — это процесс настройки цветов изображения. Некоторые примеры Перекрашивание изменение одного цвета на другой, Настройка интенсивности цвета относительно другого цвета, настройка яркости или контраста всех цветов и преобразование цветов в оттенки серого.
