### <a name="to-create-an-mdi-parent-form-at-design-time"></a><span data-ttu-id="83c8a-109">Создание родительской MDI-формы во время разработки</span><span class="sxs-lookup"><span data-stu-id="83c8a-109">To create an MDI parent form at design time</span></span>  
  
1.  <span data-ttu-id="83c8a-110">Создайте проект приложения Windows.</span><span class="sxs-lookup"><span data-stu-id="83c8a-110">Create a Windows Application project.</span></span>  
  
2.  <span data-ttu-id="83c8a-111">В **свойства** задайте <xref:System.Windows.Forms.Form.IsMdiContainer%2A> свойства **true**.</span><span class="sxs-lookup"><span data-stu-id="83c8a-111">In the **Properties** window, set the <xref:System.Windows.Forms.Form.IsMdiContainer%2A> property to **true**.</span></span>  
  
     <span data-ttu-id="83c8a-112">При этом форма назначается в качестве MDI-контейнера для дочерних окон.</span><span class="sxs-lookup"><span data-stu-id="83c8a-112">This designates the form as an MDI container for child windows.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="83c8a-113">При необходимости, при настройке свойств в окне **Свойства** для свойства `WindowState` также можно задать значение **Maximized**, так как управлять дочерними MDI-окнами проще, когда родительская форма развернута.</span><span class="sxs-lookup"><span data-stu-id="83c8a-113">While setting properties in the **Properties** window, you can also set the `WindowState` property to **Maximized**, if you like, as it is easiest to manipulate MDI child windows when the parent form is maximized.</span></span> <span data-ttu-id="83c8a-114">Кроме того, следует помнить, что граница родительской MDI-формы будет окрашена в системный цвет (заданный на панели управления Windows), а не в черный цвет, заданный с помощью свойства <xref:System.Windows.Forms.Control.BackColor%2A?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="83c8a-114">Additionally, be aware that the edge of the MDI parent form will pick up the system color (set in the Windows System Control Panel), rather than the back color you set using the <xref:System.Windows.Forms.Control.BackColor%2A?displayProperty=nameWithType> property.</span></span>  
  
3.  <span data-ttu-id="83c8a-115">Перетащите элемент управления **MenuStrip** из **панели элементов** в форму.</span><span class="sxs-lookup"><span data-stu-id="83c8a-115">From the **Toolbox**, drag a **MenuStrip** control to the form.</span></span> <span data-ttu-id="83c8a-116">Создайте пункт меню верхнего уровня — для свойства **Text** задайте значение **&File**, пункты меню должны называться **&New** и **&Close**.</span><span class="sxs-lookup"><span data-stu-id="83c8a-116">Create a top-level menu item with the **Text** property set to **&File** with submenu items called **&New** and **&Close**.</span></span> <span data-ttu-id="83c8a-117">Также создайте пункт меню верхнего уровня **&Window**.</span><span class="sxs-lookup"><span data-stu-id="83c8a-117">Also create a top-level menu item called **&Window**.</span></span>  
  
     <span data-ttu-id="83c8a-118">Первое меню будет создавать и скрывать пункты меню во время выполнения, а второе меню будет отслеживать открытые дочерние MDI-окна.</span><span class="sxs-lookup"><span data-stu-id="83c8a-118">The first menu will create and hide menu items at run time, and the second menu will keep track of the open MDI child windows.</span></span> <span data-ttu-id="83c8a-119">На этом этапе вы создали родительское MDI-окно.</span><span class="sxs-lookup"><span data-stu-id="83c8a-119">At this point, you have created an MDI parent window.</span></span>  
  
4.  <span data-ttu-id="83c8a-120">Нажмите клавишу **F5** для запуска приложения.</span><span class="sxs-lookup"><span data-stu-id="83c8a-120">Press **F5** to run the application.</span></span> <span data-ttu-id="83c8a-121">Подробнее о создании дочерних MDI-окон, работающих в родительской MDI-форме, см. в разделе [Практическое руководство. Создание дочерних MDI-форм](../../../../docs/framework/winforms/advanced/how-to-create-mdi-child-forms.md).</span><span class="sxs-lookup"><span data-stu-id="83c8a-121">For information about creating MDI child windows that operate within the MDI parent form, see [How to: Create MDI Child Forms](../../../../docs/framework/winforms/advanced/how-to-create-mdi-child-forms.md).</span></span>
