# <a name="control-help-using-tooltips"></a>Отображение справки по элементам управления с помощью всплывающих подсказок
Можно использовать <xref:System.Windows.Forms.ToolTip> компонент для отображения кратких специализированных справочных сообщений для отдельных элементов управления в формах Windows Forms. <xref:System.Windows.Forms.ToolTip> Компонент предоставляет свойство, указывающее текст, отображаемый для каждого элемента управления в форме. Дополнительные сведения о работе с <xref:System.Windows.Forms.ToolTip> компонент таким образом, см. в разделе [как: Определение всплывающих подсказок для элементов управления формы Windows во время разработки](../controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md). Кроме того, можно настроить <xref:System.Windows.Forms.ToolTip> компонент таким образом, возникает задержка перед его отображением. Дополнительные сведения см. в разделе [Как Изменение значения задержки для компонента ToolTip в Windows Forms](../controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
