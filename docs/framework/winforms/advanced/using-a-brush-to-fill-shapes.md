# Использование кисти для заливки фигур
Объект [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Brush> служит для заливки внутренней области замкнутых фигур.  В [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] определены следующие стили заливки: сплошной цвет, шаблон штриховки, текстура с изображением и цветовой градиент.
