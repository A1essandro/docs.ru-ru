# <a name="cardinal-splines-in-gdi"></a>Фундаментальные сплайны в GDI+
Фундаментальный сплайн — это последовательность отдельных кривых, объединенных в одну большую кривую. Сплайн задается массивом точек и параметром натяжение. Фундаментальный сплайн проходящую через все точки в массиве. Существуют не острого угла и внезапные изменения tightness кривой. Ниже показан набор точек и фундаментальный сплайн, который проходит через все точки в наборе.  
  
 ![Фундаментальный сплайн](../../../../docs/framework/winforms/advanced/media/aboutgdip02-art09.gif "Aboutgdip02_art09")  
  
## <a name="physical-and-mathematical-splines"></a>Физические и математические сплайны  
 Физический сплайн — это тонкий фрагмент из дерева или другого гибкого материала. До появления математические сплайны конструкторы физические сплайны использовались для рисования кривых. Конструктор следует размещать сплайн на листе бумаги и заметку можно прикрепить к заданные точки. Конструктор может создать кривую вдоль сплайн пера или карандаша. Заданные точки может иметь множество различных кривых в зависимости от свойств физического сплайна. Например сплайн с высокой устойчивости к ломаный приведет к получению отличаться от кривой очень гибкого сплайна.  
  
 Формулы для математических сплайнов основаны на свойствах гибкие стержни, кривые, образованные математическими сплайнами, похожи на кривые, нарисованные с помощью физических сплайнов. Так же, как физические сплайны различных натяжение будут порождать различные кривые одного набора точек, математические сплайны с разными значениями параметра натяжения будет порождать различные кривые заданные точки. На следующем рисунке четыре основных сплайна, проходящих через один и тот же набор точек. Для каждого сплайна показано натяжение. Натяжение 0 соответствует бесконечной физической натяжение, принудительное кривой к кратчайшему пути (прямой) между точками. Нет физического натяжение, позволяя сплайн занимает положение наименьшим суммарным изгибом соответствует натяжение, равное 1. С натяжение значения больше 1 кривой ведет себя как сжатый spring, принудительно отправить занять более длинный путь.  
  
 ![Фундаментальные сплайны](../../../../docs/framework/winforms/advanced/media/aboutgdip02-art10.gif "Aboutgdip02_art10")  
  
 Все четыре сплайна на предыдущем рисунке обладают одинаковой касательной в начальной точке. Тангенс — это линия, производными от начальной точки до следующей точки кривой. Аналогично общей касательной конечной точки является линия, проведенная через конечную точку к предыдущей точке кривой.
