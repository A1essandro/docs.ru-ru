# <a name="graphics-overview-windows-forms"></a>Общие сведения о графике (Windows Forms)

В службах [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] , предоставляются через набор управляемых классов.
