# <a name="coreresponsedata-class"></a>Класс CoreResponseData

`CoreResponseData` Класс представляет синтаксический анализ HTTP-заголовки и текст ответа.

## <a name="syntax"></a>Синтаксис
  
```csharp
internal class CoreResponseData
```

В разделе [DiagnosticSource в руководстве пользователя](https://github.com/dotnet/corefx/blob/master/src/System.Diagnostics.DiagnosticSource/src/DiagnosticSourceUsersGuide.md).
