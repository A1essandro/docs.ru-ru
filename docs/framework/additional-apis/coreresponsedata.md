# <a name="coreresponsedata-class"></a>Класс CoreResponseData

`CoreResponseData` Класс представляет синтаксический анализ HTTP-заголовки и текст ответа.

## <a name="syntax"></a>Синтаксис
  
```csharp
internal class CoreResponseData
```

> Этот API является внутренним и не предназначен для непосредственного использования в коде. Вместо этого следует использовать <xref:System.Diagnostics.DiagnosticSource> отключают сетевого кода. В разделе [DiagnosticSource в руководстве пользователя](https://github.com/dotnet/corefx/blob/master/src/System.Diagnostics.DiagnosticSource/src/DiagnosticSourceUsersGuide.md).
