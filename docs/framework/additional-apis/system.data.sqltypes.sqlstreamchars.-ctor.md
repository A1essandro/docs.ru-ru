# <a name="sqlstreamchars-constructor"></a>Конструктор SqlStreamChars

Инициализирует новый экземпляр класса `SqlStreamChars`. Дружественные сборки, содержащей этот конструктор связан с SQLAccess.dll. Он предназначен для использования с SQL Server. Для других баз данных используйте механизм размещения, предоставляемый этой базы данных.

```csharp
protected SqlStreamChars ();
```
