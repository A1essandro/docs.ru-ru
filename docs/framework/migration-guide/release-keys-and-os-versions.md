# <a name="net-framework-release-keys-and-windows-operating-system-versions"></a>Ключи выпуска .NET Framework и версии операционной системы Windows

В следующей таблице перечислены ключи выпуска в отдельных операционных системах для .NET Framework 4.5 и более поздних версий.


 |Версия платформы .NET Framework|Значение DWORD "Release"|
 |--------------------------------|-------------|
|.NET Framework 4,5|Все версии операционной системы Windows: 378389|
|.NET Framework 4.5.1|Windows 8.1 и Windows Server 2012 R2: 378675<br />Все другие версии операционной системы Windows: 378758|
|.NET Framework 4.5.2|Все версии операционной системы Windows: 379893|
|.NET Framework 4.6|Windows 10: 393295<br />Все другие версии операционной системы Windows: 393297|
|.NET Framework 4.6.1|Windows 10 с ноябрьским обновлением: 394254<br />Все другие версии операционной системы Windows: 394271|
|.NET Framework 4.6.2|В юбилейном обновлении Windows 10 и Windows Server 2016: 394802<br />Все другие версии операционной системы Windows: 394806|
|.NET Framework 4.7|Windows 10 Creators Update: 460798<br />Все другие версии операционной системы Windows: 460805| 
|.NET Framework 4.7.1|Windows 10 Fall Creators Update и Windows Server версии 1709: 461308<br/>Все другие версии операционной системы Windows: 461310|
|.NET Framework 4.7.2|Windows 10 за апрель 2018 г. Update и Windows Server версии 1803: 461808<br/>Все другие версии операционной системы Windows: 461814|
