 <span data-ttu-id="ae7d1-108">На среднем уровне сущности создаются при помощи контекста данных, который отслеживает их состояние и управляет отложенной загрузкой из базы данных и передачей изменений в базу данных.</span><span class="sxs-lookup"><span data-stu-id="ae7d1-108">On the middle tier, entities are created by the data context, which tracks their state, and manages deferred loading from class="sxs-lookup"><span data-stu-id="ae7d1-111">Entities that the client sends back for updates must be reattached to the data context before  <span data-ttu-id="ae7d1-113">В приложениях ASP.NET <xref:System.Web.UI.WebControls.LinqDataSource> управляет большей частью данного сложного процесса.</span><span class="sxs-lookup"><span data-stu-id="ae7d1-113">In ASP.NET applications, the <xref:System.Web.UI.WebControls.LinqDataSource> manages most of this complexity.</span></span> <span data-ttu-id="ae7d1-114">Дополнительные сведения см. в разделе [NIB: Обзор управления LinqDataSource Web Server](https://msdn.microsoft.com/library/104cfc3f-7385-47d3-8a51-830dfa791136).</span><span class="sxs-lookup"><span data-stu-id="ae7d1-114">For more information, see [NIB: LinqDataSource Web Server Control Overview](https://msdn.microsoft.com/library/104cfc3f-7385-47d3-8a51-830dfa791136).</span></span>
