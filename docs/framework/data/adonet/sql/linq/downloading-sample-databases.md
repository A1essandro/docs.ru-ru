# Загрузка образцов баз данных
В ряде примеров и пошаговых руководств документации [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] используется база данных «Northwind» и выпуск SQL Server Express Edition.  Эти продукты можно бесплатно загрузить в Центре загрузки Майкрософт.  
  
## Загрузка базы данных «Northwind».  
  
#### Загрузка и установка базы данных Northwind для SQL Server  
  
1.  Запустите Internet Explorer.  
  
2.  Перейдите на веб\-сайт [демонстрационных базы данных Northwind и Pubs](http://go.microsoft.com/fwlink?linkid=64296).  
  
3.  Нажмите **Загрузить**.  
  
4.  В диалоговом окне **Загрузка файла** выберите пункт **Сохранить**.  
  
5.  После загрузки файла дважды щелкните файл **Nwind.exe**, чтобы установить базу данных.  
  
     По умолчанию база данных устанавливается в папку *диск*:\\SQL Server 2000 Sample Databases.  
  
## Загрузка SQL Server Express Edition  
 SQL Server Express Edition доступен бесплатно и его можно распространять вместе с приложениями.  При использовании среды [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)] SQL Server Express Edition включается в версию Pro и более поздние версии.  
  
#### Загрузка и установка выпуска SQL Server Express  
  
1.  Запустите Internet Explorer.  
  
2.  Перейдите на веб\-сайт [Центра загрузки Майкрософт](http://go.microsoft.com/fwlink?linkid=74602).  
  
3.  В поле ключевых слов введите SQL Server Express.  
  
4.  Нажмите **Перейти**.  
  
5.  На странице результатов щелкните ссылку на страницу загрузки **Microsoft SQL Server 2005, экспресс\-выпуск**.  
  
6.  Следуйте указаниям по установке, содержащимся на веб\-узле.  
  
## Загрузка среды Management Studio Express  
 Если требуется изменить загруженную базу данных, можно открыть ее в **обозревателе сервера** в интегрированной среде разработки [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] или воспользоваться программой Microsoft SQL Server Management Studio Express \(SSMSE\).  
  
#### Загрузка среды Management Studio Express  
  
-   Следуйте инструкциям на [сайте SSMSE](http://go.microsoft.com/fwlink/?LinkId=95933).
