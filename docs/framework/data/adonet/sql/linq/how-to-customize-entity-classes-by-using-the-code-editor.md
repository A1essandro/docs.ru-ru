# Как настроить классы сущностей с помощью редактора кода

 Можно также воспользоваться редактором кода [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)] для написания своего кода сопоставления или для настройки уже сформированного кода.
