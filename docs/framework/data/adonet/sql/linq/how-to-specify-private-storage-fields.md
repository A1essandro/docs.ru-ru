# Как указать поля закрытого хранения
Чтобы задать имя базового поля хранения, используйте свойство [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A> атрибута <xref:System.Data.Linq.Mapping.DataAttribute>.  
  
 Примеры кода см. в разделе <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.  
  
### Задание имени базового поля хранения  
  
1.  Добавьте свойство <xref:System.Data.Linq.Mapping.ColumnAttribute> атрибуту <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.  
  
2.  Укажите имя поля в качестве значения свойства <xref:System.Data.Linq.Mapping.DataAttribute.Storage%2A>.
