## <a name="determining-the-transaction-type"></a><span data-ttu-id="8da92-108">Определение типа транзакции</span><span class="sxs-lookup"><span data-stu-id="8da92-108">Determining the Transaction Type</span></span>

 <span data-ttu-id="8da92-111">Для выполнения локальных транзакций каждый поставщик данных платформы [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] имеет свой собственный объект `Transaction`.</span><span class="sxs-lookup"><span data-stu-id="8da92-111">Each of the [!INCLUDE[dnprdnshort](../../../../includes/dnprdnshort-md.md)] data providers has its own `Transaction` object for performing local transactions.</span></span> <span data-ttu-id="8da92-112">Если требуется выполнить транзакцию в базе данных SQL Server, выбирается транзакция <xref:System.Data.SqlClient>.</span><span class="sxs-lookup"><span data-stu-id="8da92-112">If you require a transaction to be performed in a SQL Server database, select a <xref:System.Data.SqlClient> transaction.</span></span> <span data-ttu-id="8da92-113">Для транзакции Oracle используйте поставщик <xref:System.Data.OracleClient>.</span><span class="sxs-lookup"><span data-stu-id="8da92-113">For an Oracle transaction, use the <xref:System.Data.OracleClient> provider.</span></span> <span data-ttu-id="8da92-114">Кроме того, имеется <xref:System.Data.Common.DbTransaction> класс, доступный для написания независимого от поставщика кода с использованием транзакций.</span><span class="sxs-lookup"><span data-stu-id="8da92-114">In addition, there is a <xref:System.Data.Common.DbTransaction> class that is available for writing provider-independent code that requires transactions.</span></span>
