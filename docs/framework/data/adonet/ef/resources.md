# <a name="entity-framework-resources"></a>Ресурсы Entity Framework
Следующие внешние ресурсы предоставляют дополнительные сведения и поддержку для создания приложений Entity Framework.  
  
 [Центр разработчиков данных](http://go.microsoft.com/fwlink/?LinkId=213876)  
 Центральный узел разработки данных с использованием технологий Microsoft.  
  
 [Блог команды ADO.NET](http://go.microsoft.com/fwlink/?LinkId=91905)  
 Блог, содержащий обновления и обсуждение характеристик и возможностей ADO.NET.  
  
 [Блог разработчиков платформы Entity Framework](http://go.microsoft.com/fwlink/?LinkId=186888)  
 Обсуждение конструкций и обзоры возможностей следующих версий платформы Entity Framework.  
  
 [Как мне платформы данных? Видео: Серия Framework сущности](http://go.microsoft.com/fwlink/?LinkId=124600)  
 Содержит набор видеоматериалов, показывающих, как можно создавать приложения, используя различные компоненты платформы Entity Framework.  
  
 [Entity Framework вопросы и ответы](http://go.microsoft.com/fwlink/?LinkID=213877)  
 Блог, содержащий часто задаваемые вопросы о платформе Entity Framework.
