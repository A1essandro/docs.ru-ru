# Приступая к работе (платформа Entity Framework)
Платформа [!INCLUDE[vstecado](../../../../../includes/vstecado-md.md)] [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] поддерживает приложения и службы, ориентированные на данные, и обеспечивает программирование с данными, позволяющее поднять уровень абстракции с логического реляционного до концептуального.  Предлагая разработчикам возможность работать с данными на более высоком уровне абстракции, платформа [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] поддерживает код, который является независимым от конкретного типа подсистемы хранилища данных или реляционной схемы. Дополнительные сведения см. в разделе [Общие сведения о платформе Entity Framework](../../../../../docs/framework/data/adonet/ef/overview.md).  
  
 Чтобы быстро приступить к работе с самой последней версией платформы Entity Framework, см. раздел [Начало работы](http://go.microsoft.com/fwlink/?LinkId=235280)
