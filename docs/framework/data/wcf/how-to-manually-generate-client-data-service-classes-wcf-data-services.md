### <a name="to-generate-c-classes-that-support-data-binding"></a><span data-ttu-id="fc968-114">Формирование классов C#, поддерживающих привязку данных</span><span class="sxs-lookup"><span data-stu-id="fc968-114">To generate C# classes that support data binding</span></span>

  
    > [!NOTE]
