 <span data-ttu-id="69bc9-110">В целях обеспечения предсказуемого поведения все транзакции должны соответствовать базовым свойствам ACID (атомарность, согласованность, изолированность и  <span data-ttu-id="69bc9-119">Транзакция может ограничиваться одним ресурсом данных, например базой данных или очередью сообщений.</span><span class="sxs-lookup"><span data-stu-id="69bc9-119">A transaction can be confined to a single data resource, such as a database or message queue.</span></span> <span data-ttu-id="69bc9-120">В этом сценарии локальной транзакцией управляет диспетчер транзакций, предоставляемый пространством имен <xref:System.Transactions> и обеспечивающий повышение производительности.</span><span class="sxs-lookup"><span data-stu-id="69bc9-120">In this scenario, the local transaction is managed by the Transaction Manager provided by <xref:System.Transactions> , which generates performance gain.</span></span> <span data-ttu-id="69bc9-121">Транзакциями, ограниченными одним ресурсом данных, можно легко и эффективно управлять.</span><span class="sxs-lookup"><span data-stu-id="69bc9-121">Controlled by the data resource, these transactions are efficient and easy to manage.</span></span>
