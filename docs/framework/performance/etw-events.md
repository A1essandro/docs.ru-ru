# ETW Events in the .NET Framework
Трассировка событий для Windows \(ETW\) — это высокопроизводительная масштабируемая система трассировки с низкими издержками, предоставляемая операционными системами.  Она служит дополнением поддержки профилирования и отладки, предоставляемой платформой .NET Framework, и может использоваться для устранения неисправностей в различных сценариях.  Общие сведения о трассировке событий Windows см. в разделе [Улучшение производительности в ходе отладки и трассировки событий Windows](http://go.microsoft.com/fwlink/?LinkID=161142) в библиотеке MSDN.  
  
 В платформе .NET Framework трассировка событий ETW доступна для среды CLR, [библиотеки параллельных задач](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) и [Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).
