# <a name="icordebugchainenum-interface1"></a>ICorDebugChainEnum Interface1
Реализует методы ICorDebugEnum и выполняет перечисление массивов ICorDebugChain.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Метод Next](../../../../docs/framework/unmanaged-api/debugging/icordebugchainenum-next-method.md)|Возвращает заданное число `ICorDebugChain` экземпляров из перечисления, начиная с текущей позиции.|  
  
## <a name="remarks"></a>Примечания  
  
> [!NOTE]
>  Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
