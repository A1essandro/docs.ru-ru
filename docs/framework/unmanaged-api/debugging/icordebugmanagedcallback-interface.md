## <a name="remarks"></a><span data-ttu-id="64f21-160">Примечания</span><span class="sxs-lookup"><span data-stu-id="64f21-160">Remarks</span></span>  
 <span data-ttu-id="64f21-161">Все обратные вызовы сериализации, вызывается в том же потоке и вызываются с использованием процессов в синхронизированном состоянии.</span><span class="sxs-lookup"><span data-stu-id="64f21-161">All callbacks are serialized, called in the same thread, and called with the process in the synchronized state.</span></span>  
  
 <span data-ttu-id="64f21-162">Каждая реализация обратного вызова необходимо вызвать [ICorDebugController::Continue](../../../../docs/framework/unmanaged-api/debugging/icordebugcontroller-continue-method.md) для возобновления выполнения.</span><span class="sxs-lookup"><span data-stu-id="64f21-162">Each callback implementation must call [ICorDebugController::Continue](../../../../docs/framework/unmanaged-api/debugging/icordebugcontroller-continue-method.md) to resume execution.</span></span> <span data-ttu-id="64f21-163">Если `ICorDebugController::Continue` не вызывается до функция обратного вызова, процесс не запустился и дальнейшие обратные вызовы событий не будет происходить, пока `ICorDebugController::Continue` вызывается.</span><span class="sxs-lookup"><span data-stu-id="64f21-163">If `ICorDebugController::Continue` is not called before the callback returns, the process will remain stopped and no more event callbacks will occur until `ICorDebugController::Continue` is called.</span></span>  
  
 <span data-ttu-id="64f21-164">Отладчик должен реализовать [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) при отладке приложений .NET Framework версии 2.0.</span><span class="sxs-lookup"><span data-stu-id="64f21-164">A debugger must implement [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) if it is debugging .NET Framework version 2.0 applications.</span></span> <span data-ttu-id="64f21-165">Экземпляр `ICorDebugManagedCallback` или `ICorDebugManagedCallback2` передается как объект обратного вызова для [ICorDebug::SetManagedHandler](../../../../docs/framework/unmanaged-api/debugging/icordebug-setmanagedhandler-method.md).</span><span class="sxs-lookup"><span data-stu-id="64f21-165">An instance of `ICorDebugManagedCallback` or `ICorDebugManagedCallback2` is passed as the callback object to [ICorDebug::SetManagedHandler](../../../../docs/framework/unmanaged-api/debugging/icordebug-setmanagedhandler-method.md).</span></span>
