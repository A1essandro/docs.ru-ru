# <a name="icordebugvaluebreakpoint-interface1"></a>ICorDebugValueBreakpoint интерфейс1
Расширяет интерфейс ICorDebugBreakpoint для предоставления доступа к определенным значениям.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[GetValue-метод](../../../../docs/framework/unmanaged-api/debugging/icordebugvaluebreakpoint-getvalue-method.md)|Получает указатель на интерфейс ICorDebugValue объекта, который представляет значение объекта, на которой задана точка останова.|  
  
## <a name="remarks"></a>Примечания  
  
> [!NOTE]
>  Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
