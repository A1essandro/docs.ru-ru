# <a name="icordebugstringvalue-interface1"></a>ICorDebugStringValue интерфейс1
Подкласс ICorDebugHeapValue, применяемый к строковым значениям.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[GetLength-метод](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getlength-method.md)|Возвращает число символов в строке, упоминаемой в этом `ICorDebugStringValue`.|  
|[Метод GetString](../../../../docs/framework/unmanaged-api/debugging/icordebugstringvalue-getstring-method.md)|Возвращает строку, упоминаемой в этом `ICorDebugStringValue`.|  
  
## <a name="remarks"></a>Примечания  
  
> [!NOTE]
>  Этот интерфейс не поддерживает удаленные вызовы между компьютерами или между процессами.
