# <a name="icordebugstepperdeactivate-method"></a>Метод ICorDebugStepper::Deactivate
ICorDebugStepper для отмены последней команды шага, полученное в результате.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT Deactivate ();  
```  
  
## <a name="remarks"></a>Примечания  
 Новая команда пошагового выполнения может быть выдан после последнего полученного шаг команда была отменена.
