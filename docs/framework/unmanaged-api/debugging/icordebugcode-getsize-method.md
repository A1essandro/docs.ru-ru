# <a name="icordebugcodegetsize-method"></a>Метод ICorDebugCode::GetSize
Возвращает размер в байтах двоичного кода, представленного «ICorDebugCode».  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT GetSize (  
    [out] ULONG32    *pcBytes  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pcBytes`  
 [out] Указатель на размер в байтах двоичного кода, который `ICorDebugCode` представляет.
