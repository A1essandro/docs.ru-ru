# <a name="cleardownloadcache-function"></a>Функция ClearDownloadCache
Очищает глобальный кэш сборок загруженных сборок.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT ClearDownloadCache ();  
```  
  
## <a name="requirements"></a>Требования  
 **Платформы:** разделе [требования к системе для](../../../../docs/framework/get-started/system-requirements.md).  
  
 **Заголовок:** Fusion.h  
  
 **Библиотека:** Fusion.dll и Mscorwks.dll. Используйте Fusion.dll вместо Mscorwks.dll, чтобы целевая правильную версию платформы .NET Framework.
