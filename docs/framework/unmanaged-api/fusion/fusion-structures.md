# <a name="fusion-structures"></a>Структуры Fusion
В этом разделе описываются неуправляемые структуры, используемые API Fusion.  
  
## <a name="in-this-section"></a>Содержание  
 [ASSEMBLY_INFO-структура](../../../../docs/framework/unmanaged-api/fusion/assembly-info-structure.md)  
 Содержит сведения о сборке, которая зарегистрирована в глобальном кэше сборок.  
  
 [Fusion_install_reference-структура](../../../../docs/framework/unmanaged-api/fusion/fusion-install-reference-structure.md)  
 Представляет ссылку, когда приложение устанавливает на сборку, установленную приложением в глобальном кэше сборок.  
  
 [Identity_attribute-структура](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-structure.md)  
 Содержит сведения об атрибутах метаданных о [IDefinitionIdentity](../../../../docs/framework/unmanaged-api/fusion/idefinitionidentity-interface.md) экземпляра.  
  
 [IDENTITY_ATTRIBUTE_BLOB-структура](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-blob-structure.md)  
 Содержит сведения о смещении для ассоциированного [IDENTITY_ATTRIBUTE](../../../../docs/framework/unmanaged-api/fusion/identity-attribute-structure.md) структуры.
