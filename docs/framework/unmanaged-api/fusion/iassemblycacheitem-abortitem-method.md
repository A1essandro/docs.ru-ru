# <a name="iassemblycacheitemabortitem-method"></a>Метод IAssemblyCacheItem::AbortItem
Разрешает сборку в глобальный кэш сборок, чтобы выполнить операции очистки, перед выпуском.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT AbortItem ();  
```
