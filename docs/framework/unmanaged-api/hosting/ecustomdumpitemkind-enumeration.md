# <a name="ecustomdumpitemkind-enumeration"></a>Перечисление ECustomDumpItemKind

## <a name="syntax"></a>Синтаксис  
  
```

    DUMP_ITEM_None = 0  
} ECustomDumpItemKind;  
```
