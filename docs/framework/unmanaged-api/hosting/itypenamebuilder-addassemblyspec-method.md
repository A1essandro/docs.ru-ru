Этот метод поддерживает инфраструктуру .NET Framework и не предназначен для использования непосредственно из программного кода.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT AddAssemblySpec (  
    [in] LPCWSTR szAssemblySpec  
);  
```
