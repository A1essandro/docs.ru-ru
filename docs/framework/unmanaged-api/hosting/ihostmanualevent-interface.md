# <a name="ihostmanualevent-interface"></a>Интерфейс IHostManualEvent
Предоставляет реализацию узла представления событие ручного сброса.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание|  
|------------|-----------------|  
|[Reset-метод](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-reset-method.md)|Сбрасывает текущий `IHostManualEvent` несигнальное состояние экземпляра.|  
|[Метод Set](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-set-method.md)|Задает текущий `IHostManualEvent` экземпляр сигнальное состояние.|  
|[Wait-метод](../../../../docs/framework/unmanaged-api/hosting/ihostmanualevent-wait-method.md)|Вызывает текущий `IHostManualEvent` экземпляр ожидать его признания или определенного времени.|
