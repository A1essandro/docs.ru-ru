 <span data-ttu-id="701c3-144">`IHostTaskManager` позволяет среде CLR создавать задачи и управлять ими, чтобы обеспечить обработчиков предпринять действия при передаче управления из управляемого в неуправляемый код и обратно, а также указать определенные действия узла может и не может принимать во время выполнения кода.</span><span class="sxs-lookup"><span data-stu-id="701c3-144">`IHostTaskManager` allows the CLR to create and manage tasks, to provide hooks for the host to take action when control transfers from managed to unmanaged code and vice versa, and to specify certain actions the host can and cannot take during code execution.</span></span>
