# <a name="iappdomainbinding-interface"></a>Интерфейс IAppDomainBinding
Предоставляет метод, который вызывается общеязыковой среды выполнения (CLR), для уведомления ведущего приложения о том, что домен приложения был создан.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание:|  
|------------|-----------------|  
|[Метод OnAppDomain](../../../../docs/framework/unmanaged-api/hosting/iappdomainbinding-onappdomain-method.md)|Вызывается общеязыковой среды выполнения (CLR), для уведомления узла о том, что домен приложения был создан.|
