# <a name="ehostapplicationpolicy-enumeration"></a>Перечисление EHostApplicationPolicy
Указывает, как изменить [IHostAssemblyManager](../../../../docs/framework/unmanaged-api/hosting/ihostassemblymanager-interface.md) объект интерфейса. Это перечисление является устаревшим.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
typedef enum _HostApplicationPolicy {  
    HOST_APPLICATION_BINDING_POLICY = 1  
} EHostApplicationPolicy;  
```
