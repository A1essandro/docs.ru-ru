# <a name="idebuggerinfo-interface"></a>Интерфейс IDebuggerInfo
Предоставляет методы для получения сведений о состоянии служб отладки.  
  
## <a name="methods"></a>Методы  
  
|Метод|Описание:|  
|------------|-----------------|  
|[Метод IsDebuggerAttached](../../../../docs/framework/unmanaged-api/hosting/idebuggerinfo-isdebuggerattached-method.md)|Получает значение, показывающее, присоединен ли управляемый отладчик этого процесса.|
