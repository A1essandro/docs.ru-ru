# <a name="hosting-coclasses"></a>Размещение коклассов
В этом разделе описываются неуправляемые компонентные классы API, которые использует API размещения.  
  
## <a name="in-this-section"></a>В этом разделе  
 [Кокласс CLRRuntimeHost](../../../../docs/framework/unmanaged-api/hosting/clrruntimehost-coclass.md)  
 Предоставляет интерфейсы для выполнения управляемого кода средой выполнения.  
  
 [Кокласс ComCallUnmarshal](../../../../docs/framework/unmanaged-api/hosting/comcallunmarshal-coclass.md)  
 Предоставляет интерфейсы для управления маршалингом указателей интерфейса.  
  
 [Кокласс CorRuntimeHost](../../../../docs/framework/unmanaged-api/hosting/corruntimehost-coclass.md)  
 Предоставляет интерфейсы для управления приложениями, которые выполняются в среде CLR.  
  
 [Кокласс TypeNameFactory](../../../../docs/framework/unmanaged-api/hosting/typenamefactory-coclass.md)  
 Предоставляет интерфейс для управления деконструкцией имени типа.
