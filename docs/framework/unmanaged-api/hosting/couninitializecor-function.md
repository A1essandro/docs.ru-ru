# <a name="couninitializecor-function"></a>Функция CoUninitializeCor
`CoUninitializeCor` устарел.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
STDAPI_(void) CoUninitializeCor(void);  
```  
  
## <a name="remarks"></a>Примечания  
 Общеязыковая среда выполнения не может быть выгружен из процесса. Чтобы полностью удалить среду выполнения из запущенного процесса, необходимо завершить работу этого процесса.
