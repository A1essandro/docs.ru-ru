# <a name="cormarkthreadinthreadpool-function"></a>Функция CorMarkThreadInThreadPool
Помечает текущий выполняемый поток из пула потоков для выполнения управляемого кода. Начиная с .NET Framework версии 2.0, эта функция не делает ничего. Он не является обязательным и может быть удален из кода. Эта функция устарела в [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)].  
  
## <a name="syntax"></a>Синтаксис  
  
```  
void CorMarkThreadInThreadPool ();  
```
