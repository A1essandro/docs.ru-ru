# <a name="imetadataemitsetmoduleprops-method"></a>Метод IMetaDataEmit::SetModuleProps
Обновляет ссылки на модуль, определяются в предыдущем вызове [IMetaDataEmit::DefineModuleRef](../../../../docs/framework/unmanaged-api/metadata/imetadataemit-definemoduleref-method.md).  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT SetModuleProps (   
    [in]  LPCWSTR     szName  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `szName`  
 [in] Имя модуля в кодировке Юникод. Это только имя файла, а не полное имя пути.
