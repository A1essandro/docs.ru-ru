# <a name="imetadataemitsavetostream-method"></a>Метод IMetaDataEmit::SaveToStream
Сохраняет все метаданные в текущей области в указанный `IStream`.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
HRESULT SaveToStream (   
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>Параметры  
 `pIStream`  
 [in] Поток для записи для сохранения.  
  
 `dwSaveFlags`  
 [in] Зарезервировано. Должен равняться нулю.
