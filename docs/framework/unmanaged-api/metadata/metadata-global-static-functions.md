# <a name="metadata-global-static-functions"></a>Глобальные статические функции метаданных
Начиная с версии [!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)], документация по `_Cor*`, `CoInitialize*`, `CoUninitialize*`, и `CoEEShutDownCOM` функций, которые относятся к размещению, находится в папке [устаревшие функции размещения CLR](../../../../docs/framework/unmanaged-api/hosting/deprecated-clr-hosting-functions.md). Не существует глобальных статических функций, относящихся к метаданным.
