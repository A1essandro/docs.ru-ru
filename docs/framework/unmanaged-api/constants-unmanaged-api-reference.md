# Константы (справочные сведения о неуправляемых API)
В этом разделе описывается тип языка, поставщика языка и константы типа документа, которые определены в файле CorSym.idl.  
  
## Константы типов языков  
 Следующая таблица показывает язык константы типа, которые представляют идентификаторы GUID, обозначающие языки программирования.  
  
|Символ|Описание|  
|------------|--------------|  
|CorSym\_LanguageType\_C|Указывает язык C.|  
|CorSym\_LanguageType\_CPlusPlus|Указывает язык C\+\+.|  
|CorSym\_LanguageType\_CSharp|Указывает язык C\#.|  
|CorSym\_LanguageType\_Basic|Указывает основной язык.|  
|CorSym\_LanguageType\_Java|Обозначает язык Java.|  
|CorSym\_LanguageType\_Cobol|Указывает язык COBOL.|  
|CorSym\_LanguageType\_Pascal|Указывает языка Pascal.|  
|CorSym\_LanguageType\_ILAssembly|Указывает код сборки Microsoft промежуточного языка MSIL.|  
|CorSym\_LanguageType\_JScript|Указывает язык JScript.|  
|CorSym\_LanguageType\_SMC|Обозначает язык SMC.|  
|CorSym\_LanguageType\_MCPlusPlus|Указывает язык C\+\+ включена для платформы .NET Framework.|  
  
## Константы поставщиков языков  
 Следующая таблица показывает язык константы поставщиков, которые представляют идентификаторы GUID, обозначающие поставщиков языков программирования.  
  
|Символ|Описание|  
|------------|--------------|  
|CorSym\_LanguageVendor\_Microsoft|Обозначает корпорацию Майкрософт.|  
  
## Константы типа документа  
 Следующая таблица показывает документ константы типа, которые представляют идентификаторы GUID, обозначающие типы документов.  
  
|Символ|Описание|  
|------------|--------------|  
|CorSym\_DocumentType\_Text|Указывает текстовый документ.|  
|CorSym\_DocumentType\_MC|Указывает нетекстовый документ.|
