# UI Automation Specification and Community Promise
Модель автоматизации пользовательского интерфейса обеспечивает программный доступ к большинству элементам интерфейса пользователя, включая вспомогательные технологические продукты, такие как средства чтения с экрана, для предоставления конечным пользователям данных о пользовательском интерфейсе и управления им другими средствами, отличными от стандартного ввода данных.
