# Загрузка пакета обработчика веб-токенов JSON
В этом разделе обсуждается порядок загрузки и использования обработчика веб\-токенов JSON в проекте.  
  
## Загрузка обработчика веб\-токенов JSON  
 Расширение обработчика токена Интернета JSON доступно как пакет NuGet, который добавляет необходимые сборки и ссылки в проект.  Если NuGet еще не установлен, перейдите н [nuget.org](http://nuget.org) и загрузите его.  Журнал управления версиями для расширения можно просмотреть, посетив его страницу в NuGet с описанием [веб\-обработчика токенов JSON в NuGet](http://www.nuget.org/packages/System.IdentityModel.Tokens.Jwt/)  
  
#### Загрузка обработчика веб\-токенов JSON с помощью графического интерфейса диспетчера пакетов  
  
1.  В **обозревателе решений** Visual Studio щелкните проект правой кнопкой мыши и выберите **Управление пакетами NuGet**.  
  
2.  В поле **Управление пакетами NuGet** щелкните поле поиска и введите `JWT Token Handler` и нажмите клавишу **ВВОД**.  
  
3.  В области результатов нажмите кнопку **Установить** для первого результата.  
  
4.  Пакет начнет скачиваться.  Прежде чем он будет добавлен в проект, появится диалоговое окно принятия лицензионного соглашения.  Если вы согласны с условиями лицензионного соглашения, нажмите кнопку **Принимаю**.  
  
5.  Последние сборки обработчика веб\-токенов JSON скачаны и будут добавлены в проект.  
  
#### Загрузка обработчика веб\-токенов JSON с помощью консоли диспетчера пакетов  
  
1.  В Visual Studio выберите **Сервис**, **Library Package Manager** и **Package Manager Console**.  
  
2.  Откроется окно **Консоль диспетчера пакетов**.  Выберите следующий текст и нажмите клавишу **ВВОД**:  
  
    ```powershell  
    Install-Package System.IdentityModel.Tokens.Jwt  
    ```  
  
3.  Последние сборки обработчика веб\-токенов JSON скачаны и будут добавлены в проект.
