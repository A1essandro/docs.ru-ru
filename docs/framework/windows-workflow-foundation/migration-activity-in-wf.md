# Действие &#171;Migration&#187; в WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] предоставляет <xref:> System.Activities.Statements.Interop?qualifyHint=False&autoUpgrade=True действия для выполнения действий, которые являются производными от <xref:System.WorkflowModel.Activities.Activity> в рамках рабочего процесса, который основан на <xref:System.Activities.Activity>. [!INCLUDE[crdefault](../../../includes/crdefault-md.md)]  [руководство по миграции](../../../docs/framework/windows-workflow-foundation//migration-guidance.md) раздел.  
  
> [!NOTE]
>   <xref:System.Activities.Statements.Interop> действия не отображается в области элементов конструктора рабочих процессов, если проект рабочего процесса содержит его **Требуемая версия .NET Framework** задано значение **.Net Framework 4** или более поздней версии.
