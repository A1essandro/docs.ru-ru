# 1450 - WfMessageSent
## Свойства  
  
|||  
|-|-|  
|Идентификатор|1450|  
|Ключевые слова|WFServices|  
|Уровень|Сведения|  
|Канал|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Описание  
 Указывает, что сообщение было отправлено рабочим процессом.  
  
## Сообщение  
 Сообщение, отправленное из рабочего процесса
