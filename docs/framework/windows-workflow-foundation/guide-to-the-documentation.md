# Руководство по документации по рабочим процессам Windows
В этом разделе описывается использование документации по Windows Workflow Foundation.Документы, на которые указывают ссылки — это рекомендуемые отправные точки, сгруппированные по интересам и уровням подготовки.  
  
## Знакомство с программированием Windows Workflow Foundation  
  
-   При отсутствии опыта в программировании в Windows Workflow Foundation и желании ознакомиться с образцом работающих приложений см. разделы в [Образцы Windows Workflow](../../../docs/framework/windows-workflow-foundation/samples/index.md).  
  
-   Описание различных моделей управления потоками Windows Workflow Foundation см. в разделе [Разработка рабочих процессов](../../../docs/framework/windows-workflow-foundation//designing-workflows.md).  
  
-   Учебники с пошаговыми руководствами по созданию приложений Windows Workflow Foundation см. в разделе [Учебник по началу работы](../../../docs/framework/windows-workflow-foundation//getting-started-tutorial.md).  
  
-   Описание принципов, лежащих в основе Windows Workflow Foundation, см. в разделе [Общие сведения о рабочих процессах Windows](../../../docs/framework/windows-workflow-foundation//conceptual-overview.md).  
  
-   Список терминов, используемых в документации Windows Workflow Foundation, см. в разделе [Глоссарий Windows Workflow Foundation для .NET Framework 4.5](../../../docs/framework/windows-workflow-foundation//glossary.md).  
  
-   Список новых концепций и функций см. в разделе [Новые возможности в Windows Workflow Foundation \(WF\)](../../../docs/framework/windows-workflow-foundation//whats-new.md).  
  
## Программирование изнутри  
  
-   Более подробное описание объектной модели Windows Workflow Foundation см. в разделе [Архитектура рабочих процессов Windows](../../../docs/framework/windows-workflow-foundation//architecture.md).  
  
-   Если вы готовы перейти к разработке приложения, см. раздел [Программирование в Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation//programming.md).  
  
-   Если планируется расширить или настроить Windows Workflow Foundation в соответствии с конкретными требованиями, см. раздел [Расширение Windows Workflow Foundation](../../../docs/framework/windows-workflow-foundation//extend.md).
