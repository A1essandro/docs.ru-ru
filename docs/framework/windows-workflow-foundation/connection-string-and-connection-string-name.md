# Строка соединения и имя строки соединения
Свойство **Строка соединения** указывает строку соединения, которую должно использовать хранилище экземпляров рабочего процесса SQL для соединения с базой данных сохраняемости.Этот параметр является необязательным.Свойство **Имя строки соединения** указывает имя именованной строки соединения, которую должно использовать хранилище экземпляров рабочего процесса SQL для соединения с базой данных сохраняемости.Этот параметр является необязательным.Необходимо задать значение свойства «Имя строки соединения» или свойства «Строка соединения», чтобы хранилище экземпляров рабочих процессов SQL не использовало именованную строку соединения по умолчанию **DefaultSqlWorkflowInstanceStoreConnectionString**.
