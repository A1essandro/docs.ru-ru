# <a name="validation"></a>Проверка
В этом разделе содержатся образцы, демонстрирующие проверку действий.  
  
## <a name="in-this-section"></a>Содержание  
 [Проверка внешнего действия](../../../../docs/framework/windows-workflow-foundation/samples/external-activity-validation.md)  
 В этом образце показано, как добавить логику проверки во встроенное действие, созданное другим пользователем.  
  
 [Базовая проверка](../../../../docs/framework/windows-workflow-foundation/samples/basic-validation.md)  
 Демонстрируется действие, `CreateProduct`, проверяющее, что аргумент `Cost` меньше или равен аргументу `Price`.  
  
 [OverloadGroups](../../../../docs/framework/windows-workflow-foundation/samples/overloadgroups.md)  
 Демонстрирует действие, `CreateLocation`, содержащее обязательные и необязательные аргументы и позволяющее пользователю предоставить один из двух разных наборов аргументов.  
  
 [Проверка связей действий](../../../../docs/framework/windows-workflow-foundation/samples/activity-relationships-validation.md)  
 Демонстрирует, как проверять связь между действиями.
