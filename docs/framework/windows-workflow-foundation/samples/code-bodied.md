# Образцы в коде
В этом разделе содержатся образцы настраиваемых действий, созданные декларативно в коде.  
  
## В этом подразделе

 В этом образце показано, как разработать настраиваемое действие, которое создает закладку для получения внешних входных данных.  
  
 [Пользовательское составное действие, использующее собственное действие](../../../../docs/framework/windows-workflow-foundation/samples/custom-composite-using-native-activity.md)

 [Свойства выполнения](../../../../docs/framework/windows-workflow-foundation/samples/execution-properties.md)  
 В этом образце показано, как определить и использовать свойство выполнения в настраиваемом действии.  
  
 [Предоставление и вызов действий ActivityActions](../../../../docs/framework/windows-workflow-foundation/samples/exposing-and-invoking-activityactions.md)  
 В этом образце показано, как разработать настраиваемое действие, имеющее <xref:System.Activities.ActivityAction>.  
  
 [Использование AsyncOperationContext в действии](../../../../docs/framework/windows-workflow-foundation/samples/using-asyncoperationcontext-in-an-activity-sample.md)

 [Пользовательское действие Hello World](../../../../docs/framework/windows-workflow-foundation/samples/hello-world-custom-activity.md)  
 В этом образце показано несколько ключевых возможностей [!INCLUDE[wf](../../../../includes/wf-md.md)], включая создание простого настраиваемого действия.  
  
 [Динамические аргументы](../../../../docs/framework/windows-workflow-foundation/samples/dynamic-arguments.md)  
 В этом образце показано, как реализовать действие, аргументы которого определяются объектом\-получателем, а не автором действия.
