# Настраиваемые действия
Образцы в этом разделе демонстрируют, как создавать пользовательские действия в [!INCLUDE[wf](../../../../includes/wf-md.md)].  
  
## В этом подразделе  
 [Образцы в коде](../../../../docs/framework/windows-workflow-foundation/samples/code-bodied.md)  
 В разделе содержатся образцы пользовательских действий, созданные декларативно в коде.  
  
 [Составные](../../../../docs/framework/windows-workflow-foundation/samples/composite.md)  
 Также содержатся образцы пользовательских действий, состоящих из других действий.  
  
 [Пользовательские конструкторы действий](../../../../docs/framework/windows-workflow-foundation/samples/custom-activity-designers.md)  
 Также содержатся образцы действий, использующих пользовательские конструкторы для отображения в конструкторе рабочих процессов.  
  
 [Проверка](../../../../docs/framework/windows-workflow-foundation/samples/validation.md)  
 Также содержатся образцы, демонстрирующие проверку действий во время разработки.
