# <a name="bookmarks"></a>Закладки
В этом образце показано, как разработать настраиваемое действие, которое создает закладку для получения внешних входных данных. В образец также входит простое консольное приложение, которое использует настраиваемое действие в рабочем процессе и показывает, как обнаруживать и возобновлять закладки, связанные с выполняющимся экземпляром рабочего процесса. [!INCLUDE[crabout](../../../../includes/crabout-md.md)]закладки, в разделе [закладки](../../../../docs/framework/windows-workflow-foundation/bookmarks.md).  
  
### <a name="to-set-up-build-and-run-the-sample"></a>Настройка, сборка и выполнение образца  
  
1.  Откройте образец решения Bookmarks.sln в среде [!INCLUDE[vs2010](../../../../includes/vs2010-md.md)].  
  
2.  Для построения решения нажмите CTRL+SHIFT+B.  
  
3.  Чтобы запустить образец, нажмите клавиши CTRL+F5.

>  Образцы уже могут быть установлены на компьютере. Перед продолжением проверьте следующий каталог (по умолчанию).
