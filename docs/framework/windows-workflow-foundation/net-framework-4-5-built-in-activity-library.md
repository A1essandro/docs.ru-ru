# <a name="net-framework-45-built-in-activity-library"></a>Встроенная библиотека действий .NET Framework 4.5

[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] включает новую библиотеку действий с расширенными функциональными возможностями. Типы действий включают следующее:

- [Поток управления](../../../docs/framework/windows-workflow-foundation/control-flow-activities-in-wf.md)

- [Блок-схема](../../../docs/framework/windows-workflow-foundation/flowchart-activities-in-wf.md)

- [Конечный автомат](../../../docs/framework/windows-workflow-foundation/state-machine-activities-in-wf.md)

- [Действия обмена сообщениями](../../../docs/framework/wcf/feature-details/messaging-activities.md)

- [Среда выполнения](../../../docs/framework/windows-workflow-foundation/runtime-activities-in-wf.md)

- [Примитивы](../../../docs/framework/windows-workflow-foundation/primitives-activities-in-wf.md)

- [Транзакция](../../../docs/framework/windows-workflow-foundation/transaction-activities-in-wf.md)

- [Коллекция](../../../docs/framework/windows-workflow-foundation/collection-activities-in-wf.md)

- [Обработка ошибок](../../../docs/framework/windows-workflow-foundation/error-handling-activities-in-wf.md)

- [Миграция](../../../docs/framework/windows-workflow-foundation/migration-activity-in-wf.md)

Действия в библиотеку встроенных действий может быть рабочие процессы или использовать совместно с пользовательскими действиями. Действия из библиотеки действий запечатаны; они не предназначены для использования для создания новой функциональности с помощью наследования.
