# Разработка рабочих процессов
Подразделы данного раздела описывают принципы управления потоками, используемые при разработке рабочих процессов.  
  
## В этом подразделе  
 [Рабочие процессы блок\-схемы](../../../docs/framework/windows-workflow-foundation//flowchart-workflows.md)  
 Демонстрирует процесс моделирования рабочих процессов с помощью известных принципов блок\-схем.  
  
 [Процедурные рабочие процессы](../../../docs/framework/windows-workflow-foundation//procedural-workflows.md)  
 Демонстрирует процесс создания рабочих процессов с помощью базового принципа последовательного выполнения.  
  
 [Рабочие процессы конечного автомата](../../../docs/framework/windows-workflow-foundation//state-machine-workflows.md)  
 Описывает процесс использования рабочих процессов блок\-схемы для реализации рабочих процессов, управляемых событиями, в [!INCLUDE[netfx40_short](../../../includes/netfx40-short-md.md)], в качестве замены модели конечного автомата, используемой в прежних версиях.  
  
 [Разработка рабочих процессов, действий и выражений с помощью императивного кода](../../../docs/framework/windows-workflow-foundation//authoring-workflows-activities-and-expressions-using-imperative-code.md)  
 Описывает способы создания рабочего процесса с императивным кодом.
