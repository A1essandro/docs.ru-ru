# Использование протоколов приложений
Платформа .NET Framework поддерживает часто используемые протоколы интернет\-приложений.  Этот раздел включает сведения об использовании протокола [HTTP](../../../docs/framework/network-programming/http.md), [TCP](../../../docs/framework/network-programming/tcp-udp.md) и [UDP](../../../docs/framework/network-programming/tcp-udp.md), а также сведения об использовании интерфейса [Windows sockets](../../../docs/framework/network-programming/sockets.md) для реализации пользовательских протоколов.
