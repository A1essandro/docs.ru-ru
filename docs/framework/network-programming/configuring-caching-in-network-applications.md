# Настройка кэширования в сетевых приложениях
Чтобы настроить кэширование, необходимо задать политику кэширования на уровне приложения или <xref:System.Net.WebRequest>.  В следующих разделах содержатся примеры кода, демонстрирующие настройка приложения и запросы использовать кэширование.  
  
-   [Практическое руководство. Задайте расположение хранения\- На политику кэша для приложения](../../../docs/framework/network-programming/how-to-set-a-location-based-cache-policy-for-an-application.md)  
  
-   [Практическое руководство. Установите Time\- Based значение по умолчанию политику кэша для приложения](../../../docs/framework/network-programming/how-to-set-the-default-time-based-cache-policy-for-an-application.md)  
  
-   [Практическое руководство. Настройка Time\- на основе политики кэша](../../../docs/framework/network-programming/how-to-customize-a-time-based-cache-policy.md)  
  
-   [Практическое руководство. Задана политика кэша для запроса](../../../docs/framework/network-programming/how-to-set-cache-policy-for-a-request.md)  
  
 Можно также настроить политику кэширования с помощью приложения или подвергнуть файлы конфигурации механической обработке.  Дополнительные сведения см. в разделе &#124; [Элемент \<requestCaching\> \(параметры сети\)](../../../docs/framework/configure-apps/file-schema/network/requestcaching-element-network-settings.md).
