 <span data-ttu-id="0445a-157">Обработчик инерции позволяет приложениям экстраполировать расположение, ориентацию и другие свойства элемента пользовательского интерфейса, имитируя поведение, существующее в реальном мире.</span><span class="sxs-lookup"><span data-stu-id="0445a-157">The inertia processor enables applications to extrapolate location, orientation, and other properties of a UI element by simulating real-world behavior.</span></span>
