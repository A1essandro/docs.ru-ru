<span data-ttu-id="e0a51-103">Классы процессоров *манипуляций* и *инерции* помогают приложению перемещать элементы пользовательского интерфейса естественным и интуитивно понятным способом.</span><span class="sxs-lookup"><span data-stu-id="e0a51-103">The *manipulations* and *inertia* processor classes assist your application in moving user interface (UI) elements on the screen in a natural and intuitive way.</span></span> <span data-ttu-id="e0a51-104">Манипуляции позволяют пользователям перемещать и поворачивать компоненты, а также изменять их размер, используя манипуляторы, такие как устройство мыши, перо или (в сценарии непосредственного касания) пальцы пользователей.</span><span class="sxs-lookup"><span data-stu-id="e0a51-104">Manipulations enable users to move, rotate, and resize components by using manipulators, such as a mouse device, a stylus, or (in a direct touch-enabled scenario) their fingers.</span></span> <span data-ttu-id="e0a51-105">*Инерция* моделирует поведение, характерное для реального мира, для движущихся элементов пользовательского интерфейса, имитируя силы трения в элементах.</span><span class="sxs-lookup"><span data-stu-id="e0a51-105">*Inertia* emulates real-world behavior for UI elements that are in motion by simulating friction forces on the elements.</span></span> <span data-ttu-id="e0a51-106">Это позволяет элементам постепенно замедлять свое движение (линейное и угловое) перед остановкой.</span><span class="sxs-lookup"><span data-stu-id="e0a51-106">This enables elements to gradually slow their movement (both linear and angular) before coming to a stop.</span></span>  
  
 <span data-ttu-id="e0a51-107">Обработка манипуляций и инерции может использоваться в различных платформах пользовательских интерфейсов, например в Microsoft Windows Presentation Foundation (WPF) и Microsoft XNA.</span><span class="sxs-lookup"><span data-stu-id="e0a51-107">Manipulations and inertia processing can be used in a variety of UI frameworks, such as Microsoft Windows Presentation Foundation (WPF) or Microsoft XNA.</span></span>
