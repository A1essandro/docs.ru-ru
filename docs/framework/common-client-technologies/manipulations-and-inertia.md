# <a name="manipulations-and-inertia"></a>Манипуляции и инерция
Классы процессоров *манипуляций* и *инерции* помогают приложению перемещать элементы пользовательского интерфейса естественным и интуитивно понятным способом. Манипуляции позволяют пользователям перемещать и поворачивать компоненты, а также изменять их размер, используя манипуляторы, такие как устройство мыши, перо или (в сценарии непосредственного касания) пальцы пользователей. *Инерция* моделирует поведение, характерное для реального мира, для движущихся элементов пользовательского интерфейса, имитируя силы трения в элементах. Это позволяет элементам постепенно замедлять свое движение (линейное и угловое) перед остановкой.  
  
 Обработка манипуляций и инерции может использоваться в различных платформах пользовательских интерфейсов, например в Microsoft Windows Presentation Foundation (WPF) и Microsoft XNA.
