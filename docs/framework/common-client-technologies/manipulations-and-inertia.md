# Manipulations and Inertia
Классы процессоров *манипуляций* и *инерции* помогают приложению перемещать элементы пользовательского интерфейса естественным и интуитивно понятным способом.  Манипуляции позволяют пользователям перемещать и поворачивать компоненты, а также изменять их размер, используя манипуляторы, такие как устройство мыши, перо или \(в сценарии непосредственного касания\) пальцы пользователей.  *Инерция* эмулирует поведение реального мира для движущихся элементов пользовательского интерфейса, имитируя силы трения для этих элементов.  Это позволяет элементам перед остановкой постепенно замедлять свое движение \(как линейное, так и вращательное\).  
  
 Обработка манипуляций и инерции может использоваться в различных платформах пользовательских интерфейсов, например в Microsoft Windows Presentation Foundation \(WPF\) и Microsoft XNA.
