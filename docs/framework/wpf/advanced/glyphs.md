# Глифы
Глифы являются изображениями знаков нижнего уровня, которые должны быть нарисованы на экране.  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] предоставляет прямой доступ к глифам для клиентов, которым требуется перехватывать и сохранять текст после форматирования.
