# XAML в WPF
[!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)] — это язык разметки для декларативной разработки приложений.  В [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] реализован обработчик языка XAML и имеется поддержка языка XAML.  Типы [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] реализованы таким образом, что они могут служить необходимой платформой типов для представления XAML. Как правило, большую часть пользовательского интерфейса приложения [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] можно создать с помощью разметки XAML.
