### <a name="graphics-rendering-tiers"></a><span data-ttu-id="e006a-118">Уровни графической отрисовки</span><span class="sxs-lookup"><span data-stu-id="e006a-118">Graphics Rendering Tiers</span></span>

 <span data-ttu-id="e006a-125">Ниже перечислены возможности графического оборудования, которые сильнее всего влияют на уровни отрисовки.</span><span class="sxs-lookup"><span data-stu-id="e006a-125">The features of the graphics hardware that most impact the rendering tier levels are:</span></span>  
  
-   <span data-ttu-id="e006a-126">**Видеопамять**. -   <span data-ttu-id="e006a-130">**Поддержка мультитекстурирования**.  <span data-ttu-id="e006a-132">Построитель текстуры, вершинный построитель текстуры и возможности мультитекстурирования используются для определения конкретных [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] уровни версий, которые в свою очередь, используются для определения различных уровней отрисовки в [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span><span class="sxs-lookup"><span data-stu-id="e006a-132">The pixel shader, vertex shader, and multitexture features are used to define specific [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version levels, which, in turn, are used to define the different rendering tiers in [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span></span>  
  
 <span data-ttu-id="e006a-133">Возможности графического оборудования определяют возможности отрисовки приложения [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].</span><span class="sxs-lookup"><span data-stu-id="e006a-133">The features of the graphics hardware determine the rendering capability of a [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] application.</span></span> <span data-ttu-id="e006a-134">Система [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] определяет три уровня отрисовки.</span><span class="sxs-lookup"><span data-stu-id="e006a-134">The [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] system defines three rendering tiers:</span></span>

-   <span data-ttu-id="e006a-137">**Подготовка к просмотру уровня 1** Частичное аппаратное ускорение графики.</span><span class="sxs-lookup"><span data-stu-id="e006a-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="e006a-138">[!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] Версия больше или равна 7.0, и **меньшей** версии 9.0.</span><span class="sxs-lookup"><span data-stu-id="e006a-138">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>  
  
-   <span data-ttu-id="e006a-139">**Уровень отрисовки 2**. Большинство графических функций используют аппаратное ускорение графики.</span><span class="sxs-lookup"><span data-stu-id="e006a-139">**Rendering Tier 2** Most graphics features use graphics hardware acceleration.</span></span> <span data-ttu-id="e006a-140">Уровень версии [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] больше или равен версии 9.0.</span><span class="sxs-lookup"><span data-stu-id="e006a-140">The [!INCLUDE[TLA2#tla_dx](../../../../includes/tla2sharptla-dx-md.md)] version level is greater than or equal to version 9.0.</span></span>
