# Ресурсы WPF.
Ресурс \- это объект, который можно использовать в разных местах в приложении.  WPF поддерживают различные типы ресурсов.  Эти ресурсы в основном 2 типа ресурсов: Ресурсы XAML и файлы данных resource.  Примерами ресурсов XAML входят кисти и стили.  Файлы данных resource файлы данных non\-исполняемого файла, приложению требуется.
