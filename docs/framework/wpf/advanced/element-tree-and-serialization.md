# Дерево элементов и сериализация
Связь между элементами программирования WPF представлена в виде дерева.  Например, пользовательский интерфейс приложения, созданный в XAML, можно представить как дерево объектов.  Дерево элементов можно подразделить на два дискретных и в то же время параллельных дерева: [логическое дерево](GTMT) и [визуальное дерево](GTMT).  Сериализация в WPF приводит к тому, что состояния этих двух деревьев и состояние приложения сохраняются и записываются в файл в формате XAML.
