# Базовые элементы
Четыре ключевых класса ― <xref:System.Windows.UIElement>, <xref:System.Windows.ContentElement>, <xref:System.Windows.FrameworkElement> и <xref:System.Windows.FrameworkContentElement> ― реализуют значительный процент от общих функциональных возможностей элементов, доступных в программировании WPF.  В данном пакете SDK эти четыре класса называются базовыми классами элементов.
