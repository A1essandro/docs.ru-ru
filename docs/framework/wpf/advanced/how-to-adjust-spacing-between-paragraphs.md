# Практическое руководство. Изменение интервалов между абзацами
В этом примере демонстрируется настройка или устранение интервалов между абзацами в содержимом нефиксированного формата.  
  
 В содержимом нефиксированного формата дополнительное расстояние, возникающее между абзацами, является результатом набора полей этих абзацев. Поэтому интервалами между абзацами можно управлять при помощи корректировки полей этих абзацев.  Чтобы полностью убрать дополнительное расстояние между двумя абзацами, установите поля для абзацев в значение **0**.  Для достижения одинакового интервала между абзацами во всем <xref:System.Windows.Documents.FlowDocument>, используйте стилизацию для установки одинакового значения поля для всех абзацев в <xref:System.Windows.Documents.FlowDocument>.  
  
 Важно обратить внимание, что поля для двух соседних абзацев заменены на большее поле, а не увеличены вдвое.  Таким образом, если два соседних абзаца имеют поля 20 пикселей и 40 пикселей соответственно, результирующий интервал между абзацами будет 40 пикселей, что является большим из двух значений полей.
