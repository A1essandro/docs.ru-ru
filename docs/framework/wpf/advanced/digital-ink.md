# <a name="digital-ink"></a>Рукописный ввод
В этом разделе обсуждается использование рукописного ввода в [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]. В большинстве случаев найден только в SDK для планшетных ПК, рукописного теперь доступен в ядро Windows Presentation Foundation. Это означает, что теперь можно разрабатывать полноценные приложений для планшетных ПК с помощью возможностей Windows Presentation Foundation.
