# Рукописный ввод
В этом разделе обсуждается использование рукописного ввода в среде [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].  Традиционная составляющая набора SDK для Tablet PC, функция рукописного ввода, теперь доступна в Windows Presentation Foundation.  Это означает, что теперь можно разрабатывать полноценные приложения Tablet PC, используя возможности Windows Presentation Foundation.
