# Фигуры
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] обеспечивает поддержку рисования основных фигур с помощью класса <xref:System.Windows.Shapes.Shape>.  Объекты фигур являются производными от класса <xref:System.Windows.FrameworkElement>, присутствуют в системе макета и могут быть нарисованы просто с помощью [!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)].
