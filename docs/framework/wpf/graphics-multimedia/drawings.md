# Объекты рисования
Объекты <xref:System.Windows.Media.Drawing> используются для эффективного рисования фигур, изображений или текста.  Объекты рисования используются при рисовании с помощью объекта <xref:System.Windows.Media.DrawingBrush> или программирования с помощью объектов <xref:System.Windows.Media.Visual>.
