# IEnumRAWINPUTDEVIC:Reset
Сбрасывает последовательность перечисления в начало.  
  
## Синтаксис  
  
```  
HRESULT Reset();  
```  
  
## Значение свойства или возвращаемое значение  
 HRESULT: S\_OK.
