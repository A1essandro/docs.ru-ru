# Окна в приложениях WPF
Пользователи взаимодействуют с приложениями через окна.  Основным предназначением окна является размещение и отображение содержимого окна.  Тип содержимого, размещаемого в окне, зависит от типа данных, с которым работает приложение. Они могут включать файлы мультимедиа, страницы XAML, веб\-страницы, документы, таблицы и записи базы данных, сведения о системе.
