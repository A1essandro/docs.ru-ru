# Размещение WPF-приложений
Приложения браузера XAML \(XBAP\) системы WPF — это многофункциональные клиентские приложения, которые можно разворачивать на веб\-сервере и запускать в браузере.  Ведущее приложение WPF \(PresentationHost.exe\) зарегистрировано как оболочка и обработчик MIME для файлов XBAP и XAML.  Поэтому Internet Explorer понимает, что при запуске приложения XBAP необходимо запустить ведущее приложение WPF.  Пользователи Firefox могут установить дополнительные компоненты Firefox, чтобы обеспечить поддержку размещения приложений XBAP в этом браузере.  Приложения XBAP можно размещать в других браузерах или изолированных приложениях, используя собственные размещающие API браузеров, предоставляемые системой WPF.
