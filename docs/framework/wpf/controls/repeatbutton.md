# <a name="repeatbutton"></a>RepeatButton
<xref:System.Windows.Controls.Primitives.RepeatButton> Аналогичен <xref:System.Windows.Controls.Button>. Тем не менее <xref:System.Windows.Controls.Primitives.RepeatButton> обеспечивает элементы, можно контролировать, когда и как <xref:System.Windows.Controls.Primitives.ButtonBase.Click> событием.  
  
 На рисунке показан пример три состояния кнопок повтора управления, по умолчанию, PointerFocused и нажата кнопка. Первая кнопка показывает состояние по умолчанию <xref:System.Windows.Controls.Primitives.RepeatButton>. Во втором показан как внешний вид кнопки меняется при наведении указателя мыши на кнопку, предоставив ему фокус. Последняя кнопка показывает внешний вид <xref:System.Windows.Controls.Primitives.RepeatButton> когда пользователь нажимает кнопку мыши над элементом управления.
