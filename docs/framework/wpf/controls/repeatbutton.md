# RepeatButton
<xref:System.Windows.Controls.Primitives.RepeatButton> аналогична <xref:System.Windows.Controls.Button>.  Тем не менее, элементы <xref:System.Windows.Controls.Primitives.RepeatButton> дают возможность контролировать, когда и как будет возникать событие <xref:System.Windows.Controls.Primitives.ButtonBase.Click>.  
  
 На следующем рисунке показан пример трех состояний элемента управления repeat button: Default, PointerFocused и Pressed.  Первая кнопка показывает состояние <xref:System.Windows.Controls.Primitives.RepeatButton> по умолчанию.  Вторая показывает, как изменяется внешний вид кнопки, когда указатель мыши наводится на нее, предоставляя ей фокус.  Последняя кнопка показывает внешний вид <xref:System.Windows.Controls.Primitives.RepeatButton>, когда пользователь нажимает кнопку мыши над элементом управления.
