# ComboBox
Элемент управления <xref:System.Windows.Controls.ComboBox> предоставляется пользователям в виде списка параметров.  Этот список является открытым и скрытые, когда элемент управления развернут и свернут.  В состоянии по умолчанию список свернут, отображая при этом только один вариант.  Пользователь нажимает кнопку, чтобы просмотреть полный список параметров.
