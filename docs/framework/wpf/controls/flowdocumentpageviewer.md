# FlowDocumentPageViewer
Элемент управления <xref:System.Windows.Controls.FlowDocumentPageViewer> используется для просмотра содержимого <xref:System.Windows.Documents.FlowDocument> на основе страниц.  В свою очередь <xref:System.Windows.Controls.FlowDocumentScrollViewer> отображает содержимое <xref:System.Windows.Documents.FlowDocument> в области с прокруткой.
