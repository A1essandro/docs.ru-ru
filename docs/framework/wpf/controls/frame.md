# Frame
Элемент управления <xref:System.Windows.Controls.Frame> поддерживает навигацию по содержимому внутри содержимого.  Размещение <xref:System.Windows.Controls.Frame> может выполнять такой корневой элемент, как <xref:System.Windows.Window>, <xref:System.Windows.Navigation.NavigationWindow>, <xref:System.Windows.Controls.Page>, <xref:System.Windows.Controls.UserControl>, <xref:System.Windows.Documents.FlowDocument>. Кроме того, данный объект может быть размещен в виде острова внутри дерева содержимого, которое принадлежит корневому элементу.
