# FlowDocumentScrollViewer
Элемент управления <xref:System.Windows.Controls.FlowDocumentScrollViewer> используется для просмотра содержимого <xref:System.Windows.Documents.FlowDocument> в прокручивающемся контейнере.  Он отличается от <xref:System.Windows.Controls.FlowDocumentPageViewer>, который отображает содержимое по страницам.
