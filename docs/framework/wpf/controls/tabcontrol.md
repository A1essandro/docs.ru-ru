# TabControl
Элементы <xref:System.Windows.Controls.TabControl> отображают содержимое на отдельных страницах, доступ к которым осуществляется выбором соответствующей вкладки.  Каждая вкладка содержит <xref:System.Windows.Controls.TabItem>.
