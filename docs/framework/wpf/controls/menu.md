# Меню
<xref:System.Windows.Controls.Menu> является элементом управления, который обеспечивает иерархическую организацию элементов, связанных с помощью команд или обработчиков событий.  Каждый элемент <xref:System.Windows.Controls.Menu> может содержать несколько элементов управления <xref:System.Windows.Controls.MenuItem>.  Каждый <xref:System.Windows.Controls.MenuItem> может вызвать команду или обработчик событий `Click`.  <xref:System.Windows.Controls.MenuItem> может также иметь несколько элементов <xref:System.Windows.Controls.MenuItem> в качестве дочерних. Это позволяет создать подменю.  
  
 Ниже показаны три различных состояния элемента управления меню.  Состоянием по умолчанию называется состояние, при котором указатель мыши не находится над <xref:System.Windows.Controls.Menu>.  Состояние фокуса возникает, когда указатель мыши перемещается через <xref:System.Windows.Controls.Menu>, а состояние нажатия возникает при нажатии кнопки мыши над <xref:System.Windows.Controls.Menu>.
