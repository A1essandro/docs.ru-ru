# TextBlock
Элемент управления <xref:System.Windows.Controls.TextBlock> обеспечивает гибкую поддержку текста для приложений [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].  Элемент в основном предназначен для основных скриптов [!INCLUDE[TLA2#tla_ui](../../../../includes/tla2sharptla-ui-md.md)], которые не требуют более одного абзаца текста.
