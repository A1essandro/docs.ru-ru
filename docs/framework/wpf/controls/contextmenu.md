# ContextMenu
<xref:System.Windows.Controls.ContextMenu> позволяет элементу управления отобразить <xref:System.Windows.Controls.Menu>, которое определено для контекста элемента управления.  Как правило, <xref:System.Windows.Controls.ContextMenu> предоставляется в [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] через правую кнопку мыши или через кнопку клавиатурного меню.  
  
 На следующем рисунке демонстрируется действие <xref:System.Windows.Controls.ContextMenu> в двух различных состояниях: состоянии по умолчанию и открытом состоянии.  В состоянии по умолчанию элемент управления свернут.  При нажатии правой кнопки мыши родительского элемента меню элемент управления раскрывается и отображает элементы меню.
