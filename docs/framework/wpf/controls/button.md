# Кнопка
Элемент управления <xref:System.Windows.Controls.Button> реагирует на пользовательский ввод с мыши, клавиатуры, пера или другого устройства ввода и инициирует событие <xref:System.Windows.Controls.Primitives.ButtonBase.Click>.  Элемент управления <xref:System.Windows.Controls.Button> является основным компонентом пользовательского интерфейса [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)], который может содержать как простое содержимое, например, текст, так и сложное, например, изображения и элементы управления <xref:System.Windows.Controls.Panel>.
