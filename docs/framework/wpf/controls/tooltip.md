# ToolTip
Всплывающая подсказка представляет собой небольшое всплывающее окно, которое появляется, когда пользователь задерживает указатель мыши на элементе, например, на <xref:System.Windows.Controls.Button>.  
  
 На следующем рисунке показан указатель мыши, указывающий на **Закрыть** <xref:System.Windows.Controls.Button>, который затем отображает отождествленную <xref:System.Windows.Controls.ToolTip>.
