# BulletDecorator
Объект <xref:System.Windows.Controls.Primitives.BulletDecorator> имеет два свойства содержимого: <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> и <xref:System.Windows.Controls.Decorator.Child%2A>.  Свойство <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> определяет <xref:System.Windows.UIElement> для использования в качестве маркера.  Свойство <xref:System.Windows.Controls.Decorator.Child%2A> определяет <xref:System.Windows.UIElement>, который визуально выравнивается с маркером.
