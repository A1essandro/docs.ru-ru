 <span data-ttu-id="262ca-107">Начиная с [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], доступен только полный распространяемый пакет.</span><span class="sxs-lookup"><span data-stu-id="262ca-107">Starting with the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], the Client Profile has been discontinued and only the full redistributable package is available.</span></span> <span data-ttu-id="262ca-108">Оптимизации, предоставляемые платформой [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], такие как меньший размер загружаемых файлов и повышенная скорость развертывания, исключили необходимость в отдельном пакете.</span><span class="sxs-lookup"><span data-stu-id="262ca-108">Optimizations provided by the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], such as smaller download size and faster deployment, have eliminated the need for a separate deployment package.</span></span> <span data-ttu-id="262ca-109">Один распространяемый пакет упрощает процесс установки и параметры развертывания приложения.</span><span class="sxs-lookup"><span data-stu-id="262ca-109">The single redistributable streamlines the installation process and simplifies your app's deployment options.</span></span>  
  
 <span data-ttu-id="262ca-110">Но если приложение предназначено для .NET Framework 4 или 3.5 и вы хотите больше узнать о клиентском профиле и его использовании, см. раздел [Клиентский профиль .NET Framework](https://msdn.microsoft.com/library/cc656912\(v=vs.100\).aspx) в документации по платформе .NET Framework 4.</span><span class="sxs-lookup"><span data-stu-id="262ca-110">However, if you are targeting the .NET Framework 4 or 3.5 and want to learn more about the Client Profile and when to use it, see [.NET Framework Client Profile](https://msdn.microsoft.com/library/cc656912\(v=vs.100\).aspx) in the .NET Framework 4 documentation.</span></span>  
  
 <span data-ttu-id="262ca-111">При установке [!INCLUDE[net_v45](../../../includes/net-v45-md.md)] клиентский профиль платформы .NET Framework 4 обновляется до полной версии платформы .NET Framework.</span><span class="sxs-lookup"><span data-stu-id="262ca-111">When you install the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], the .NET Framework 4 Client Profile is updated to the full version of the .NET Framework.</span></span> <span data-ttu-id="262ca-112">Сведения об установке [!INCLUDE[net_v45](../../../includes/net-v45-md.md)] см. в разделе [Установка .NET Framework для разработчиков](../../../docs/framework/install/guide-for-developers.md).</span><span class="sxs-lookup"><span data-stu-id="262ca-112">For information about installing the [!INCLUDE[net_v45](../../../includes/net-v45-md.md)], see [Install the .NET Framework for developers](../../../docs/framework/install/guide-for-developers.md).</span></span>
