# <a name="visual-basic-intellisense-code-snippets"></a>Фрагменты кода IntelliSense в Visual Basic
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]включает библиотеку кода, состоящую из нескольких сотен фрагментов кода, называемых фрагментами кода IntelliSense, которые готовы для вставки в приложении. Каждый фрагмент выполняет завершенную задачу программирования, например создание пользовательского исключения, отправку сообщения электронной почты или рисование круга. Можно вставить фрагмент в исходный код с помощью нескольких щелчков мыши. Можно также создать свои собственные фрагменты, удовлетворяющие вашим потребностям, добавить их в библиотеку и затем использовать их при необходимости. Дополнительные сведения см. в статье [Фрагменты кода](https://docs.microsoft.com/visualstudio/ide/code-snippets).  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Средство повышения производительности для пользователей Visual Basic  
 Эти фрагменты кода повышают производительность за счет сокращения времени, затрачиваемого на изучение примеров, за счет сокращения времени, необходимого для изучения использования незнакомых возможностей и поддерживая повторного использования кода.  
  
 Библиотека кода поддерживает следующие функции:  
  
-   Вставка одного из фрагментов в **редактор кода**.  
  
-   Создайте новые задачи, которые можно использовать в своих проектах.  
  
-   Создание новых задач для совместного использования с рабочей группой и коллегами.  
  
-   Редактирование задач.  
  
-   Загрузка дополнительных задач со сторонними производителями.  
  
 Эти блоки кода доступны на протяжении [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   Можно добавлять их через контекстное меню **редактор кода**.  
  
-   Можно перетаскивать XML-файлы из **проводнике** в файл исходного кода.
