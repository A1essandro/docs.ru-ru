# <a name="visual-basic-settings"></a>Параметры Visual Basic
Параметры Visual Basic предназначены для оптимизации работы пользователей и повышения производительности разработчиков Visual Basic. Он является частью [Настройка параметров разработки в Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3) функция, которая позволяет настроить и сохранить параметры в интегрированной разработки среды (IDE) Visual Studio.  
  
 Можно также перенести или перезагрузить параметров IDE на других компьютерах. Дополнительные сведения см. в разделе [как: общий ресурс параметров между компьютерами](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882).

|Функция|Отображается по умолчанию?|Примечания|  
|---|---|---|    
|[Представление классов](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|Нет|Скрывает **скрытые типы и члены**|  
|[Командное окно](https://docs.microsoft.com/visualstudio/ide/reference/command-window)|Нет||  
|**Динамическая справка** окна|Нет|Не отображается при нажатии клавиши F1.|  
|[Окно интерпретации](https://docs.microsoft.com/visualstudio/ide/reference/immediate-window)|Да, при запуске отладки.||  
|[Обозреватель объектов](https://docs.microsoft.com/visualstudio/ide/viewing-the-structure-of-code)|Нет|По умолчанию отображает:<br /><br /> -Пространства имен<br />-Общие члены<br />-Наследуемые члены|  
|[Окно выходных данных](https://docs.microsoft.com/visualstudio/ide/reference/output-window)|Да, при запуске построения.||  
|[Окно "Свойства"](https://docs.microsoft.com/visualstudio/ide/reference/properties-window)|Да||  
|**Обозреватель решений**|Да||  
|**Обозреватель серверов**|Нет|Отображает подключения данных и доступные системные службы.|  
|[Начальная страница](https://docs.microsoft.com/visualstudio/ide/customizing-the-start-page-for-visual-studio)|Да|Открывается при первом запуске интегрированной среды разработки. Его можно настраивать.|  
|**Список задач**|Да, при компиляции ошибки или предупреждения существует.||  
|[Team Explorer](http://msdn.microsoft.com/library/1372e56c-b34f-42c2-b72c-94b57620c75c)|Да|Можно использовать командный обозреватель для доступа к командным проектам и использовать возможности управления жизненным циклом приложения Team Foundation. |[Панель элементов](https://docs.microsoft.com/visualstudio/ide/reference/toolbox)|Да|Отображает элементы управления и компоненты в алфавитном порядке.|  
  
 Кроме того, следующие элементы имеют определенные характеристики при выборе **параметров разработки Visual Basic**.  
  
## <a name="dialog-boxes"></a>Диалоговые окна  
  
|Функция|Поведение|
|---|---|  
|**Новый проект** диалоговое окно|Если на компьютере установлены другие платформы, интегрированной среды разработки включает поле со списком для выбора целевой платформы.|  
|[Диалоговое окно "Параметры"](https://docs.microsoft.com/visualstudio/ide/reference/options-dialog-box-visual-studio)|Visual Basic обеспечивает упрощенное **параметры** страниц. Щелкните **Показать все параметры** флажок, чтобы просмотреть полный список параметров.|
