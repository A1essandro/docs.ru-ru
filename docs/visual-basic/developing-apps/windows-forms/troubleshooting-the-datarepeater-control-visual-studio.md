 <span data-ttu-id="1cada-110">Чтобы это исключение возникает, когда <xref:System.Windows.Forms.Control.Padding%2A> свойству большой достаточно значение для предоставления края <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> элемента управления.</span><span class="sxs-lookup"><span data-stu-id="1cada-110">The exception to this is when the <xref:System.Windows.Forms.Control.Padding%2A> property is set to a large enough value to expose the edges of the <xref:Microsoft.VisualBasic.PowerPacks.DataRepeater> control.</span></span> <span data-ttu-id="1cada-111">Таким образом щелкнув в предоставленном поле вызывает события мыши.</span><span class="sxs-lookup"><span data-stu-id="1cada-111">In this case, clicking in the exposed margin will raise mouse events.</span></span>
