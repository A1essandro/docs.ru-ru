 <span data-ttu-id="591a6-105">Для успешного развертывания приложения необходимо также развернуть все компоненты, на которые ссылается приложение.</span><span class="sxs-lookup"><span data-stu-id="591a6-105">To successfully deploy an application, you must also deploy all components that are referenced by the application.</span></span> <span data-ttu-id="591a6-106">Процесс установки необходимых компонентов называется *начальной загрузкой*.</span><span class="sxs-lookup"><span data-stu-id="591a6-106">The process of installing prerequisite components is known as *bootstrapping*.</span></span>  
  
 <span data-ttu-id="591a6-107">При установке Visual Studio на компьютере разработчика, пакет начального загрузчика Power Packs добавляется в папку начального загрузчика Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="591a6-107">When Visual Studio is installed on your development computer, a Power Packs bootstrapper package is added to the Visual Studio bootstrapper directory.</span></span> <span data-ttu-id="591a6-108">Этот пакет доступен при добавлении необходимых компонентов для [!INCLUDE[ndptecclick](~/includes/ndptecclick-md.md)] или для развертывания установщика Windows.</span><span class="sxs-lookup"><span data-stu-id="591a6-108">This package is then available when you follow the procedures for adding prerequisites for either [!INCLUDE[ndptecclick](~/includes/ndptecclick-md.md)] or Windows Installer deployment.</span></span>  
  
 <span data-ttu-id="591a6-109">По умолчанию компоненты начальной загрузки развертываются из расположения пакета установки.</span><span class="sxs-lookup"><span data-stu-id="591a6-109">By default, bootstrapped components are deployed from the same location as the installation package.</span></span> <span data-ttu-id="591a6-110">Кроме того, можно выбрать для развертывания этих компонентов URL-адрес или общую папку, откуда пользователь сможет скачать их при необходимости.</span><span class="sxs-lookup"><span data-stu-id="591a6-110">Alternatively, you can choose to deploy the components from a URL or file share location from which users can download them as necessary.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="591a6-111">Чтобы установить компоненты начальной загрузки, пользователю могут потребоваться права администратора или аналогичные им на компьютере.</span><span class="sxs-lookup"><span data-stu-id="591a6-111">To install bootstrapped components, the user might need administrative or similar user permissions on the computer.</span></span> <span data-ttu-id="591a6-112">Для приложений [!INCLUDE[ndptecclick](~/includes/ndptecclick-md.md)] это означает, что пользователю могут потребоваться права администратора для установки приложения независимо от уровня безопасности, заданного приложением.</span><span class="sxs-lookup"><span data-stu-id="591a6-112">For [!INCLUDE[ndptecclick](~/includes/ndptecclick-md.md)] applications, this means that the user will need administrative permissions to install the application, regardless of the security level specified by the application.</span></span> <span data-ttu-id="591a6-113">После установки приложения пользователь может запустить приложение без прав администратора.</span><span class="sxs-lookup"><span data-stu-id="591a6-113">After the application is installed, the user can run the application without administrative permissions.</span></span>  
  
 <span data-ttu-id="591a6-114">Во время установки пользователям будет предложено установить элементы управления пакетов Power Packs, если они не присутствуют на конечном компьютере.</span><span class="sxs-lookup"><span data-stu-id="591a6-114">During installation, users will be prompted to install the Power Packs controls if they are not present on the destination computer.</span></span>
