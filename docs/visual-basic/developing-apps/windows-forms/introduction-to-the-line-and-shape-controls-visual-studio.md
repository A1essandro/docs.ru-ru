<span data-ttu-id="a21cf-103">Элементы управления Visual Basic Power Packs, Line и Shape — это набор из трех графических элементов управления, которые позволяют рисовать линии и фигуры в формах и контейнерах.</span><span class="sxs-lookup"><span data-stu-id="a21cf-103">The Visual Basic Power Packs Line and Shape controls are a set of three graphical controls that enable you to draw lines and shapes on forms and containers.</span></span> <span data-ttu-id="a21cf-104"><xref:Microsoft.VisualBasic.PowerPacks.LineShape> Управления используется для рисования горизонтальные, вертикальные и диагональные линии.</span><span class="sxs-lookup"><span data-stu-id="a21cf-104">The <xref:Microsoft.VisualBasic.PowerPacks.LineShape> control is used to draw horizontal, vertical, and diagonal lines.</span></span> <span data-ttu-id="a21cf-105"><xref:Microsoft.VisualBasic.PowerPacks.OvalShape> Управления используется для рисования окружностей и овалов и <xref:Microsoft.VisualBasic.PowerPacks.RectangleShape> управления используется для рисования прямоугольников и квадратов.</span><span class="sxs-lookup"><span data-stu-id="a21cf-105">The <xref:Microsoft.VisualBasic.PowerPacks.OvalShape> control is used to draw circles and ovals, and the <xref:Microsoft.VisualBasic.PowerPacks.RectangleShape> control is used to draw rectangles and squares.</span></span>  
  
## <a name="line-and-shape-controls"></a><span data-ttu-id="a21cf-106">Элементы управления Line и Shape</span><span class="sxs-lookup"><span data-stu-id="a21cf-106">Line and Shape Controls</span></span>  
 <span data-ttu-id="a21cf-107">Элементы управления Line и Shape инкапсулируют множество графических методов, которые содержатся в <xref:System.Drawing> пространства имен.</span><span class="sxs-lookup"><span data-stu-id="a21cf-107">Line and Shape controls encapsulate many of the graphics methods that are contained in the <xref:System.Drawing> namespace.</span></span> <span data-ttu-id="a21cf-108">Это позволяет рисовать линии и фигуры за один шаг не требуется создавать графические объекты, перья и кисти.</span><span class="sxs-lookup"><span data-stu-id="a21cf-108">This enables you to draw lines and shapes in a single step without having to create graphics objects, pens, and brushes.</span></span> <span data-ttu-id="a21cf-109">Достаточно настроить некоторые параметры могут выполняться сложные графические методы, такие как градиентные заливки.</span><span class="sxs-lookup"><span data-stu-id="a21cf-109">Complex graphics techniques such as gradient fills can be accomplished by just setting some properties.</span></span>  
  
 <span data-ttu-id="a21cf-110">Несмотря на то, что это также можно рисовать линии и фигуры с помощью графических методов, существует несколько преимуществ использования элементов управления Line и Shape.</span><span class="sxs-lookup"><span data-stu-id="a21cf-110">Although it is also possible to draw lines and shapes by using graphics methods, there are several advantages to using the Line and Shape controls:</span></span>  
  
-   <span data-ttu-id="a21cf-111">Графические методы могут вызываться только во время выполнения.</span><span class="sxs-lookup"><span data-stu-id="a21cf-111">Graphics methods can be called only at run time.</span></span> <span data-ttu-id="a21cf-112">Элементы управления Line и Shape можно добавить в форму во время разработки.</span><span class="sxs-lookup"><span data-stu-id="a21cf-112">Line and Shape controls can be added to a form at design time.</span></span> <span data-ttu-id="a21cf-113">Это позволяет увидеть, как они выглядят и точного их; они также могут быть добавлены во время выполнения.</span><span class="sxs-lookup"><span data-stu-id="a21cf-113">This enables you to see what they look like and to position them exactly; they can also be added at run time.</span></span>  
  
-   <span data-ttu-id="a21cf-114">Элементы управления Line и Shape, могут быть выделены во время выполнения, предоставляющий события, такие как <xref:Microsoft.VisualBasic.PowerPacks.Shape.Click> и <xref:Microsoft.VisualBasic.PowerPacks.Shape.OnDoubleClick%2A>.</span><span class="sxs-lookup"><span data-stu-id="a21cf-114">Line and Shape controls are selectable at run time, providing events such as <xref:Microsoft.VisualBasic.PowerPacks.Shape.Click> and <xref:Microsoft.VisualBasic.PowerPacks.Shape.OnDoubleClick%2A>.</span></span> <span data-ttu-id="a21cf-115">Выходные данные графические методы не могут быть выбраны и предоставляют события.</span><span class="sxs-lookup"><span data-stu-id="a21cf-115">The outputs of graphics methods are not selectable and do not provide events.</span></span>  
  
-   <span data-ttu-id="a21cf-116">Элементы управления Line и Shape предоставляют <xref:Microsoft.VisualBasic.PowerPacks.Shape.BringToFront%2A> и <xref:Microsoft.VisualBasic.PowerPacks.Shape.SendToBack%2A> методы, которые позволяют управлять их z порядок во время разработки и во время выполнения.</span><span class="sxs-lookup"><span data-stu-id="a21cf-116">Line and Shape controls provide <xref:Microsoft.VisualBasic.PowerPacks.Shape.BringToFront%2A> and <xref:Microsoft.VisualBasic.PowerPacks.Shape.SendToBack%2A> methods that enable you to control their z-order at design time and at run time.</span></span> <span data-ttu-id="a21cf-117">Z порядок графических методов можно управлять только путем изменения их порядка выполнения во время выполнения.</span><span class="sxs-lookup"><span data-stu-id="a21cf-117">The z-order of graphics methods can be controlled only by changing their order of execution at run time.</span></span>  
  
-   <span data-ttu-id="a21cf-118">Элементы управления Line и Shape, — это элементы управления без окон; у них нет дескрипторов окон и таким образом использовать меньше системных ресурсов.</span><span class="sxs-lookup"><span data-stu-id="a21cf-118">Line and Shape controls are windowless controls; they have no window handles and therefore use less system resources.</span></span>

 <span data-ttu-id="a21cf-120">Элементы управления Line и Shape, являются производными от базового <xref:Microsoft.VisualBasic.PowerPacks.Shape> класс, который определяет свои общие свойства, методы и события.</span><span class="sxs-lookup"><span data-stu-id="a21cf-120">Line and Shape controls derive from a base <xref:Microsoft.VisualBasic.PowerPacks.Shape> class that defines their shared properties, methods, and events.</span></span>
