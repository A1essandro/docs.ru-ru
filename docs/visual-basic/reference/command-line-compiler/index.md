# <a name="visual-basic-command-line-compiler"></a>Компилятор командной строки Visual Basic

Компилятор командной строки Visual Basic представляет собой альтернативу компиляции в среде разработки Visual Studio (IDE). Этот раздел содержит описания для параметров компилятора Visual Basic.

Каждый параметр компилятора доступен в двух формах:-option и/Option. В документации по Visual Basic отображаются только форма/Option.
