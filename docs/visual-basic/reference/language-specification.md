# <a name="visual-basic-language-specification"></a>Спецификация языка Visual Basic
[!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Спецификация языка является полномочным источником для получения ответов на все вопросы о [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] грамматике и синтаксисе. Она содержит подробные сведения о языке, включая многие области, не описанные в [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] справочной документации.  
  
 Спецификация доступна на [центра загрузки Майкрософт](http://go.microsoft.com/fwlink/?LinkId=188623).
