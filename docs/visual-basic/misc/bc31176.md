# <a name="character-39character39-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>Символ &#39; символ &#39; (0 x&lt;шестнадцатеричное&gt;) не допускается в XML-литерала
Неподдерживаемый символ был использован в XML-литерале.  
  
 **Идентификатор ошибки:** BC31176  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите неподдерживаемый символ.
