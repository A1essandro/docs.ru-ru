# <a name="mybase-cannot-be-used-with-method-methodname-because-it-is-declared-mustoverride"></a>«MyBase» не может использоваться с методом "\<имя_метода >", так как он объявлен как «MustOverride»
Вы попытались использовать `MyBase` с методом, который был объявлен `MustOverride`.  
  
 **Идентификатор ошибки:** BC30399  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите объявление `MustOverride` .
