 <span data-ttu-id="61b72-108">Хотя тип временно остается доступным в исходной сборке, он с высокой вероятностью может стать неопределенным после удаления из исходной сборки в результате рефакторинга кода.</span><span class="sxs-lookup"><span data-stu-id="61b72-108">Although the type is temporarily still available in the original assembly, it is likely to become undefined when code refactoring removes it from the original assembly.</span></span>
