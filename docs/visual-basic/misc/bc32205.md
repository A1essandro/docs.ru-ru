# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>Длина имени переменной, объявленной с модификатором WithEvents, не может превышать 1019 знаков
Имена или идентификаторы переменных, объявленных с атрибутом `WithEvents` , ограничены 1019 знаками.  
  
 **Идентификатор ошибки:** BC32205  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Уменьшите длину идентификатора.
