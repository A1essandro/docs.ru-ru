<span data-ttu-id="f500d-103">Структура наследования приводит к циклической зависимости между вложенными классами, то есть к двум классам, наследующим друг друга.</span><span class="sxs-lookup"><span data-stu-id="f500d-103">An inheritance structure results in circular dependency among nested classes, that is, two classes inheriting from each other.</span></span>  
  
 <span data-ttu-id="f500d-104">К этой ошибке может привести следующий код.</span><span class="sxs-lookup"><span data-stu-id="f500d-104">The following code can generate this error message.</span></span>  
  
```  
Public Class c1  
    Inherits c3.c4  
    Public Class c2  
    End Class  
End Class  
Public Class c3  
    Inherits c1.c2  
    Public Class c4  
    End Class  
End Class  
```  
  
 <span data-ttu-id="f500d-105">В приведенном выше коде класс `c1` наследует от класса `c4`, но класс `c4` вложен в класс `c3`, который наследует от класса `c2`, вложенного в `c1`.</span><span class="sxs-lookup"><span data-stu-id="f500d-105">In the preceding code, class `c1` inherits from class `c4`, but `c4` is nested inside `c3`, which inherits from `c2`, nested inside `c1`.</span></span>  
  
 <span data-ttu-id="f500d-106">**Идентификатор ошибки:** BC30907</span><span class="sxs-lookup"><span data-stu-id="f500d-106">**Error ID:** BC30907</span></span>
