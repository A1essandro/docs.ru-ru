# <a name="operator-without-an-39as39-clause-type-of-object-assumed"></a>Оператор без предложения «Как»; Тип объекта, предполагается, что
Процедура оператора не указывает предложение `As` .  
  
 Предложение `As` определяет тип данных, который будет связан с программным элементом. В [оператор](../../visual-basic/language-reference/statements/operator-statement.md), он указывает тип данных значения, процедура оператора возвращает в вызывающий код. Если предложение `As` не включено в оператор `Operator` , то по умолчанию для возвращаемых данных будет задан тип `Object`.  
  
 По умолчанию данное сообщение является предупреждением. Сведения о сокрытии предупреждений или обработке предупреждений как ошибок см. в разделе [Configuring Warnings in Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Идентификатор ошибки:** BC41005  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Включите предложение `As` в оператор `Operator` для указания типа данных переменной.
