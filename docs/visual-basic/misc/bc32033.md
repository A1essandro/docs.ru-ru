# <a name="end-get-must-be-the-first-statement-on-a-line"></a>End Get должен являться первым оператором в строке.
Оператор `End Get` следует за разделителем операторов в виде двоеточия (:). `End Get` должен быть единственным оператором в своей строке исходного кода.  
  
 **Идентификатор ошибки:** BC32033  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Разместите несколько операторов в отдельных строках.
