# <a name="ltimplementsclausegt-cannot-implement-lttypenamegt-because-lttypenamegt-is-a-reserved-name"></a>"&lt;implementsclause&gt;«не может реализовать»&lt;typename&gt;«, так как»&lt;typename&gt;" является зарезервированным именем
В операторе `Implements` указано зарезервированное имя.  
  
 **Идентификатор ошибки:** BC31415  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переименуйте член интерфейса, который не удалось реализовать.
