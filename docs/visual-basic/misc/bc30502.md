# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>«Shared» не может использоваться вместе с "&lt;описатель&gt;" в объявлении свойства
Предпринята попытка объединить модификатор `Shared` со спецификатором, что недопустимо в объявлении свойства.  
  
 **Идентификатор ошибки:** BC30502  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
