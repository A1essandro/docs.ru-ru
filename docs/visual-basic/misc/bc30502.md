# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39;Общие&#39; нельзя использовать вместе с &#39; &lt;описатель&gt; &#39; в объявлении свойства
Предпринята попытка объединить модификатор `Shared` со спецификатором, что недопустимо в объявлении свойства.  
  
 **Идентификатор ошибки:** BC30502  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
