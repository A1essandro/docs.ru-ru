# <a name="ltmethodgt-ltmethodname1gt-must-be-declared-overloads-because-another-ltmethodname2gt-is-declared-overloads"></a>&lt;метод&gt; "&lt;имя_метода1&gt;" должен быть объявлен как «Overloads», поскольку другой "&lt;имя_метода2&gt;" объявлен как «Overloads»
Другое свойство или метод с тем же именем объявлен с модификатором `Overloads` .  
  
 **Идентификатор ошибки:** BC31409  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Добавьте модификатор `Overloads` в это свойство или метод.  
  
2.  Удалите модификатор `Overloads` из всех свойств или методов с тем же именем.
