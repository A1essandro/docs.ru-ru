# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; Невозможно применить к &#39; Получить &#39; или &#39; Выбрать &#39;
Атрибут `DllImportAttribute` применен к процедуре свойства `Get` или `Set` .  
  
 **Идентификатор ошибки:** BC31524  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Удалите атрибут `DllImportAttribute` из процедур свойства `Get` и `Set` .
