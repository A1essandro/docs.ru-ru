# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Только один из &#39; Открытый &#39; &#39; Закрытые &#39; &#39; Защищенные &#39; &#39; Дружественные &#39; или &#39; Protected Friend &#39; можно указать
Конфликтующие уровни доступности в объявлении элемента. Можно указать только один уровень доступности.  
  
 **Идентификатор ошибки:** BC30176  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Выберите один уровень доступности и удалите остальные ключевые слова из объявления.
