# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Только одна из &#39;открытый&#39;, &#39;закрытый&#39;, &#39;Protected&#39;, &#39;Friend&#39;, или &#39;Protected Friend&#39; можно указать
Конфликтующие уровни доступности в объявлении элемента. Можно указать только один уровень доступности.  
  
 **Идентификатор ошибки:** BC30176  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Выберите один уровень доступности и удалите остальные ключевые слова из объявления.
