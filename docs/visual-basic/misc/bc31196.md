# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Литералы XML и свойства оси XML не доступны во время этого сеанса отладки, так как они не используются в этой сборке
XML-литерал или свойство оси XML содержит ссылку на **Контрольные значения** или **Интерпретация** окна во время сеанса отладки в XML в [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] функции будут недоступны. Это справедливо для сборки, либо не использовать XML-данные в [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] функции или находится в сборке выпуска.  
  
 **Идентификатор ошибки:** BC31196  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Начните новый сеанс отладки с использованием отладочной сборки.
