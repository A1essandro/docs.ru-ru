# <a name="no-method-39ltmethodnamegt39-can-accept-an-argument-of-type-39lttypenamegt39-for-parameter-39ltparameternamegt39"></a>Ни один метод &#39; &lt;имя_метода&gt;&#39; может принимать аргумент типа &#39;&lt; TypeName&gt;&#39; для параметра &#39;&lt; имя_параметра&gt;&#39;
Вызов метода содержит аргумент типа, который не является допустимым.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте типы аргументов.  
  
2.  Сопоставьте типы аргументов с типами параметров метода, чтобы убедиться в их согласованности. При необходимости измените типы.
