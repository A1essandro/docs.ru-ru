# <a name="39class39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39;Класс&#39; ограничение нельзя указывать несколько раз для одного параметра типа.
Список ограничений включает [Class (Visual Basic)](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) ограничение, более одного раза.  
  
 Список ограничений для параметра типа может указывать, что аргумент типа, переданный в параметр типа должен быть типом значения (с [Structure (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) ограничение) или должен быть ссылочным типом (с `Class` ограничение) . Нельзя указывать оба ограничения в одном и том же параметре типа, и нельзя указывать какое-либо из них более одного раза.  
  
 Идентификатор ошибки: BC32101  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите все избыточные ключевые слова `Class` . В списке ограничений должно быть только одно ключевое слово.
