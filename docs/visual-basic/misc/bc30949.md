# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-3939"></a><span data-ttu-id="d3029-102">Инициализатор массива не может быть указан для переменной размерности; Используйте пустой инициализатор &#39; {} &#39;</span><span class="sxs-lookup"><span data-stu-id="d3029-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer &#39;{}&#39;</span></span>
<span data-ttu-id="d3029-103">Массив инициализирует размерность, неизвестную во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="d3029-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="d3029-104">Следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="d3029-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="d3029-105">Следующий код позволяет избежать этой ошибки.</span><span class="sxs-lookup"><span data-stu-id="d3029-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="d3029-106">**Идентификатор ошибки:** BC30949</span><span class="sxs-lookup"><span data-stu-id="d3029-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d3029-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="d3029-107">To correct this error</span></span>  
  
-   <span data-ttu-id="d3029-108">Если это возможно, укажите постоянную размерность в объявлении массива.</span><span class="sxs-lookup"><span data-stu-id="d3029-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="d3029-109">Если вы не можете указать постоянную размерность, необходимо инициализировать массив с помощью цикла, когда переменная размерность становится известной.</span><span class="sxs-lookup"><span data-stu-id="d3029-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
