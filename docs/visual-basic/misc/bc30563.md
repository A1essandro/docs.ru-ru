# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;имя&gt; &#39; неоднозначно в объектах приложения &#39; &lt;списка&gt;&#39;
Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор Visual Basic не имеет правил разрешения конфликтов; Вы должны самостоятельно устранить неоднозначность имен. В списке, указанном в сообщении об ошибке, представлены объекты приложения, имя которых является неоднозначным.  
  
 **Идентификатор ошибки:** BC30563  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите полное имя.
