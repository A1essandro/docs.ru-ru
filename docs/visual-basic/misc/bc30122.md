# <a name="39stop39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Откл &#39; операторы недопустимы в окне интерпретации
Операторы `Stop` и `End` приостанавливают выполнение и не разрешены в контексте отладки.  
  
 **Идентификатор ошибки:** BC30122  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Не используйте оператор `Stop` или `End` в окне **интерпретации** .
