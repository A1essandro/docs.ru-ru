# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>«Finally» недопустимы в окне интерпретации
`Finally`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30716  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `Finally` из отлаживаемого кода.
