# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>XML комментарий тег &#39; включить &#39; должен иметь &#39; &lt;атрибута&gt;&#39; атрибут
Тег комментария XML «include» должен иметь \<атрибут > атрибут. Комментарий XML будет проигнорирован.  
  
 Атрибут `file` или `path` , переданный в тег `<`include`>` , отсутствует или является неполным.  
  
 **Идентификатор ошибки:** BC42310  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте или дополните требуемый атрибут для \<включают > тег.
