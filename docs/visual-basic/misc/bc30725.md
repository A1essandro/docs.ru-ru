# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>Операторы 'Throw' недопустимы в окне интерпретации
`Throw`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30725  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `Throw` из отлаживаемого кода.
