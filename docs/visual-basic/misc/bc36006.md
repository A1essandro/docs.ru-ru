<span data-ttu-id="c1704-103">Атрибут применяется к программному элементу с помощью конструктора атрибута, принимающего параметр `ByRef` .</span><span class="sxs-lookup"><span data-stu-id="c1704-103">An attribute is applied to a programming element using an attribute constructor that takes a `ByRef` parameter.</span></span>  
  
 <span data-ttu-id="c1704-104">Атрибуты применяются во время компиляции, и компилятор должен конкретизировать значения для передачи в конструктор атрибута.</span><span class="sxs-lookup"><span data-stu-id="c1704-104">Attributes are applied at compile time, and the compiler needs concrete values to pass to the attribute constructor.</span></span> <span data-ttu-id="c1704-105">Параметр `ByRef` принимает указатель на значение, которое не может быть вычислено во время компиляции.</span><span class="sxs-lookup"><span data-stu-id="c1704-105">A `ByRef` parameter takes a pointer to a value, which cannot be evaluated at compile time.</span></span>  
  
 <span data-ttu-id="c1704-106">Вы можете определить конструктор атрибута, который принимает параметр `ByRef` , и использовать его для таких целей, как наследование, но при применении атрибута вы должны использовать конструктор, который не принимает никакие параметры `ByRef` .</span><span class="sxs-lookup"><span data-stu-id="c1704-106">You can define an attribute constructor that takes a `ByRef` parameter, and you can use it for purposes such as inheriting, but when you apply the attribute you must use a constructor that does not take any `ByRef` parameters.</span></span>  
  
 <span data-ttu-id="c1704-107">**Идентификатор ошибки:** BC36006</span><span class="sxs-lookup"><span data-stu-id="c1704-107">**Error ID:** BC36006</span></span>
