# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Не задан аргумент для параметра "&lt;parametername&gt;«метода расширения»&lt;имя_метода&gt;«определен в»&lt;typename&gt;"
Не предоставлен аргумент для обязательного параметра метода расширения.  
  
 **Идентификатор ошибки:** BC36586  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Предоставьте аргумент для указанного параметра.
