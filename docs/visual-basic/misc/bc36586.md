# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Не указан аргумент для параметра &#39; &lt;имя_параметра&gt; &#39; метода расширения &#39; &lt;имя_метода&gt; &#39; определенные в &#39; &lt;typename&gt;&#39;
Не предоставлен аргумент для обязательного параметра метода расширения.  
  
 **Идентификатор ошибки:** BC36586  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Предоставьте аргумент для указанного параметра.
