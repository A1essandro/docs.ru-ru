# <a name="unable-to-apply-security-attributes-to-39ltnamegt39-lterrorgt"></a>Не удается применить атрибуты безопасности для "&lt;имя&gt;": &lt;ошибки&gt;
Произошла ошибка при применении атрибутов безопасности. Это может быть результатом неправильного использования атрибутов безопасности или может указывать на проблему с компилятором.  
  
 **Идентификатор ошибки:** BC31520  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте синтаксис и аргументы атрибута.  
  
2.  Скомпилируйте программу еще раз, чтобы узнать, повторится ли ошибка.  
  
3.  Если ошибка возникает снова, сохраните работу и перезапустите [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
4.  Если ошибка повторяется, переустановите [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
5.  Если после переустановки ошибка не устранена, уведомите службу технической поддержки Майкрософт.
