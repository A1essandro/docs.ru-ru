# <a name="type-argument-inference-failed-for-type-parameter-lttypeparameternamegt-of-ltgenericproceduresignaturegt"></a><span data-ttu-id="3b865-102">Для параметра типа ошибка определения аргумента типа "&lt;имя_параметра_типа&gt;«из»&lt;сигнатура_универсальной_процедуры&gt;"</span><span class="sxs-lookup"><span data-stu-id="3b865-102">Type argument inference failed for type parameter '&lt;typeparametername&gt;' of '&lt;genericproceduresignature&gt;'</span></span>
<span data-ttu-id="3b865-103">Для параметра типа ошибка определения аргумента типа "\<имя_параметра_типа >" из "\<сигнатура_универсальной_процедуры >".</span><span class="sxs-lookup"><span data-stu-id="3b865-103">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="3b865-104">Аргумент типа не могут быть выведены из аргумента, переданного параметру "\<имя_параметра >".</span><span class="sxs-lookup"><span data-stu-id="3b865-104">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>  
  
 <span data-ttu-id="3b865-105">Универсальная процедура вызывается без аргументов типа, и компилятор не может определить тип, который нужно передать одному из параметров.</span><span class="sxs-lookup"><span data-stu-id="3b865-105">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>  
  
 <span data-ttu-id="3b865-106">Как правило, при вызове универсальной процедуры указывается аргумент типа для каждого параметра типа, определяемого этой универсальной процедурой.</span><span class="sxs-lookup"><span data-stu-id="3b865-106">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="3b865-107">Если никакие аргументы типа не передаются, компилятор пытается вывести типы, которые должны быть переданы параметрам типов.</span><span class="sxs-lookup"><span data-stu-id="3b865-107">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="3b865-108">Если контекст вызова предоставляет конфликтующие сведения о типах данных для параметра типа, выведение типа выполнить не удастся.</span><span class="sxs-lookup"><span data-stu-id="3b865-108">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>  
  
 <span data-ttu-id="3b865-109">Эту ошибку может вызывать следующий код.</span><span class="sxs-lookup"><span data-stu-id="3b865-109">The following code can generate this error.</span></span>  
  
```  
Public Sub doSomething(Of t)(ByVal arg1 As t(), ByVal arg2 As t)  
End Sub  
Call doSomething(6, 42)  
```  
  
 <span data-ttu-id="3b865-110">В приведенном выше примере компилятор определяет тип `Integer` для `t` на основе значения 42, передаваемого `arg2`.</span><span class="sxs-lookup"><span data-stu-id="3b865-110">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="3b865-111">Однако для такого определения требуется, чтобы параметр `arg1` имел тип `Integer()`, то есть был массивом `Integer`, но значение 6, передаваемое `arg1` , не соответствует этому типу.</span><span class="sxs-lookup"><span data-stu-id="3b865-111">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>  
  
 <span data-ttu-id="3b865-112">**Идентификатор ошибки:** BC32051</span><span class="sxs-lookup"><span data-stu-id="3b865-112">**Error ID:** BC32051</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3b865-113">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="3b865-113">To correct this error</span></span>  
  
-   <span data-ttu-id="3b865-114">Укажите аргументы типа для универсальной процедуры, чтобы компилятору не приходилось их определять.</span><span class="sxs-lookup"><span data-stu-id="3b865-114">Supply type arguments to the generic procedure, so that the compiler does not have to infer them.</span></span>  
  
-   <span data-ttu-id="3b865-115">Укажите обычные аргументы типов, соответствующих аргументам типа.</span><span class="sxs-lookup"><span data-stu-id="3b865-115">Supply normal arguments with types that match those of the type arguments.</span></span>
