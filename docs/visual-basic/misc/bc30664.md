# <a name="conversions-from-39lttypename1gt39-to-39lttypename2gt39-must-be-explicit"></a>Преобразования из &#39; &lt;Имя_типа1&gt; &#39; для &#39; &lt;имя_типа2&gt; &#39; должны быть явными
Это преобразование нельзя выполнить неявно.  
  
 **Идентификатор ошибки:** BC30664  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте `CType`или одну из функций определенных преобразований, таких как `CInt` или `CDec`.
