# <a name="end-event-must-be-preceded-by-a-matching-custom-event"></a>Оператору End Event должен предшествовать соответствующий ему оператор Custom Event
Оператор `End Event` указан без соответствующего оператора `Custom Event` . Оператору`End Event` должен предшествовать соответствующий оператор `Custom Event` .  
  
 **Идентификатор ошибки:** BC31123  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь, что предшествующий оператор `Custom Event` допустим и правильно написан.
