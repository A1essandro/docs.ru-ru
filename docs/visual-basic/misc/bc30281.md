# <a name="structure-ltstructurenamegt-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Структура "&lt;имя_структуры&gt;" должен содержать по крайней мере один экземпляр переменной члена или объявление события
Обнаружена функционально пустая структура. Между оператором `Structure` и оператором `End Structure` должен быть объявлен по крайней мере один член-переменная.  
  
 **Идентификатор ошибки:** BC30281  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте оператор `Dim` или `Event` в структуру.
