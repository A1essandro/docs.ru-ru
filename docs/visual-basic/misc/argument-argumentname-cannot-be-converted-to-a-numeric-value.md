# <a name="argument-ltargumentnamegt-cannot-be-converted-to-a-numeric-value"></a>Аргумент "&lt;имя_аргумента&gt;" невозможно преобразовать в числовое значение
Предпринята попытка преобразовать переменную, например строку, в числовое значение.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Найдите преобразование, имеющее смысл.
