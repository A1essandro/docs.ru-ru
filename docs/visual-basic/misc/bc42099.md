# <a name="unused-local-constant-ltconstantgt"></a>Неиспользуемая локальная константа: "&lt;константы&gt;"
Была объявлена локальная константа, которая не используется нигде в коде.  
  
 **Идентификатор ошибки:** BC42099  
  
### <a name="to-remove-this-warning"></a>Удаление этого предупреждения  
  
1.  Эта ошибка является предупреждением, и ее можно игнорировать. Чтобы устранить это предупреждение, удалите объявления неиспользуемой константы из кода.
