# <a name="39inherits39-statement-must-precede-all-declarations-in-a-class"></a>&#39; наследует &#39; оператор должен предшествовать всем объявлениям в классе
Операторы `Inherits` должны находиться в начале определений класса.  
  
 **Идентификатор ошибки** : BC30683  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переместите оператор `Inherits` в начало класса.
