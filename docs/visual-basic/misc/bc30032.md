# События не могут иметь тип возврата
События могут принимать аргументы, но не могут возвращать значения напрямую.  
  
 **Идентификатор ошибки:** BC30032  
  
### Исправление ошибки  
  
-   Удалите тип возврата из оператора `Event`.
