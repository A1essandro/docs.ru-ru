# <a name="unable-to-sign-assembly-error-message"></a>Не удалось подписать сборку: \<сообщение об ошибке >
Компилятор Visual Basic вызывает компоновщик сборок (Al.exe, который также называется Alink) для создания сборки с манифестом. Компоновщик сообщил об ошибке подписи сборки после ее создания. Это может быть вызвано недопустимым ключом.  
  
 **Идентификатор ошибки:** BC30146  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Изучите приведенное сообщение об ошибке и обратитесь к разделу [Al.exe (компоновщик сборок)](../../framework/tools/al-exe-assembly-linker.md) для пояснения и рекомендации.  
  
2.  Если ошибка не устранена, соберите сведения об условиях ее возникновения и уведомите службу технической поддержки Майкрософт.
