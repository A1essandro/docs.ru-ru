# <a name="ltelementnamegt-is-not-a-method-parameter"></a>"&lt;elementname&gt;" не является параметром метода
В вызове процедуры указывается дополнительный аргумент, который не определен методом.  
  
 **Идентификатор ошибки:** BC36568  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите лишний аргумент из списка аргументов.
