# Свойству &lt;имя_свойства&gt; нельзя присвоить значение Nothing
Свойство требует значения, отличного от `Nothing`. Это может быть результатом неправильного вычисления значения.  
  
### Исправление ошибки  
  
-   Укажите допустимое значение.
