# <a name="conflicts-with-ltname1gt-which-is-implicitly-created-for-ltname2gt-in-lttypegt-ltname3gt"></a>Конфликтует с "&lt;name1&gt;«, неявно объявленным для»&lt;name2&gt;" в &lt;тип&gt; "&lt;name3&gt;"
Имя члена типа конфликтует с другим неявно созданным членом.  
  
 **Идентификатор ошибки:** BC31058  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переименуйте явно объявленный член, чтобы устранить конфликт имен.
