# Исходный файл &quot;&lt;имя_файла&gt;&quot; задан несколько раз
Файл указан несколько раз.  
  
 **Идентификатор ошибки:** BC2002  
  
### Исправление ошибки  
  
-   Удалите лишние спецификаторы файлов.
