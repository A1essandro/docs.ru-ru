# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>Невозможно преобразовать начальное значение типа &#39; &lt;тип1&gt;&#39;, предельное значение типа &#39; &lt;тип2&gt; &#39; и значение шага &#39; &lt;Тип3&gt; &#39; для общий тип
Указанные управляющие значения `For...Next` имеют разные типы и, следовательно, недопустимы, так как не могут быть преобразованы в общий тип.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Измените тип одного или нескольких значений, чтобы все они могли быть приведены к общему типу.
