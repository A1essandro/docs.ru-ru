# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Имя агрегатной функции не может использоваться с символом типа
Символ типа включен в конец имени агрегатной функции.  
  
 **Идентификатор ошибки:** BC36617  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Удалите символ типа из имени агрегатной функции.
