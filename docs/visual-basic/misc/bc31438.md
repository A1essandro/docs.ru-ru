# <a name="method-39ltmethodname1gt39-does-not-have-the-same-generic-constraints-as-the-partial-method-39ltmethodname2gt39"></a>Метод &#39; &lt;имя_метода1&gt;&#39; не имеет те же универсальные ограничения, как разделяемый метод &#39;&lt; имя_метода2&gt;&#39;
Вы определили реализацию разделяемого метода, имеющую универсальные ограничения, которые отличаются от ограничений в объявлении разделяемого метода. Это проиллюстрировано в следующем коде.  
  
```vb  
Partial Class Class1  
  
    Partial Private Sub Test(Of T As Class)(ByVal arg As T)  
    End Sub  
  
End Class  
  
Partial Class Class1  
  
    '' The error occurs here, for Test.  
    'Private Sub Test(Of T As Structure)(ByVal arg As T)  
    'End Sub  
  
End Class  
```  
  
 **Error ID:** BC31438
