# <a name="anonymous-type-member-or-property-ltpropertynamegt-is-already-declared"></a><span data-ttu-id="4268c-102">Анонимный член типа или свойство "&lt;propertyname&gt;" уже объявлена</span><span class="sxs-lookup"><span data-stu-id="4268c-102">Anonymous type member or property '&lt;propertyname&gt;' is already declared</span></span>
<span data-ttu-id="4268c-103">Имя свойства может быть установлено только один раз в объявлении анонимного типа.</span><span class="sxs-lookup"><span data-stu-id="4268c-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="4268c-104">Например, следующие объявления неправильны:</span><span class="sxs-lookup"><span data-stu-id="4268c-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="4268c-105">**Идентификатор ошибки:** BC36547</span><span class="sxs-lookup"><span data-stu-id="4268c-105">**Error ID:** BC36547</span></span>

```vb

```

```vb  
' Valid.  
Dim anonType4 = New With {Key .ProductName = product.Name, Key .CarName = car1.Name}  
```
