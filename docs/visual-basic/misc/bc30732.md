# Веб-методы не могут выполняться в окнах отладки
В окнах отладки нельзя проверять методы веб\-служб XML.  
  
 **Идентификатор ошибки:** BC30732  
  
### Исправление ошибки  
  
-   Проверяйте методы веб\-служб XML в редакторе кода.
