# Операторы Continue недопустимы в окне интерпретации
Операторы `Continue` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30780  
  
### Исправление ошибки  
  
-   Удалите оператор `Continue` из отлаживаемого кода.
