# <a name="39namespace39-statement-must-end-with-a-matching-39end-namespace39"></a>&#39; Пространство имен &#39; оператор должен заканчиваться соответствующим &#39; End Namespace &#39;
Блоки`Namespace` должны заканчиваться оператором `End Namespace` .  
  
 **Идентификатор ошибки:** BC30626  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте оператор `End Namespace` в конец блока `Namespace` .
