# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a><span data-ttu-id="bcbe5-102">В операторе Next указано больше переменных, чем в соответствующих операторах For</span><span class="sxs-lookup"><span data-stu-id="bcbe5-102">'Next' statement names more variables than there are matching 'For' statements</span></span>
<span data-ttu-id="bcbe5-103">Вложенные циклы завершены с помощью одного оператора `Next` , в котором задано количество переменных, превышающее количество циклов, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="bcbe5-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```  
  
 <span data-ttu-id="bcbe5-104">**Идентификатор ошибки:** BC32037</span><span class="sxs-lookup"><span data-stu-id="bcbe5-104">**Error ID:** BC32037</span></span>

1.  <span data-ttu-id="bcbe5-106">Убедитесь в том, что в операторе `Next` указаны все переменные вложенных циклов в порядке, обратном порядку создания циклов.</span><span class="sxs-lookup"><span data-stu-id="bcbe5-106">Ensure that the `Next` statement specifies all the nested loop variables in the reverse order of loop initiation.</span></span>
