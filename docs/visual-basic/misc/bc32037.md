<span data-ttu-id="cf8f4-103">Вложенные циклы завершены с помощью одного оператора `Next` , в котором задано количество переменных, превышающее количество циклов, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="cf8f4-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
