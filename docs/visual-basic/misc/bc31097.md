# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>Элемент &#39; &lt;membername&gt;&#39; не удается найти в классе &#39;&lt; className&gt;&#39;
Член "\<имя_члена >" не удается найти в классе\<имя_класса > ". Это состояние, как правило, является результатом использования несоответствующей библиотеки "Microsoft.VisualBasic.dll".  
  
 Не удалось найти определенный член.  
  
 **Идентификатор ошибки:** BC31097  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Скомпилируйте программу еще раз, чтобы узнать, повторится ли ошибка.  
  
2.  Если ошибка возникает снова, сохраните работу и перезапустите [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Если устранить ошибку не удается, переустановите [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
4.  Если после переустановки ошибка не устранена, уведомите службу технической поддержки Майкрософт.
