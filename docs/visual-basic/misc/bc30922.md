# <a name="ltderivedtypenamegt-cannot-inherit-from-lttypegt-ltconstructedbasetypenamegt-because-it-expands-the-access-of-type-ltinternaltypenamegt-outside-the-assembly"></a><span data-ttu-id="e349b-102">"&lt;имя_производного_типа&gt;" не может наследовать от &lt;тип&gt; "&lt;имя_сконструированного_базового_типа&gt;«, поскольку он расширяет доступ типа»&lt;имя_внутреннего_типа&gt;"за пределы данной сборки</span><span class="sxs-lookup"><span data-stu-id="e349b-102">'&lt;derivedtypename&gt;' cannot inherit from &lt;type&gt; '&lt;constructedbasetypename&gt;' because it expands the access of type '&lt;internaltypename&gt;' outside the assembly</span></span>
<span data-ttu-id="e349b-103">Производный класс или интерфейс пытается расширить уровень доступа ограниченного типа, используя его как аргумент типа для базового класса или интерфейса.</span><span class="sxs-lookup"><span data-stu-id="e349b-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="e349b-104">Эту ошибку может вызывать следующий код.</span><span class="sxs-lookup"><span data-stu-id="e349b-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="e349b-105">Коду вне сборки не разрешен доступ к `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="e349b-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="e349b-106">Однако доступ к `derivedClass` может осуществляться из любого кода, который может на него ссылаться.</span><span class="sxs-lookup"><span data-stu-id="e349b-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="e349b-107">Таким образом, `derivedClass` не может наследовать `baseClass` , если он использует `restrictedStructure` в качестве аргумента типа, так как это может предоставить доступ к `restrictedStructure` коду в любой сборке.</span><span class="sxs-lookup"><span data-stu-id="e349b-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="e349b-108">**Идентификатор ошибки:** BC30922</span><span class="sxs-lookup"><span data-stu-id="e349b-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e349b-109">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="e349b-109">To correct this error</span></span>  
  
-   <span data-ttu-id="e349b-110">Настройте уровни доступа классов или интерфейсов таким образом, чтобы производный тип не расширял уровень доступа ограниченного типа.</span><span class="sxs-lookup"><span data-stu-id="e349b-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="e349b-111">- или -</span><span class="sxs-lookup"><span data-stu-id="e349b-111">-or-</span></span>  
  
-   <span data-ttu-id="e349b-112">Если не удается настроить уровни доступа, не используйте ограниченный тип в качестве аргумента типа при создании базового класса или интерфейса.</span><span class="sxs-lookup"><span data-stu-id="e349b-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
