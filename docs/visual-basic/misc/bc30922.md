# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-outside-the-assembly"></a><span data-ttu-id="716f2-102">&#39; &lt;имя_производного_типа&gt;&#39; не может наследовать от &lt;тип&gt; &#39;&lt; имя_сконструированного_базового_типа&gt;&#39; поскольку он расширяет доступ типа &#39;&lt; имя_внутреннего_типа&gt;&#39; за пределы данной сборки</span><span class="sxs-lookup"><span data-stu-id="716f2-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; outside the assembly</span></span>
<span data-ttu-id="716f2-103">Производный класс или интерфейс пытается расширить уровень доступа ограниченного типа, используя его как аргумент типа для базового класса или интерфейса.</span><span class="sxs-lookup"><span data-stu-id="716f2-103">A derived class or interface attempts to expand the access level of a restricted type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="716f2-104">Приведенный ниже код может вызвать эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="716f2-104">The following code can generate this error.</span></span>  
  
```  
Public Class baseClass(Of t)  
End Class  
Public Class derivedClass  
    Inherits baseClass(Of restrictedStructure)  
End Class  
Friend Structure restrictedStructure  
    Dim firstMember As Integer  
End Structure  
```  
  
 <span data-ttu-id="716f2-105">Коду вне сборки не разрешен доступ к `restrictedStructure`.</span><span class="sxs-lookup"><span data-stu-id="716f2-105">Code outside the assembly is not allowed to access `restrictedStructure`.</span></span> <span data-ttu-id="716f2-106">Однако доступ к `derivedClass` может осуществляться из любого кода, который может на него ссылаться.</span><span class="sxs-lookup"><span data-stu-id="716f2-106">However, `derivedClass` can be accessed from any code that can reference it.</span></span> <span data-ttu-id="716f2-107">Таким образом, `derivedClass` не может наследовать `baseClass` , если он использует `restrictedStructure` в качестве аргумента типа, так как это может предоставить доступ к `restrictedStructure` коду в любой сборке.</span><span class="sxs-lookup"><span data-stu-id="716f2-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `restrictedStructure` as a type argument, because that could expose `restrictedStructure` to code in any assembly.</span></span>  
  
 <span data-ttu-id="716f2-108">**Идентификатор ошибки:** BC30922</span><span class="sxs-lookup"><span data-stu-id="716f2-108">**Error ID:** BC30922</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="716f2-109">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="716f2-109">To correct this error</span></span>  
  
-   <span data-ttu-id="716f2-110">Настройте уровни доступа классов или интерфейсов таким образом, чтобы производный тип не расширял уровень доступа ограниченного типа.</span><span class="sxs-lookup"><span data-stu-id="716f2-110">Adjust the access levels of the classes or interfaces so the derived type does not expand the access level of the restricted type.</span></span>  
  
     <span data-ttu-id="716f2-111">-или-</span><span class="sxs-lookup"><span data-stu-id="716f2-111">-or-</span></span>  
  
-   <span data-ttu-id="716f2-112">Если не удается настроить уровни доступа, не используйте ограниченный тип в качестве аргумента типа при создании базового класса или интерфейса.</span><span class="sxs-lookup"><span data-stu-id="716f2-112">If you cannot adjust the access levels, do not use the restricted type as a type argument when constructing the base class or interface.</span></span>
