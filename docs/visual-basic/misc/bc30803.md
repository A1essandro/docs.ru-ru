# <a name="39currency39-is-no-longer-a-supported-type-use-the-39decimal39-type-instead"></a>&#39; Валюта &#39; больше не является поддерживаемым типом; использовать &#39; десятичных &#39; Вместо этого тип
Тип данных `Currency` заменен на тип данных `Decimal` .  
  
 **Идентификатор ошибки:** BC30803  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте тип данных `Decimal` вместо типа данных `Currency`.
