# VbStrConv.Wide и VbStrConv.Narrow не могут использоваться для заданных языковых стандартов
Приложение пытается использовать члены перечисления `VbStrConv``Wide` или `Narrow`, которые не применимы для указанного языкового стандарта.  
  
### Исправление ошибки  
  
1.  Удалите `VbStrConv.Wide` или `VbStrConv.Narrow`.
