# <a name="project-39ltprojectnamegt39-cannot-reference-project-39ltprojectnamegt39-because-39ltprojectnamegt39-directly-or-indirectly-references-39ltprojectnamegt39"></a>Проект &#39; &lt;projectname&gt;&#39; не может ссылаться на проект &#39;&lt; имя_проекта&gt;&#39; так как &#39;&lt; имя_проекта&gt;&#39; прямо или косвенно ссылки на &#39;&lt; имя_проекта&gt;&#39;
Проект ссылается на второй проект. Второй проект, в свою очередь, содержит ссылку на проект, в котором на него есть ссылка.  
  
 **Идентификатор ошибки:** BC30761  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите одну из ссылок.
