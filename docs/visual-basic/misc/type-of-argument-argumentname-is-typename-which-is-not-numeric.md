# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>Тип аргумента "&lt;имя_аргумента&gt;«is»&lt;typename&gt;", который не является числовым
Вместо ожидаемого числового аргумента используется нечисловой.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь в том, что аргумент является числовым.
