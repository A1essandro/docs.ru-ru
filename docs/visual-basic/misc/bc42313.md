# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>Тег комментария XML &#39;возвращает&#39; не разрешен в &#39;WriteOnly&#39; свойство
Тег комментария XML "returns" недопустим для свойства "WriteOnly". Комментарий XML будет проигнорирован.  
  
 XML-комментарий для объявления свойства только для записи не может содержать \<возвращает > тег.  
  
 **Идентификатор ошибки:** BC42313  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите неподдерживаемый \<возвращает > тег.
