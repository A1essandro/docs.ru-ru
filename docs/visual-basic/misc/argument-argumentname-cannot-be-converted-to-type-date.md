# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39date39"></a>Аргумент &#39; &lt;имя_аргумента&gt;&#39; невозможно преобразовать в тип &#39; Дата &#39;
Преобразование пыталось изменить тип переменной на `Date`, однако значение не распознано как допустимая дата.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Найдите преобразование, имеющее смысл.
