# Защищенные типы могут объявляться только в теле класса.
Тип в модуле был объявлен как `Protected`.  
  
 **Идентификатор ошибки:** BC31047  
  
### Исправление ошибки  
  
1.  Объявите тип с классом.  
  
2.  Удалите модификатор `Protected`.
