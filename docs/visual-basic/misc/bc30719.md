# <a name="39case39-statements-are-not-valid-in-the-immediate-window"></a>Операторы 'Case' недопустимы в окне интерпретации
`Case`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30719  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите `Case` операторы из окна **интерпретации** .
