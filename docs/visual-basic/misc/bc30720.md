# Операторы On Error недопустимы в окне интерпретации
Операторы `On Error` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30720  
  
### Исправление ошибки  
  
-   Удалите операторы `On Error` из отлаживаемого кода.
