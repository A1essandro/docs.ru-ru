# <a name="39on-error39-statements-are-not-valid-in-the-immediate-window"></a>«При ошибке» недопустимы в окне интерпретации
`On Error`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30720  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `On Error` из отлаживаемого кода.
