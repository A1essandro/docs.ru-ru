# <a name="ltspecifiergt-is-not-valid-on-a-constant-declaration"></a>"&lt;описатель&gt;" является недопустимым объявлением константы
Инструкция `Const` содержит недопустимое ключевое слово. Оператор `Const` может содержать только ключевые слова `Friend`, `Private`, `Protected`, `Public`и `Shadows` .  
  
 **Идентификатор ошибки:** BC30233  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь в том, что ключевое слово написано правильно.  
  
2.  Удалите недопустимое ключевое слово из оператора `Const` .
