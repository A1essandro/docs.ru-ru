# <a name="label-labelname-is-already-defined-in-the-current-methodmultiline-lambda"></a>Метка '\<имя_метки >' уже определена в текущем метода/многострочного лямбда-выражения
Имя метки повторяется в рамках процедуры. Все имена в процедуре должны быть уникальными.  
  
 **Идентификатор ошибки:** BC30094  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Переименуйте повторяющуюся метку.
