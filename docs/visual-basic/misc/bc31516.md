# <a name="type-39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-declared-39public39"></a>Тип &#39; &lt;typename&gt;&#39; не может использоваться как атрибут, поскольку он не объявлен как &#39; Открытый &#39;
Атрибуты должны быть открытыми.  
  
 **Идентификатор ошибки:** BC31516  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Добавьте модификатор `Public` в класс атрибута.
