# <a name="cannot-convert-argument-ltargumentnamegt-of-type-lttype1gt-to-type-lttype2gt"></a>Не удается преобразовать аргумент "&lt;имя_аргумента&gt;«типа»&lt;тип1&gt;«в тип»&lt;тип2&gt;"
Операция преобразования пытается преобразовать аргумент в несовместимый тип.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что тип, в который вы пытаетесь преобразовать аргумент, является допустимым.
