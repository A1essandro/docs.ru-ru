# <a name="value-ltvaluename1gt-cannot-be-converted-to-ltvaluename2gt"></a>Значение "&lt;имя_значения1&gt;«невозможно преобразовать в»&lt;имя_значения2&gt;"
Предпринята попытка преобразования между двумя несовместимыми типами.  
  
 **Идентификатор ошибки:** BC30742  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте преобразуемые типы.
