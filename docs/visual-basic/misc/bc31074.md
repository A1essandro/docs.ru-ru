# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Конструктор делегата для типа "\<typename >", который требуется для реализации делегатов, не найден
Сборка, содержащая тип, может быть повреждена или недопустима.  
  
 **Идентификатор ошибки:** BC31074  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте допустимость сборки, содержащей этот тип.  
  
2.  Скомпилируйте программу еще раз, чтобы узнать, повторится ли ошибка.  
  
3.  Если ошибка продолжает появляться, сохраните работу и перезапустите Visual Studio.
