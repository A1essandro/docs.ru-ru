# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>Объект больше не существует из-за ошибки компиляции или удаления
В компиляторе [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] произошла ошибка доступа к элементу кода.  
  
 **Идентификатор ошибки:** BC32300  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Скомпилируйте программу еще раз, чтобы узнать, повторится ли ошибка.  
  
2.  Если устранить ошибку не удается, переустановите компилятор [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] .  
  
3.  Если ошибка повторяется после переустановки, соберите сведения об условиях ее возникновения и уведомите службу технической поддержки Майкрософт.
