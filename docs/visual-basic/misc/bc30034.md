# <a name="bracketed-identifier-is-missing-closing-"></a>У идентификатора в квадратных скобках отсутствует закрывающая скобка "]".
Квадратные скобки в имени должны быть парными.  
  
 **Идентификатор ошибки:** BC30034  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Поместите закрывающую скобку в конце имени.
