# <a name="bracketed-identifier-is-missing-closing-3939"></a>Идентификатора в квадратных скобках отсутствует закрывающая скобка &#39;]&#39;
Квадратные скобки в имени должны быть парными.  
  
 **Идентификатор ошибки:** BC30034  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Поместите закрывающую скобку в конце имени.
