# <a name="method-ltmethodnamegt-cannot-be-declared-partial-because-only-one-method-ltmethodnamegt-can-be-marked-partial"></a>Метод "&lt;имя_метода&gt;" не может быть объявлен «Partial», так как только один метод "&lt;имя_метода&gt;" может быть помечен как «Partial»
Не допускается иметь два объявления разделяемых методов с одним именем и сигнатурой.  
  
 **Идентификатор ошибки:** BC31433  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Измените имя или сигнатуру одного из объявлений разделяемого метода.
