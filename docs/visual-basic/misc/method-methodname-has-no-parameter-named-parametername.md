# <a name="method-39ltmethodnamegt39-has-no-parameter-named-39ltparameternamegt39"></a>Метод &#39; &lt;имя_метода&gt;&#39; имеет параметр с именем &#39;&lt; имя_параметра&gt;&#39;
Указанный именованный аргумент не соответствует ни одному из аргументов целевого метода.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что именованный аргумент был указан верно.
