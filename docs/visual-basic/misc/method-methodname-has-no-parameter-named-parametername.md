# <a name="method-ltmethodnamegt-has-no-parameter-named-ltparameternamegt"></a>Метод "&lt;имя_метода&gt;«не имеет параметра с именем»&lt;parametername&gt;"
Указанный именованный аргумент не соответствует ни одному из аргументов целевого метода.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что именованный аргумент был указан верно.
