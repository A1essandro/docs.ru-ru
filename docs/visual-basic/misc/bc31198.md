# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a>XML-литерал можно использовать здесь, только если заключить его в скобки
Объявление XML-литерала используется в неоднозначном для компилятора месте [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] . То есть [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] компилятор не может определить ли меньше-не символ (<) предназначен как оператор сравнения или запуска XML-литерала. Примером является следующий код:  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 **Идентификатор ошибки:** BC31198  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Заключите объявление XML-литерала в скобки, как показано в следующем примере:  
  
```vb  
Dim queryResult = From element In elements _  
                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
-   Измените код, чтобы он ссылался на существующий XML-литерал, как показано в следующем примере:  
  
```vb  
Dim queryResult = From element In elements _  
                  Where e.<sample>.Value = "Value" _  
                  Select element  
```
