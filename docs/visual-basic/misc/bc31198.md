# <a name="xml-literal-cannot-appear-here-unless-it-is-enclosed-in-parentheses"></a><span data-ttu-id="882c9-102">XML-литерал можно использовать здесь, только если заключить его в скобки</span><span class="sxs-lookup"><span data-stu-id="882c9-102">XML literal cannot appear here unless it is enclosed in parentheses</span></span>
<span data-ttu-id="882c9-103">Объявление XML-литерала используется в выражении в расположение, которое находится в неоднозначном для компилятора Visual Basic.</span><span class="sxs-lookup"><span data-stu-id="882c9-103">An XML literal declaration is used in an expression in a location that is ambiguous to the Visual Basic compiler.</span></span> <span data-ttu-id="882c9-104">То есть компилятор Visual Basic не может определить является ли меньше-не символ (<) предназначен как оператор сравнения или запуска XML-литерала.</span><span class="sxs-lookup"><span data-stu-id="882c9-104">That is, the Visual Basic compiler cannot determine whether the less-than character (<) is intended as a comparative operator or the start of an XML literal.</span></span> <span data-ttu-id="882c9-105">Примером является следующий код:</span><span class="sxs-lookup"><span data-stu-id="882c9-105">The following code provides an example:</span></span>  

```vb  
' Generates an error.  
Dim queryResult = From element In elements _  
                  Where <sample>Value</sample> = "Value" _  
                  Select element  
```  
  
 <span data-ttu-id="882c9-106">**Идентификатор ошибки:** BC31198</span><span class="sxs-lookup"><span data-stu-id="882c9-106">**Error ID:** BC31198</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="882c9-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="882c9-107">To correct this error</span></span>  
  
-   <span data-ttu-id="882c9-108">Заключите объявление XML-литерала в скобки, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="882c9-108">Enclose the XML literal declaration in parentheses, as shown in the following example:</span></span>  
  
```vb

                  Where (<sample> Value</sample>) = "Value" _  
                  Select element  
```  
  
-   <span data-ttu-id="882c9-109">Измените код, чтобы он ссылался на существующий XML-литерал, как показано в следующем примере:</span><span class="sxs-lookup"><span data-stu-id="882c9-109">Modify your code to refer to an existing XML literal, as shown in the following example:</span></span>

                  Select element
