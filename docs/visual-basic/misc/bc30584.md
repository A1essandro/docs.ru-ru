# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39; &lt;тип&gt;&#39; не может наследоваться несколько раз
Существует повторяющийся оператор `Inherits` .  
  
 **Идентификатор ошибки:** BC30584  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите лишний оператор `Inherits` .
