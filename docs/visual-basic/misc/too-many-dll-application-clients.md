# <a name="too-many-dll-application-clients"></a>Слишком много приложений-клиентов DLL
Библиотеку динамической компоновки (DLL) для Visual Basic может обеспечить доступ ограниченному числу ведущих приложений. Приложения и других приложений, которые являются узлами Visual Basic (некоторые из которых может обращаться ваше приложение), пытаются получить доступ к DLL Visual Basic, в то же время.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Уменьшите число открытых приложений, использующих Visual Basic.
