# <a name="39ltfunctionnamegt39-is-not-declared-visual-basic-error"></a>&#39; &lt;functionname&gt;&#39; не объявлен (ошибка Visual Basic)
"\<имя_функции >" не объявлен. Возможности ввода/вывода файлов доступны в пространстве имен Microsoft.VisualBasic.  
  
 **Идентификатор ошибки:** BC30815  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте функции файлового ввода-вывода, определенные в `My.Computer.FileSystem`, пространстве имен `System.IO` , функциях времени выполнения Visual Basic или в объекте `Microsoft.VisualBasic.FileSystem` .
