<span data-ttu-id="37a72-103">В определении унарного оператора указан параметр с типом, отличным от типа класса или структуры, в которой определен оператор.</span><span class="sxs-lookup"><span data-stu-id="37a72-103">A definition of a unary operator specifies a parameter with a type other than that of the class or structure in which the operator is defined.</span></span>  
  
 <span data-ttu-id="37a72-104">При определении оператора в классе или структуре по крайней мере один из параметров должен иметь тип класса или структуры.</span><span class="sxs-lookup"><span data-stu-id="37a72-104">When you define an operator in a class or structure, at least one of the parameters must be of the type of that class or structure.</span></span> <span data-ttu-id="37a72-105">В случае унарного оператора единственный операнд должен иметь тип данного класса или структуры.</span><span class="sxs-lookup"><span data-stu-id="37a72-105">In the case of a unary operator, the sole operand must be of the type of that class or structure.</span></span>
