# <a name="string-length-exceeds-maximum-length-of-32767-characters-for-39filesystem39-apis"></a>Длина строки превышает максимальное значение 32767 символов для &#39;FileSystem&#39; API-интерфейсы
Длина строки превышает максимальное значение 32 767 символов.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Сократите длину строки.
