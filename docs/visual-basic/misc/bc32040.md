# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>&#39; &lt;Ключевое слово&gt; &#39; ключевое слово используется для перегрузки наследуемых членов; не используйте &#39; &lt;ключевое слово&gt; &#39; ключевое слово, при перегрузке &#39;Sub New&#39;
Конструктор объявлен с ключевым словом `Overloads` .  
  
 Visual Basic не поддерживает наследование или перегрузку конструкторов.  
  
 **Идентификатор ошибки:** BC32040  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите ключевое слово `Overloads` из всех объявлений конструктора.
