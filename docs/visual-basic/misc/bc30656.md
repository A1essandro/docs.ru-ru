# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Поле &#39; &lt;fieldname&gt; &#39; имеет неподдерживаемый тип
Поле объявлено как тип, не может быть представлен Visual Studio.  
  
 **Идентификатор ошибки:** BC30656  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Укажите в определении поля поддерживаемый тип.  
  
2.  Разработайте новый тип, описывающий данные поля.
