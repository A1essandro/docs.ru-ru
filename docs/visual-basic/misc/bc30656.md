# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Поле "&lt;fieldname&gt;" имеет неподдерживаемый тип
Поле объявляется в качестве типа, который не может быть представлен [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **Идентификатор ошибки:** BC30656  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Укажите в определении поля поддерживаемый тип.  
  
2.  Разработайте новый тип, описывающий данные поля.
