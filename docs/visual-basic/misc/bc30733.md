# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>"&lt;membername&gt;'уже объявлен в'&lt;procedurename&gt;", которая была создана для этой &lt;имя_контейнера&gt;
Объявленный член имеет то же имя, что и член, объявленный средой CLR.  
  
 **Идентификатор ошибки:** BC30733  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените имя члена, чтобы устранить конфликт.
