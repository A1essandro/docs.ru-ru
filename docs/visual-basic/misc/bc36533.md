## <a name="to-correct-this-error"></a><span data-ttu-id="fa5ed-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="fa5ed-106">To correct this error</span></span>

```vb  
    Dim fv = filterValue  
    Dim queryResult = From num In collection _  
                      Where num < fv  
End Sub  
```
