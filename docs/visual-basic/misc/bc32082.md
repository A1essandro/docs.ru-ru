# <a name="type-argument-lttypeargumentnamegt-is-declared-mustinherit-and-does-not-satisfy-the-new-constraint-for-type-parameter-lttypeparameternamegt"></a><span data-ttu-id="810ef-102">Аргумент типа "&lt;имя_аргумента_типа&gt;«объявлен как «MustInherit» и не удовлетворяет ограничению «New» параметра типа»&lt;имя_параметра_типа&gt;"</span><span class="sxs-lookup"><span data-stu-id="810ef-102">Type argument '&lt;typeargumentname&gt;' is declared 'MustInherit' and does not satisfy the 'New' constraint for type parameter '&lt;typeparametername&gt;'</span></span>
<span data-ttu-id="810ef-103">Универсальный тип вызывается с классом `MustInherit` в качестве аргумента типа, в то время как соответствующий параметр типа объявлен с ограничением `New` .</span><span class="sxs-lookup"><span data-stu-id="810ef-103">A generic type is invoked with a `MustInherit` class as a type argument, while the corresponding type parameter is declared with the `New` constraint.</span></span>  
  
 <span data-ttu-id="810ef-104">Ограничение `New` требует, чтобы тип, передаваемый в соответствующий аргумент типа, поддерживал создание объектов.</span><span class="sxs-lookup"><span data-stu-id="810ef-104">The `New` constraint requires that the type passed in the corresponding type argument must support the creation of objects.</span></span> <span data-ttu-id="810ef-105">Однако *абстрактный* класс, то есть класс, объявленный как `MustInherit`, не предоставляет никакие конструкторы, поэтому вы не можете создавать из него какие-либо объекты.</span><span class="sxs-lookup"><span data-stu-id="810ef-105">However, an *abstract* class, that is, a class declared as `MustInherit`, does not expose any constructors because you cannot create any objects from it.</span></span>  
  
 <span data-ttu-id="810ef-106">**Идентификатор ошибки:** BC32082</span><span class="sxs-lookup"><span data-stu-id="810ef-106">**Error ID:** BC32082</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="810ef-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="810ef-107">To correct this error</span></span>  
  
1.  <span data-ttu-id="810ef-108">Если класс, используемый в аргументе типа, не должен быть абстрактным, удалите из его объявления ключевое слово `MustInherit` .</span><span class="sxs-lookup"><span data-stu-id="810ef-108">If the class used in the type argument does not need to be abstract, then remove the `MustInherit` keyword from its declaration.</span></span>  
  
2.  <span data-ttu-id="810ef-109">Если класс, используемый в аргументе типа, должен быть абстрактным, но не требуется использовать его для создания универсального типа, то передайте в качестве аргумента типа другой класс.</span><span class="sxs-lookup"><span data-stu-id="810ef-109">If the class used in the type argument needs to be abstract but does not need to be used to construct the generic type, then pass a different class in the type argument.</span></span>  
  
3.  <span data-ttu-id="810ef-110">Если не требуется, чтобы соответствующий параметр типа создавал объекты из переданного в него типа, удалите из его объявления ограничение `New` .</span><span class="sxs-lookup"><span data-stu-id="810ef-110">If the corresponding type parameter does not need to create any objects from the type passed to it, then remove the `New` constraint from its declaration.</span></span>
