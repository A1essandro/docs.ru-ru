# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>Метод не может одновременно содержать &#39;повторите&#39; инструкции и &#39;On Error&#39; или &#39;Resume&#39; инструкции
Вы объединили оператор `Try` с `On Error` или `Resume`.  
  
 **Идентификатор ошибки:** BC30544  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Удалите `On Error` или `Resume`.
