# <a name="ltnamegt-cannot-expose-type-lttype1gt-used-in-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-lttype2gt"></a>"&lt;имя&gt;«не может представлять тип»&lt;тип1&gt;«используется в базовом делегированном типе»&lt;тип_делегата&gt;" события, он реализует вне проекта посредством &lt;описатель&gt; "&lt;тип2&gt;"
Код представляет закрытый тип за пределами открытого класса.  
  
 **Идентификатор ошибки:** BC30924  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Объявите тип `Public` или используйте другой тип.
