# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>Номер предупреждения "&lt;номер&gt;«для параметра»&lt;параметра optionname&gt;" является недопустимым или не настраивается
Идентификатор предупреждения является недопустимым. Идентификатор должен быть целым числом больше 0.  
  
 **Идентификатор ошибки:** BC2026  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте, является ли указанный идентификатор предупреждения допустимым.
