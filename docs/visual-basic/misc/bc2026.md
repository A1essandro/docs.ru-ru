# <a name="warning-number-ltnumbergt-for-the-option-ltoptionnamegt-is-either-not-configurable-or-not-valid"></a>Номер предупреждения "&lt;номер&gt;«для параметра»&lt;имя_параметра&gt;" не настраивается или недопустим
Идентификатор предупреждения является недопустимым. Идентификатор должен быть целым числом больше 0.  
  
 **Идентификатор ошибки:** BC2026  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте, является ли указанный идентификатор предупреждения допустимым.
