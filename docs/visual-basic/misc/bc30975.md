# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Для&#39; переменной цикла уже используется во внешнем &#39;для&#39; цикла
Все вложенные циклы `For` должны использовать уникальные управляющие переменные.  
  
 **Идентификатор ошибки:** BC30975  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените имя одной из управляющих переменных.
