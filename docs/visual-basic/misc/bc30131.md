# <a name="security-attribute-39ltattributenamegt39-cannot-be-applied-to-a-module"></a>Атрибут безопасности &#39; &lt;attributename&gt;&#39; не может быть применен к модулю
Среда CLR не принимает атрибут безопасности для модуля.

**Идентификатор ошибки:** BC30131

## <a name="to-correct-this-error"></a>Исправление ошибки

- Удалите атрибут из объявления.
