# <a name="39ltspecifier1gt39-and-39ltspecifier2gt39-cannot-be-combined"></a>&#39; &lt;спецификатор1&gt;&#39; и &#39;&lt; спецификатор2&gt;&#39; нельзя использовать вместе
Два несовместимых спецификатора, например `Shadows` и `Overrides` , используются вместе.  
  
 **Идентификатор ошибки:** BC31408  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите один из спецификаторов.
