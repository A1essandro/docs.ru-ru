# <a name="comma-or-3939-expected-type-parameter-list"></a>Запятая или &#39;) &#39; Ожидаемый (список параметров типа)
Список параметров содержит непредвиденный элемент. Используйте запятые (`,`) для разделения параметров типа в таком списке и закрывающие скобки (`)`) для завершения списка.  
  
 **Идентификатор ошибки:** BC32099  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте запятую, чтобы продолжить список параметров типа, или закрывающую скобку, чтобы его завершить.
