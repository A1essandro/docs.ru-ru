# <a name="nesting-of-parentheses-in-lambda-expression39s-parameter-list-exceeds-maximum-allowable-depth"></a>Вложенность скобок в лямбда-выражение &#39; список параметров s превышает максимально допустимую глубину
Вложенные скобки в списке параметров лямбда-выражения не могут превышать глубину 128.  
  
 **Идентификатор ошибки:** BC36664
