# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Аргумент &#39; Доступ к &#39; не допускается. Допустимые значения для режима Input являются &#39;OpenAccess.Read &#39; и &#39;OpenAccess.Default &#39;
Значение доступа недопустимо в режиме `Input` .  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените значение на `OpenAccess.Read` или на `OpenAccess.Default`.
