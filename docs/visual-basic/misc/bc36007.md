# <a name="39end-using39-must-be-preceded-by-a-matching-39using39"></a>&#39;Завершить с помощью&#39; должен предшествовать соответствующий &#39;Using&#39;
Оператору `End Using` не предшествует соответствующее объявление `Using` .  
  
 **Идентификатор ошибки:** BC36007  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите оператор `End Using` , если он лишний.  
  
-   Укажите отсутствующий [инструкции Using](../../visual-basic/language-reference/statements/using-statement.md) если она отсутствует.  
  
-   Переместите оператор `End Using` в соответствующее место кода.
