# Требуется Is
Оператор `TypeOf` указан без предложения `Is`.  
  
 **Идентификатор ошибки:** BC30224  
  
### Исправление ошибки  
  
-   Добавьте предложение `Is` после оператора `TypeOf`, например `TypeOf MyControl Is ComboBox`.
