# <a name="operator-39operator39-is-not-defined-for-types-39type139-and-39type239"></a>Оператор &#39;оператор&#39; не определен для типов &#39;тип1&#39; и &#39;тип2&#39;
Оператор "оператор" для типов "тип1" и "тип2" не определен. Свойство «Значение» для получения строкового значения первого элемента "\<Родительскийэлемент >".  
  
 XML-литерал использовался в двоичном или `Boolean` сравнении.  
  
 **Идентификатор ошибки:** BC31195  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте свойство `Value` XML-литерала для двоичного или `Boolean` сравнения.
