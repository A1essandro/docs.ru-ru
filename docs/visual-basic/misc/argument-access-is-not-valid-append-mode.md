# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Аргумент &#39; Доступ к &#39; не допускается. Допустимые значения находятся в режиме Append &#39;OpenAccess.Write &#39; и &#39;OpenAccess.Default &#39;
Значение доступа недопустимо в режиме Append.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Измените значение на `OpenAccess.Write` или на `OpenAccess.Default`.
