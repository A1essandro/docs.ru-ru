# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>Класс&lt;classname&gt;"не удалось создать: &lt;ошибки&gt;
Ошибка инициализации класса. Одной из возможных причин является поврежденным [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] установки.  
  
 **Идентификатор ошибки:** BC32400  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте указанное сообщение об ошибке и выполните соответствующее действие.  
  
2.  Переустановите [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] и выполните повторную компиляцию программы.  
  
3.  Если устранить ошибку не удается, переустановите компилятор [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Если ошибка не устранена после переустановки, соберите сведения об условиях ее возникновения и уведомите службу технической поддержки Майкрософт.
