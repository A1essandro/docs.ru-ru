 <span data-ttu-id="ee51c-116">Первый параметр в следующем примере является единственным указанием, что метод `Print` расширяет тип данных `String` .</span><span class="sxs-lookup"><span data-stu-id="ee51c-116">The first parameter in the following example is the only indication that the `Print` method extends the `String` data type.</span></span> <span data-ttu-id="ee51c-117">Таким образом, он не может быть необязательным.</span><span class="sxs-lookup"><span data-stu-id="ee51c-117">Therefore, it cannot be optional.</span></span>
