## <a name="example"></a><span data-ttu-id="ce35e-115">Пример</span><span class="sxs-lookup"><span data-stu-id="ce35e-115">Example</span></span>  
 <span data-ttu-id="ce35e-116">Первый параметр в следующем примере является единственным указанием, что метод `Print` расширяет тип данных `String` .</span><span class="sxs-lookup"><span data-stu-id="ce35e-116">The first parameter in the following example is the only indication that the `Print` method extends the `String` data type.</span></span> <span data-ttu-id="ce35e-117">Таким образом, он не может быть необязательным.</span><span class="sxs-lookup"><span data-stu-id="ce35e-117">Therefore, it cannot be optional.</span></span>  
  
```

    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="ce35e-118">Когда метод расширения вызывается указанным ниже образом, параметр `str` в методе привязывается к `greeting`, экземпляру `String` , который вызывает метод `Print`.</span><span class="sxs-lookup"><span data-stu-id="ce35e-118">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="ce35e-119">Компилятор использует `greeting` в качестве аргумента для метода расширения `Print`.</span><span class="sxs-lookup"><span data-stu-id="ce35e-119">The compiler uses `greeting` as the argument to extension method `Print`.</span></span>

