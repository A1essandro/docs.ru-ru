# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>Длина аргумента &#39; &lt;имя_аргумента&gt;&#39; должен быть больше нуля
Аргумент имеет значение, которое меньше или равно нулю.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите аргумент больше нуля.
