# <a name="length-of-argument-ltargumentnamegt-must-be-greater-than-zero"></a>Длина аргумента "&lt;имя_аргумента&gt;" должно быть больше нуля
Аргумент имеет значение, которое меньше или равно нулю.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите аргумент больше нуля.
