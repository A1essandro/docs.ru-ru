# <a name="project-level-conditional-compilation-constant-39ltconstantnamegt39-is-not-valid-lterrorgt"></a>Константа условной компиляции на уровне проекта &#39; &lt;имя_константы&gt; &#39; является недопустимым: &lt;ошибки&gt;
Для проекта указан недопустимый параметр условной компиляции.  
  
 **Идентификатор ошибки:** BC31030  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте константы условной компиляции, используемые в этом проекте.
