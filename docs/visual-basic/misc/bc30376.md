# <a name="39new39-cannot-be-used-on-class-39ltclassnamegt39-because-it-contains-a-39mustoverride39-member-that-has-not-been-overridden"></a>&#39; Создать &#39; не может использоваться в классе &#39; &lt;classname&gt;&#39; так как он содержит &#39; MustOverride &#39; элемент, который не был переопределен
Предпринята попытка использования `New` в классе, содержащем член `MustOverride` , который не был переопределен.  
  
 **Идентификатор ошибки:** BC30376  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите оператор `New` .
