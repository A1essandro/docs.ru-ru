# <a name="new-cannot-be-used-on-class-ltclassnamegt-because-it-contains-a-mustoverride-member-that-has-not-been-overridden"></a>«New» не может использоваться для класса&lt;classname&gt;", так как он содержит член «MustOverride», который не был переопределен
Предпринята попытка использования `New` в классе, содержащем член `MustOverride` , который не был переопределен.  
  
 **Идентификатор ошибки:** BC30376  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите оператор `New` .
