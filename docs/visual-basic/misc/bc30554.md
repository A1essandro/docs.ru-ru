# <a name="39ltnamespacenamegt39-is-ambiguous"></a>"&lt;Имя_пространства_имен&gt;" является неоднозначным
Вы указали неоднозначное имя, которое конфликтует с другим именем. [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Компилятора нет никаких правил разрешения конфликтов, необходимо устранить неоднозначность имен самостоятельно.  
  
 **Идентификатор ошибки:** BC30554  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите полное имя.
