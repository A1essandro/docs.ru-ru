# <a name="ltnamespacenamegt-is-ambiguous"></a>"&lt;namespacename&gt;" является неоднозначным
Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор Visual Basic не имеет правила разрешения конфликтов; Вы должны самостоятельно устранить неоднозначность имен.  
  
 **Идентификатор ошибки:** BC30554  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите полное имя.
