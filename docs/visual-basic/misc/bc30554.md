# <a name="39ltnamespacenamegt39-is-ambiguous"></a>&#39; &lt;Имя_пространства_имен&gt;&#39; является неоднозначным
Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] не имеет правил разрешения конфликтов; вы должны самостоятельно устранить неоднозначность имен.  
  
 **Идентификатор ошибки:** BC30554  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите полное имя.
