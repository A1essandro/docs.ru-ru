# &quot;&lt;пространство_имен &gt;&quot; является неоднозначным
Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] не имеет правил разрешения конфликтов; вы должны самостоятельно устранить неоднозначность имен.  
  
 **Идентификатор ошибки:** BC30554  
  
### Исправление ошибки  
  
-   Укажите полное имя.
