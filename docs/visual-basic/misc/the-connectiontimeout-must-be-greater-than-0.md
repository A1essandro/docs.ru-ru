# <a name="the-connectiontimeout-must-be-greater-than-0"></a>Значение ConnectionTimeout должно быть больше 0.
При отправке и загрузке файлов с помощью [My.Computer.Network](xref:Microsoft.VisualBasic.Devices.Network), необходимо указать `connectionTimeout` больше, чем `0`.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите `connectionTimeout` больше `0`.
