# <a name="arrays-cannot-be-declared-with-39new39"></a><span data-ttu-id="96aba-102">Массивы не могут объявляться с &#39; Создать &#39;</span><span class="sxs-lookup"><span data-stu-id="96aba-102">Arrays cannot be declared with &#39;New&#39;</span></span>
<span data-ttu-id="96aba-103">Ключевое слово `New` может содержаться только в части инициализации объявления массива.</span><span class="sxs-lookup"><span data-stu-id="96aba-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="96aba-104">Это означает, что ключевое слово `New` должно быть справа от знака равенства (`=`), чтобы можно было создать новый тип массива, назначаемый переменной массива.</span><span class="sxs-lookup"><span data-stu-id="96aba-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="96aba-105">Краткий путь для инициализации класса недоступен для массивов.</span><span class="sxs-lookup"><span data-stu-id="96aba-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="96aba-106">Обе следующие строки кода допустимы и эквивалентны, поскольку они инициализируют объект из класса.</span><span class="sxs-lookup"><span data-stu-id="96aba-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="96aba-107">Тем не менее, при инициализации массива нельзя использовать тот же краткий путь, что и для инициализации класса.</span><span class="sxs-lookup"><span data-stu-id="96aba-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="96aba-108">Обратите внимание, что предложение `New` для массива должно содержать как круглые скобки `()`, так и фигурные скобки `{}`.</span><span class="sxs-lookup"><span data-stu-id="96aba-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="96aba-109">Круглые скобки указывают, что новый тип является массивом, а фигурные скобки передают значения инициализации.</span><span class="sxs-lookup"><span data-stu-id="96aba-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="96aba-110">Компилятор требует наличия фигурных скобок, даже если они пустые, то есть даже если вы не инициализируете какое-либо значение массива.</span><span class="sxs-lookup"><span data-stu-id="96aba-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="96aba-111">**Идентификатор ошибки:** BC30053</span><span class="sxs-lookup"><span data-stu-id="96aba-111">**Error ID:** BC30053</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="96aba-112">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="96aba-112">To correct this error</span></span>  
  
-   <span data-ttu-id="96aba-113">Замените инструкцию, например `Dim myDates() As New Date` , на такую инструкцию, как `Dim myDates() As Date = New Date() {}`.</span><span class="sxs-lookup"><span data-stu-id="96aba-113">Replace a statement such as `Dim myDates() As New Date` with a statement such as `Dim myDates() As Date = New Date() {}`.</span></span>
