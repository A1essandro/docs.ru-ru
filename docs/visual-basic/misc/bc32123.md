# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-it-could-be-identical-to-interface-39ltinterfacename2gt39-from-which-the-interface-39ltinterfacename3gt39-inherits-for-some-type-arguments"></a><span data-ttu-id="c9787-102">Невозможно наследовать интерфейс &#39; &lt;имя_интерфейса1&gt;&#39; поскольку он может совпадать с интерфейсом &#39;&lt; имя_интерфейса2&gt;&#39; из которого интерфейс &#39;&lt; имя_интерфейса3&gt;&#39; наследует некоторые аргументы типа</span><span class="sxs-lookup"><span data-stu-id="c9787-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because it could be identical to interface &#39;&lt;interfacename2&gt;&#39; from which the interface &#39;&lt;interfacename3&gt;&#39; inherits for some type arguments</span></span>
<span data-ttu-id="c9787-103">Универсальный интерфейс наследует от двух или более универсальных интерфейсов, и два из этих наследований могут конфликтовать для некоторых значений аргументов типов.</span><span class="sxs-lookup"><span data-stu-id="c9787-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="c9787-104">Следующие операторы могут привести к этой ошибке.</span><span class="sxs-lookup"><span data-stu-id="c9787-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="c9787-105">Если `derivedInterface` создается или реализуется, предоставляя один и тот же тип как для `t1` , так и для `t2`, он должен наследовать две версии `interfaceA` с одинаковыми аргументами типов.</span><span class="sxs-lookup"><span data-stu-id="c9787-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="c9787-106">Это приведет к неоднозначности определения, к какой версии следует получить доступ.</span><span class="sxs-lookup"><span data-stu-id="c9787-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="c9787-107">**Идентификатор ошибки:** BC32123</span><span class="sxs-lookup"><span data-stu-id="c9787-107">**Error ID:** BC32123</span></span>
