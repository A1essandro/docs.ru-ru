# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>С поздним связыванием присвоение полю значения типа "&lt;typename&gt;«является недопустимым при»&lt;имя&gt;" является результатом выражения с поздним связыванием
Вы попытались выполнить присвоение с поздним связыванием, что недопустимо.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Создайте присвоение с ранним связыванием.
