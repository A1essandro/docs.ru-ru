# <a name="assembly-attribute-ltattributenamegt-is-not-valid-lterrorgt"></a>Атрибут сборки "&lt;attributename&gt;" является недопустимым: &lt;ошибка&gt;
Среда CLR не принимает атрибут для сборки.

**Идентификатор ошибки:** BC30129

## <a name="to-correct-this-error"></a>Исправление ошибки

1. Удалите атрибут из объявления или замените его атрибутом, который допустим в данном контексте.

2. Предпримите действия согласно сообщению об ошибке.
