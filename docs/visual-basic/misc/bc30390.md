# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>"&lt;уточненный_метод&gt;«недоступен в этом контексте, так как он является»&lt;модификатор&gt;"
Предпринята попытка обращения к методу, который недоступен в данном контексте из-за примененного к нему модификатора.  
  
 **Идентификатор ошибки:** BC30390  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите этот модификатор.
