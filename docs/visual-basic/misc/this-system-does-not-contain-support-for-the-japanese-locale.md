# Эта система не содержит поддержки для стандартов японского языка
Вы пытаетесь использовать элемент перечисления `VbStrConv.Japanese`, но ваша система не поддерживает японский язык.  
  
### Исправление ошибки  
  
1.  Обновите систему для поддержки японского языка или запустите приложение на компьютере, который поддерживает этот языковой стандарт.
