# <a name="39microsoftvisualbasiccomclassattribute39-is-specified-for-class-39ltclassnamegt39-but-it-has-no-public-members-that-can-be-exposed-to-com-therefore-no-com-interfaces-are-generated"></a>Указанный для класса «Microsoft.VisualBasic.ComClassAttribute»&lt;classname&gt;", но у него нет открытых членов, доступные для COM; поэтому интерфейсы COM не генерируются
Класс, использующий блок атрибутов `COMClassAttribute` , не определяет никакие свойства или методы `Public` . Если класс должен быть предоставлен как COM-объект, все его свойства и методы должны быть объявлены с уровнем доступа `Public` .  
  
 Данное сообщение является предупреждением по умолчанию. Дополнительные сведения о скрытии предупреждений или обработке предупреждений как ошибок см. в разделе [в Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Идентификатор ошибки:** BC40011  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте ключевое слово `Public` в одно или несколько свойств или методов в классе или удалите блок атрибутов `COMClassAttribute` .
