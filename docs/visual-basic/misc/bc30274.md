# <a name="parameter-39ltparameternamegt39-of-39ltmethodnamegt39-already-has-a-matching-argument"></a>Параметр &#39; &lt;имя_параметра&gt; &#39; из &#39; &lt;имя_метода&gt; &#39; уже имеет соответствующий аргумент
Аргумент передается по имени несколько раз в одном вызове процедуры.  
  
 **Идентификатор ошибки:** BC30274  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте правильность написания имен аргументов.  
  
-   Удалите повторяющийся аргумент из списка аргументов.
