# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>"&lt;имя_интерфейса1&gt;«не может реализовать»&lt;имя_метода&gt;", так как нет соответствующего &lt;метод&gt; в интерфейсе "&lt;имя_интерфейса2&gt;"
Предпринята попытка реализации метода в интерфейсе при отсутствии соответствующего метода в интерфейсе, откуда получен первый интерфейс.  
  
 **Идентификатор ошибки:** BC30401  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь, что имя метода указано верно, или определите отсутствующий метод.
