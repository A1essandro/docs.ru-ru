# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39; &lt;имя_интерфейса1&gt;&#39; не может реализовать &#39;&lt; имя_метода&gt;&#39; поскольку отсутствует соответствующий &lt;метод&gt; на интерфейс &#39;&lt; имя_интерфейса2&gt;&#39;
Предпринята попытка реализации метода в интерфейсе при отсутствии соответствующего метода в интерфейсе, откуда получен первый интерфейс.  
  
 **Идентификатор ошибки:** BC30401  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь, что имя метода указано верно, или определите отсутствующий метод.
