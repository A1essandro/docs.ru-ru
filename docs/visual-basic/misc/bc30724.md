# <a name="39error39-statements-are-not-valid-in-the-immediate-window"></a>«Ошибка» недопустимы в окне интерпретации
`Error`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30724  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `Error` из отлаживаемого кода.
