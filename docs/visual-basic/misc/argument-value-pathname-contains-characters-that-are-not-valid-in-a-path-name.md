# <a name="argument-value-39ltpathnamegt39-contains-characters-that-are-not-valid-in-a-path-name"></a>Значение аргумента &#39; &lt;pathname&gt;&#39; содержит знаки, недопустимые в имени пути
Путь содержит недопустимые символы.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите символ из пути.
