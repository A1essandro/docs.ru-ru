# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>Значение аргумента "&lt;pathname&gt;" содержит символы, недопустимые в имени пути
Путь содержит недопустимые символы.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите символ из пути.
