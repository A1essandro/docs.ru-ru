# <a name="name-cannot-expose-type-type-used-in-the-underlying-delegate-type-delegatetype-of-the-event-it-is-implementing-to-specifier1-type1-through-specifier2-type2"></a>"\<имя >" не может представлять тип "\<тип >" используется в базовом делегированном типе "\<тип_делегата >" события, которое он реализует, для \<спецификатор1 > "\<тип1 >" через \<спецификатор2 > "\<тип2 >"
Код представляет закрытый тип за пределами открытого класса.  
  
 **Идентификатор ошибки:** BC30923  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Объявите тип `Public` или используйте другой тип.
