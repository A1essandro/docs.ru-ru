# <a name="too-few-type-arguments"></a>Слишком мало аргументов типа.
Универсальный метод был вызван с меньшим числом аргументов типа, чем число параметров типа.  
  
 При вызове универсального метода вы должны представить по одному аргументу типа для каждого параметра типа, определяемого этим методом.  
  
 **Идентификатор ошибки:** BC36578  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте аргументы типа в список аргументов типа так, чтобы имелось по одному аргументу типа для каждого параметра типа вызываемого универсального метода.
