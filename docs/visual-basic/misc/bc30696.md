# <a name="39ltmembername1gt39-and-39ltmembername2gt39-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>&#39;&lt;имя_члена1&gt; &#39; и &#39; &lt;имя_члена2&gt; &#39; не могут перегружать друг друга, так как они отличаются только типами необязательных параметров.
Перегруженные члены должны иметь разные типы данных для одного или нескольких обязательных параметров.  
  
 **Идентификатор ошибки:** BC30696  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Определите по крайней мере один обязательный параметр с другим типом данных.
