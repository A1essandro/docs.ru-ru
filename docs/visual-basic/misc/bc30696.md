# <a name="ltmembername1gt-and-ltmembername2gt-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>"&lt;имя_члена1&gt;«и»&lt;имя_члена2&gt;" не могут перегружать друг друга, так как они отличаются только типами необязательных параметров
Перегруженные члены должны иметь разные типы данных для одного или нескольких обязательных параметров.  
  
 **Идентификатор ошибки:** BC30696  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Определите по крайней мере один обязательный параметр с другим типом данных.
