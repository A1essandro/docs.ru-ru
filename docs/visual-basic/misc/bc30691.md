# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;TypeName&gt; &#39; является типом в &#39; &lt;containername&gt; &#39; и не может использоваться как выражение
Одно из используемых имен уже было объявлено как имя типа и не может использоваться в выражении.  
  
 **Идентификатор ошибки:** BC30691  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите уникальное имя члена.  
  
-   Заключите это имя члена в квадратные скобки.
