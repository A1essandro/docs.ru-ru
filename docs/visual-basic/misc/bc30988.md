<span data-ttu-id="50948-103">При объявлении экземпляра класса за ключевым словом `New` должно следовать имя типа или `With`.</span><span class="sxs-lookup"><span data-stu-id="50948-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="50948-104">Например, в следующих инструкциях объявляется `client` как экземпляр класса `Customer` .</span><span class="sxs-lookup"><span data-stu-id="50948-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="50948-105">Имя типа `Customer` , следующее за `New`.</span><span class="sxs-lookup"><span data-stu-id="50948-105">The type name `Customer` follows `New`.</span></span>  
  
```

' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="50948-106">Начиная с версии [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], вы можете объявлять объект как экземпляр анонимного типа, и в этом случае не указывать тип данных.</span><span class="sxs-lookup"><span data-stu-id="50948-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="50948-107">В объявлениях анонимных типов за `With` следует ключевое слово `New`.</span><span class="sxs-lookup"><span data-stu-id="50948-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>

