# <a name="type-or-39with39-expected"></a><span data-ttu-id="e7e4e-102">Тип или &#39; С &#39; Ожидается</span><span class="sxs-lookup"><span data-stu-id="e7e4e-102">Type or &#39;With&#39; expected</span></span>
<span data-ttu-id="e7e4e-103">При объявлении экземпляра класса за ключевым словом `New` должно следовать имя типа или `With`.</span><span class="sxs-lookup"><span data-stu-id="e7e4e-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="e7e4e-104">Например, в следующих инструкциях объявляется `client` как экземпляр класса `Customer` .</span><span class="sxs-lookup"><span data-stu-id="e7e4e-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="e7e4e-105">Имя типа `Customer` , следующее за `New`.</span><span class="sxs-lookup"><span data-stu-id="e7e4e-105">The type name `Customer` follows `New`.</span></span>  
  
```  
' Dim client As New Customer()  
' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="e7e4e-106">Начиная с версии [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], вы можете объявлять объект как экземпляр анонимного типа, и в этом случае не указывать тип данных.</span><span class="sxs-lookup"><span data-stu-id="e7e4e-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="e7e4e-107">В объявлениях анонимных типов за `With` следует ключевое слово `New`.</span><span class="sxs-lookup"><span data-stu-id="e7e4e-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>  
  
```  
Dim person = New With {.Name ="Mike Nash", .Age = 27}  
```
