# <a name="expression-of-type-39lttypename1gt39-can-never-be-of-type-39lttypename2gt39"></a><span data-ttu-id="6c16d-102">Выражение типа &#39; &lt;Имя_типа1&gt;&#39; никогда не может иметь тип &#39;&lt; имя_типа2&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="6c16d-102">Expression of type &#39;&lt;typename1&gt;&#39; can never be of type &#39;&lt;typename2&gt;&#39;</span></span>
<span data-ttu-id="6c16d-103">Выражение `TypeOf`...`Is` проверяет переменную, ссылающуюся на объект, на тип данных, который она не может содержать.</span><span class="sxs-lookup"><span data-stu-id="6c16d-103">A `TypeOf`...`Is` expression tests an object reference variable to a data type that it cannot hold.</span></span>  
  
 <span data-ttu-id="6c16d-104">В некоторых случаях компилятор может определить, что проверка `TypeOf`...`Is` может завершиться только ошибкой, например, если отсутствует отношение наследования между двумя классами.</span><span class="sxs-lookup"><span data-stu-id="6c16d-104">In some cases the compiler can determine that a `TypeOf`...`Is` test can only fail, for example if there is no inheritance relationship between two classes.</span></span>  
  
 <span data-ttu-id="6c16d-105">Эту ошибку может вызывать следующий код.</span><span class="sxs-lookup"><span data-stu-id="6c16d-105">The following code can generate this error.</span></span>  
  
 `Dim refVar as System.Windows.Forms.Form`  
  
 `If TypeOf refVar Is System.Array`  
  
 `End If`  
  
 <span data-ttu-id="6c16d-106">Так как <xref:System.Windows.Forms.Form> и <xref:System.Array> являются полностью несвязанными типами, компилятор может определить, что выражение `TypeOf`...`Is` возвращает `False` для любого значения `refVar`.</span><span class="sxs-lookup"><span data-stu-id="6c16d-106">Because <xref:System.Windows.Forms.Form> and <xref:System.Array> are totally unrelated types, the compiler can determine that the `TypeOf`...`Is` expression returns `False` for any value of `refVar`.</span></span>  
  
 <span data-ttu-id="6c16d-107">**Идентификатор ошибки:** BC31430</span><span class="sxs-lookup"><span data-stu-id="6c16d-107">**Error ID:** BC31430</span></span>
