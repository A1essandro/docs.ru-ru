# <a name="xml-elements-cannot-be-selected-from-type-type"></a><span data-ttu-id="568a2-102">Невозможно выбрать элементы XML из типа "тип"</span><span class="sxs-lookup"><span data-stu-id="568a2-102">XML elements cannot be selected from type 'type'</span></span>
<span data-ttu-id="568a2-103">На дочерний элемент XML имеется ссылка для получения объекта, который не относится к типу <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>или `IEnumerable(Of XElement)`.</span><span class="sxs-lookup"><span data-stu-id="568a2-103">An XML child element has been referenced for an object that is not of type <xref:System.Xml.Linq.XElement>, <xref:System.Xml.Linq.XDocument>, or `IEnumerable(Of XElement)`.</span></span> <span data-ttu-id="568a2-104">Для получения дополнительной информации см. [XML Child Axis Property](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span><span class="sxs-lookup"><span data-stu-id="568a2-104">For more information, see [XML Child Axis Property](../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md).</span></span>  
  
```vb  
' Generates an error.  
Dim var = "sample text".<child>  
```  
  
 <span data-ttu-id="568a2-105">**Идентификатор ошибки:** BC36807</span><span class="sxs-lookup"><span data-stu-id="568a2-105">**Error ID:** BC36807</span></span>
