# <a name="parameter-already-declared-with-name-ltargumentnamegt"></a>Параметр уже объявлен с именем "&lt;имя_аргумента&gt;"
Объявление процедуры определяет два аргумента с одним именем. Все имена аргументов в объявлении процедуры должны быть уникальными.  
  
 **Идентификатор ошибки:** BC30237  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените имя одного из аргументов.
