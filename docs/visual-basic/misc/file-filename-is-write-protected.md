# Файл &quot;&lt;имя_файла&gt;&quot; защищен от записи
Записываемый файл защищен от записи и поэтому не может быть изменен.  
  
### Исправление ошибки  
  
1.  Задайте разрешения на чтение и запись файла или выполните запись в другой файл.
