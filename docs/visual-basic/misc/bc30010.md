# <a name="reference-required-to-module-39ltmodulenamegt39-containing-the-implemented-interface-39ltinterfacenamegt39"></a>Требуется ссылка на модуль &#39; &lt;modulename&gt; &#39; содержит реализованный интерфейс &#39; &lt;interfacename&gt;&#39;
Требуется ссылка на модуль "\<имя_модуля >" содержит реализованный интерфейс "\<имя_интерфейса >". Добавьте эту ссылку в проект.  
  
 Интерфейс определяется в модуле, на который в проекте нет прямой ссылки. Компилятор Visual Basic требует ссылку в целях уточнения, если событие определено в нескольких модулях.  
  
 **Идентификатор ошибки:** BC30010  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Включите в ссылки проекта имя модуля, на который нет ссылки.
