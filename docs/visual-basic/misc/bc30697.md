# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39;&lt;имя_члена1&gt; &#39; не может переопределить &#39; &lt;имя_члена2&gt; &#39; , так как они отличаются типами необязательных параметров.
Перегруженные члены должны иметь разные типы данных для одного или нескольких обязательных параметров.  
  
 **Идентификатор ошибки:** BC30697  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Определите по крайней мере один обязательный параметр с другим типом данных.
