# <a name="ltmembername1gt-cannot-override-ltmembername2gt-because-they-differ-by-the-types-of-optional-parameters"></a>"&lt;имя_члена1&gt;«не может переопределить»&lt;имя_члена2&gt;" так как они отличаются типами необязательных параметров
Перегруженные члены должны иметь разные типы данных для одного или нескольких обязательных параметров.  
  
 **Идентификатор ошибки:** BC30697  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Определите по крайней мере один обязательный параметр с другим типом данных.
