# <a name="type-parameter-not-allowed-in-implements-clause"></a><span data-ttu-id="ba100-102">Параметр типа нельзя использовать в предложении Implements</span><span class="sxs-lookup"><span data-stu-id="ba100-102">Type parameter not allowed in 'Implements' clause</span></span>
<span data-ttu-id="ba100-103">Предложение `Implements` в универсальном типе определяет параметр типа как реализуемый член.</span><span class="sxs-lookup"><span data-stu-id="ba100-103">An `Implements` clause in a generic type specifies a type parameter as the member to be implemented.</span></span>  
  
 <span data-ttu-id="ba100-104">В предложении `Implements` должны указываться интерфейс и член.</span><span class="sxs-lookup"><span data-stu-id="ba100-104">An `Implements` clause must specify an interface and a member.</span></span> <span data-ttu-id="ba100-105">Параметр типа может быть передан в интерфейс, но его нельзя передать члену или использовать в качестве имени члена.</span><span class="sxs-lookup"><span data-stu-id="ba100-105">It can pass a type parameter to the interface, but it cannot pass it to the member, nor use it as the name of the member.</span></span>  
  
 <span data-ttu-id="ba100-106">Следующие операторы могут привести к этой ошибке.</span><span class="sxs-lookup"><span data-stu-id="ba100-106">The following statements can generate this error.</span></span>  
  
```

    Implements i1(Of t)  
    Public Sub doSomething() Implements t  
End Class  
```  
  
 <span data-ttu-id="ba100-107">**Идентификатор ошибки:** BC32056</span><span class="sxs-lookup"><span data-stu-id="ba100-107">**Error ID:** BC32056</span></span>
