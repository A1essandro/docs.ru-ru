# <a name="lttypenamegt-ltmembernamegt-must-be-declared-shadows-because-another-member-with-this-name-is-declared-shadows"></a>&lt;TypeName&gt; "&lt;membername&gt;" должен быть объявлен как «Shadows», поскольку другой член с таким именем объявлен «Shadows»
Это свойство или процедура имеет то же имя, что и другое свойство или процедура, помеченная модификатором `Shadows` .  
  
 **Идентификатор ошибки:** BC30695  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте модификатор `Shadows` .
