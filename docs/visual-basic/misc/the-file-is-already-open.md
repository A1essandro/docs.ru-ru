# Файл уже открыт.
Предпринята попытка открыть файл, который уже открыт.  
  
### Исправление ошибки  
  
-   Закройте файл, прежде чем пытаться открыть его.  
  
     \-или\-  
  
-   Удалите лишний оператор `FileOpen` из вашего кода.
