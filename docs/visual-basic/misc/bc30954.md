 <span data-ttu-id="02c02-107">*Ограничение* для параметра типа ограничивает аргументы типов, которые могут быть переданы в него.</span><span class="sxs-lookup"><span data-stu-id="02c02-107">A *constraint* on a type parameter limits what type arguments can be passed to it.</span></span> <span data-ttu-id="02c02-108">Например, параметр типа может быть ограничен классом, который реализует интерфейс <xref:System.IComparable%601> .</span><span class="sxs-lookup"><span data-stu-id="02c02-108">For example, a type parameter might be constrained to be a class that implements the <xref:System.IComparable%601> interface.</span></span> <span data-ttu-id="02c02-109">Дополнительные сведения см. в подразделе "Ограничения" раздела [Generic Procedures in Visual Basic](../../visual-basic/programming-guide/language-features/data-types/generic-procedures.md).</span><span class="sxs-lookup"><span data-stu-id="02c02-109">For more information, see "Constraints" in [Generic Procedures in Visual Basic](../../visual-basic/programming-guide/language-features/data-types/generic-procedures.md).</span></span>
