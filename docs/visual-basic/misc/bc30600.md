# <a name="39withevents39-variable-does-not-raise-any-instance-events-that-are-accessible-to-39ltcontainernamegt39"></a>&#39;WithEvents&#39; переменной не порождает событий экземпляра, доступных для &#39; &lt;containername&gt;&#39;
Объект, на который вы ссылаетесь, не инициирует доступных событий.  
  
 **Идентификатор ошибки:** BC30600  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Измените модификаторы доступа для события или выберите событие, которое доступно в текущем контексте.
