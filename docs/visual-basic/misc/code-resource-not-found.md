# <a name="code-resource-not-found"></a>Не найден ресурс кода
Выполнен вызов процедуры в ресурсе кода, но не удалось найти ресурс кода.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте, что ресурс доступен и имеет правильную ссылку.
