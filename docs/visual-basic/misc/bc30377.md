# <a name="method-ltmethodnamegt-is-already-declared-in-interface-ltinterfacenamegt"></a>Метод "&lt;имя_метода&gt;«уже объявлен в интерфейсе»&lt;interfacename&gt;"
Вы объявили метод интерфейса второй раз, дублируя его.  
  
 **Идентификатор ошибки:** BC30377  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите повторяющееся объявление.
