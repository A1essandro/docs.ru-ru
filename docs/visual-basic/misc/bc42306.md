# <a name="xml-comment-tag-39lttaggt39-is-not-permitted-on-a-39ltelementgt39-language-element"></a>Тег комментария XML &#39; &lt;тега&gt;&#39; не разрешена на &#39;&lt; элемент&gt;&#39; элемент языка
Тег комментария XML \<тега > не разрешен для '\<элемент > "элемента. Комментарий XML будет проигнорирован.  
  
 Тег комментариев XML не поддерживается элементом языка, к которому он применяется.  
  
 **Идентификатор ошибки:** BC42306  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите неподдерживаемый тег комментариев XML.
