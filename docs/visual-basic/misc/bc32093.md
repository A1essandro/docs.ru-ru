# <a name="39of39-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>«», Необходимых при указании аргументов типа для универсального типа или метода
Оператор пытается построить тип из универсального типа или вызвать универсальный метод, без использования [из](../../visual-basic/language-reference/statements/of-clause.md) предложения.  
  
 [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] Вызывает синтаксиса для универсальных типов для параметров типа и аргументов типа должен быть представлен по `Of` ключевое слово. Кроме того, список параметров типов или список аргументов типов должен быть заключен в скобки.  
  
 **Идентификатор ошибки:** BC32093  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Включите ключевое слово `Of` в начало списка аргументов типов и заключите весь список в скобки.
