# <a name="39end-region39-must-be-preceded-by-a-matching-39region39"></a>&#39;#End region&#39; должен предшествовать соответствующий &#39;#Region&#39;
С помощью `#Region` можно указать блок кода, чтобы разворачивать и сворачивать с помощью возможности структурирования в редакторе кода Visual Studio. Начало и конец операторов `#Region` должны находиться в одном блоке кода.  
  
 **Идентификатор ошибки** : BC30680  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Вставить `#Region` в нужное место до соответствующего `#End Region` инструкции.
