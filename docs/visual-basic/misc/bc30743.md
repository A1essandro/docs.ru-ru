# <a name="type-typename-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Тип '\<typename >' не определен или содержащий его модуль не загружен во время сеанса отладки
Используется ссылка на тип, который недоступен или не определен.  
  
 **Идентификатор ошибки:** BC30743  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь в том, что имя типа указано правильно.  
  
2.  Определите тип в текущем классе или модуле.
