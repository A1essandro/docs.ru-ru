# <a name="39end-function39-must-be-the-first-statement-on-a-line"></a>&#39;Функция End&#39; должен быть первым оператором в строке
Оператор `End Function` следует за разделителем операторов в виде двоеточия (:). `End Function` должен быть единственным оператором в строке исходного кода.  
  
 **Идентификатор ошибки:** BC32032  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Разместите несколько операторов в отдельных строках.
