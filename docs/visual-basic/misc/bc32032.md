# <a name="end-function-must-be-the-first-statement-on-a-line"></a>End Function должен быть первым оператором в строке
Оператор `End Function` следует за разделителем операторов в виде двоеточия (:). `End Function` должен быть единственным оператором в своей строке исходного кода.  
  
 **Идентификатор ошибки:** BC32032  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Разместите несколько операторов в отдельных строках.
