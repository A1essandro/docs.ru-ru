# <a name="type-parameter-39lttypeparametername1gt39-must-have-either-a-39new39-constraint-or-a-39structure39-constraint-to-satisfy-the-39new39-constraint-for-type-parameter-39lttypeparametername2gt39"></a><span data-ttu-id="8a246-102">Параметр типа &#39; &lt;имя_параметра_типа1&gt;&#39; должен иметь либо &#39; Создать &#39; ограничение или &#39; Структура &#39; с ограничением &#39; Создать &#39; ограничения для параметра типа &#39; &lt;имя_параметра_типа2&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="8a246-102">Type parameter &#39;&lt;typeparametername1&gt;&#39; must have either a &#39;New&#39; constraint or a &#39;Structure&#39; constraint to satisfy the &#39;New&#39; constraint for type parameter &#39;&lt;typeparametername2&gt;&#39;</span></span>
<span data-ttu-id="8a246-103">Оператор создает универсальный тип передачи параметра типа, который не ограничен для удовлетворения ограничения `New` .</span><span class="sxs-lookup"><span data-stu-id="8a246-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>  
  
 <span data-ttu-id="8a246-104">Ограничение `New` означает, что аргумент типа, предоставленный для данного параметра типа, должен предоставлять коду доступ к конструктору без параметров, который создает объекты из него.</span><span class="sxs-lookup"><span data-stu-id="8a246-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="8a246-105">Конструкторы без параметров имеются для всех типов значений, но не для всех ссылочных типов.</span><span class="sxs-lookup"><span data-stu-id="8a246-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="8a246-106">Поэтому ограничение `Structure` удовлетворяет ограничению `New` , а ограничение `Class` или имя класса или интерфейса — нет.</span><span class="sxs-lookup"><span data-stu-id="8a246-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>  
  
 <span data-ttu-id="8a246-107">Эта ошибка может быть вызвана указанными ниже операторами.</span><span class="sxs-lookup"><span data-stu-id="8a246-107">The following statements can generate this error.</span></span>  
  
```  
Public Class c1(Of t As New)  
End Class  
Public Class c2(Of u)  
    Public testObject As New c1(Of u)  
End Class  
```  
  
 <span data-ttu-id="8a246-108">Когда класс `c2` создает объект из `c1`, параметр типа `u` не удовлетворяет ограничению `New` для параметра типа `t`.</span><span class="sxs-lookup"><span data-stu-id="8a246-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>  
  
 <span data-ttu-id="8a246-109">**Идентификатор ошибки:** BC32084</span><span class="sxs-lookup"><span data-stu-id="8a246-109">**Error ID:** BC32084</span></span>

-   <span data-ttu-id="8a246-111">Если передаваемый в универсальный тип параметр типа может удовлетворять ограничению `Structure` или `New` , следует добавить к его определению соответствующее ограничение.</span><span class="sxs-lookup"><span data-stu-id="8a246-111">If the type parameter to be passed to the generic type can satisfy the `Structure` or `New` constraint, then add the appropriate constraint to its definition.</span></span>

