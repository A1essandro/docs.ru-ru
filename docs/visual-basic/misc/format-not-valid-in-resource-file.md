# Недопустимый формат в файле ресурсов
Файл ресурсов в проекте не соответствует стандартному формату для файла ресурса Windows, или существует несоответствие между файлом ресурсов и проектом [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
### Исправление ошибки  
  
1.  Повторно создайте источник ресурсов и файлы ресурсов, соблюдая синтаксис, задокументированный в пакете средств разработки программного обеспечения для Windows.  
  
2.  Удалите недопустимый файл ресурсов из проекта.
