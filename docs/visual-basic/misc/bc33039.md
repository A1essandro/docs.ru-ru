 <span data-ttu-id="4c1d1-104">Переменная счетчика должна иметь тип, поддерживающий операторы сложения (`+`) и вычитания (`-`), которые полностью функциональны в содержащем их типе.</span><span class="sxs-lookup"><span data-stu-id="4c1d1-104">The counter variable must be of a type that supports addition (`+`) and subtraction (`-`) operators that operate completely on their containing type.</span></span> <span data-ttu-id="4c1d1-105">Это означает, что оба операнда и возвращаемое значение должны иметь тип переменной счетчика.</span><span class="sxs-lookup"><span data-stu-id="4c1d1-105">This means both of the operands and the return value must be of the type of the counter variable.</span></span>  
  
 <span data-ttu-id="4c1d1-106">Если для переменной счетчика используется числовой тип данных, операторы `+` и `-` поддерживаются в содержащем типе.</span><span class="sxs-lookup"><span data-stu-id="4c1d1-106">If you use a numeric data type for the counter variable, the `+` and `-` operators are supported on the containing type.</span></span> <span data-ttu-id="4c1d1-107">При использовании пользовательского класса или структуры необходимо определить оба оператора с операндами и возвращаемое значение с типом этого класса или структуры.</span><span class="sxs-lookup"><span data-stu-id="4c1d1-107">If you use a user-defined class or structure, you must define both operators with operands and return value of the type of your class or structure.</span></span>
