<span data-ttu-id="87c66-103">Если размерность, используемая для инициализации массива, не соответствует размерности в объявлении, компилятор не может вывести тип данных для массива.</span><span class="sxs-lookup"><span data-stu-id="87c66-103">If the dimensions used to initialize an array do not match the dimensions in the declaration, the compiler cannot infer a data type for the array.</span></span> <span data-ttu-id="87c66-104">Например, следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="87c66-104">For example, the following code causes this error.</span></span>  
  
```vb

' Not valid.  
'Dim exampleArray2(,) = New Integer() {1, 2, 3}  
'Dim exampleArray3(,) = New Integer() {}  
```
