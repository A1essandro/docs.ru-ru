# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>Параметр типа комментария XML &lt;параметр&gt; не совпадает с параметром типа, соответствующего &lt;ключевое слово&gt; инструкции
Параметр комментария XML \<параметр > не совпадает с параметром типа, соответствующего \<ключевое слово > инструкции. Комментарий XML будет проигнорирован.  
  
 Тег XML-комментария указан для некоторых, но не для всех параметров типа в методе.  
  
 **Идентификатор ошибки:** BC42317  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Укажите теги XML-комментариев для всех параметров типа в методе.
