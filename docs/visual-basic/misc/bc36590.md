<span data-ttu-id="fbfe3-103">Универсальный метод расширения был вызван с меньшим числом аргументов типа, чем число параметров типа в определении метода.</span><span class="sxs-lookup"><span data-stu-id="fbfe3-103">A generic extension method has been invoked with fewer type arguments than there are type parameters in the method definition.</span></span>  
  
 <span data-ttu-id="fbfe3-104">При вызове универсального метода вы должны представить по одному аргументу типа для каждого параметра типа, определяемого этим методом.</span><span class="sxs-lookup"><span data-stu-id="fbfe3-104">When you invoke a generic method, you must supply one type argument for each type parameter defined by that method.</span></span>  
  
 <span data-ttu-id="fbfe3-105">**Идентификатор ошибки:** BC36590</span><span class="sxs-lookup"><span data-stu-id="fbfe3-105">**Error ID:** BC36590</span></span>

-   <span data-ttu-id="fbfe3-107">Добавьте аргументы типа в список аргументов типа так, чтобы имелось по одному аргументу типа для каждого параметра типа вызываемого метода расширения.</span><span class="sxs-lookup"><span data-stu-id="fbfe3-107">Add type arguments to your type argument list so that there is one for each type parameter of the extension method that you are invoking.</span></span>
