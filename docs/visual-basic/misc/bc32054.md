# <a name="39ltmembernamegt-has-the-same-name-as-a-type-parameter"></a>&#39;&lt;имя пользователя&gt; имеет то же имя, что и параметр типа
Член универсального класса или структуры имеет то же имя, что и параметр типа этого класса или структуры.  
  
 У каждой переменной, константы, перечисления, процедуры, свойства, события и делегата должно быть имя, отличное от любого параметра типа.  
  
 **Идентификатор ошибки:** BC32054  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените имя члена класса или структуры так, чтобы оно отличалось от имени любого параметра типа.
