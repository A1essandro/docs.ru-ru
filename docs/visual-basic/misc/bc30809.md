# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39;Wend&#39; операторы больше не поддерживаются; Используйте &#39;End While&#39; инструкции вместо
Конструкция `End While` заменяет ключевое слово `Wend` .  
  
 **Идентификатор ошибки:** BC30809  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте `End While` для завершения блока `While` .
