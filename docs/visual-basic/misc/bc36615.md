# <a name="into-expected"></a><span data-ttu-id="25a02-102">Требуется Into.</span><span class="sxs-lookup"><span data-stu-id="25a02-102">'Into' expected</span></span>
<span data-ttu-id="25a02-103">Предложение `Aggregate`, `Group By` или `Group Join` было указано без оператора `Into`.</span><span class="sxs-lookup"><span data-stu-id="25a02-103">An `Aggregate`, `Group By`, or `Group Join` clause has been specified without an `Into` operator.</span></span> <span data-ttu-id="25a02-104">Оператор `Into` используется для идентификации агрегатных функций, которые должны применяться к результатам запроса, и для определения члена типа результата запроса, который должен содержать сгруппированные результаты (с помощью агрегатной функции `Group` ).</span><span class="sxs-lookup"><span data-stu-id="25a02-104">You use the `Into` operator to identify aggregate functions to apply to the query result and to identify the member of the query result type to contain the grouped results (by using the `Group` aggregate function).</span></span>  
  
 <span data-ttu-id="25a02-105">**Идентификатор ошибки:** BC36615</span><span class="sxs-lookup"><span data-stu-id="25a02-105">**Error ID:** BC36615</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="25a02-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="25a02-106">To correct this error</span></span>
