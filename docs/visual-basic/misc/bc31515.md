# <a name="project-39ltprojectnamegt39-makes-an-indirect-reference-to-assembly-39ltassemblynamegt39-which-contains-39ltclassnamegt39"></a>Проект "&lt;projectname&gt;«делает косвенную ссылку на сборку»&lt;assemblyname&gt;«которого содержит»&lt;classname&gt;"
Проект "\<projectname настроек" косвенную ссылку на сборку "\<assemblyname настроек", содержащее "\<classname настроек". Добавьте ссылку на "\<assemblyname настроек" в проект.  
  
 Этот проект содержит элементы, зависящие от сборки, которая не содержится в ссылках проекта.  
  
 **Идентификатор ошибки:** BC31515  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте ссылку на сборку, щелкнув правой кнопкой мыши элемент **Ссылки** в **обозревателе решений** и выбрав **Добавить ссылку**.
