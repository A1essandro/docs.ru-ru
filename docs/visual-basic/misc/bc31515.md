# <a name="project-ltprojectnamegt-makes-an-indirect-reference-to-assembly-ltassemblynamegt-which-contains-ltclassnamegt"></a>Проект "&lt;имя_проекта&gt;«делает косвенную ссылку на сборку»&lt;assemblyname&gt;«которого содержит»&lt;classname&gt;"
Проект "\<имя_проекта >" делает косвенную ссылку на сборку "\<имя_сборки >", содержащее "\<имя_класса >". Добавьте ссылку на "\<имя_сборки >" в проект.  
  
 Этот проект содержит элементы, зависящие от сборки, которая не содержится в ссылках проекта.  
  
 **Идентификатор ошибки:** BC31515  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Добавьте ссылку на сборку, щелкнув правой кнопкой мыши элемент **Ссылки** в **обозревателе решений** и выбрав **Добавить ссылку**.
