# <a name="cannot-inherit-interface-ltinterfacename1gt-because-it-could-be-identical-to-interface-ltinterfacename2gt-for-some-type-arguments"></a><span data-ttu-id="0f9f9-102">Невозможно наследовать интерфейс "&lt;имя_интерфейса1&gt;«, так как он может совпадать с интерфейсом»&lt;имя_интерфейса2&gt;" для некоторых аргументов типа</span><span class="sxs-lookup"><span data-stu-id="0f9f9-102">Cannot inherit interface '&lt;interfacename1&gt;' because it could be identical to interface '&lt;interfacename2&gt;' for some type arguments</span></span>
<span data-ttu-id="0f9f9-103">Универсальный интерфейс наследует более одного раза от другого универсального интерфейса, и два этих наследования могут конфликтовать для некоторых значений аргументов типов.</span><span class="sxs-lookup"><span data-stu-id="0f9f9-103">A generic interface inherits more than once from another generic interface, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="0f9f9-104">Эта ошибка может быть вызвана указанными ниже операторами.</span><span class="sxs-lookup"><span data-stu-id="0f9f9-104">The following statements can generate this error.</span></span>  
  
 `Public Interface interfaceA(Of u)`  
  
 `End Interface`  
  
 `Public Interface derivedInterface(Of t1, t2)`  
  
 `Inherits interfaceA(Of t1), interfaceA(Of t2)`  
  
 `End Interface`  
  
 <span data-ttu-id="0f9f9-105">Если `derivedInterface` создается или реализуется, предоставляя один и тот же тип как для `t1` , так и для `t2`, он должен наследовать две версии `interfaceA` с одинаковыми аргументами типов.</span><span class="sxs-lookup"><span data-stu-id="0f9f9-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="0f9f9-106">Это приведет к неоднозначности определения, к какой версии следует получить доступ.</span><span class="sxs-lookup"><span data-stu-id="0f9f9-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="0f9f9-107">**Идентификатор ошибки:** BC32120</span><span class="sxs-lookup"><span data-stu-id="0f9f9-107">**Error ID:** BC32120</span></span>
