# ReadOnly и WriteOnly нельзя использовать вместе
Модификаторы `ReadOnly` и `WriteOnly` являются взаимоисключающими.  
  
 **Идентификатор ошибки:** BC30635  
  
### Исправление ошибки  
  
-   Укажите только один модификатор: `ReadOnly` или `WriteOnly`.
