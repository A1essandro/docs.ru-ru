# <a name="39lttypenamegt39-is-an-unsupported-type"></a>&#39; &lt;typename&gt;&#39; имеет неподдерживаемый тип
Указанный тип не может быть представлен [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
 **Идентификатор ошибки** : BC30649  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте поддерживаемый тип.  
  
-   Разработайте новый тип.
