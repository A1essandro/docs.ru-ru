# <a name="lttypenamegt-is-an-unsupported-type"></a>"&lt;typename&gt;" является неподдерживаемым типом
Visual Studio не может быть представлено типом, указанным.  
  
 **Идентификатор ошибки:** BC30649  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте поддерживаемый тип.  
  
-   Разработайте новый тип.
