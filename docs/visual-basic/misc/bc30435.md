# <a name="members-in-a-structure-cannot-be-declared-ltspecifiergt"></a>Члены структуры, не может объявляться как "&lt;описатель&gt;"
Вы использовали спецификатор, который недопустим для члена в операторе `Structure` .  
  
 **Идентификатор ошибки:** BC30435  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
