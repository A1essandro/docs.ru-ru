# <a name="members-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>Члены в структуре не могут объявляться &#39; &lt;спецификатор&gt;&#39;
Вы использовали спецификатор, который недопустим для члена в операторе `Structure` .  
  
 **Идентификатор ошибки:** BC30435  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
