# <a name="expression-ltnamegt-is-not-a-procedure-but-occurs-as-the-target-of-a-procedure-call"></a>Выражение "&lt;имя&gt;" не является процедурой, но используется как целевой объект вызова процедуры
Выполнен вызов процедуры для выражения, которое не является процедурой.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь в правильности написания имени процедуры.  
  
2.  Добавьте процедуру, которую вызывает <`name`>.
