# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-type-39lttypenamegt39"></a>Аргумент &#39; &lt;имя_аргумента&gt;&#39; невозможно преобразовать в тип &#39;&lt; TypeName&gt;&#39;
Преобразование пытается изменить аргумент на тип, который является недопустимым.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените тип на допустимый.
