# <a name="39using39-statements-are-not-valid-in-the-immediate-window"></a>Операторы 'Using' недопустимы в окне интерпретации
`Using`операторы могут использоваться только в исходном коде.  
  
 **Идентификатор ошибки:** BC30186  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите оператор `Using` из окна **Интерпретация** .
