# Операторы Using недопустимы в окне интерпретации
Операторы `Using` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30186  
  
### Исправление ошибки  
  
-   Удалите оператор `Using` из окна **Интерпретация**.
