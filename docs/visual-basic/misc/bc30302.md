# <a name="type-character-ltcharactergt-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Символ типа "&lt;символ&gt;" не может использоваться в объявлении с явно заданным типом
Используется символ типа, такой как `$`, при явном указании типа.  
  
 **Идентификатор ошибки:** BC30302  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите символ типа или удалите предложение `As <Type>` из объявления.
