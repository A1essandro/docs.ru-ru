<span data-ttu-id="7a3eb-103">В разделяемом методе, который включает один или несколько параметров типа, имена параметров типа должны быть одинаковыми в объявлении метода и в реализации метода.</span><span class="sxs-lookup"><span data-stu-id="7a3eb-103">In a partial method that includes one or more type parameters, the names of the type parameters must be the same in the declaration of the method and in the implementation of the method.</span></span>
