# <a name="39assembly39-or-39module39-expected"></a><span data-ttu-id="5af61-102">&#39; Сборка &#39; или &#39; Модуль &#39; Ожидается</span><span class="sxs-lookup"><span data-stu-id="5af61-102">&#39;Assembly&#39; or &#39;Module&#39; expected</span></span>
<span data-ttu-id="5af61-103">Глобальный атрибут задан без указания того, применяется ли он для всей сборки или только для текущего модуля.</span><span class="sxs-lookup"><span data-stu-id="5af61-103">A global attribute is specified without indicating whether it applies to the entire assembly or only to the current module.</span></span> <span data-ttu-id="5af61-104">Для глобальных атрибутов необходимо указать `Assembly` или `Module`.</span><span class="sxs-lookup"><span data-stu-id="5af61-104">Global attributes must specify either `Assembly` or `Module`.</span></span>  
  
 <span data-ttu-id="5af61-105">Глобальным атрибутом является тот, который присутствует в строке исходного кода сам по себе, а не применяется к объявлению конкретного программного элемента.</span><span class="sxs-lookup"><span data-stu-id="5af61-105">A global attribute is one that appears on a source line by itself, rather than being applied to the declaration of a particular programming element.</span></span>  
  
 <span data-ttu-id="5af61-106">**Идентификатор ошибки:** BC32015</span><span class="sxs-lookup"><span data-stu-id="5af61-106">**Error ID:** BC32015</span></span>

1.  <span data-ttu-id="5af61-108">Если необходимо сделать атрибут глобальным, добавьте в начало блока атрибута ключевое слово `Assembly` или `Module` , за которым следует двоеточие (:).</span><span class="sxs-lookup"><span data-stu-id="5af61-108">If you intend the attribute to be global, add the `Assembly` or `Module` keyword to the beginning of the attribute block, followed by a colon (:).</span></span>
