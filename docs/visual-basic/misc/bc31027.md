# <a name="unable-to-open-file-ltfilenamegt-lterrorgt"></a>Не удается открыть файл "&lt;filename&gt;": &lt;ошибка&gt;
Произошла ошибка при открытии файла в этом проекте.  
  
 **Идентификатор ошибки:** BC31027  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что файл существует и не поврежден другим приложением.
