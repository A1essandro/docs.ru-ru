# <a name="unable-to-open-file-filename-error"></a>Не удается открыть файл "\<имя_файла >": \<ошибка >
Произошла ошибка при открытии файла в этом проекте.  
  
 **Идентификатор ошибки:** BC31027  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что файл существует и не поврежден другим приложением.
