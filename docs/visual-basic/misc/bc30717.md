# Операторы Try недопустимы в окне интерпретации
Операторы `Try` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30717  
  
### Исправление ошибки  
  
-   Удалите операторы `Try` из отлаживаемого кода.
