# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;С&#39; контексты и операторы недопустимы в окнах отладки
Операторы`With` можно использовать только в исходном коде.  
  
 **Идентификатор ошибки:** BC30726  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите операторы `With` из отлаживаемого кода.
