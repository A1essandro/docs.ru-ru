# <a name="property-39ltpropertynamegt39-cannot-be-initialized-in-an-object-initializer-expression-because-it-requires-arguments"></a><span data-ttu-id="af74b-102">Свойство &#39; &lt;propertyname&gt;&#39; не удается инициализировать в выражении инициализатора объекта, поскольку оно требует аргументов</span><span class="sxs-lookup"><span data-stu-id="af74b-102">Property &#39;&lt;propertyname&gt;&#39; cannot be initialized in an object initializer expression because it requires arguments</span></span>
<span data-ttu-id="af74b-103">Члены, инициализированные в списке инициализаторов объекта, должны быть полями или свойствами, и у членов свойства не может быть параметров.</span><span class="sxs-lookup"><span data-stu-id="af74b-103">The members initialized in an object initializer list must be fields or properties, and property members cannot have parameters.</span></span> <span data-ttu-id="af74b-104">Например, для свойств по умолчанию требуются аргументы, поэтому они не могут быть инициализированы в списке инициализаторов объекта.</span><span class="sxs-lookup"><span data-stu-id="af74b-104">For example, default properties require arguments, so they cannot be initialized in an object initializer list.</span></span> <span data-ttu-id="af74b-105">Дополнительные сведения см. в разделе [не в СБОРКЕ: свойства по умолчанию](http://msdn.microsoft.com/en-us/a70f2a27-8176-4858-935e-f25afdd43ab5).</span><span class="sxs-lookup"><span data-stu-id="af74b-105">For more information, see [NOT IN BUILD: Default Properties](http://msdn.microsoft.com/en-us/a70f2a27-8176-4858-935e-f25afdd43ab5).</span></span>  
  
 <span data-ttu-id="af74b-106">**Идентификатор ошибки:** BC30992</span><span class="sxs-lookup"><span data-stu-id="af74b-106">**Error ID:** BC30992</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="af74b-107">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="af74b-107">To correct this error</span></span>  
  
-   <span data-ttu-id="af74b-108">Удалите из списка инициализации все свойства, имеющие аргументы.</span><span class="sxs-lookup"><span data-stu-id="af74b-108">Remove from the initialization list all properties that have arguments.</span></span>  
  
## <a name="example"></a><span data-ttu-id="af74b-109">Пример</span><span class="sxs-lookup"><span data-stu-id="af74b-109">Example</span></span>  
 <span data-ttu-id="af74b-110">В приведенном ниже классе содержится используемое по умолчанию свойство `defaultProp`, для которого требуется целочисленный аргумент.</span><span class="sxs-lookup"><span data-stu-id="af74b-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="af74b-111">Так как для свойства по умолчанию требуется аргумент, следующее объявление вызывает ошибку:</span><span class="sxs-lookup"><span data-stu-id="af74b-111">Because the default property requires an argument, the following declaration causes an error.</span></span>  
  
```  
' Dim strs As New SomeStrings(2) With { .defaultProp = "One" }  
```
