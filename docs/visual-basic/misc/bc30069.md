# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>Управляющая переменная цикла for "&lt;variablename&gt;" уже используется вложенным циклом For в
Во всех вложенных циклах `For` должны использоваться уникальные управляющие переменные.  
  
 **Идентификатор ошибки:** BC30069  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените имя одной из управляющих переменных.
