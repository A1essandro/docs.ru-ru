# <a name="39declare39-statements-in-a-structure-cannot-be-declared-39ltspecifiergt39"></a>&#39; объявлять &#39; операторы в структуре не могут объявляться &#39; &lt;описатель&gt;&#39;
Объявление `Declare` содержит спецификатор, который недопустим в объявлении `Structure` .  
  
 **Идентификатор ошибки:** BC30791  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите спецификатор.
