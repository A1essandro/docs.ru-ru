# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>Префикс пространства имен XML &#39; префикс &#39; Зарезервировано для использования в XML и пространство имен URI не может быть изменено
В XML-литерале использовался зарезервированный префикс пространства имен XML, такой как `xmlns`.  
  
 **Идентификатор ошибки:** BC31152  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените префикс пространства имен XML на не зарезервированное значение.
