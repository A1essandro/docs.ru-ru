# <a name="39set39-method-cannot-have-more-than-one-parameter"></a>&#39; Выбрать &#39; метод не может иметь более одного параметра
Для процедур свойства `Set` определено несколько параметров.  
  
 **Идентификатор ошибки:** BC31063  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите лишние параметры из процедур свойства `Set` .
