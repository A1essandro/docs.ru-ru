# <a name="value-of-type-lttype1gt-cannot-be-converted-to-lttype2gt-because-the-array-types-have-different-numbers-of-dimensions"></a>Значение типа "&lt;тип1&gt;«невозможно преобразовать в»&lt;тип2&gt;", так как в типах массивов указано различное количество измерений
Вы попытались преобразовать массив в другой массив с иным числом измерений.  
  
 **Идентификатор ошибки:** BC30414  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените один или оба массива таким образом, чтобы число измерений в них совпадало.
