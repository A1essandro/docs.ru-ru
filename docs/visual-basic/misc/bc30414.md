# <a name="value-of-type-39lttype1gt39-cannot-be-converted-to-39lttype2gt39-because-the-array-types-have-different-numbers-of-dimensions"></a>Значение типа &#39; &lt;тип1&gt; &#39; не может быть преобразован &#39; &lt;тип2&gt; &#39; , так как в типах массивов указано различное количество измерений
Вы попытались преобразовать массив в другой массив с иным числом измерений.  
  
 **Идентификатор ошибки:** BC30414  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените один или оба массива таким образом, чтобы число измерений в них совпадало.
