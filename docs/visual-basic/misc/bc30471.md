# <a name="expression-is-not-an-array-or-a-method-and-cannot-have-an-argument-list"></a>Выражение не является массивом или методом и не может иметь список аргументов
Вы предоставили список аргументов для выражения, не являющегося массивом или методом. Списки аргументов могут быть только у массивов и методов.  
  
 **Идентификатор ошибки:** BC30471  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите список аргументов.
