<span data-ttu-id="12c3b-103">Когда предоставляются параметры в объявлении и реализации разделяемого метода, имена соответствующих параметров должны совпадать.</span><span class="sxs-lookup"><span data-stu-id="12c3b-103">When parameters are supplied for the declaration and implementation of a partial method, the names of the corresponding parameters must be the same.</span></span> <span data-ttu-id="12c3b-104">Например, следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="12c3b-104">For example, the following code causes this error.</span></span>
