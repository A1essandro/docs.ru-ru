# <a name="property-propertyname-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-name"></a>Свойство "\<имя_свойства >" должен быть объявлен как «Default», поскольку оно переопределяет свойство по умолчанию в базовом "\<имя >"
Оператор `Property` в коде переопределяет свойство `Default` базового класса, структуры или интерфейса, и поэтому переопределяющее свойство должно быть объявлено как `Default`.  
  
 **Идентификатор ошибки:** BC30360  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Объявите свойство `Default`.
