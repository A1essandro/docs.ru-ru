# Аргумент не может быть меньше нуля
Использовано значение меньше нуля, но аргумент не может быть меньше нуля.  
  
### Исправление ошибки  
  
-   Проверьте, как вычисляется значение в выражении, чтобы убедиться в том, что оно правильно.
