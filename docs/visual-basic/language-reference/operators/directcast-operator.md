 <span data-ttu-id="33bb6-106">Использовании `DirectCast` аналогично тому, как можно использовать ключевое слово [функция CType](../../../visual-basic/language-reference/functions/ctype-function.md) и [оператор TryCast](../../../visual-basic/language-reference/operators/trycast-operator.md) ключевое слово.</span><span class="sxs-lookup"><span data-stu-id="33bb6-106">You use the `DirectCast` keyword similar to the way you use the [CType Function](../../../visual-basic/language-reference/functions/ctype-function.md) and the [TryCast Operator](../../../visual-basic/language-reference/operators/trycast-operator.md) keyword.</span></span> <span data-ttu-id="33bb6-107">Необходимо указать в качестве первого аргумента и тип для преобразования его в качестве второго аргумента выражение.</span><span class="sxs-lookup"><span data-stu-id="33bb6-107">You supply an expression as the first argument and a type to convert it to as the second argument.</span></span> <span data-ttu-id="33bb6-108">`DirectCast` требуется отношение наследования или реализации между типами данных обоих аргументов.</span><span class="sxs-lookup"><span data-stu-id="33bb6-108">`DirectCast` requires an inheritance or implementation relationship between the data types of the two arguments.</span></span> <span data-ttu-id="33bb6-109">Это означает, что один тип должен наследовать или реализовывать другой.</span><span class="sxs-lookup"><span data-stu-id="33bb6-109">This means that one type must inherit from or implement the other.</span></span>
