# <a name="mycomputerfilesystemspecialdirectories-object"></a>Объект My.Computer.FileSystem.SpecialDirectories
Включает свойства для доступа к часто используемым каталогам.  
  
## <a name="remarks"></a>Примечания  
 Дополнительные сведения о методах и свойствах `My.Computer.FileSystem.SpecialDirectories` см. в разделе <xref:Microsoft.VisualBasic.FileIO.SpecialDirectories>.  
  
 Дополнительные сведения см. в разделе [как: получение содержимого каталоге Мои документы](../../../visual-basic/developing-apps/programming/drives-directories-files/how-to-retrieve-the-contents-of-the-my-documents-directory.md).
