# <a name="mycomputerports-object"></a>Объект My.Computer.Ports
Предоставляет свойства и методы для доступа к последовательным портам компьютера.  
  
## <a name="remarks"></a>Примечания  
 Дополнительные сведения о методах и свойствах `My.Computer.Ports` объекта см. в разделе <xref:Microsoft.VisualBasic.Devices.Ports>.</xref:Microsoft.VisualBasic.Devices.Ports>  
  
 Дополнительные сведения см. в разделе [доступ к портам компьютера](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
>  Можно также использовать свойства и методы <xref:System.IO.Ports.SerialPort?displayProperty=fullName>класса для доступа к последовательным портам компьютера.</xref:System.IO.Ports.SerialPort?displayProperty=fullName>
