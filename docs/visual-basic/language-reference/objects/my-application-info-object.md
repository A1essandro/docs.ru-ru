# <a name="myapplicationinfo-object"></a>Объект My.Application.Info
Предоставляет свойства для получения сведений о приложении, например номер версии, описание, загруженные сборки и т. д.  
  
## <a name="remarks"></a>Примечания  
 Сведения о методах и свойствах `My.Application.Info` объекта, см. в разделе <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>.  
  
> [!NOTE]
>  Можно использовать свойства <xref:System.Diagnostics.FileVersionInfo?displayProperty=nameWithType> для получения сведений о файле на диске.  
  
## <a name="requirements"></a>Требования  
 **Пространство имен:** <xref:Microsoft.VisualBasic.ApplicationServices>  
  
 **Класс:** <xref:Microsoft.VisualBasic.ApplicationServices.AssemblyInfo>  
  
 **Сборка:** библиотеки времени выполнения Visual Basic (в Microsoft.VisualBasic.dll)
