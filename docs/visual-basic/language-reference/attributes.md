# <a name="attributes-visual-basic"></a>Атрибуты (Visual Basic)
Visual Basic предоставляет несколько атрибутов, которые позволяют объектам взаимодействовать с неуправляемым кодом и один атрибут, который позволяет членам модуля должен осуществляться без имени модуля. В следующей таблице перечислены атрибуты, используемые в Visual Basic.  
  
|||  
|---|---|  
|<xref:Microsoft.VisualBasic.ComClassAttribute>|Указывает компилятору на необходимость добавления метаданных, позволяющих класс должен быть предоставлен как COM-объектом.|  
|<xref:Microsoft.VisualBasic.HideModuleNameAttribute>|Позволяет членам модуля, доступ к используя только квалификацию, необходимую для модуля.|  
|<xref:Microsoft.VisualBasic.VBFixedArrayAttribute>|Указывает, что массив в структуре или внешней переменной следует считать массив фиксированной длины.|  
|<xref:Microsoft.VisualBasic.VBFixedStringAttribute>|Указывает, что строки должны рассматриваться, как если бы фиксированной длины.|
