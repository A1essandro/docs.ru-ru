# <a name="module-ltkeywordgt-visual-basic"></a>Модуль &lt;ключевое слово&gt; (Visual Basic)
Указывает, что атрибут в начале исходного файла применяется к текущему модулю сборки.  
  
## <a name="remarks"></a>Примечания  
 Многие атрибуты относятся к отдельному элементу, такие как класс или свойство. Применяйте такой атрибут путем присоединения блока атрибута в угловых скобках (`< >`), непосредственно в операторе объявления.  
  
 Если атрибут принадлежит не только к следующему элементу, но к текущему модулю сборки, поместите блок атрибута в начале исходного файла и определения атрибута с `Module` ключевое слово. Если он применяется ко всей сборке, используйте [сборки](../../../visual-basic/language-reference/modifiers/assembly.md) ключевое слово.  
  
 `Module` Модификатор не совпадает с [оператор Module](../../../visual-basic/language-reference/statements/module-statement.md).
