# <a name="identifier-is-too-long"></a>Слишком длинный идентификатор
Имя или идентификатор любого программируемого элемента ограничен 1023 знаков. Кроме того полное доменное имя не может превышать 1023 знаков. Это означает, что вся строка идентификатора (`<namespace>.<...>.<namespace>.<class>.<element>`) не может содержать более 1023 знаков, включая оператор доступа к членам (`.`) символов.  
  
 **Идентификатор ошибки:** BC30033  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Уменьшите длину идентификатора.
