# <a name="expression-does-not-produce-a-value"></a>Выражение не выдает значение
Была предпринята попытка использовать выражение, которое формирует значение в контексте, например, вызов `Sub` в контексте, где `Function` ожидается.  
  
 **Идентификатор ошибки:** BC30491  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Измените выражение на то, что создает значение.
