# <a name="internal-error-happened-at-ltlocationgt"></a>Произошла внутренняя ошибка в &lt;расположение&gt;
Произошла внутренняя ошибка. Строка, в которой произошло содержится в сообщении об ошибке.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Убедитесь, что эта ошибка не была создана `Error` инструкции или `Raise` метод; Если не удалось, обратитесь в службу технической поддержки Майкрософт и сообщите условия, при которых появилось это сообщение.
