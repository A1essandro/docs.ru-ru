# <a name="type-mismatch-visual-basic"></a>Несоответствие типа (Visual Basic)
Предпринята попытка преобразования значения в другой тип, в результате которого является недопустимым.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте назначение, чтобы убедиться в том, что он является допустимым.  
  
2.  Убедитесь, что в процедуру, ожидающую отдельное свойство или значение, не был передан объект.  
  
3.  Убедитесь, что вы не использовали имя модуля или проекта где ожидалось выражение.
