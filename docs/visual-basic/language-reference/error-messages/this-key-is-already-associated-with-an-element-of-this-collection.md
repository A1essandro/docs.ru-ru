# <a name="this-key-is-already-associated-with-an-element-of-this-collection"></a>Ключ уже связан с элементом коллекции
Указан ключ для элемента коллекции, уже определяет другой член коллекции. Ключ является строкой, указанной в `Add` метод, который уникально определяет отдельный член коллекции.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте другой ключ для этого элемента.
