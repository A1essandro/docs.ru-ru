# <a name="object-required-visual-basic"></a>Требуется объект (Visual Basic)
Ссылки на свойства и методы часто требуют явного квалификатора объекта. Это в этом случае.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что ссылки на свойство или метод имеют допустимый квалификатор объекта. Укажите квалификатор объекта, если не указывается.  
  
2.  Проверьте правильность написания квалификатора объекта и убедитесь, что объект является видимым в части программы, в которой содержится ссылка на него.  
  
3.  Если путь предоставляется ведущее приложение **Открытие файла** команды, проверьте правильность аргументов.  
  
4.  Просмотрите документацию объекта и убедитесь, что это действие допустимо.
