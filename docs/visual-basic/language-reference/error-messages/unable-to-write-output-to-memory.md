# Не удается выполнить запись выходных данных в память
[!INCLUDE[vs2017banner](../../../visual-basic/includes/vs2017banner.md)]

Возникла проблема с выводом в память.  
  
 **Идентификатор ошибки:** BC31020  
  
### Исправление ошибки  
  
1.  Скомпилируйте программу еще раз, чтобы узнать, повторится ли ошибка.  
  
2.  Если ошибка возникает снова, сохраните работу и перезапустите [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)].  
  
3.  Если устранить ошибку не удается, переустановите [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Если после переустановки ошибка не устранена, уведомите службу технической поддержки Майкрософт.
