# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>&#39; &lt;name1&gt;&#39; является неоднозначным, импортированные из пространств имен или типов &#39;&lt; Name2&gt;&#39;
Вы указали неоднозначное имя, которое конфликтует с другим именем. Компилятор [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] не имеет правил разрешения конфликтов; вы должны самостоятельно устранить неоднозначность имен.  
  
 **Идентификатор ошибки:** BC30561  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Устраните неоднозначность имени, удалив импортированных пространств имен.  
  
2.  Укажите полное имя.
