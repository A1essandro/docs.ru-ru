# <a name="39optional39-expected"></a>&#39;Необязательный&#39; ожидается
Обязательный аргумент следует за необязательным аргументом в объявлении процедуры. Каждый аргумент после необязательного аргумента также должен быть необязательным.  
  
 **Идентификатор ошибки:** BC30202  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Если аргумент является обязательным, переместите его на предшествуют первому необязательному аргументу в списке аргументов.  
  
2.  Если аргумент является необязательным, используйте `Optional` ключевое слово.
