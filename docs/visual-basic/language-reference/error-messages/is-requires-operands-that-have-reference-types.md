# <a name="39is39-requires-operands-that-have-reference-types-but-this-operand-has-the-value-type-39lttypenamegt39"></a>&#39; — &#39; требуются операнды, имеющие ссылочные типы, но этот операнд имеет тип значения &#39; &lt;typename&gt;&#39;
`Is` Оператор сравнения определяет, ссылаются ли две объектные переменные с тем же экземпляром. Это сравнение не определен для типов значений.  
  
 **Идентификатор ошибки:** BC30020  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Используйте соответствующий арифметический оператор сравнения или `Like` оператор для сравнения двух типов значений.
