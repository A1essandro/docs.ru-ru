# <a name="path-not-found"></a>Путь не найден
Во время операции доступа к файлам или доступа к диску операционной системы не удалось найти указанный путь. Путь к файлу содержит спецификацию диска, а также каталоги и подкаталоги, которые нужно пройти, чтобы найти файл. Путь может быть как относительным, так и абсолютным.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Проверьте и укажите путь повторно.
