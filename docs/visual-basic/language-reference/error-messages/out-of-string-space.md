# <a name="out-of-string-space-visual-basic"></a>Отсутствует место в строке (Visual Basic)
С помощью Visual Basic можно использовать очень больших строк. Однако требования других программ и способы работы со строками можно по-прежнему вызывать эту ошибку.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что требует создания временной строки во время вычисления выражения не вызывает ошибку.  
  
2.  Удалите все ненужные приложения из памяти, чтобы освободить место.
