# <a name="xml-namespace-uri-39httpwwww3orgxml1998namespace39-can-be-bound-only-to-39xmlns39"></a>URI пространства имен XML &#39; http://www.w3.org/XML/1998/namespace &#39; можно привязать только к &#39; xmlns &#39;
URI http://www.w3.org/XML/1998/namespace используется в объявлении пространства имен XML. Этот URI является зарезервированным пространством имен и не может быть включен в объявление пространства имен XML.  
  
 **Идентификатор ошибки:** BC31183  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Удалите объявление пространства имен XML или замените URI http://www.w3.org/XML/1998/namespace с допустимым URI пространства имен.
