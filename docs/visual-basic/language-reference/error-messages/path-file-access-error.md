# <a name="pathfile-access-error"></a>Ошибка доступа к пути/файлу
В ходе операции доступа к файлам или доступа к диску операционной системы не может установить подключение между путь и имя файла.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что спецификация файла имеет правильный формат. Имя файла может содержать полный (абсолютный) или относительный путь. Полный URL-адрес начинается с имени диска (если путь находится на другом диске) и содержит явный путь от корня к файлу. Любой путь, который не является полным образом задается относительно текущего диска и каталога.  
  
2.  Убедитесь, что вы не была предпринята попытка сохранить файл, который заменит существующий файл только для чтения. Если это так, измените атрибут только для чтения целевого файла или сохраните файл с другим именем.  
  
3.  Убедитесь, что не была предпринята попытка открыть файл только для чтения в последовательных `Output` или `Append` режиме. Если это так, откройте файл в `Input` режиме или изменить атрибут только для чтения файла.  
  
4.  Убедитесь, что не была предпринята попытка изменить проект Visual Basic в пределах базы данных или документа.
