# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>CAN &#39; открыть t &#39; &lt;filename&gt;&#39; для записи
Указанный файл не удается открыть для записи, возможно, так как он уже открыт.  
  
 **Идентификатор ошибки:** BC2012  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Закройте файл и откройте его снова.  
  
2.  Проверка разрешений для файла.
