# <a name="can39t-open-39ltfilenamegt39-for-writing"></a>Можно&#39;t откройте &#39; &lt;filename&gt; &#39; для записи
Указанный файл не удалось открыть для записи, возможно, так как он уже открыт.  
  
 **Идентификатор ошибки:** BC2012  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Закройте файл и откройте его снова.  
  
2.  Проверка разрешений для файла.
