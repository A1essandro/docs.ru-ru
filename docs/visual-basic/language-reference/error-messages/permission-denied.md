# <a name="permission-denied-visual-basic"></a>Доступ запрещен (Visual Basic)
Была предпринята попытка записи на диск защищен от записи или доступа к заблокированному файлу.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Чтобы открыть файл защищен от записи, измените атрибут защиту от записи файла.  
  
2.  Убедитесь, что файл не заблокирован другим процессом и ожидания для открытия файла, пока не будет освобожден другим процессом.  
  
3.  Для доступа к реестру, убедитесь, что разрешения пользователя включают этот тип доступа к реестру.
