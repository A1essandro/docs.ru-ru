# <a name="property-let-procedure-not-defined-and-property-get-procedure-did-not-return-an-object"></a>Процедура свойства let не определена, а процедура свойства get не вернула объект
Некоторые свойства, методы и операции применимы только к `Collection` объектов. Указанные операция или свойство, которое применимы только к коллекциям, но объект не является коллекцией.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте правильность написания имени объекта или свойства или убедитесь, что объект является `Collection` объекта.  
  
2.  Рассмотрим `Add` метод, используемый для добавления объекта в коллекцию, чтобы убедиться, синтаксис правильный, и все идентификаторы были написаны без орфографических ошибок.
