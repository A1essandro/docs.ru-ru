# <a name="unable-to-load-information-for-class-classname"></a>Не удалось загрузить сведения для класса\<имя_класса > "
Использовалась ссылка на класс, который не поддерживается.  
  
 **Идентификатор ошибки:** BC30712  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Убедитесь, что класс был определен и что имя указано правильно.  
  
2.  Попробуйте обратиться к одному из членов, объявленных в модуле. В некоторых случаях среда отладки не может найти члены, потому что модули, в которых они объявлены, еще не были загружены.
