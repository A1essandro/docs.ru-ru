# <a name="property-not-found"></a>Свойство не найдено
Этот объект не поддерживает указанное свойство.  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
1.  Проверьте правильность написания имени свойства.  
  
2.  Проверьте документацию объекта и убедитесь, что вы не пытаетесь получить доступ примерно свойством «text», если объект фактически поддерживает «заголовок» или свойство с похожим именем.
