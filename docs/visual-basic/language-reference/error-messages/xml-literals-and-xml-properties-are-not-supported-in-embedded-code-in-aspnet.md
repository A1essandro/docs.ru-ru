# <a name="xml-literals-and-xml-properties-are-not-supported-in-embedded-code-within-aspnet"></a>XML литералы и XML свойства не поддерживаются во встроенном в ASP.NET коде
XML литералы и XML свойства не поддерживаются во встроенном в ASP.NET коде. Чтобы использовать функции XML, переместите код для кода.  
  
 XML-литерал или свойство оси XML определено во встроенном коде (`<%= =>`) в файле ASP.NET.  
  
 **Идентификатор ошибки:** BC31200  
  
## <a name="to-correct-this-error"></a>Исправление ошибки  
  
-   Переместите код, который содержит XML-литерала или свойства оси XML к файлу фонового кода ASP.NET.
