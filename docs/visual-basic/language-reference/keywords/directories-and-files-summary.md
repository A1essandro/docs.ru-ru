# <a name="directories-and-files-summary-visual-basic"></a>Сводка по каталогам и файлам (Visual Basic)
Ключевые слова языка Visual Basic и члены библиотек времени выполнения упорядочены по назначению и использованию.  
  
 С помощью `My` возможность предоставляет большую производительность и быстродействие операций файлового ввода-вывода, чем использование этих элементов. Дополнительные сведения см. в разделе <xref:Microsoft.VisualBasic.FileIO.FileSystem>.  
  
|**Действие**|**Элемент языка**|  
|----------------|--------------------------|  
|Изменение папки или каталога.|<xref:Microsoft.VisualBasic.FileSystem.ChDir%2A>|  
|Выбор диска.|<xref:Microsoft.VisualBasic.FileSystem.ChDrive%2A>|  
|Скопируйте файл.|<xref:Microsoft.VisualBasic.FileSystem.FileCopy%2A>|  
|Сделайте папки или каталога.|<xref:Microsoft.VisualBasic.FileSystem.MkDir%2A>|  
|Удалите папки или каталога.|<xref:Microsoft.VisualBasic.FileSystem.RmDir%2A>|  
|Переименуйте файл, каталог или папку.|<xref:Microsoft.VisualBasic.FileSystem.Rename%2A>|  
|Возвращает текущий путь.|<xref:Microsoft.VisualBasic.FileSystem.CurDir%2A>|  
|Возвращает метку даты и времени файла.|<xref:Microsoft.VisualBasic.FileSystem.FileDateTime%2A>|  
|Возвращение атрибутов метки, файла или каталога.|<xref:Microsoft.VisualBasic.FileSystem.GetAttr%2A>|  
|Возвращает длину файла.|<xref:Microsoft.VisualBasic.FileSystem.FileLen%2A>|  
|Возвращает для файла имя или метки тома.|<xref:Microsoft.VisualBasic.FileSystem.Dir%2A>|  
|Задайте сведения об атрибутах файла.|<xref:Microsoft.VisualBasic.FileSystem.SetAttr%2A>|
