<span data-ttu-id="6ef85-103">Если процедура объявляет параметр как [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic предоставляет код процедуры прямую ссылку на элемент программирования в аргументе в вызывающем коде.</span><span class="sxs-lookup"><span data-stu-id="6ef85-103">If a procedure declares a parameter as [ByRef](../../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic gives the procedure code a direct reference to the programming element underlying the argument in the calling code.</span></span> <span data-ttu-id="6ef85-104">Это позволяет процедуре для изменения значения, содержащегося в аргументе в вызывающем коде.</span><span class="sxs-lookup"><span data-stu-id="6ef85-104">This permits the procedure to change the value underlying the argument in the calling code.</span></span> <span data-ttu-id="6ef85-105">В некоторых случаях вызывающий код может потребоваться защитить от таких изменений.</span><span class="sxs-lookup"><span data-stu-id="6ef85-105">In some cases the calling code might want to protect against such a change.</span></span>
