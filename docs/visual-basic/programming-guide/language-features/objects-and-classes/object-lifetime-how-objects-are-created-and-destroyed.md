### <a name="implementing-idisposable"></a><span data-ttu-id="fdfc0-164">Использование IDisposable</span><span class="sxs-lookup"><span data-stu-id="fdfc0-164">Implementing IDisposable</span></span>

    ```  
    Protected disposed As Boolean = False  
    ```  
  
-   <span data-ttu-id="fdfc0-167">Перегрузка <xref:System.IDisposable.Dispose%2A>для высвобождения ресурсов класса.</xref:System.IDisposable.Dispose%2A></span><span class="sxs-lookup"><span data-stu-id="fdfc0-167">An overload of the <xref:System.IDisposable.Dispose%2A> that frees the class's resources.</span></span> <span data-ttu-id="fdfc0-168">Этот метод должен вызываться <xref:System.IDisposable.Dispose%2A>и `Finalize` методы базового класса:</xref:System.IDisposable.Dispose%2A></span><span class="sxs-lookup"><span data-stu-id="fdfc0-168">This method should be called by the <xref:System.IDisposable.Dispose%2A> and `Finalize` methods of the base class:</span></span>

            End If  
            ' Insert code to free unmanaged resources.  
        End If  
        Me.disposed = True  
    End Sub  
    ```  
  
-   <span data-ttu-id="fdfc0-169">Реализация <xref:System.IDisposable.Dispose%2A>, содержащая только следующий код:</xref:System.IDisposable.Dispose%2A></span><span class="sxs-lookup"><span data-stu-id="fdfc0-169">An implementation of <xref:System.IDisposable.Dispose%2A> that contains only the following code:</span></span>

        GC.SuppressFinalize(Me)  
    End Sub  
    ```  
  
-   <span data-ttu-id="fdfc0-170">Переопределение метода `Finalize`, содержащее только следующий код:</span><span class="sxs-lookup"><span data-stu-id="fdfc0-170">An override of the `Finalize` method that contains only the following code:</span></span>  
  
    ```

        MyBase.Finalize()  
    End Sub  
    ```  
  
### <a name="deriving-from-a-class-that-implements-idisposable"></a><span data-ttu-id="fdfc0-171">Производное создание от класса, реализующего IDisposable</span><span class="sxs-lookup"><span data-stu-id="fdfc0-171">Deriving from a Class that Implements IDisposable</span></span>

 <span data-ttu-id="fdfc0-175">Производный класс не должен переопределять базовый класс <xref:System.IDisposable.Dispose%2A>и `Finalize` методы.</xref:System.IDisposable.Dispose%2A></span><span class="sxs-lookup"><span data-stu-id="fdfc0-175">A derived class should not override the base class's <xref:System.IDisposable.Dispose%2A> and `Finalize` methods.</span></span> <span data-ttu-id="fdfc0-176">Когда эти методы вызываются из экземпляра производного класса, реализация этих методов в базовом классе вызывает переопределение метода `Dispose(disposing)` производного класса.</span><span class="sxs-lookup"><span data-stu-id="fdfc0-176">When those methods are called from an instance of the derived class, the base class's implementation of those methods call the derived class's override of the `Dispose(disposing)` method.</span></span>
