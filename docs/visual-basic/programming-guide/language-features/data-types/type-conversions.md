# <a name="type-conversions-in-visual-basic"></a>Преобразование типов в Visual Basic
Процесс изменения типа значения из одного типа данных к другому типу называется *преобразование*. Преобразования являются либо *расширяющие* или *сужающие*, в зависимости от диапазонов данных типов. Они также могут *неявное* или *явных*, в зависимости от синтаксиса в исходном коде.
