 <span data-ttu-id="4939e-109">Структуры полезны в тех случаях, когда требуется одну переменную для хранения связанных части сведений.</span><span class="sxs-lookup"><span data-stu-id="4939e-109">Structures are useful when you want a single variable to hold several related pieces of information.</span></span> <span data-ttu-id="4939e-110">Например может потребоваться сохранить вместе имя, телефон и заработной платы сотрудника.</span><span class="sxs-lookup"><span data-stu-id="4939e-110">For example, you might want to keep an employee's name, telephone extension, and salary together.</span></span> <span data-ttu-id="4939e-111">Эти сведения можно использовать несколько переменных, или можно определить структуру и использовать ее для одной переменной сотрудника.</span><span class="sxs-lookup"><span data-stu-id="4939e-111">You could use several variables for this information, or you could define a structure and use it for a single employee variable.</span></span> <span data-ttu-id="4939e-112">Преимущества структуры становится очевидным, когда имеется много сотрудников и поэтому многие экземпляры переменной.</span><span class="sxs-lookup"><span data-stu-id="4939e-112">The advantage of the structure becomes apparent when you have many employees and therefore many instances of the variable.</span></span>
