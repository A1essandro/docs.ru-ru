# <a name="structures-visual-basic"></a>Структуры (Visual Basic)
Объект *структуры* является обобщением определяемых пользователем типов (UDT), поддерживаемых предыдущими версиями [!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]. В дополнение к полям структуры могут предоставлять свойства, методы и события. Структура может реализовать один или несколько интерфейсов, и можно объявлять индивидуальные уровни доступа для каждого поля.  
  
 Можно комбинировать элементы данных различных типов для создания структуры. Структуры связывают один или несколько *элементы* друг с другом и с самой структурой. При объявлении структуры она становится *составной тип данных*, и можно объявлять переменные этого типа.  
  
 Структуры полезны в тех случаях, когда требуется одну переменную для хранения нескольких связанных частей данных. Например может потребоваться сохранить вместе имя, телефон и заработной платы сотрудника. Эти сведения можно использовать несколько переменных, или можно определить структуру и использовать ее для одной переменной сотрудника. Преимущества структуры становится очевидным, когда имеется множество сотрудников, поэтому большое количество экземпляров переменной.
