# <a name="how-to-hold-more-than-one-value-in-a-variable-visual-basic"></a>Практическое руководство. Хранение нескольких значений в переменной (Visual Basic)
Переменная содержит более одного значения, если объявить ее из *составной тип данных*.  
  
 [Составные типы данных](../../../../visual-basic/programming-guide/language-features/data-types/composite-data-types.md) включают структуры, массивы и классы. Переменная составного типа данных может содержать сочетание простые типы данных и других составных типов. Классы и структуры могут содержать код, а также данные.  
  
### <a name="to-hold-more-than-one-value-in-a-variable"></a>Для хранения более одного значения в переменной  
  
1.  Определите, какой тип составных данных, которые вы хотите использовать для переменной.  
  
2.  Если составной тип данных не определен, определите его, чтобы его можно было использовать переменную.  
  
    -   Определите структуру при помощи [оператор Structure](../../../../visual-basic/language-reference/statements/structure-statement.md).  
  
    -   Определить массив с [оператор Dim](../../../../visual-basic/language-reference/statements/dim-statement.md).  
  
    -   Определение класса с [оператор Class](../../../../visual-basic/language-reference/statements/class-statement.md).  
  
3.  Объявите переменную при помощи `Dim` инструкции.  
  
4.  После имени переменной `As` предложения.  
  
5.  Выполните `As` ключевое слово с именем соответствующего составного типа.
