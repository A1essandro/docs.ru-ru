# <a name="creating-xml-in-visual-basic"></a>Создание XML в Visual Basic
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]позволяет использовать *XML-литералы* непосредственно в коде. Синтаксис литерала XML представляет [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] объекты и напоминает синтаксис XML 1.0. Это упрощает программное создание элементов, документов и фрагментов XML, так как ваш код имеет такую же структуру, как окончательный XML.
