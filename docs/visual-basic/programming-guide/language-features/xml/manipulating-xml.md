# <a name="manipulating-xml-in-visual-basic"></a>Обработка XML в Visual Basic
Можно использовать *XML-литералы* для загрузки XML из внешнего источника, таких как строка, файл или поток. Затем можно использовать [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] управления XML и использовать [!INCLUDE[vbteclinqext](../../../../csharp/getting-started/includes/vbteclinqext_md.md)] для составления запросов XML.
