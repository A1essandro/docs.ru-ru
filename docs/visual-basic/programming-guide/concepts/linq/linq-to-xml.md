# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML предоставляет интерфейс программирования XML в памяти, который использует платформу запросов запросы (LINQ). LINQ to XML использует новейшие возможности языка .NET Framework и сравним с обновленным, переработанным объектной модели документов (DOM) XML интерфейсом программирования.  
  
 Семейство технологий LINQ обеспечивает согласованное функционирование запросов для объектов (LINQ to Objects), реляционных баз данных (LINQ to SQL) и XML (LINQ to XML).
