# <a name="how-to-query-an-assembly39s-metadata-with-reflection-linq-visual-basic"></a><span data-ttu-id="b39c8-102">Практическое руководство: запрос сборки метаданных с помощью отражения (LINQ) (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="b39c8-102">How to: Query An Assembly&#39;s Metadata with Reflection (LINQ) (Visual Basic)</span></span>
<span data-ttu-id="b39c8-103">Следующий пример показывает, как LINQ можно с помощью отражения для извлечения определенных метаданных о методах, соответствующих условиям поиска.</span><span class="sxs-lookup"><span data-stu-id="b39c8-103">The following example shows how LINQ can be used with reflection to retrieve specific metadata about methods that match a specified search criterion.</span></span> <span data-ttu-id="b39c8-104">В этом случае запрос будет искать имена всех методов в сборке, которые возвращают перечислимые типы, такие как массивы.</span><span class="sxs-lookup"><span data-stu-id="b39c8-104">In this case, the query will find the names of all the methods in the assembly that return enumerable types such as arrays.</span></span>  
  
## <a name="example"></a><span data-ttu-id="b39c8-105">Пример</span><span class="sxs-lookup"><span data-stu-id="b39c8-105">Example</span></span>  
  
```vb  
Imports System.Reflection  
Imports System.IO  
Imports System.Linq  
Module Module1  
  
    Sub Main()  
        Dim asmbly As Assembly =   
            Assembly.Load("System.Core, Version=3.5.0.0, Culture=neutral, PublicKeyToken= b77a5c561934e089")  
  
        Dim pubTypesQuery = From type In asmbly.GetTypes()   
                            Where type.IsPublic   
                            From method In type.GetMethods()   
                            Where method.ReturnType.IsArray = True   
                            Let name = method.ToString()   
                            Let typeName = type.ToString()   
                            Group name By typeName Into methodNames = Group  
  
        Console.WriteLine("Getting ready to iterate")  
        For Each item In pubTypesQuery  
            Console.WriteLine(item.methodNames)  
  
            For Each type In item.methodNames  
                Console.WriteLine(" " & type)  
            Next  
        Next  
        Console.ReadKey()  
    End Sub  
  
End Module  
```

an array of types in the specified assembly.</span></span> <span data-ttu-id="b39c8-107">[Предложение Where](../../../../visual-basic/language-reference/queries/where-clause.md) применяется фильтр, applied so that only public types are returned.</span></span> <span data-ttu-id="b39c8-108">Для каждого открытого типа создается вложенный запрос с помощью <xref:System.Reflection.MethodInfo>массив, возвращаемый из <xref:System.Type.GetMethods%2A>вызова.</xref:System.Type.GetMethods%2A> </xref:System.Reflection.MethodInfo></span><span class="sxs-lookup"><span data-stu-id="b39c8-108">For each
