# <a name="linq-and-reflection-visual-basic"></a>LINQ и отражение (Visual Basic)
API отражения библиотеки классов .NET Framework можно использовать для просмотра метаданных в сборке .NET и создания коллекций типов, членов типов, параметров и т. д, которые находятся в этой сборке. Так как эти коллекции поддерживают универсальный `IEnumerable` интерфейса, они могут быть запрошены с помощью LINQ.
