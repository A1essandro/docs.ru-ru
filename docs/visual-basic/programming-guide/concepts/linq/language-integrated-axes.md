# <a name="language-integrated-axes-in-visual-basic-linq-to-xml"></a>Загрузка встроенных в язык осей в Visual Basic (LINQ to XML)
В этом разделе описываются встроенные возможности [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] языка, чтобы облегчить доступ к XML. Во многих примерах в документации LINQ to XML используются интегрированные оси [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)].  
  
## <a name="in-this-section"></a>Содержание  
  
|Раздел|Описание|  
|-----------|-----------------|  
|[Свойство дочерней оси XML](../../../../visual-basic/language-reference/xml-axis/xml-child-axis-property.md)|Предоставляет доступ к дочерним элементам одного из следующих: объекта <xref:System.Xml.Linq.XElement>, объекта <xref:System.Xml.Linq.XDocument>, коллекции объектов <xref:System.Xml.Linq.XElement> или коллекции объектов <xref:System.Xml.Linq.XDocument>. Эта ось эквивалентна оси <xref:System.Xml.Linq.XContainer.Elements%2A>.|  
|[Свойство дочерней оси XML](../../../../visual-basic/language-reference/xml-axis/xml-descendant-axis-property.md)|Предоставляет доступ к потомкам из следующих: объекта <xref:System.Xml.Linq.XElement>, объекта <xref:System.Xml.Linq.XDocument> или коллекции объектов <xref:System.Xml.Linq.XElement>. Эта ось эквивалентна оси <xref:System.Xml.Linq.XContainer.Descendants%2A>.|  
|[Свойство оси атрибута XML](../../../../visual-basic/language-reference/xml-axis/xml-attribute-axis-property.md)|Обеспечивает доступ к атрибуту объекта <xref:System.Xml.Linq.XElement>. Эта ось примерно эквивалентна оси <xref:System.Xml.Linq.XElement.Attribute%2A>. Эта ось отличается от оси <xref:System.Xml.Linq.XElement.Attribute%2A> в том, что она возвращает значение атрибута, а не объект <xref:System.Xml.Linq.XAttribute>.|  
|[Свойство индексатора расширения](../../../../visual-basic/language-reference/xml-axis/extension-indexer-property.md)|Обеспечивает доступ к отдельным элементам коллекции.|
