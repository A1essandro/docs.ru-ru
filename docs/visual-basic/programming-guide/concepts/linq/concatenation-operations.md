# <a name="concatenation-operations-visual-basic"></a>Операции объединения (Visual Basic)

 На рисунке показана операция объединения двух последовательностей символов.  
  
 ![График, показывающий объединение двух последовательностей. ](../../../../csharp/programming-guide/concepts/linq/media/concat.png "Concat")
