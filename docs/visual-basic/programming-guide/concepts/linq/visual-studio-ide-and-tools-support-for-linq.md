# <a name="visual-studio-ide-and-tools-support-for-linq-visual-basic"></a>Интегрированная среда разработки Visual Studio и поддержка средств для LINQ (Visual Basic)
Среда разработки Visual Studio (IDE) предоставляет следующие возможности, поддерживающие разработку приложений LINQ:  
  
## <a name="object-relational-designer"></a>Object Relational Designer  
 Реляционный конструктор объектов это средство визуальной разработки, которое можно использовать в [LINQ to SQL](https://msdn.microsoft.com/library/bb386976) приложения для создания классов в Visual Basic, представляющие реляционные данные в основной базе данных. Дополнительные сведения см. в разделе [средства LINQ to SQL в Visual Studio](https://docs.microsoft.com/visualstudio/data-tools/linq-to-sql-tools-in-visual-studio2).  
  
## <a name="sqlmetal-command-line-tool"></a>Средства командной строки SQLMetal  
 SQLMetal — программа командной строки, который может использоваться в процессах построения для создания классов из существующих баз данных для использования в запросах LINQ для приложений SQL. Дополнительные сведения см. в разделе [SqlMetal.exe (средство создания кода)](http://msdn.microsoft.com/library/819e5a96-7646-4fdb-b14b-fe31221b0614).  
  
## <a name="linq-aware-code-editor"></a>Поддержкой LINQ редактор кода  
 Редактор кода Visual Basic поддерживает LINQ широко с технологией IntelliSense и возможностями форматирования.  
  
## <a name="visual-studio-debugger-support"></a>Поддержка отладчика Visual Studio  
 Отладчик Visual Studio поддерживает отладку выражений запросов. Дополнительные сведения см. в разделе [отладки LINQ](https://docs.microsoft.com/visualstudio/debugger/debugging-linq).
