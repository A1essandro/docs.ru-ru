## <a name="remarks"></a><span data-ttu-id="152fa-145">Примечания</span><span class="sxs-lookup"><span data-stu-id="152fa-145">Remarks</span></span>  
 <span data-ttu-id="152fa-146">Если генератор XML-сериализатора не используется, <xref:System.Xml.Serialization.XmlSerializer> создает код сериализации и сборку сериализации для каждого типа при каждом запуске приложения.</span><span class="sxs-lookup"><span data-stu-id="152fa-146">When the XML Serializer Generator is not used, a <xref:System.Xml.Serialization.XmlSerializer> generates serialization code and a serialization assembly for each type every time an application is run.</span></span> <span data-ttu-id="152fa-147">Чтобы повысить производительность при запуске сериализации XML, инструмент Sgen.exe заранее создать эти сборки.</span><span class="sxs-lookup"><span data-stu-id="152fa-147">To improve the performance of XML serialization startup, use the Sgen.exe tool to generate those assemblies in advance.</span></span> <span data-ttu-id="152fa-148">Эти сборки можно развернуть вместе с приложением.</span><span class="sxs-lookup"><span data-stu-id="152fa-148">These assemblies can then be deployed with the application.</span></span>  
  
 <span data-ttu-id="152fa-149">Инструмент создания XML-сериализатора также повышает производительность клиентов, использующих прокси XML-веб-служб для связи с серверами, поскольку производительность процесса сериализации не снижается при первой загрузке типа.</span><span class="sxs-lookup"><span data-stu-id="152fa-149">The XML Serializer Generator can also improve the performance of clients that use XML Web service proxies to communicate with servers because the serialization process will not incur a performance hit when the type is loaded the first time.</span></span>  
  
 <span data-ttu-id="152fa-150">Эти созданные сборки нельзя использовать на стороне сервера веб-службы.</span><span class="sxs-lookup"><span data-stu-id="152fa-150">These generated assemblies cannot be used on the server side of a Web service.</span></span> <span data-ttu-id="152fa-151">Этот инструмент предназначен только для клиентов веб-служб и сценариев ручной сериализации.</span><span class="sxs-lookup"><span data-stu-id="152fa-151">This tool is only for Web service clients and manual serialization scenarios.</span></span>  
  
 <span data-ttu-id="152fa-152">Если сборка, содержащая сериализуемый тип, называется "MyType.dll", связанная сборка сериализации будет называться "MyType.XmlSerializers.dll".</span><span class="sxs-lookup"><span data-stu-id="152fa-152">If the assembly containing the type to serialize is named MyType.dll, then the associated serialization assembly will be named MyType.XmlSerializers.dll.</span></span>
