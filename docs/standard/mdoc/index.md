# <a name="building-documentation-with-mdoc"></a>Создание документации с помощью mdoc

mdoc — это система создания управляемой справочной документации. Отражая ваши сборки, mdoc создает базовый формат управляемой справочной документации, которая далее дополняется разрабатываемым содержимым.
