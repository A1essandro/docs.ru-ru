# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Модернизация существующих приложений .NET на оптимизированную для облачных приложений

> Концепция. Модернизация существующих приложений .NET Framework оптимизированную для облачных приложений, чтобы радикально улучшить гибкости процесса развертывания, чтобы вы могли быстрее доставляйте и снизить затраты на доставку приложения.

Чтобы воспользоваться преимуществами облачных и новых технологий, таких как контейнеры, следует хотя бы частично модернизировать существующие приложения .NET. В конечном счете обновление корпоративных приложений приведет к снижению общей стоимости владения.

Частичная модернизация приложений не обязательно означает полной миграции и изменять архитектуру. Вы можете изначально модернизировать существующие приложения с важной, но просто сделать модернизации. При этом вы сохраняете текущую базу кода, написанную в существующих версиях .NET Framework, со всеми зависимостями Windows и IIS. На рисунке 4-1 показано, как оптимизированную для облачных приложений располагаются в моделях зрелости модернизации приложений Azure.
