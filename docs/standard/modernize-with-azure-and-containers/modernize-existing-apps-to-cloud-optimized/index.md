# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Модернизировать существующие приложения .NET для приложений, оптимизированными для облака

> Концепция: Модернизировать существующие приложения .NET Framework в приложения, оптимизированными для облака, чтобы значительно улучшить гибкость вашего развертывания, чтобы можно было отправить быстрее и снижение расходов доставки приложения.

Чтобы воспользоваться преимуществами облачных и новых технологий, таких как контейнеры, следует хотя бы частично модернизировать существующие приложения .NET. В конечном счете обновление корпоративных приложений приведет к снижению общей стоимости владения.

Частично модернизация приложения не обязательно означает полной миграции и rearchitecture. Изначально можно модернизировать существующие приложения с важных, но легко сделать изменения. При этом вы сохраняете текущую базу кода, написанную в существующих версиях .NET Framework, со всеми зависимостями Windows и IIS. Рис. 4-1 выделяет как оптимизированную для облачных приложений располагаются в модели зрелости модернизацию приложения Azure.
