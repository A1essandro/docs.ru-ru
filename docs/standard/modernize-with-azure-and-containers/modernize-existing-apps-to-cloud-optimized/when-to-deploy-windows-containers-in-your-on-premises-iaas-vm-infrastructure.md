# <a name="when-to-deploy-windows-containers-in-your-on-premises-iaas-vm-infrastructure"></a>Если развернуть контейнеры Windows в своей локальной инфраструктуре ВМ IaaS

-   Ваша организация не может быть готовы перейти в облако или не может быть возможность перемещать в облако бизнес-цель. Но по-прежнему можно получить преимущества использования контейнеры Windows в собственных центрах обработки данных.

-   Возможно, другие артефакты, они используются локально, и который может замедлить работу при попытке перейти к облаку. Например безопасности или проверки подлинности зависимости с локальной Active Directory Windows Server или другие ресурсы в локальной среде.

-   Если можно начать использовать контейнеры Windows, которые можно выполнить поэтапной миграции в облако завтра гораздо более высокую позицию. Контейнеры Windows становится элементом развертывания для любой облачной без блокировки в систему.
