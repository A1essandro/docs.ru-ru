# <a name="introduction-tothe-docker-application-life-cycle"></a>Введение в жизненный цикл приложения Docker

*Жизненный цикл контейнерных приложений — это процесс, который начинается с разработчика. Разработчик выбирает предпочитают реализовывать контейнеры и Docker, так как он оперативности развертываний и ИТ-операций, которое в конечном счете помогает для большей гибкости, продуктивности end-to-end и быстрее.*
