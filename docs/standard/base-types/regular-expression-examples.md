# Примеры регулярных выражений
В этом разделе представлены примеры кодов, иллюстрирующих использование регулярных выражений в обычных приложениях.  
  
> [!NOTE]
>  Пространство имен <xref:System.Web.RegularExpressions> содержит некоторые объекты регулярных выражений, реализующих предопределенные шаблоны регулярных выражений для разбора строк из документов HTML, XML и ASP.NET.  Например, класс <xref:System.Web.RegularExpressions.TagRegex> определяет открывающие теги в строке, а класс <xref:System.Web.RegularExpressions.CommentRegex> определяет комментарии ASP.NET в строке.
