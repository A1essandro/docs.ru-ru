<span data-ttu-id="b2044-103">Объект <xref:System.TimeSpan> строка формата определяет строковое представление <xref:System.TimeSpan> значение, полученное в результате операции форматирования.</span><span class="sxs-lookup"><span data-stu-id="b2044-103">A <xref:System.TimeSpan> format string defines the string representation of a <xref:System.TimeSpan> value that results from a formatting operation.</span></span> <span data-ttu-id="b2044-104">Строка настраиваемого формата состоит из одного или нескольких пользовательских <xref:System.TimeSpan> описателях вместе с любым числом буквенных символов.</span><span class="sxs-lookup"><span data-stu-id="b2044-104">A custom format string consists of one or more custom <xref:System.TimeSpan> format specifiers along with any number of literal characters.</span></span> <span data-ttu-id="b2044-105">Любая строка, которая не является [стандартную строку формата TimeSpan](../../../docs/standard/base-types/standard-timespan-format-strings.md) интерпретируется как строка настраиваемого <xref:System.TimeSpan> строка формата.</span><span class="sxs-lookup"><span data-stu-id="b2044-105">Any string that is not a [standard TimeSpan format string](../../../docs/standard/base-types/standard-timespan-format-strings.md) is interpreted as a custom <xref:System.TimeSpan> format string.</span></span>  
  
> [!IMPORTANT]
>  <span data-ttu-id="b2044-106">Пользовательский <xref:System.TimeSpan> описателей формата не включают разделяющие символы, символы, которые отделяют дни от часов, часов, минут или секунд от долей секунд.</span><span class="sxs-lookup"><span data-stu-id="b2044-106">The custom <xref:System.TimeSpan> format specifiers do not include placeholder separator symbols, such as the symbols that separate days from hours, hours from minutes, or seconds from fractional seconds.</span></span> <span data-ttu-id="b2044-107">Вместо этого эти символы должны быть включены в строку настраиваемого формата как строковые литералы.</span><span class="sxs-lookup"><span data-stu-id="b2044-107">Instead, these symbols must be included in the custom format string as string literals.</span></span> <span data-ttu-id="b2044-108">Например, строка `"dd\.hh\:mm"` определяет точку (.) как разделитель дней и часов и двоеточие (:) как разделитель часов и минут.</span><span class="sxs-lookup"><span data-stu-id="b2044-108">For example, `"dd\.hh\:mm"` defines a period (.) as the separator between days and hours, and a colon (:) as the separator between hours and minutes.</span></span>  
>   
>  <span data-ttu-id="b2044-109">Настраиваемые спецификаторы формата <xref:System.TimeSpan> также не содержат символ знака, позволяющий различать положительные и отрицательные временные интервалы.</span><span class="sxs-lookup"><span data-stu-id="b2044-109">Custom <xref:System.TimeSpan> format specifiers also do not include a sign symbol that enables you to differentiate between negative and positive time intervals.</span></span> <span data-ttu-id="b2044-110">Чтобы включить символ знака, необходимо построить строку формата с помощью условной логики.</span><span class="sxs-lookup"><span data-stu-id="b2044-110">To include a sign symbol, you have to construct a format string by using conditional logic.</span></span> <span data-ttu-id="b2044-111">См. пример в разделе [Остальные символы](#Other).</span><span class="sxs-lookup"><span data-stu-id="b2044-111">The [Other Characters](#Other) section includes an example.</span></span>
