# Общие шаблоны разработки
Существует большое количество книг на шаблоны программного обеспечения, шаблон языки и antipatterns пояснениями очень обширная тема шаблонов. Таким образом в этой главе приводятся рекомендации и обсуждения, связанные с очень ограниченный набор шаблонов, которые часто используются при разработке интерфейсов API платформы .NET Framework.
