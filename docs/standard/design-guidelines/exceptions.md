# Правила разработки исключений
Обработка исключений имеет множество преимуществ перед отчеты об ошибках на основе возвращаемого значения. Хорошей платформы разработки позволяет разработчику использовать преимущества исключения. В этом разделе обсуждаются преимущества исключения и представлены рекомендации по эффективному использованию.
