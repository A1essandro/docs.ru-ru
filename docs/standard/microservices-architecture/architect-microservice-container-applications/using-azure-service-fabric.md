## <a name="stateless-versus-stateful-microservices"></a><span data-ttu-id="ba3cb-149">Микрослужбы с отслеживанием и без отслеживания состояния</span><span class="sxs-lookup"><span data-stu-id="ba3cb-149">Stateless versus stateful microservices</span></span>
