# <a name="choosing-between-net-core-and-net-framework-for-docker-containers"></a>Выбор между .NET Core и .NET Framework для контейнеров Docker

Серверные контейнерные приложения Docker на платформе .NET можно разрабатывать в двух поддерживаемых реализациях: [.NET Framework](https://www.microsoft.com/net/download/framework) и [.NET Core](https://www.microsoft.com/net/download/core). В них используется множество одинаковых стандартных компонентов .NET, а код можно использовать как в одной среде, так и в другой. Но между этими двумя средами также существуют и фундаментальные различия. Поэтому ваш выбор будет зависеть от поставленной задачи. В этом разделе приводятся рекомендации по выбору каждой из реализаций.
