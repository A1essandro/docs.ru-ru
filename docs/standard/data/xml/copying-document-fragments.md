# <a name="copying-document-fragments"></a>Копирование фрагментов документа
Можно создать **XmlDocumentFragment** узел и затем добавлять узлы в нем. При **XmlDocumentFragment** вставляется с **InsertNode** метода **XmlDocumentFragment** узел не копируются, но его дочерние узлы вставляются в XML Модель объектов документов (DOM).
