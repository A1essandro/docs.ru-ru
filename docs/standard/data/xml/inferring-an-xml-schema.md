# Выведение XML-схемы
Описывает, как использовать класс <xref:System.Xml.Schema.XmlSchemaInference> модели SOM, чтобы выводить схемы на языке XSD из структуры XML\-документа.  
  
 Класс <xref:System.Xml.Schema.XmlSchemaInference> модели SOM в пространстве имен <xref:System.Xml.Schema?displayProperty=fullName> позволяет выводить схемы на языке XSD из структуры XML\-документа.  Класс <xref:System.Xml.Schema.XmlSchemaInference> выводит схему XML, которая может проверять XML\-документ.
