# Удаление узлов, содержимого и значений из XML-документа
Как только в памяти создана модель XML DOM, можно удалить узлы дерева или удалить содержимое и значения из узлов определенного типа.  Сведения об удалении из документа конечного узла или всего поддерева узла см. в разделе [Удаление узлов из модели DOM](../../../../docs/standard/data/xml/removing-nodes-from-the-dom.md).  Сведения об удалении атрибутов элемента см. в разделе [Удаление атрибутов из узла элемента в модели DOM](../../../../docs/standard/data/xml/removing-attributes-from-an-element-node-in-the-dom.md).  Сведения об удалении содержимого узла с сохранением самого узла в дереве см. в разделе [Удаление содержимого узла в модели DOM](../../../../docs/standard/data/xml/removing-node-content-in-the-dom.md).
