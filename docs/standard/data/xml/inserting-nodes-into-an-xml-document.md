# Вставка узлов в XML-документ
Можно изменить документ, вставив в него узлы.  Эти узлы могут быть только что созданными, уже существующими в том же документе или импортированными из другого документа.  Дополнительные сведения о создании узлов см. в разделе [Создание новых узлов в модели DOM](../../../../docs/standard/data/xml/create-new-nodes-in-the-dom.md).  Дополнительные сведения о копировании существующих узлов см. в разделах [Копирование существующих узлов](../../../../docs/standard/data/xml/copy-existing-nodes.md), [Копирование существующих узлов из одного документа в другой](../../../../docs/standard/data/xml/copying-existing-nodes-from-one-document-to-another.md) и [Копирование фрагментов документа](../../../../docs/standard/data/xml/copying-document-fragments.md).
