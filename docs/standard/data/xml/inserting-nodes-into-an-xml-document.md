# <a name="inserting-nodes-into-an-xml-document"></a>Вставка узлов в XML-документ
Можно изменить документ, вставив в него узлы. Эти узлы могут быть только что созданными, уже существующими в том же документе или импортированными из другого документа. См. дополнительные сведения о [создании узлов в модели DOM](../../../../docs/standard/data/xml/create-new-nodes-in-the-dom.md). Дополнительные сведения о копировании существующих узлов см. в руководствах по [копированию существующих узлов](../../../../docs/standard/data/xml/copy-existing-nodes.md), [копированию существующих узлов из одного документа в другой](../../../../docs/standard/data/xml/copying-existing-nodes-from-one-document-to-another.md) и [копированию фрагментов документа](../../../../docs/standard/data/xml/copying-document-fragments.md).
